# 生成台面

1. 选择柜体，工具会自动识别需要生成台面的柜体，也可**长按shift框选多个柜体**进行生成，需要识别全部柜体可以点击**智能选择**

![](https://qhstaticssl.kujiale.com/image/png/1739346005982/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 在右侧面板的基础参数中选择添加**前后挡水、台面材质**，调整具体参数后，点击**生成台面**

![](https://qhstaticssl.kujiale.com/image/png/1739346351740/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 如果需要自由调整台面生成的范围，点击顶部栏的**自由编辑**，可以进入自由绘制模式

![](https://qhstaticssl.kujiale.com/image/png/1739346512360/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. 在自由绘制界面中，在2D下使用顶部栏的绘制工具绘制台面线条与矩形

![](https://qhstaticssl.kujiale.com/image/png/1739347176250/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1739347140609/B68E0A7A5A6D0BAFEBDDA5CFC76F9AB7.gif)

5. 可以手动编辑点线面，或输入数值进行精准调整，调整完毕后点击**开始生成**

![](https://qhstaticssl.kujiale.com/image/gif/1739347388496/A438130944EE7208823119BEEC8AE091.gif)

6. 生成台面后，点击**完成**，退出生成台面界面

![](https://qhstaticssl.kujiale.com/image/png/1739347704341/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
