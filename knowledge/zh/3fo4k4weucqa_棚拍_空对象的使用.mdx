# 画布中支持直接添加“空对象”

概念：空对象是一种特殊的对象类型，它在3D场景中没有实际的几何形状，不直接渲染显示。可以将其看作是一个不可见的模型或者一个坐标参考点

作用：

* 对象管理

  + 空对象在层级结构中非常有用。它可以作为其他对象的父对象，用于组织和管理场景中的多个对象。例如：当制作一个复杂的动画场景，包含多个模型、灯光和摄像机时，可以创建空对象，将相关的对象放置在其下，形成一个逻辑清晰的层级关系
* PSR变换操作传递（批量操作&群组管理）

  + 空对象的变换属性（如位置、旋转、缩放）会影响其所有子对象。这使得对一组对象进行统一的变换操作变得更加容易。比如：要对一组分散的模型进行整体的旋转移动操作，将这些模型设置为空对象的子对象后，只需对空对象进行旋转或移动操作，其子对象就会按照相同的方式进行变换，可以大大提供操作效率
* 动画驱动

  + 空对象常用于建立约束关系，例如：可以将一个对象约束到空对象上，通过控制空对象的运动来间接控制被约束对象的运动。由于空对象没有几何形状，在渲染过程中它不会对最终的渲染结果产生任何直接的视觉影响，这意味着可以在场景中自由地使用空对象来构建场景结构和进行动画控制，而不必担心它会在渲染画面中出现。比如使用 “目标” 约束，使一个模型的某个部分始终指向空对象的位置
