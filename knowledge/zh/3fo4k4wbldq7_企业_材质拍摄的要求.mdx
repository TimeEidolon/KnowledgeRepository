# 拍摄一张适合自己的纹理照片

**您所需要的：良好的照明和专业的单反相机**

* 您需要的最佳专业摄影设备是数码单反相机
* 不建议使用手机，因为手机拍摄的照片质量不够好，且手机相机中的滤镜会造成色差，后期调整很困难

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733886123251/C825B9F6E621AFA5ADE616F0201EDB4D.png)

**专业相机设备的优缺点:**

* **优点：**纹理显示清晰，底图会更大
* **难点：**您需要擅长拍照，拍摄时注意光线和细节，要使中心轮廓清晰，边缘模糊。拍摄后还需要进行修图

# 拍摄样片要求

1. 板材尺寸在 600mm\*600mm 以上，纹理完整

2. 板材上没有瑕疵或污垢

3. 样品纹理精美，能体现产品的质感

# 照明要求

1. 我们推荐专业摄影棚，因为它们的光线控制更好

* 三个柔光灯箱以直角排列，从三个方向照亮产品
* 相机用三脚架固定，以90度拍摄
* 还可以拍摄周围的物体以进行后期处理

![](https://qhstaticssl.kujiale.com/image/png/1748501023648/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 如果附近找不到摄影棚，可以寻找任何阴凉的空旷场地。使用合适的模式（例如自动模式），关闭闪光灯，不使用任何人工照明。确保拍摄对象光照均匀，避免高光或阴影，否则后期处理会非常困难

* 注意：不要忘记检查图像质量设置以获得最佳图像
* 竖拍：用相机镜头垂直拍摄物体，距离要适中，把物体全部拍进去，每个细节都要清晰，视野要开阔，不用担心周围的环境，这些都可以在后期处理中去掉

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733886129624/702562FDE60ABD2F345D776C04C54608.png)

# 处理照片

**方式一：插件处理**

方便快捷，适合简单、规则的图案纹理，但会降低照片质量

**方式二：Photoshop 处理**

比较复杂，但能保持照片质量

* 图片分辨率是多少? ---- 不超过5000\*5000像素
* 文件有多大? ---- 纹理文件不应超过 5M
