# 面向用户

所有企业用户

# 入口

1. 选择**厨卫/全屋家具/门窗定制**，进入定制设计界面

2. 选择顶层工具栏中的**文件-导出JSON/XML数据**

3. 选择需要导出JSON/XML数据的工具线

![](https://qhstaticssl.kujiale.com/image/png/1738820769919/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 导出JSON/XML数据

1. 设置**报价方式**，导出范围可选择**单独房间、单独楼层及所有楼层**导出，设置完成点击**生成**

![](https://qhstaticssl.kujiale.com/image/png/1738812728532/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 点击**下载**后，下载完成会压缩成单独的压缩包

![](https://qhstaticssl.kujiale.com/image/png/1738813194470/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
