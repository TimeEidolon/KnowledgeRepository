为明确系统操作权限、规范企业管理流程，本文对**报价、产品、订单、图纸、模型、灵感空间**等模块的角色功能进行统一说明，以便企业合理分配账号权限

# 面向用户

企业用户

# 角色介绍

以下为报价、产品、订单、图纸、模型及灵感空间等模块中，各角色对应的详细功能说明：

| 类型 | 角色 | 功能说明 |
| --- | --- | --- |
| ****报价管理**** | 厨卫产品及报价管理员 | 厨卫定制商品管理、参数化建模、报价规则设置 |
|  | 全屋定制产品及报价管理员 | 全屋定制商品管理、参数化建模、报价规则设置 |
|  | 门窗产品及报价管理员 | 门窗定制商品管理、参数化建模、报价规则设置 |
| **定制产品管理** | 厨卫产品管理员 | 厨卫定制商品管理、参数化建模 |
|  | 全屋定制产品管理员 | 全屋定制商品管理、参数化建模 |
|  | 门窗产品及报价管理员 | 门窗定制商品管理、参数化建模 |
| **工业 4.0 接口** | JSON/XML 文件下载操作员 | 可通过设计工具下载JSON/XML文件 |
| **订单管理** | 审核单分配员 | 可手动设置分配规则，将**审核单**分配给指定审核员 |
|  | 生产单分配员 | 可手动设置分配规则，将**生产单**分配给指定审核员 |
|  | 审核员 | 通过主动认领/被动分配获取**审核单**，对审核单进行审核、修改、通过审核、创建生产单 |
|  | 生产审核员 | 通过主动认领/被动分配获取**生产单**，对生产单进行审核、修改、通过审核/下单、填写单号 |
|  | 订单管理员 | 可以看到所有订单(客户订单、审核单、生产 单)信息，并对可以对这些订单进行**转交**操作 (除审核中状态订单外) |
|  | 订单查看员 | 可以**看**到所有的客户订单和审核单，但是不能 操作除了自己提交的订单之外的其他审核单 |
|  | 订单提交员 | 允许**创建**客户订单/**提交**审核单 |
| **施工图** | 施工图设计师 | 在设计工具**使用/导出**施工图功能 |
| **灵感空间** | 灵感空间设计师 | 灵感空间方案创建 |
|  | 灵感空间管理员 | 灵感空间管理 |
| **模型管理** | 建模系统操作员 | 参数化建模 |
|  | 参数化铺贴编辑器 | 该角色允许具有管理员角色的帐户在企业目录 中创建参数化铺贴产品 |
