在基础材质库中，您只需要上传材质图片，就能实现将一个新创建的材质同时发布到各个行业线，包括家具、铺贴、厨卫、家具定制等行业线，大幅节省材质的维护时间和精力。本文会为您详细讲解如何在基础材质库中创建材质

# 面向用户

开通了“基础材质库”功能的企业用户

# 前提条件

您需要联系您的客户成功顾问，来试用或者购买“基础材质库”相关服务

# 创建新材质并发布

功能开通后，您可以直接创建一个新的基础材质

## 1. 创建基础材质

进入**工作台**，选择**“企业库 - 基础材质库”**，点击**【创建基础材质】**，进入材质制作工具

![](https://qhstaticssl.kujiale.com/image/png/1753757666527/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 2. 替换材质贴图

先从左侧面板选择材质的类型，然后根据具体需求，在右侧面板调节材质的贴图和颜色、效果属性

![](https://qhstaticssl.kujiale.com/image/png/1753757917696/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 3. 保存基础材质

完成**材质属性编辑**后，可以点击**保存**，补充剩余的材质信息

![](https://qhstaticssl.kujiale.com/image/png/1753758023339/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 4. 发布基础材质

基础材质创建完成后，无法直接在工具使用，您需要选择这些基础材质，将其发布到需要的行业线

1. 选择需要在工具中使用的**基础材质**，点击**材质发布**

![](https://qhstaticssl.kujiale.com/image/png/1753758430861/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 设置发布材质所在的**商品库和文件夹**，设置完成后点击**下一步**

![](https://qhstaticssl.kujiale.com/image/png/1753758543070/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 设置发布材质的**渲染分类**，设置完成后点击**发布**

![](https://qhstaticssl.kujiale.com/image/png/1753758684901/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 从已有材质提取

您也可以从已有材质的中提取材质到**基础材质库**，并发送到其它行业线

## 1. 选择一个非基础材质

从**企业库**中的任意一个行业线中，选择一个或多个已有的材质，点击**批量操作/编辑**，选择**材质提取**

如下图，分别展示了**纹理、铺贴、定制商品库**内的材质提取方式

![](https://qhstaticssl.kujiale.com/image/png/1753759085898/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1753759197708/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1753759325079/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 2. 发布基础材质

材质提取完成后，您可以进入**基础材质库**，找到对应的基础材质，将其发布到需要的行业线

* 操作步骤同上，详细步骤参考上文内容：**“****创建新材质并发布 —— 4. 发布基础材质”**
