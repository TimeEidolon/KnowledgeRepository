# 问题

在调整渲染效果时，发现调整凹凸参数后颗粒感太强

# 解决方法

1. 降低凹凸强度：可以尝试在实时渲染中材质编辑中进一步微调，自定义调整法线贴图凹凸强度。

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762134893640/8F3BDF68BFFEDA2569C8E146AD480476.png)

2. 调整材质：有时候，颗粒感可能与材质本身有关。尝试更换材质或调整材质的其他属性，如光泽度或反射率。

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762134893531/3A64130960B7E4E7C452D6827CA04E17.png)  
3. 光源调整：适当调整光源的数量和位置，可能会影响到材质的表现，缓解颗粒感。  
  
4. 渲染设置：检查渲染设置，确保没有过度锐化或后处理等其他影响材质表现的设置。
