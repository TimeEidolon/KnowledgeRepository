# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 入口

1. 进入**工作台**，选择**厨房和衣橱设计**

![](https://qhstaticssl.kujiale.com/image/png/1779701521850/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 进入全屋定制设计后，选择**装饰**

![](https://qhstaticssl.kujiale.com/image/png/1742882330894/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 查找模型

## 方式一：类目搜索

根据所需要的模型类型，按照类目所属，进行一级、二级、三级分类进行查找

* 如：使用双人床模型，通过 “家具 - 床 - 双人床” 进行查找

|  |  |
| --- | --- |
|  |  |

## 方式二：关键词搜索

如果不想按照类目一层层搜索，可以在模型库中上方搜索栏里直接输入关键词进行查找，也有热门搜索关键词推荐可进行选择

|  |  |
| --- | --- |
|  |  |

## 方式三：标签搜索

如果已经有明确类型的模型，可以通过材质、形状或者尺寸等标签进行筛选

![](https://qhstaticssl.kujiale.com/image/png/1742889748880/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1742889977065/AC8A6D6A6DFF3C292CA33D5D0F0E50FC.gif)

## 方式四：品牌馆查找

如果方案里的模型希望使用品牌家具，可以直接在品牌馆里搜索相应品牌进行挑选

|  |  |
| --- | --- |
|  |  |

# 查看最新模型

选择模型类别后，左侧排序中选择**最新**，即可查看最新上架的模型

|  |  |
| --- | --- |
|  |  |

# 模型专题

模型专题集合了公共库中里最新颖的模型，以专题的形式定期更新当季最热主题模型、节日热点等相关内容
