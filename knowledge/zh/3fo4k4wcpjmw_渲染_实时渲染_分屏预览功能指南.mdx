在移动画布、调整材质和灯光参数时，实时预览可以实时观察设计效果。其中的分屏预览功能可以实现双向协作 —— 您可以在一个电脑屏幕上专注调整渲染参数、移动画布或优化材质与灯光，另一个屏幕则实时呈现渲染效果，让设计调整与效果预览互不干扰，大幅提升操作效率~

# 实时渲染入口

**入口一：**进入室内设计后，选择**“渲染 - 实时渲染”**

![](https://qhstaticssl.kujiale.com/image/png/1754363172904/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**入口二：**进入**渲染**工具后，左侧点击**添加实时渲染**

![](https://qhstaticssl.kujiale.com/image/png/1754363281895/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 分屏预览

1. 进入实时渲染工具后，点击上方工具栏中的**“分屏预览”**或右上角预览框中的**“切换至分屏预览”**，可以切换到另一个屏幕实时预览

|  |  |
| --- | --- |
| **入口一** | **入口二** |
|  |  |
|  |  |

2. 点击**“合并预览页面”**，便可以切换为原始页面

![](https://qhstaticssl.kujiale.com/image/png/1754364137710/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1754365867848/9D46AC8CC60F4C89BC440E70C5B643BA.gif)

3. 分屏预览提供**锁定**功能

需要分屏视窗为某个固定视角时，可在当前视角下点击分屏视窗的**“锁定当前视角”**，即可固定分屏视窗视角；需要分屏视窗跟随画布变化时，单击“**解锁当前视角”**即可
