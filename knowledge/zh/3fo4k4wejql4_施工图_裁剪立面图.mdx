# 面向用户

所有用户

# 入口

1. 进入设计工具后，点击左侧栏的**图纸—施工图纸**

![](https://qhstaticssl.kujiale.com/image/png/1768375026227/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 在编辑视图中点击**立面索引图**

![](https://qhstaticssl.kujiale.com/image/png/1768383813237/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 裁剪立面图

1. 选择立面索引标记，然后点击浮窗中的**查看立面图**

![](https://qhstaticssl.kujiale.com/image/png/1768383894685/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 在左下角可以选择关闭或者打开**视图裁剪区域**，当显示视图裁剪时，可以直接拖动裁剪框调整立面图大小

![](https://qhstaticssl.kujiale.com/image/gif/1768384035853/DCE71965D19C1438840BBBDE410C6B53.gif)

* 也可以点击底部的**调整立面范围**，然后手动拖动裁剪框改变大小，完成后再次点击**查看立面图**，就可看到调整后的效果

![](https://qhstaticssl.kujiale.com/image/gif/1768384651410/BDD03DBD7B143CFD44CBE3156FF860F8.gif)

3. 调整完成后，点击左上角的**返回立面索引**按钮返回即可

![](https://qhstaticssl.kujiale.com/image/png/1768384906379/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
