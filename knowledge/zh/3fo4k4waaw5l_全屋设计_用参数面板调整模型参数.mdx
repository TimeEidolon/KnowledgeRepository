在参数面板中，所有参数以分组的形式展示，基础参数和高级参数分开，在设计和审核的过程中更方便查看和修改对应的参数，提高修改模型参数的效率；还具有搜索功能，一键搜索即筛选出参数栏，调整参数更快速高效

⭐在阅读文章前，建议您先查看视频教程，便于快速上手~

🎥 [Video URL](https://www.youtube.com/watch?v=Wrge7953GD8)

# 面向用户

所有用户

# 参数面板

1、选中模型，右侧参数面板被激活

**注意：若关闭了尺寸限制，部分可选项类型的参数，文字会变为数字显示，重新开启尺寸限制即可！**

![](https://qhstaticssl.kujiale.com/image/png/1737082447588/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

|  |  |  |
| --- | --- | --- |
| 可修改**模型名称** | 可将修改过的参数**恢复默认** | 可开启**拓展面板** |
|  |  |  |

2、参数面板以分组的形式展示可**一键展开/折叠**

![](https://qhstaticssl.kujiale.com/image/gif/1737084956027/AB8853C73610A95BED5FBB086BD7E920.gif)

3、**开启拓展面板**以双面板的形式呈现，所有参数一目了然

4、在**高级参数面板**中点击**面板常驻**，选中其他模型也会保持双面板的呈现形式

![](https://qhstaticssl.kujiale.com/image/gif/1737085703103/CB425447CE12698A336A13DF4D619B60.gif)

 5、参数面板具有搜索功能，可**搜索****参数名/参数值**，即可筛选出对应的参数栏，调整参数更高效

![](https://qhstaticssl.kujiale.com/image/png/1737086013784/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1737086424803/55333C8E5CE2E6454E893CD401848F46.gif)

 6、手动修改过参数的参数栏字体会加粗，方便用户进行模型审查和方案审核

![](https://qhstaticssl.kujiale.com/image/png/1737095591289/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* 点击右上角**恢复默认**则会清除加粗样式

![](https://qhstaticssl.kujiale.com/image/png/1737095939550/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
