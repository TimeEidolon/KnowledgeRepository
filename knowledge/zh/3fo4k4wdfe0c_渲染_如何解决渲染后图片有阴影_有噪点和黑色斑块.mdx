# 原因1

模型的两个面发生重叠，就会出现下图中的情况

1. 在第一张图中，可以看到，天空的位置并没有物体，但是也出现类似的情况，这是由于在相同位置有一块透明的玻璃，这样渲染的时候，玻璃的面和吊顶的面发生重叠，就会出现下图中的情况。
2. 在一些定制板材或者柜子中经常出现下图中的情况，这是由于模型在建模的时候就出现了两个面重叠的情况，所以在渲染的时候会出现有阴影的情况。
3. 由于光源和模型之间发生重叠，当吊顶高度和面光源高度重叠的时候，渲染就会出现下图中的情况，吊顶会丢失。

# 解决方法1

1. 如果是图一中的情况，只需要删除自由造影的重叠面，即可，如果还是不能解决问题，可以尝试清空吊顶设计来解决。
2. 把模型中重叠的面删除，如果是定制柜体的模型，可以尝试在资源列表中找到重叠的板件，删除即可。
3. 调整下面光源的高度，让吊顶高度和面光源不重叠。

# 问题2

画面很干净，但是局部某一个材质出现噪点，如下图

# 原因2

1.上传的模型附带的参数中有一个选项设置为低于1，就会出现下图中的情况，如果是公库的材质，那么就是上传的素材有问题，

2.照射的光线过暗

# 解法2

1.如果是自己上传的材质，只需要在max中将材质属性中的模糊值，调整为默认值1，然后重新上传即可，如果是公库的素材，推荐换一个素材或者使用新材质上传自己重新上传一个材质即可
