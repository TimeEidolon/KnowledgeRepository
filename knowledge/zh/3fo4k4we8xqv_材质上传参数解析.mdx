# 参数解析

|  |  |  |
| --- | --- | --- |
| **漫反射** |  | 效果1：颜色 - 如果您选择颜色作为漫反射效果，则只能修改材质表面的颜色  VR 颜色：用于指定或拾取漫反射颜色（在未使用贴图时有效）。 |
|  | 效果2：贴图 - 如果您选择贴图作为漫反射效果，则可以同时修改材质表面的颜色和纹理  VR 颜色：指定或拾取漫反射颜色并叠加在贴图上  色相：用于调整材质贴图的整体颜色的色调方向，例如黄色的贴图改为粉色  饱和度：用于调节颜色的鲜艳程度，默认数值0，数值为负，色彩变灰暗，数值为正，则色彩越鲜艳。  高级伽玛对比度：用于调节贴图的中间调对比度，默认数值1.0，大于1.0，对比变弱；小于1.0，对比变强。  高级亮度：用于调节材质贴图整体亮度，默认数值1.0，数值越大越亮，越小则越暗。 |
| **反射** |  | 效果1：颜色-反射颜色用于控制反射的强度与色彩倾向。  颜色的明度决定反射强弱：   * 黑色 = 0，完全无反射 * 白色 = 1，反射最强（镜面反射） * 灰色 = 中等反射   颜色本身的色调会给反射染色  金属：反射通常带金属色，例如铜是黄色  非金属：反射通常是黑白中间的数值 |
|  | 效果2：贴图-用贴图表示不同区域产生不同的反射强度/颜色。   * 贴图明度代表反射强弱 * 贴图的色彩（如果使用彩色贴图）会给反射染色   通常是黑白的贴图 |
| **反射光泽度** |  | 贴图：一般为黑白通道图，黑色表示反射完全模糊，白色为镜面效果  反射：物体表面的反射模糊程度，数值越大反射越清晰 |
| **凹凸/法线** |  | 效果1：凹凸贴图：用灰度贴图表示物体表面的凹凸  一般为黑白通道图，白色的凸起，黑色的凹陷  凹凸比例代表凹凸的深浅值，数值越大凹凸感越强  通常小颗粒纹理、砖墙表面小裂缝、布料细纹等会使用凹凸  效果2：法线贴图：模拟表面细节（凹凸、高光、阴影），光照和反射方向更真实  一般为彩色贴图，通过RGB 三个通道表示法线的 XYZ 偏移信息  通常金属划痕、木纹、布料纹理、小石头表面细节会用法线 |
| **置换** |  | 置换会通过贴图将模型的顶点进行偏移，在原本的平面上创造出凹凸的效果，使用置换贴图会大量增加渲染时间。  置换贴图一般为黑白图，白色凸起，黑色凹陷。  常见材质：石墙凹凸、雕刻图案、地面裂缝、地砖高低差 |
| **折射** |  | 用来控制物体的折射颜色和透明属性。  颜色越接近白色，物体越透明。颜色越接近黑色，物体则不透。在折射颜色为白色时，漫反射不起作用  折射率（IOR ）:折射率描述了光线穿过材料表面时的弯曲方式，值为1.0表示灯光不会改变方向等于空气，IOR 越高，表面越“亮”、反射越强；IOR 越低，反射越弱、表面越“糙。  开启光线追踪后才能实时预览折射率的效果”。  IOR =1.3–1.5，木头、塑料、布料等弱反射材质  IOR =1.5–1.6，玻璃  IOR =2.0–3.0，强反射材质，如钻石、金属氧化层 |
| **折射光泽度** |  | 用来控制折射的锐度。  贴图: 一般为黑白通道图，黑色表示完全模糊白色为完美的玻璃折射  数值: 值为1表示完美的玻璃状折射;较低的值会产生模糊或有光泽的折射  开启光线追踪后才能实时预览折射光泽度效果 |
| **菲涅耳** |  | 启用后，材质反射强度将取决于表面的视角，侧面角度较正面反射强  禁用后，材质在所有角度均具有相同的反射情况  基本上所有材质都有菲涅尔属性，菲涅耳折射率越高，物体表面反射越强 |
| **不透明度** |  | 和折射的参数只能二选一  贴图:贴图为白色时材质不透明，颜色越黑透明程度越高，黑色时材质全透明。  数值:数值为1.0时材质不透明，数值越小透明程度越高，数值为0时材质全透明。 |
| **自发光** |  | 颜色：决定物体发光的颜色  贴图：控制不同区域的发光强度和颜色  发光能量：控制自发光的亮度强弱。数值越高 ，物体发光越亮  不透明度：控制自发光材质的透明程度半透明发光材，光线透过物体散射；完全透明， 发光仅在边缘或纹理存在。 |
