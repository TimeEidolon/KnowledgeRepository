# **约束动画功能概述**

## 关于约束

**【约束】是什么？**

即 给不同模型物体间 建立**联系。**（为了方便阅读：以下我们称目标对象为A，约束对象为B）

当物体A与B之间建立约束关系后，可通过A的运动，驱动B进行对应运动，而无需对B进行动画设置

**【约束】能给我们做什么？**

掌握约束功能后可以在动画制作中做到：

* 减少重复操作，减少关键帧设置
* 使动画更佳丝滑流畅，新增更多层次

## 三种约束的定义及设置

在酷家乐棚拍中为大家上新了三种【约束】功能，详情如下：

**（1） 父子约束**

定义：即A跟B彷佛之间有一条线；  B（子）以A（父）的轴心为轴心进行旋转、缩放、移动

应用场景：办公椅动画视频（轮子仅作旋转动画帧，设置车身为父对象，可让轮子跟着车身位移移动的同时自身旋转）

设置入口：3D棚拍设计界面左侧“层级”操作面板-拖动B到A层级下

![](https://qhstaticssl.kujiale.com/image/png/1770014319035/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1770014511240/28147E8A9D68903126C703EB9498C759.gif)

**（2）跟随约束**

定义：B模仿A的行为动作

应用场景：办公椅其它轮子(B)跟随已经做好了动效的轮子(A)进行运动

设置入口：选择约束对象B-右侧操作面板-模型-**添加约束**-**跟随约束**-设置目标A-设置范围

![](https://qhstaticssl.kujiale.com/image/png/1770014619963/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1770014942609/85B748DD9CB2E062722F545F922B5A77.gif)

**（3）面向约束**

定义：不论A/B做什么运动，B都会一直朝向A，B会因为A的运动而产生旋转但不会由于A的运动产生位移

应用场景：相机镜头始终朝向商品方向、摄像头始终面向物体

入口设置：选择被约束对象B-右侧操作面板-模型-添加约束-**面向约束**-设置目标A-设置轴向

![](https://qhstaticssl.kujiale.com/image/png/1770014794304/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1770014927854/04F58D89884C87A4CAE199E8CE7E2C6D.gif)

# 约束效果演示

## 三种约束的概念演示

为方便大家理解：下图以红物体为目标对象A，蓝物体为约束对象B

在不同的约束关系下，蓝物体会因红物体的运动而产生对应的运动效果：

|  |  |  |  |
| --- | --- | --- | --- |
| **对红物体进行的操作** | **父子约束** | **跟随约束** | **面向约束** |
| **位移** |  |  |  |
| **旋转** |  |  |  |
| **缩放** |  |  |  |
|  | 给红物体设置关键帧动画，蓝物体会以红物体（父对象）的轴心为轴心进行对应运动 | 给红物体设置关键帧动画，蓝物体会以自身轴心“模仿”红物体进行运动 | 给红物体设置关键帧动画，当红物体位置发生改变时，蓝物体会对应运动以朝向红物体 |
|  | 这三种约束里面B的运动都不会影响A的运动 | | |
