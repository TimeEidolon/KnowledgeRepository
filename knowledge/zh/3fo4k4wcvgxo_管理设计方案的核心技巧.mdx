# 面向用户

所有用户

# 进入Coohom设计方案

1. **[登录](https://www.coohom.com/user/signin?kjl_source=hc)**您的Coohom账户

2. 点击右上角的**工作台**

3. 在**[方案](https://www.coohom.com/pub/saas/apps/project/list?kjl_source=hc)**页面，将鼠标悬停在您想要编辑的方案上，点击**进入方案**

* **详细操作可见教程文档：[点击查看](https://www.coohom.com/zh_CN/helpcenter/quickly-enter-the-coohom-design-project?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1752651223619/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752828486278/C709A4E4F6A47B36A86D49088F73D677.png)

# 将您的设计方案分享到社交社区

1. 进入**工作台-方案**，选择需要分享的方案，点击**分享**

2. 在弹窗中，选择您需要分享的社交社区

* **详细操作可见教程文档：[点击查看](https://www.coohom.com/zh_CN/helpcenter/2-steps-to-share-your-projects-to-social-community?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1752739495933/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752828486251/2D37C2734C50A6837B78ACB4959B66FF.png)

# 与其他人分享您的方案链接

1. 进入**工作台-方案**，选择想要与其他人共享的方案，点击**合作**

2. 您可以将设计方案链接分享给其他人，并可以在弹窗中对方案链接设置查看权限、有效期限和访问密码

* **详细操作可见教程文档：[点击查看](https://www.coohom.com/zh_CN/helpcenter/share-your-project-links-with-others?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1752739545414/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752828486280/2D37C2734C50A6837B78ACB4959B66FF.png)

# 重命名设计方案

进入**工作台-方案**，选择需要重命名的方案，点击**重命名**，重命名方案即可

* **详细操作可见教程文档：[点击查看](https://www.coohom.com/zh_CN/helpcenter/rename-your-projects?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1752739916530/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752828486021/C709A4E4F6A47B36A86D49088F73D677.png)

# 复制和删除设计方案

进入**工作台-方案**，选择需要**复制/删除**的方案，点击**复制/删除**即可

* **详细操作可见教程文档：[点击查看](https://www.coohom.com/zh_CN/helpcenter/duplicate-and-delete-your-projects?kjl_source=hc)**
