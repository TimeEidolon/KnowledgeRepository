定制模型及材质支持收藏，可分类收藏到自定义命名的文件夹中，随时取用

# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 收藏定制模型

## 方式一

1. 选择**厨房/衣帽间**，进入定制设计界面

2. 点击定制模型，浮窗工具栏中点击**☆收藏**

![](https://qhstaticssl.kujiale.com/image/png/1744704533986/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 方式二

选择**公库**中的定制模型，点击右上角☆一键**收藏**

|  |  |
| --- | --- |
|  |  |

# 收藏模型材质

## 入口一

在参数面板的**风格替换**中，选择材质点击**收藏**

![](https://qhstaticssl.kujiale.com/image/png/1744705367015/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1744705902211/C63B3EF8EFE0434A0B02F4463A2F1022.gif)

## 入口二

选择**公库**中的定制材质，点击右上角☆一键**收藏**

|  |  |
| --- | --- |
|  |  |

# 收藏夹分类

1. 浮窗工具栏中点击**收藏**，可**新建文件夹**，自定义命名进行收藏夹分类

![](https://qhstaticssl.kujiale.com/image/png/1744706573933/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 在对应工具线下，切换为**个人库**，**整体**栏下为定制模型收藏夹，**组件**栏下为定制组件收藏夹，**材质**栏下为定制材质收藏夹，选择文件夹下的素材即可拖出来使用

![](https://qhstaticssl.kujiale.com/image/png/1744706995973/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1744707268589/21CD7E179214935F1C7A8EBB848FBF95.gif)
