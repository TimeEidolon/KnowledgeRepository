# 添加手动灯光

1. 点击**添加手动灯光**

2. **命名**灯光方案，选择**基础灯光模板**可以在模板基础上进行手动调整

![](https://qhstaticssl.kujiale.com/image/png/1742961564890/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 进入手动打光界面，左侧面板可调整阳光的**色温、亮度和位置**

4. 点击基础模板内的灯光，可在右侧面板调整灯光的**色温****、亮度和高度**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1742982556283/2D37C2734C50A6837B78ACB4959B66FF.png)

5. 光源工具栏中，可以自行选择需要添加的**基础光源、IES光源、辉光和体积光**，拖出即可完成添加

![](https://qhstaticssl.kujiale.com/image/png/1742969768045/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1742982556558/FA4828BB27B835553782B9211E462BD1.gif)

6. 灯光方案设置完成后，点击**完成**

![](https://qhstaticssl.kujiale.com/image/png/1742970198688/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 批量处理灯光

## 方式一：按住**Shift / ⇧**多选光源+筛选

当大空间筒灯、射灯数量多的时候，这个方法可以更好地提升效率

* **按住Shift / ⇧键**可以框选或点选多个灯光，进行批量调整；还可以在工具栏中筛选光源类型

![](https://qhstaticssl.kujiale.com/image/png/1743647924891/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1743647538771/9E87848E191E9E1CC3938CEBDF4CD8EE.gif)

## 方式二：属性应用至同款灯

可以将属性参数快速应用至所有同类型的光源

* 选择一个光源，调整色温、亮度等属性后，点击**属性应用至同款灯**

![](https://qhstaticssl.kujiale.com/image/png/1743648411937/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 方式三：复制属性

可以将属性参数快速应用至指定的某几个同类型光源

* 选择一个光源，点击**复制属性**，再点击其他同类型光源，即可粘贴属性

![](https://qhstaticssl.kujiale.com/image/gif/1743651230295/78060F9996D5278C97AF3AA3A19DABED.gif)

# 替换灯光

选中**IES光源**，在浮窗工具栏中点击**替换**，替换其他款式的IES光源

![](https://qhstaticssl.kujiale.com/image/png/1743651622624/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 三轴定向复制灯光

在放置柜子层板灯时，有助于提升效率

* 选中光源，**按住Alt / ⌥键**并拖动三轴控件，即可沿坐标轴方向复制
