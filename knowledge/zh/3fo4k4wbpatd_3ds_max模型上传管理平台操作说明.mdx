Coohom支持上传3ds Max模型，本文会详细讲解如何使用3ds Max模型上传管理平台，便于您更顺利地上传模型

# 面向用户

所有用户，功能逐步开放中

# 插件安装

**1. 下载**

① 进入**工作台**，选择**模型**，点击**上传模型**

![](https://qhstaticssl.kujiale.com/image/png/1747276429710/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

② 进入上传模型界面，点击3ds Max模型上传管理平台中的**下载插件**，下载.exe文件

![](https://qhstaticssl.kujiale.com/image/png/1747276550280/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**2. 安装**

点击.exe文件进行安装操作，选择安装时使用的语言，插件会安装在3ds Max里

* **可支持3ds Max版本为：3ds Max 2021、3ds Max 2023**

![](https://qhstaticssl.kujiale.com/image/png/1747276948137/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1747279621793/0C0EBF9A038651E4440C51B706A54D96.png)

# 插件使用

**1. 插件功能**

① 安装成功后，打开**3ds Max**，菜单栏新增**CooVerse(K)**

![](https://qhstaticssl.kujiale.com/image/png/1747277685902/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

② 点击**CooVerse(K)**，打开功能弹窗，包括三个主要部分：

* **用户账号**
* **项目**
* **模型**

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1747275523346/E4CEB17D6589187948BE1DCD682B5796.png)

**2.模型上传**

**① 登录用户账号（必选）**

点击**登录**，唤起弹窗，输入Coohom账号密码，之后点击刷新按钮完成登录

![](https://qhstaticssl.kujiale.com/image/png/1747280003376/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**② 新建项目（必选）**

点击**创建**，新建项目文件夹，上传的模型会在相应的文件夹内

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1747275523297/E35B15ADF44B665E560766E13CBD60E3.png)

**③ 传输模型**

打开电脑本地的max模型，点击**上传模型**，云端处理进度100%即为成功

![](https://qhstaticssl.kujiale.com/image/png/1740659633122/1B02D6131CA86B26A0F549D261FC1998.png?x-oss-process=image/format,webp)

**④ 入库**

传输完成后，打开**“模型 - 上传模型 - 3ds Max模型上传管理平台”**，选择模型，点击**确认入库**

![](https://qhstaticssl.kujiale.com/image/png/1747277304462/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1747277196105/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**⑤ 上传**

入库后，选择**模型类目、位置**，进行模型上传操作即可

![](https://qhstaticssl.kujiale.com/image/png/1747277389172/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 注意事项

1. 使用插件需先在本地安装3ds Max，目前仅支持**2021**和**2023版本**

2. 目前**仅支持V-Ray for 3ds Max**，模型传输时需将渲染引擎更改为V-Ray

3. 建议使用**V-Ray 4**，效果更准确，**V-Ray 5及以上**会有效果损失

4. 该功能**仅对部分用户开放**

# 故障排除：安装后插件未响应

如果在**安装好CooVerse插件后，单击插件没有反应**，可能是以下几种原因导致，可通过下述方式操作解决:

| **问题原因** | **解决方法** |
| --- | --- |
| 插件文件损坏 | 1. 下载**Python文件夹**并解压——[点击下载](https://user-platform-oss.kujiale.com/upms/direct/63f9d2eadd6896b1-1740621401692-1.zip?kjl_source=hc)          2. 找到自己电脑下载的3DsMax安装路径，把Python文件夹替换为               刚解压好的Python文件夹          3. 重启3DsMax |
| 缺少必要的依赖项 | 确保安装了**最新版本的 Visual C++Redistributable**——[点此下载](https://user-platform-oss.kujiale.com/upms/direct/6168acb493447cdb-1741921563578-1.exe?kjl_source=hc) |
| 插件与3ds Max版本不兼容 | 换**3dsMax2023安装包版本**，初始版更合适 |
