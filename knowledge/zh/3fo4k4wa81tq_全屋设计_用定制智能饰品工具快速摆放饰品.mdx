# 面向用户

所有用户

# 入口

1. 选择**厨卫/全屋家具定制**，进入定制设计界面

2. 选择顶层工具栏中的**AI-定制智能饰品**

![](https://qhstaticssl.kujiale.com/image/png/1739254171463/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# **定制智能饰品**

1. 选中需要摆放饰品的柜子

2. 从智能饰品面板中选择对应的定制工具线

3. 选择适合的饰品模板，点击应用

![](https://qhstaticssl.kujiale.com/image/png/1739254497704/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
