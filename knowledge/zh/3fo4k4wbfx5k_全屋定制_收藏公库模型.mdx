公库中的模型素材支持收藏，可分类收藏到自定义命名的文件夹中，随时取用

# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 收藏公库模型

## 方式一

1. 选择**装饰**，在**公库**中选择模型素材

2. 选择**公库**中的模型素材，点击左上角☆一键**收藏**

|  |  |
| --- | --- |
|  |  |

## 方式二

点击公库中拖出的模型素材，浮窗工具栏中点击**☆收藏**

![](https://qhstaticssl.kujiale.com/image/png/1744708658121/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 收藏夹分类

1. 浮窗工具栏中点击**收藏**，可**新建文件夹**，自定义命名进行收藏夹分类

![](https://qhstaticssl.kujiale.com/image/png/1744710413208/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 选择**个人库 - 收藏**，选择文件夹下的素材即可拖出来使用

![](https://qhstaticssl.kujiale.com/image/png/1744711330310/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1744711901256/7F2F1E634F5739C982B586242400DF50.gif)
