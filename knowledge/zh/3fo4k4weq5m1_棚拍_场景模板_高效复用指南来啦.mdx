## 一、功能背景

**1. 你是否也曾遇到这种情况：**

* 轴心调了半小时，没法复用至其他场景？
* 灯光/模型动画做了一天，运营突然要换场景？
* 多店铺出多个视频，换个场景就得重打关键帧？

**2.「场景模板」来解决：**

* **场景模板可将灯光、相机、动画、模型参数保存为成模板，支持跨场景、跨子账号复用，减少重复操作，大幅提升设计效率。**

## 二、面向对象

* 面向棚拍付费企业客户，主账号及绑定电商棚拍管理员角色的子账号

## 三、功能详情

## 1. 如何使用

* 功能入口：在层级面板选中目标对象后右键点击，选择「保存为场景模板」，输入模板名称并选择存储文件夹，确认后完成保存。

![](https://qhstaticssl.kujiale.com/image/png/1773627590022/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* **模版调用：**左侧「企业库」面板 -「场景模版」- 选择任一模版放置到场景内适当位置即可

![](https://qhstaticssl.kujiale.com/image/png/1773627636535/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* 多选的对象中至少包含一个模型/空对象（单个模型/空对象也支持另存为企业场景模板库）
* 多选对象的数量需控制在50个以内

## 2. 不同角色操作权限

|  |  |  |
| --- | --- | --- |
| **角色** | **操作权限** | **限制** |
| 主账号 / 电商棚拍管理员 | 创建、重命名、删除、迁移模板；  管理企业场景模版文件夹； | 无 |
| 企业内子账号 | 调用模板、替换模型、调整已解锁的动画参数 | 不可修改、删除模板及管理文件夹 |
