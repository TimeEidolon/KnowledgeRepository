# 设置户型单位

1. 进入室内设计界面，将鼠标移动至头像处，选择**偏好设置**

![](https://qhstaticssl.kujiale.com/image/png/1745920856881/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 在偏好设置界面中，选择**通用 - 单位**，单位可选择**毫米、米、分数英寸、小数英寸、英尺/分数英寸、英尺/小数英寸**

![](https://qhstaticssl.kujiale.com/image/png/1745921275301/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1745921438070/801C56CA0B546DDD5138F0AB73509376.gif)

3. 设置成功后，以设置成英制单位为例，楼层属性的基础参数中单位会变为英制单位

![](https://qhstaticssl.kujiale.com/image/png/1756367579019/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. 点击导入图纸，比例尺上面便会显示已设置的单位，英制输入规则：支持按 “英尺+英寸” 输入

![](https://qhstaticssl.kujiale.com/image/png/1756367628504/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1756367874020/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
