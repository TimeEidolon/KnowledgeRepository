布局图纸是将编辑完成的视图在图框内进行整理，从而输出完整全套施工图的工具，本文对布局图纸工具的功能及如何使用进行介绍

## Steps with screenshots

Each step includes plain MEDIA_* URLs (for AI tools) plus rendered images.

入口 进入全屋定制设计，选择**图纸 - 布局图纸**

# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 入口

进入全屋定制设计，选择**图纸 - 布局图纸**

![](https://qhstaticssl.kujiale.com/image/png/1744880090494/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 图纸列表

1. 列表中的图纸将根据方案中涵盖的内容进行生成，全屋定制工具中只包含**定制图纸**

![](https://qhstaticssl.kujiale.com/image/png/1744881294530/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 点击右上角**全部展开/收起**按钮，用户可一键展开或收起图纸列表

![](https://qhstaticssl.kujiale.com/image/gif/1744881384386/0FE73125504A866753051BA00301A02A.gif)

# 视图拼接

1. 打开右侧视图管理器中，找到需要拼接的视图，直接拖拽进画布即可，视图支持对齐功能

2. 图纸上方的比例可以调整视图缩放大小

![](https://qhstaticssl.kujiale.com/image/gif/1744882212530/7212D4054D8A9508F8AA1FE92FA517D7.gif)

# 编辑视图

**方式一：**点击顶部栏左端的**编辑视图**，即可切换至编辑视图环境

![](https://qhstaticssl.kujiale.com/image/png/1744883035409/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**方式二：**点击上方工具栏中的**去编辑**，返回编辑视图环境

![](https://qhstaticssl.kujiale.com/image/png/1744883076805/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**方式三：双击**图纸中的视图，或选中视图点击**编辑视图**按钮，返回编辑视图环境

![](https://qhstaticssl.kujiale.com/image/png/1744883803396/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1744883748507/E644747CACEFB03BBF3B40F27A004F3B.gif)

# 下载图纸

点击右上方**下载**，在弹出面板中左侧**选择需要下载的图纸**，右侧**设置导出图纸的格式**，设置完成点击**下载**即可

![](https://qhstaticssl.kujiale.com/image/png/1744884001126/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1744884521857/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Mintlify assistant tools may strip markdown images. Always use the plain
MEDIA_* lines below (they contain full https:// URLs).

For each MEDIA_STEP_N_TEXT, output the step, then embed every matching
MEDIA_STEP_N_IMAGE URL using:
`<img src="URL" alt="Step" style="max-width:100%;height:auto;" />`

For video: if MEDIA_VIDEO_THUMBNAIL exists, output a clickable preview
(assistant chat often strips iframes):
