# 生成全屋漫游

1. 根据您的方案生成多张全景渲染图

![](https://qhstaticssl.kujiale.com/image/png/1748420401266/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 进入**方案**，点击**方案详情**，选择**全屋漫游**，点击**生成全屋漫游**

![](https://qhstaticssl.kujiale.com/image/png/1748420503948/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1748420558865/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 选择**全景图**，点击**生成**

![](https://qhstaticssl.kujiale.com/image/png/1748420760502/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. 生成全屋漫游后，点击**查看全屋漫游**

![](https://qhstaticssl.kujiale.com/image/png/1748420810660/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 建议

* 确保每个房间都是封闭的.
* 您可以[重命名全景图](https://www.coohom.com/en_US/helpcenter/article/how-to-rename-panorama?kjl_source=hc)，使其在漫游中显示正确
* 您的方案中只能生成一个全屋漫游，以防万一它被新的覆盖
