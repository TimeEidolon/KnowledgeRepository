# 设计转角飘窗

1. 点击**户型**

2. **定制门窗**中选择**L型/U型飘窗**，放置到墙体上

* 教程演示以L型飘窗为例

![](https://qhstaticssl.kujiale.com/image/png/1740724570592/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 点击飘窗模型，进入**门窗定制**

![](https://qhstaticssl.kujiale.com/image/png/1740724711147/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. 选中飘窗模型，右侧参数面板可调整飘窗的具体参数

![](https://qhstaticssl.kujiale.com/image/png/1740730563390/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. 在**风格替换**中，可替换**窗、套线、套板、窗台和材质样式**

![](https://qhstaticssl.kujiale.com/image/png/1740730670509/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
