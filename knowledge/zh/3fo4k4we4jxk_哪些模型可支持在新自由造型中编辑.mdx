# 问题

用户发现部分模型可以进行自由造型编辑

![](https://qhstaticssl.kujiale.com/image/png/1764731807796/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 原因

1.上传到个人库skp文件，支持自由造型编辑，进入自由造型编辑页面后，可对模型各个部分进行删除、拉伸、变形等操作。（[skp文件的上传要求](https://www.coohom.com/zh_CN/helpcenter/sketchup%E6%A8%A1%E5%9E%8B%E4%B8%8A%E4%BC%A0%E8%A6%81%E6%B1%82?kjl_source=hc)）

2.部分公库模型支持自由造型编辑。

# 使用方法

将模型“转为可编辑造型”

转为可编辑造型是一个不可逆的过程，会丢失该模型的商品数据，但转化后可对造型进行多次编辑

![](https://qhstaticssl.kujiale.com/image/png/1764211474381/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1764211805820/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

进入自由造型编辑页面后，可对模型各个部分进行删除、拉伸、变形等操作。

![](https://qhstaticssl.kujiale.com/image/png/1764214232946/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
