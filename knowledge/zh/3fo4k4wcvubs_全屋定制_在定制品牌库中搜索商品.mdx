在定制品牌库的搜索框中，搜索所需商品相关的关键词，便可以一键搜索到所有商品，本文会为您详细讲解如何在定制品牌库中搜索商品

# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 入口

进入全屋定制设计，选择**厨房**

![](https://qhstaticssl.kujiale.com/image/png/1753253434092/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 切换目录

点击右上角切换目录标志，在展开栏中可切换不同目录

![](https://qhstaticssl.kujiale.com/image/png/1753254005474/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 搜索商品

在搜索框中搜索所需商品的**模型型号或品牌**，可以一键搜索到您想找的商品

![](https://qhstaticssl.kujiale.com/image/png/1753431417254/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

|  |  |
| --- | --- |
| **搜索商品的模型型号** | **搜索商品的品牌** |
|  |  |

在搜索结果中，不仅可选择检索到的商品，还可以点击商品所属的**品牌目录**，切换至**品牌目录**下查看商品

![](https://qhstaticssl.kujiale.com/image/gif/1753254844562/6289FE0F5FDCAE8BD4F9B2A7EA930262.gif)
