在厨房和衣帽间设计中，设计师可以自行上传模型材质，上传的材质可以替换到定制柜体上，本文会为您详细讲解如何在厨房和衣帽间设计中上传模型材质

# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 上传模型材质

## 方式一

1. 进入**工作台**，选择**模型**，点击**上传材质**

![](https://qhstaticssl.kujiale.com/image/png/1778470812356/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 有两种新建材质的方式，可制作**实时材质**，或按照要求的格式自行**上传材质**图片

![](https://qhstaticssl.kujiale.com/image/png/1778470876731/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 方式二

1. 进入全屋定制设计，选择**“厨房/衣帽间”**

![](https://qhstaticssl.kujiale.com/image/png/1753258650624/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 点击右上角切换库标志，切换为**个人库**

![](https://qhstaticssl.kujiale.com/image/png/1753259987127/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 进入**厨房/衣帽间**后，有两个入口可进行**模型材质上传**，详见下表：

|  |  |
| --- | --- |
| **入口一** | **入口二** |
| 打开**厨房/衣帽间的个人库**，选择**“上传”**，点击**材质上传** | 点击定制柜体，选择参数面板中的**“风格替换-柜体材质-我的上传”**，点击**材质上传** |
|  |  |

3. 点击**上传**后，会跳转到**工作台**页面，再按照方式一的操作流程上传材质即可

![](https://qhstaticssl.kujiale.com/image/png/1778470812356/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 查看已上传的模型材质

材质上传成功后，可以在**个人库**中查看

|  |  |
| --- | --- |
| **查看方式一** | **查看方式二** |
| 打开**厨房/衣帽间的个人库**，选择**“我的-上传”** | 点击定制柜体，选择参数面板中的**“风格替换-柜体材质-我的上传”** |
|  |  |

# 替换已上传的模型材质

**方式一：**打开**厨房/衣帽间的个人库**，选择**“我的-上传”**，将上传的模型材质拖出便可以一键替换到定制柜体上

![](https://qhstaticssl.kujiale.com/image/gif/1753261744113/2B82777CAC3BF765272E49F23752F161.gif)

**方式二：**点击定制柜体，选择右侧参数面板中的**“风格替换-柜体材质-我的上传”**，点击上传的模型材质即可

![](https://qhstaticssl.kujiale.com/image/gif/1753261802178/B62342E261719BE94780D023B7117897.gif)
