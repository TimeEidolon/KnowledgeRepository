为了便于您更顺畅地在Coohom中上传3ds Max模型，本文会为您详细讲解3ds Max模型上传要求

# 面向用户

所有用户

# 支持格式

1. 2021及更早的版本
2. .max格式，可以直接上传，若有贴图需要进行归档，打包成zip、rar格式，归档的方式：（链接）
3. .3ds / .dae / .prt / .asm等格式，需要进行打包成zip、rar

# 模型要求

| 类别 | 具体要求 | 注意事项 |
| --- | --- | --- |
| 模型尺寸 | 单位设为毫米（mm），大小在30000mm\*30000mm以内 | 1）一个压缩包中仅能包含一个 **.max** 源文件。  2）压缩包不得设置解压密码。  3）**.max** 源文件不得损坏，必须能用 3ds Max 正常打开。  4）源文件不得包含 **ALC、ALC2、CRP、ADSL** 等第三方脚本。  5）源文件中必须包含模型，不允许上传空场景。  6）模型文件过大和三角形面数过多会导致上传时间过长渲染失败及破面。  7）不同的材质必须分开附给不同模型部位，多维子材质一定要拆分，便于后期材质替换。  8）支持发光材质与VRay灯光，但是使用VRay灯光时请确保是商品(灯具，灯箱等)所需，但是渲染所用的辅助照明灯光需要删除掉。  9）避免在材质编辑器中更改贴图尺寸，贴图不要使用真实世界比例及裁剪 |
| 模型大小 | 200M以内 |
| 模型面数 | 200万三角形面内 |
| 模型对象 | 200个OBJ部位以内 |
| 场景内物体 | 只保留需要的模型本身，删除摄像机等辅助对象，如果需要保留灯具光源，可以解冻、取消隐藏后一起上传 |
| 模型朝向 | 在max的正视图视角，模型正面要朝前，避免模型上传后默认角度错误。 |
| 模型修改器 | 如果模型叠加多个修改器命令，建议右键将模型转为可编辑多边形/可编辑网格。 |
| 模型材质 | VRay材质（VRayMtl） |
| 模型贴图 | 贴图文件大小在5M内  像素在5000\*5000以内，且不能低于1000\*1000  格式支持 jpg / png / tga，颜色模式为RGB  贴图只能使用通道1，且不勾选真实世界比例 |
| 伽马值要求 | 勾选启用Gamma/LUT 校正，max文件Gamma设置为2.2；  打开文件-查看图像文件，确保所有贴图为自动（推荐）或覆盖2.2 |

# 材质支持情况

可支持大多数材质的上传，但不支持以下贴图和材质：ColorCorrect、VaryFastSSS2、Vraysky、Perlin Marble等
