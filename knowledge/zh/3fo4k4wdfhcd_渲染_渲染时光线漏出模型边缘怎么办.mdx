# 问题

墙角、地缝或天花边缘出现异常光条或亮边。

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762136031450/43CA081BC2833BA37D008814AE7F1E1C.png)

# 解决方法

1.检查导入的模型是否存在开口或墙体间隙。

2.打开灯光模板中的漏光修复。

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762136031416/DAA19F2E6CD423718B5AFF13A6B650A3.png)
