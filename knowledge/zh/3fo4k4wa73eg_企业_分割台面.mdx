生成台面后，可对台面进行分割，对分割后的每块台面单独计算价格，减少了台面报价误差

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
进入台面分割界面后，可自由添加台面分割线，点击分割线可以删除

### Step 2
可拖拽点或线进行移动，调整分割的角度 * 吸附内外轮廓的差异： | | | | --- | --- | | **吸附内轮廓** | **吸附外轮廓** | | 不包含前挡水深度在内的，仅台面板的轮廓区域 | 包含前挡水深度在内的，台面板+前挡水深的轮廓区域 | | | |

# 面向用户

所有企业用户

# 入口

**入口一：**生成台面后，选中台面，浮窗工具栏中选择**台面分割**

* 生成台面的操作，**可见教程文档：[生成台面](https://www.coohom.com/zh_CN/helpcenter/article/%E7%94%9F%E6%88%90%E5%8F%B0%E9%9D%A2?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1739863642557/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**入口二：**进入自由编辑界面，顶部工具栏中选择**台面分割**

![](https://qhstaticssl.kujiale.com/image/png/1739863858046/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1739863582779/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 分割台面

1. 进入台面分割界面后，可自由添加台面分割线，点击分割线可以删除

![](https://qhstaticssl.kujiale.com/image/gif/1739864250625/214735FD43FB9A99EE5A0116F2CEBEE8.gif)

2. 可拖拽点或线进行移动，调整分割的角度

![](https://qhstaticssl.kujiale.com/image/gif/1739864626073/AF8322157298DDC0AC3AED18B96DA1BB.gif)

* 吸附内外轮廓的差异：

|  |  |
| --- | --- |
| **吸附内轮廓** | **吸附外轮廓** |
| 不包含前挡水深度在内的，仅台面板的轮廓区域 | 包含前挡水深度在内的，台面板+前挡水深的轮廓区域 |
|  |  |

3. 分割线添加完成后，点击**完成分割**

![](https://qhstaticssl.kujiale.com/image/png/1739865575270/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 后挡水分割

在台面分割界面，顶部工具栏中选择**后挡水分割**，即可在后挡水线条上添加点，完成后挡水的切割

![](https://qhstaticssl.kujiale.com/image/png/1739865444784/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
