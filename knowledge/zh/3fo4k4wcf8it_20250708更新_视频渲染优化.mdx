# 优化具体表现

**1. 引擎升级**

渲染视频经过升级，在使用极速&室内系列3.0及以上版本的灯光时，极速渲染的视频支持了更多动态模型的类型，材质支持新增了半透明材质、覆盖材质、混合材质

**![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1751872058819/0BB7DA08FED6DC84C3E8D24BB4EAFCD1.gif)**

**2. 视频和图片的效果保持一致**

根据用户选择不同灯光模板提供对应的视频渲染方式，可支持的分辨率会自动改变，渲染出的视频灯光效果可以和普通图保持一致

**![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1751872058831/6D18299F18874012AD6483269D5F4948.gif)**

# 面向用户

所有用户

# 操作方式影响

1. **视频分辨率的选择方式发生了变化：**原有的分辨率并没有被取消，而是根据所选灯光模板自动进行匹配，确保视频效果更一致

2. 模板视频也将根据所选灯光模板自动匹配渲染方式，功能将逐步上线

# 可能带来的新疑问

**【****极速类&写实类3.0及以上版本】模板的用户：**

**Q1：**我使用【极速类&写实类3.0及以上版本】，发现不能渲染【HQ720P&HQ1080P】了，但是我觉得这些分辨率的视频效果更好，该怎么办？

**A1：**如果觉得之前的效果视频更好，建议使用【写实类&均衡类】灯光模板，然后渲染视频会保持和效果一致

**【写实类&均衡类】灯光模板的用户：**

**Q1：**我使用【写实类&均衡类】灯光模板，无法当日出720P、1080P视频了，该怎么办？

**A1：**这类灯光模板的参数配置渲染时间确实比较久，只能隔夜出，如果希望可以当天出视频建议更换【极速类&写实类3.0及以上版本】灯光模板

**Q2：**我使用【写实类&均衡类】灯光模板渲染480P的时候，发现视频卡顿，是否是故障？

**A2：**480P作为预览档位，为了保证当日出图，所以是低帧率视频，但是升级成高分辨率时，会保证正常帧率
