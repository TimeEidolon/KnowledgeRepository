定制门窗及材质支持收藏，可分类收藏到自定义命名的文件夹中，随时取用

⭐在阅读文章前，建议您先查看视频教程，便于快速上手~

🎥 [Video URL](https://www.youtube.com/watch?v=66IXzRhBDv0)

# 面向用户

所有用户

# 收藏定制门窗

## 入口一

1. 选择**门窗定制**，进入定制设计界面

2. 点击定制门窗模型，浮窗工具栏中点击**收藏**

![](https://qhstaticssl.kujiale.com/image/png/1739944286700/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 入口二

选择**公共库**中的定制门窗模型，点击左上角☆一键**收藏**

|  |  |
| --- | --- |
|  |  |

# 收藏门窗材质

## 入口一

参数面板的风格替换中，选择门窗材质点击**收藏**

![](https://qhstaticssl.kujiale.com/image/png/1739944999194/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 入口二

选择公共库中的定制门窗材质，点击左上角☆一键**收藏**

![](https://qhstaticssl.kujiale.com/image/png/1739947128182/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 收藏夹分类

1. 浮窗工具栏中点击**收藏**，可**新建文件夹**，自定义命名进行收藏夹分类

![](https://qhstaticssl.kujiale.com/image/png/1739947248302/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 点击**我的**，**整体**栏下为定制门窗收藏夹，**材质**栏下为门窗材质收藏夹，选择文件夹下的素材即可拖出来使用

![](https://qhstaticssl.kujiale.com/image/png/1739947454223/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1739947819871/B398A598A0FA51A01EAC17B84F1D1171.gif)
