# 下载施工图

1. 在**编辑视图/布局图纸**，点击右上角的**下载**

* 编辑视图详见下图：

![](https://qhstaticssl.kujiale.com/image/png/1769423640775/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* 布局图纸详见下图：

![](https://qhstaticssl.kujiale.com/image/png/1769423733593/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 在下载界面中，可以根据需要勾选需要的图纸，以及文件格式。点击**下载**即可下载文件

* 左侧图纸目录勾选需要下载的图纸
* 下载格式可分别选择CAD、JPG或PDF。当格式为CAD时，可选择是否保留彩图；为jpg时，可选择彩色或黑白及清晰度；为PDF时，可选择彩色或黑白

![](https://qhstaticssl.kujiale.com/image/gif/1769423854054/435B495D2CC0DB3743BE6523FD081D5D.gif)

注意：在下载页面，当选择CAD格式时，可进一步选择是否需要采用**布局空间出图**

![](https://qhstaticssl.kujiale.com/image/png/1769425429691/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* 选择布局出图的情况下，图纸会展示在布局空间（见下图），通过布局出图可以实现部分视图修改的联动，修改模型空间的视图，布局空间里应用了该视口的图纸都会联动修改，目前默认联动视图分为以下几组：

1. * 组一：墙体定位图、平面布置图、家具尺寸图、地坪布置图
   * 组二：综合天花图、天花尺寸图、灯具定位图
   * 组三：强电点位图、弱电点位图、水路施工图

![](https://qhstaticssl.kujiale.com/image/png/1769422059241/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 点击**下载**后，会弹出进度条，方便查看下载进度（弹窗中会有随机操作技巧介绍）

![](https://qhstaticssl.kujiale.com/image/png/1769423890443/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. 下载结束后，方案中可看到上一次下载的结果，可根据需要进行下载

![](https://qhstaticssl.kujiale.com/image/png/1769423951393/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 指定默认导出的图纸（企业功能）

在施工图环境中进行**下载/存档**时，如果图纸文件夹配置了默认导出，那么在**下载/存档**弹窗中，文件夹内的图纸会自动选中，不用手动勾选

* 注意：如果企业用户需要此功能，可以联系客户成功顾问咨询
