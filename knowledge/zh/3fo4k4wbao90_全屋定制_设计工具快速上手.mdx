全屋定制设计工具是一种适合定制设计师的专业设计工具。与原始的室内设计工具相比，该工具简化了设计过程，并丰富了厨房、浴室和衣柜模型，使设计师能够更高效快捷地完成全屋定制设计~

# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 入口

进入**工作台**，选择**厨房和衣橱设计**

![](https://qhstaticssl.kujiale.com/image/png/1779701521850/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

进入**开始设计**界面后

* **方式一：**直接点击**“开始设计”**，用墙体工具自行绘制户型
* **方式二：**可以选择**设计样板**，点击**应用**，包括了户型、硬装和软装的全屋设计方案一键生成
* **方式三：**可以选择**户型模板**，点击**应用**，工具界面会生成对应的户型

![](https://qhstaticssl.kujiale.com/image/png/1745745930674/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 工具界面

进入全屋定制设计界面后，您会注意到它分为三个模块：左侧导航栏、顶部工具栏和右侧面板

* **①** **左侧导航栏：**按照设计顺序分为五个工具模块：**房间、厨房、衣帽间、装饰、渲染与图纸**
* **②** **顶部工具栏：**提供了【保存】【撤销】【工具】等快速操作功能入口
* **③** **右侧面板：**可预览房间的不同视角，以及选中修改对象后可进行参数设置

![](https://qhstaticssl.kujiale.com/image/png/1745747845869/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# **绘制户型**

在**户型设计**中，您可以使用矩形墙工具快速绘制闭合的户型，添加柱子、梁、烟道和门窗到户型中，还可以对不同房间进行命名区分

* **⭐建议您在设计户型前，先阅读：[【全屋定制】户型设计基础入门](https://www.coohom.com/zh_CN/helpcenter/kitchen-closet-room-design-basics?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1745748707286/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 设计厨房和衣帽间

1. 在**厨房设计**中，您可自行组合地柜、吊柜、台上柜，并添加到设计方案中，可以调整柜体模型的尺寸、材质及样式，一键生成台面顶线~

* **⭐建议您在设计厨房前，先阅读：[【全屋定制】厨房设计基础入门](https://www.coohom.com/zh_CN/helpcenter/kitchen-closet-kitchen-design-basics?kjl_source=hc)**

2. 在**衣帽间设计**中，您可自行选择需要的全屋家具素材添加到设计方案中，可以调整柜体模型的尺寸、材质及样式，自由设计柜体内部结构~

* **⭐建议您在设计衣帽间前，先阅读：[【全屋定制】衣帽间设计基础入门](https://www.coohom.com/zh_CN/helpcenter/kitchen-closet-closet-design-basics?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1745749128559/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 添加装饰

Coohom公库里集合了上百万个模型资源，为用户搭建多样化方案保驾护航，可快速获取模型，一键收藏或直接使用至方案中~

* **⭐建议您在添加装饰前，先阅读：[【全屋定制】公库使用指南](https://www.coohom.com/zh_CN/helpcenter/kitchen-closet-public-library-usage-guide?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1745749367431/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 渲染效果图

搭建完设计场景后，渲染工具能帮助我们快速生成好看的效果图。从选择渲染模式开始，依次添加灯光、外景，调整相机位置，设置渲染清晰度与高级参数，最后完成效果图渲染~

* **⭐建议您在渲染效果图前，先阅读：[【全屋定制】渲染工具基础入门](https://www.coohom.com/zh_CN/helpcenter/kitchen-closet-images-tools-basics?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1742956313863/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 下载施工图

在全屋定制设计过程中，图纸设计是连接创意与实际施工的关键环节。您需要掌握从视图编辑到图纸输出的基础操作~

* **⭐建议您在下载施工图前，先阅读：[【全屋定制】图纸设计基础入门](https://www.coohom.com/zh_CN/helpcenter/kitchen-closet-drawing-design-basics?kjl_source=hc)**
