# 问题1

找不到max

# 解决办法1

本地解压，打开文件夹查看是否有max文件。多数情况把其他类型文件，上传到模型入口造成的原因。

# 问题2

MAX解析失败

# 解决办法2

1.Max压缩包超过200MB，如上传整个空间（如一个卧室），这种情况需要拆分模型。

2.超面模型（超过平台200W面的限制）。

3.存在空物体和辅助对象，并且数量过多，导致解析失败，如果自己找不到辅助对象，需要导出FBX格式文件再打开就可以看见。

4.MAX文件缓存过多或者是MAX文件有病毒。

5.检查MAX文件的单位设置是否为mm，若不是，把单位设置为mm，再重新上传下，看是否能成功。

# 问题3

未知错误

1.压缩文件有解压密码。

2.模型综合问题过多。

3.网络问题，重新上传就可以。

# 解决办法3

如果以上情况商家不能解决可以咨询在线的模型专家，或者寻找对应的模型运营解决问题。
