# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 设置墙体相交自动拆分

1. 鼠标放到右上角头像处，选择**偏好设置**

![](https://qhstaticssl.kujiale.com/image/png/1740982192498/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 进入偏好设置界面，**户型＆布置**中的**墙体相交自动拆分**一栏，选择**自动拆分**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1740987853515/2D37C2734C50A6837B78ACB4959B66FF.png)

3. 墙体相交自动拆分与不拆分效果如下：

|  |  |
| --- | --- |
| **自动拆分** | **无自动拆分** |
|  |  |

* 拆分后的墙体可单独调整，未拆分的墙体仍为整体

![](https://qhstaticssl.kujiale.com/image/gif/1740995925475/C5288AF57DA506A93D69776CEED88407.gif)
