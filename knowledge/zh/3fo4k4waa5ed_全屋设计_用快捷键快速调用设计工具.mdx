# 面向用户

所有用户

# 入口

进入定制工具线，将鼠标拖到头像处展开栏中选择**快捷键和鼠标**

![](https://qhstaticssl.kujiale.com/image/png/1738981782148/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 快捷键面板设置

1. 进入快捷键面板，左上方选择相应的工具线

![](https://qhstaticssl.kujiale.com/image/png/1738982328199/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 为了延续用户使用CAD、SU快捷键的操作习惯，有三种模板可供选择：**默认模板、CAD模板、SU模板**

![](https://qhstaticssl.kujiale.com/image/png/1738985499464/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 左侧可选择不同功能的快捷键进行查看，上方有关键词搜索功能可更快速便捷地找到需查找的快捷键

![](https://qhstaticssl.kujiale.com/image/png/1738986309814/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. 可自行编辑修改快捷键，如果出现与已有快捷键重复的会出现相应提示，以避免设置重复的快捷键，设置完成点击**保存**即可

* 修改编辑后需要恢复默认快捷键，点击左下角**恢复默认快捷键**即可

![](https://qhstaticssl.kujiale.com/image/png/1739000594880/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
