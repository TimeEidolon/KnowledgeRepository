# 面向用户

所有企业用户

**Q: 当用户查看图像时，URL 中的数据是否可以显示场景/模型的名称？**

**A:** URL 可以显示模型和棚拍场景的 ID

**⭐请阅读以下细节：**

1. 影棚照片图库与特定产品型号相关。浏览这些照片时，您可以从顶部 URL 中的元数据中判断出该影棚照片是为哪个产品型号拍摄的，然后您可以根据BrandGoodId搜索目标型号

* 在下图中，照片对应的模型 ID，即 BrandGoodId，图中红框标注处的“3FO48E5RLI8B”
* 同时，相册 ID便是该相册的影棚 ID，显示为“3FO4K58VANHC”

![](https://qhstaticssl.kujiale.com/image/png/1748486938925/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* 通过输入 BrandGoodId，您可以在企业目录的模型库中查找与照片相关的产品型号。搜索完成后，产品型号的名称也会显示在其预览下方

![](https://qhstaticssl.kujiale.com/image/png/1748487012127/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1748487109840/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 虽然每个影棚方案都有其专属的影棚 ID，并显示在顶部的 URL 中，但您无法根据该 ID 在影棚相册中搜索照片所属的影棚。换句话说，影棚 ID 无法帮助您查找或定位影棚。如果您想获取搜索结果，请输入您创建的影棚名称

## Assistant step guide

Mintlify assistant tools may strip markdown images. Always use the plain
MEDIA_* lines below (they contain full https:// URLs).

For each MEDIA_STEP_N_TEXT, output the step, then embed every matching
MEDIA_STEP_N_IMAGE URL using:
`<img src="URL" alt="Step" style="max-width:100%;height:auto;" />`

For video: if MEDIA_VIDEO_THUMBNAIL exists, output a clickable preview
(assistant chat often strips iframes):
`<a href="MEDIA_VIDEO"><img src="MEDIA_VIDEO_THUMBNAIL" alt="Video" /></a>`
For MEDIA_VIDEO_FILE (.mp4), use `<video controls src="URL"></video>`.
