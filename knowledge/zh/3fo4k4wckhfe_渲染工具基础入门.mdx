# 面向用户

所有用户

# 入口

1. 进入**工作台**，右上角选择**创建新项目**

![](https://qhstaticssl.kujiale.com/image/png/1779344824290/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 进入室内设计后，选择**渲染**

![](https://qhstaticssl.kujiale.com/image/png/1750328278740/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 设置构图

1. 选择渲染模式，有**普通图、全景图、俯视图**可选

2. 左侧面板设置**构图比例**

* 全景图渲染出来为360°效果，无需设置构图比例

![](https://qhstaticssl.kujiale.com/image/png/1750650012124/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 添加灯光

## 选择灯光模板

选择**灯光模板**，可设置**亮度和效果**参数

![](https://qhstaticssl.kujiale.com/image/png/1750650102725/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 添加手动灯光

1. 点击**添加手动灯光**

2. **命名**灯光方案，选择**基础灯光模板**可以在模板基础上进行手动调整

* **与添加手动打光相关的具体操作，可见教程文档：[三步设置自定义灯光](https://www.coohom.com/zh_CN/helpcenter/3-steps-to-light-customization?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1750650296839/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 添加外景

选择合适的外景，可设置**外景亮度、色调、位置和环境光亮度**

![](https://qhstaticssl.kujiale.com/image/png/1750650355629/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 调整相机位置

选中相机，右侧面板可以设置**相机的位置、视角和剪裁效果**

* **与设置相机参数相关的具体操作，可见教程文档：[设置相机参数和视角](https://www.coohom.com/zh_CN/helpcenter/set-camera-properties-and-views?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1750651106310/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1750651239527/699406343916512CFD3FDBCA4E93DAEF.gif)

# 设置渲染清晰度

有四种清晰度可以选择，清晰度越高渲染的时间会越长

* **建议：可以先选择1K标清渲染小图，初步确定渲染效果，调整后再选择清晰度高的进行渲染**

![](https://qhstaticssl.kujiale.com/image/png/1750651277296/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 设置高级参数

选择高级设置，可以对渲染效果进行高级参数调整

* **与设置高级参数相关的具体操作，可见教程文档：[设置渲染高级参数](https://www.coohom.com/zh_CN/helpcenter/set-advanced-render-parameters?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1750399337599/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 查看效果图

调整完所有渲染参数后，点击**立即渲染**，渲染完成的效果图可以在右上角的**图册**中查看

![](https://qhstaticssl.kujiale.com/image/png/1750658639010/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1750651758538/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
