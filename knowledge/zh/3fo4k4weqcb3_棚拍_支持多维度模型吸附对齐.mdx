# **一、功能概述**

* 模型吸附对齐功能包含对齐（轴心点对齐、定界框对齐）与吸附（定界框吸附、物体表面吸附）两大类操作，并且支持一键落地快速定位，可实现多对象位置校准与高效摆放，覆盖场景搭建中各类位置调整需求。

# **二、功能价值**

* 效率提升：批量处理多对象位置，减少 80% 手动调整的重复劳动，场景搭建速度显著加快；
* 精准度保障：实时高亮吸附/对齐提示，降低位置误差至毫米级，避免手动摆放的偏移问题；
* 专注创意：解放设计师从繁琐校准中脱身，更聚焦场景整体搭配与创意设计，间接提升图片/视频质量。

# **三、功能介绍**

## 1. 模型对齐

* 支持通过「轴心点」或「定界框」两种对齐方式，将多个对象快速校准到统一位置基准。
* 操作入口： 3D 棚拍工作台 - 按住【Shift】键选中需要对齐的多个对象 - 右侧操作面板 - 【对象】-【对齐】

![](https://qhstaticssl.kujiale.com/image/png/1773626939694/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## （1）轴心点对齐

* 将被对齐模型的 X/Y/Z 轴轴心点，精准匹配参考对象对应轴的轴心点坐标，实现「点对点」精准对齐。​

* 示意图（黄色模型为参考对象）：

|  |  |  |
| --- | --- | --- |
| X轴心对齐 | Y轴心对齐 | Z轴心对齐 |
| 主视图 | 俯视图 | 主视图 |

## （2）定界框对齐

* 以参考模型的边界框（三维空间中的最小包裹框）为基准，支持X/Y/Z轴上的最小值、最大值、中心点对齐

  + 以X轴为例，简单来说就是能让 A 物体的左边对齐 B 物体的左边，或 A 的右边对齐 B 的左边，按需进行模型对齐操作。

* 示意图（黄色模型为参考对象）

|  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- |
| X轴 |  |  |  |  |  |
| Y轴 |  |  |  |  |  |
| Z轴 |  |  |  |  |  |

💡 参考对象设置 ：

* 默认参考对象：框选多个对象的时候，默认最后一个选中的对象为参考对象；

* 参考对象调整：按住【Ctrl】+ 左键单击目标对象，当对象出现蓝绿色线框时，即表示参考对象切换成功

## 2.模型吸附

## （1）定界框吸附

* 操作入口：3D棚拍工作台 -顶部工具栏-【吸附】-【定界框】

* 模型移动时靠近其他对象的定界框，会自动吸附
* 支持同时吸附多个面

* 支持通过【Ctrl】+ 拖动模型取消定界框吸附

## （2）物体表面吸附

* 移动吸附对象时，系统会自动吸附到距离最近的物体表面，模型随表面角度旋转并精准贴合物体表面；​
* 吸附过程中，实时高亮「模型吸附面」与「目标物体表面」，直观提示贴合位置。
* 常见场景：​

  + 挂画 → 自动贴合墙面；​

* + 吊灯 → 吸附到吊顶；​
  + 沙发 → 贴合地板；
  + ​摆件 → 灵活吸附到桌面、台面等任意表面。

## **3.一键落地**

* 用于将选中的模型快速对齐到场景中的特定平面，一般为地面或其他基准面（比如展台、桌面等）

* 操作入口：3D棚拍工作台 - 选中模型对象 - 工具栏 - 一键落地
* 快捷键：G

![](https://qhstaticssl.kujiale.com/image/png/1773626995919/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

⚠️ 注：若模型已绑定「父子关系」「跟随约束」「面向约束」，则无法使用一键落地和模型对齐。
