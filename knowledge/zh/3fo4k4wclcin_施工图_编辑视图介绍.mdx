编辑视图是通过BIM技术将设计方案的各个面转为图纸的视图，并支持修改这些视图内容，如标注添加等功能操作，本文对编辑视图工具的功能及如何使用进行介绍

⭐在阅读文章前，建议您先查看视频教程，便于快速上手~

🎥 [Video URL](https://www.youtube.com/watch?v=oXOnM3jorhs)

# 面向用户

所有用户

# 入口

进入室内设计，选择**图纸 - 施工图纸，**点击**编辑视图**

![](https://qhstaticssl.kujiale.com/image/png/1750733097795/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1750733039584/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 视图列表

1. 列表中的视图将根据方案中涵盖的内容进行生成，默认包含：装修视图

![](https://qhstaticssl.kujiale.com/image/png/1750745444043/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 点击右上角**全部展开/收起**按钮，用户可一键展开或收起视图列表

![](https://qhstaticssl.kujiale.com/image/gif/1750733905659/C0C6DA730DF93F5C3AD3CE059D464E51.gif)

# 视图与方案实时联动

# 添加视图标注

## 平面视图

**1.** **自动标注**

![](https://qhstaticssl.kujiale.com/image/png/1750734408524/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1750734572072/B47F8A045E686A911F7DFC24D6264197.gif)

**2. 尺寸标注**

|  |  |
| --- | --- |
| **线性标注** | **对齐标注** |
|  |  |

**3. 文本标注**

文本标注可以**自定义编辑文本信息**，点击生成**商品信息、房间信息**和**定制信息**

![](https://qhstaticssl.kujiale.com/image/png/1750736439782/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 立面视图

**1. 放置立面标记**

在顶部工具栏选择**生成视图 - 立面图**
