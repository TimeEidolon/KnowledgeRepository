满足用户用不同工具线导出JSON/XML数据的需求，导出范围可选择单独房间、单独楼层及所有楼层导出，一键生成下载JSON/XML数据，帮助用户应对不同设计场景

# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 入口

1. 进入全屋定制设计，选择**厨房/衣帽间**

2. 选择顶层工具栏中的**文件-导出JSON/XML数据**

3. 选择需要导出JSON/XML数据的工具线

![](https://qhstaticssl.kujiale.com/image/png/1744290184750/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 导出JSON/XML数据

1. 设置**报价方式**，导出范围可选择**单独房间、单独楼层及所有楼层**导出，设置完成点击**生成**

![](https://qhstaticssl.kujiale.com/image/png/1744290230369/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 点击**下载**后，下载完成会压缩成单独的压缩包

![](https://qhstaticssl.kujiale.com/image/png/1744290452571/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
