# 在谷歌浏览器中保存HAR记录

1. 准备一个干净的新打开的Google Chrome 无痕浏览环境：

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1751880473512/B9416EF9C8DFCAE0AE1F40690AC1E8CA.png)

2. 按 F12键（Win）或者 Option + Command + I 键（Mac） 开启当前Tab的 DevTools

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1751880473526/41751468A2F5E8B698F8603FF4F53A84.png)

3. 在浏览器地址栏中输入想要记录的页面，例如 www.coohom.com， 等待页面加载完毕， DevTools侧在Network栏目观察不再有变化

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1751880473528/3FB2C9EF1D8118FA75D6572C7430FB5F.png)

4. 找到并点击 "Export HAR (...." 按钮，并选择保存地点

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1751880473540/2A4F6B071F27F43B65D4200C552741A9.png)![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1751880473555/F1B5D5B389EEF4A48EE45E5D714CD39F.png)

5. 找到保存的HAR文件，并传输给Coohom客服或者技术支持同事，后续会交给测试研发进行进一步分析

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1751880473520/97B272B3A71A63AE276EFE7FC6D5F3F6.png)
