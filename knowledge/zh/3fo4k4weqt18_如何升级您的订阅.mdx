# 方式1：

1. **[用](https://www.coohom.com/user/signin)**你的 Coohom 账户登录

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137188110/86E1A01A9854AC2601F47F37512E7C59.png)

* 如果你是老用户可以直接用账号登录，如果你是新用户的话要先**注册**，注册后在登录

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137188018/1512F7B47BF7ECB7C582D5C0CAB358EC.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137188159/84341AF1326016A13AF763A1AA38BFEC.png)

2. 登录后在工作台页面，点击左上方的**立即升级**，然后选择所需套餐升级/续订您的订阅

* 注意：升级为Elite时，你可以选择单次订购或连续订阅，然后在右侧根据需要选择席位（一个席位对应在团队成员管理中可邀请一个成员）

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137188044/1955984AA2E83646412BDC404E6EF2EE.png)

* 您还可以点击右上角的头像选择**我的订阅**，点击**立即升级**，然后选择所需套餐升级/续订您的订阅

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137188213/D701041D39AB785DEAF964C886982733.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137188114/2C1F4D4E4B60F19F0CCF0D8EAE1ADD48.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137187955/AF20F3DC7917E5B826085812A8F0BB15.png)

3. 填写以下结账页面的所有必填项

4. 点击**完成订单**以确认

# 方式2：

1. 您可以在首页顶部点击**定价**，然后选择需要的套餐

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137188091/B9AC83543C4C83ACC5D0C30E2EB95603.png)

* 您也可以在工作台页面点击右上角的头像选择**我的订阅**，点击**套餐及定价**

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137188162/4686885AEA188E365FD013A456F92740.png)

2. 点击所需套餐升级/续订您的订阅

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1773137187955/AF20F3DC7917E5B826085812A8F0BB15.png)

3. 填写以下结账页面的所有必填项并支付

4. 点击**完成订单**以确认

5. 购买成功后，您的订单信息会在新页面显示，并且 PDF 发票会发送到您的登录邮箱。

6. 如需升级到 Coohom 企业版，请填写 **[此表格](https://www.coohom.com/b2b/book-a-consultation?kpm=qkWL.10444e360755181f.3a6a173.1655351096784)**，我们的销售人员会与您联系（通常为 1–2 个工作日内）。  
**请注意**：一旦订阅费用被扣除，概不退还。更多详情请参见 **[无退款政策](https://www.coohom.com/en_US/helpcenter/article/no-refund-policy?kjl_source=hc).**
