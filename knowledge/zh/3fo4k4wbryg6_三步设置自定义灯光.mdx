# 选择基础模板

1. 点击**添加手动灯光**

2. **命名**灯光方案，选择**基础灯光模板**可以在模板基础上进行手动调整

* 建议：使用前建议先了解基础灯光模板的功能效果，[点击查看](https://www.coohom.com/helpcenter/default-light-settings?kjl_source=hc)

![](https://qhstaticssl.kujiale.com/image/png/1746693988010/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 调整阳光

进入手动打光界面，左侧面板可调整阳光的**色温、亮度和位置**

![](https://qhstaticssl.kujiale.com/image/png/1746696989395/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**以下与阳光参数相关按钮的具体作用:**

* **状态** ：打开/关闭阳光效果
* **色温** ：决定整体色温为暖色还是冷色
* **阴影的柔和度** ：调节阳光阴影的柔和程度
* **亮度**：视觉感知的一种属性，用于表示光源反射的效果，建议亮度不超过 50%。如需更改亮度单位，请在**“偏好设置 - 渲染 - 灯光单位”**中更改，如下图所示：

![](https://qhstaticssl.kujiale.com/image/png/1746695447486/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* **阳光投射至每个房间** ：开启后，阳光位置会自动变化，使得渲染每个房间时都有阳光投射效果
* **自定义阳光位置 ：**可自定义调整太阳的方位角和俯仰角

# 添加灯光效果

1. 光源工具栏中，可以自行选择需要添加的**基础光源、IES光源、辉光和体积光**，拖出即可完成添加

* **基础光源**
  + **矩形面光源：**区域光源/面光源
  + **点光源：**聚光灯/球光
* **IES 光源：**IES 代表了照明行业中的一种文件格式，它描述了光源的分布
  + 许多制造商为他们的光源提供 IES 文件，如果您需要指定品牌的灯光，可以上传自己的 IES 光源
* **体积光：**能够增强场景的真实感，使光线看起来更加立体和有深度

![](https://qhstaticssl.kujiale.com/image/png/1746697807854/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 点击基础模板内的灯光，可在右侧面板调整灯光的**色温****、亮度和高度**

![](https://qhstaticssl.kujiale.com/image/png/1746697667922/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 灯光方案设置完成后，点击**完成**

![](https://qhstaticssl.kujiale.com/image/png/1746697862590/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
