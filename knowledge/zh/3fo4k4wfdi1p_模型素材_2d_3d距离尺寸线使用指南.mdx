# 斜墙测量

这里有两种方法，详细内容如下：

**方法一**

你可以点击底部的**显示/隐藏**图标，选择**2D距离尺寸线**，点击**中心点距离**，然后选择模型，此时就会显示模型距墙的垂直距离

![](https://qhstaticssl.kujiale.com/image/gif/1773804350106/8B92EAD58843012015ACE668614C773A.gif)

**方法二**

2D视角下可直接使用快捷键**shift/⇧+空格**，将**边距距离**模式切换至**中心点距离**模式来测量模型距墙的垂直距离

![](https://qhstaticssl.kujiale.com/image/gif/1773804457964/CD8FC63C91BC49078853BAFD719BE561.gif)

# 2D距离尺寸线隔离显示

1. 把鼠标移动到头像，选择**偏好设置**

![](https://qhstaticssl.kujiale.com/image/png/1773804515786/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 在弹出的面板勾选**允许墙面/顶面/地面模型之间触发尺寸线识别**时，墙面模型、顶面模型、地面模型在2D视图下（平面视图/顶面视图）距离尺寸线相互识别，即顶面模型可以识别和墙面模型/顶面模型/地面模型之间的距离尺寸线

![](https://qhstaticssl.kujiale.com/image/png/1773730243941/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1773804910381/BA3CF5EA969D0747CC1A36BC94ED33E3.gif)
