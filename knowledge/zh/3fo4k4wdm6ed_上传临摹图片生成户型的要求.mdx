# 上传临摹图片生成户型的要求

## 1. 格式要求

支持上传JPG/JPEG/PNG/PDF格式的图纸进行识别

## 2. 墙体为双线结构

上传的图纸中，墙体需采用双线结构，即墙体通过两条平行线进行绘制，便于识别墙体区域

![](https://qhstaticssl.kujiale.com/image/png/1760516214035/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 3. 上传真实的户型图

需要上传真实户型图，该类图纸应能准确反映房屋实际布局，例如装修公司提供的房屋户型规划图、房产中介出具的房屋户型图等；非房屋户型相关图像，如现场施工照等，均不属于合规户型图，不支持上传

|  |  |
| --- | --- |
| ✅真实户型图 | ❌非户型图 |
|  |  |

## 4、户型图四周需要保留一定的空白

在上传户型图时，建议确保户型图四周保留适量空白区域，避免户型线条紧贴图片边缘，这样能让系统更清晰、准确地识别户型结构，提升自动生成户型的成功率与精度

|  |  |
| --- | --- |
| ✅四周留白 | ❌四周未留白 |
|  |  |

## 5. 不支持手绘图

不支持上传手绘图，手绘户型图由于其线条、比例易存在不标准问题，会导致无法准确识别

![](https://qhstaticssl.kujiale.com/image/png/1760515608894/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 6. 不支持现场拍摄图

不支持上传现场拍摄图，现场拍摄的房屋实景图，如墙面、地面、天花板实景等，不能清晰呈现房屋整体户型结构，无法基于此类图像完成临摹识别

![](https://qhstaticssl.kujiale.com/image/png/1760515683086/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 7. 不支持二次翻拍图

建议优先选择图纸原图上传，不支持上传二次翻拍图。二次拍摄过程中易出现画面变形、透视偏差等问题导致识别图纸失败

![](https://qhstaticssl.kujiale.com/image/png/1760515851106/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 8. 图像画面元素清晰可见，避免低分辨率的图片

上传图像需保证画面元素清晰可见，避免上传低分辨率的图片。若图像模糊不清，将无法识别图纸中的线条、尺寸等关键信息

![](https://qhstaticssl.kujiale.com/image/png/1760515965441/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
