# 版本推荐

**64位浏览器下载地址如下：**

* 谷歌浏览器下载链接（Mac适用）：[https://www.google.com/chrome/](https://www.google.com/chrome/?kjl_source=hc)
* Edge浏览器下载链接（Mac适用）：[https://www.microsoft.com/edge/](https://www.microsoft.com/edge/)

**温馨提示:**

* 网络和系统软件环境也是影响Coohom设计工具流畅运行的重要因素
* 系统软件环境：建议定期清理本地计算机缓存和系统垃圾，以确保流畅运行
* 网络故障排除：1. 网络不稳定，请检测网速  2. 网页浏览痕迹太多，需要清理缓存
* 如果仅用于做户型，可以降低要求，32位CPU+4GB内存即可，显卡和操作系统无特殊要求

# Coohom配置要求

| **类别** | **标准配置** | **推荐配置** |
| --- | --- | --- |
| 网络 | **个人用户4M**，**企业100M** | **个人用户8M**或以上，**企业100M**或以上 |
| 浏览器 | 满足以下几点：  1. **浏览器：**64位  2. **Chrome：**133及以上（最推荐）  3. **Microsoft Edge：**133及以上 | |
| 操作系统 | Win 10，Win 11，Mac OS 10.5 | |
| CPU | 64位CPU | |
| 内存 | 8GB | 16GB |
| 显卡 | 最低集成显卡型号：Intel(R) UHD Graphics 770，[Intel Iris Xe Graphics G7](https://www.notebookcheck.net/Intel-Tiger-Lake-U-Xe-Graphics-G7-96EUs-GPU-Benchmarks-and-Specs.462145.0.html) |  |
| 独立显卡：  1. NVIDIA GeForce GTX 1650  2. NVIDIA GeForce GTX 1660  3. NVIDIA GeForce GTX 1060 | 独立显卡：  1. NVIDIA GeForce RTX 3060/3060Laptop  2. NVIDIA GeForce RTX 3060 Ti   3. NVIDIA GeForce RTX 4060 |

如果您在网页上操作Coohom，**谷歌浏览器**是强烈推荐的浏览器
