| **室内白天**  **2.0** | 1. 修复已知问题  2. 调整自动灯光生成的窗口面光源强度，减少部分窗口曝光的情况  3. 使用了新的曝光控制，此模式会对画面中曝光的部分适当降低亮度，让原本出现曝光的画面看起来更自然，较大程度的抑制了画面曝光问题 |  |
| **室内白天**  **2.1** | 1. 装饰灯高度限制，2.1版本中生成的装饰灯高度进行限制，不会出现超出层高情况。  2. 定制层板灯规则优化，删除2.0版本中的限制逻辑，解决部分定制层板灯不生成问题。使用新的判断自发光材质标准，判断更准确。 新增补光最少保留一个原则，避免场景中部分空间出现过暗的问题。 |  |

| **室内白天**  **3.1** | 1. 基于上一版本的规则降低了室内补光的强度，从而有效避免了画面出现通亮、缺少过渡的问题，增强了空间体量感  2. 调整了室内补光的规则，减少了光斑的出现 |  |

|  |  |
| --- | --- |
| **室内白天/夜晚  1.0/2.0/2.1** | **室内白天/夜晚 3.1/3.0** |
| 手动开启 | 自动开启 |
|  |  |

**2）漏光修复**

| **未勾选** | **勾选** |
| --- | --- |
|  |  |

| **室内白天/夜晚  1.0** | **室内白天/夜晚  2.0/2.1/3.0** | **室内白天/夜晚 3.0** | **室内白天/夜晚  3.1** |
| --- | --- | --- | --- |
| 无此功能 | 新增了漏光修复，需要手动勾选。 | 移除效果设置【漏光修复】按钮，改为默认开启漏光修复 | 99%的场景默认设置已有效避免了漏光问题。本版提供了可选功能，启用该功能目的是为了提供更精确的灯光缓存计算；因此需要注意的是，勾选后渲染耗时增加且渲染效果在低分辨率下差异不大 |

**3）**从3.0版本起，全局照明反弹次数提升，暗部细节提升****

由于GI升级，一样的灯光亮度可以照亮更大面积范围，如下图所示。自3.0起，自动灯光亮度已做相应降低处理，手动灯光也建议适度降低在一个合理的范围内，避免曝光问题出现

| **GI升级前** | **GI升级后** |

# 版本3.1的主要更新

**1）****引入实时渲染：调整参数即调即渲，仅需1s即可预览效果减少等待。点击了解更多：****[实时渲染](https://www.coohom.com/helpcenter/article/real-time-rendering-is-launched-in-coohom?kjl_source=hc)**

在移动画布、调整材质和灯光参数时，都会对镜头视角进行实时渲染，可以实时观察设计效果

![](https://qhstaticssl.kujiale.com/image/gif/1724826283344/2A968EBC26C6C6EFA8316648A0DD8900.gif)

**2）入口更新**

手动模板移动到“实时渲染”标签下，渲染弹窗入口位置如下:

![](https://qhstaticssl.kujiale.com/image/png/1743061062195/55C4CA0B71CE630769992E95FE49D51A.png)

**3）相比3.0版本，3.1版本的**默认参数升级，最终渲染效果不仅有效减少了噪点问题，且更好地保留了渲染细节。同时，解决了3.0版本部分材质模糊问题****

| **室内白天  3.0** | **室内白天   3.1** |
| --- | --- |
|  |  |

**4）****新增后处理【曝光压制】：实时渲染-效果-后处理参数内开放了曝光压制参数**

该参数用于控制曝光区域的亮度和细节，参数越低可以使高光变柔和，从而保留更多的细节；用户可按需调整

| **参数低** | **参数高** |
| --- | --- |
|  |  |

**5）**其他功能升级：写实模板首次支持LUT滤镜、景深、材质替换等****

**a. LUT滤镜**

手动开启滤镜 → 在公共库可自由切换不同滤镜

* **注意：也可自行上传SRGB模式滤镜，目前支持cube格式**

![](https://qhstaticssl.kujiale.com/image/gif/1724826439455/27971027C2D0C3CCBDCD30B0C883A8A2.gif)

**b. 景深**

手动开启景深，通过编辑中心点以及模糊度可以实现多种景深效果

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1733998473862/53F46318C4C9A48738678ED4572BFD6E.gif)

**c. 材质替换**

点击材质 ，在左侧栏进行材质替换，也可以在右侧栏编辑原材质属性

![](https://qhstaticssl.kujiale.com/image/gif/1724829918960/0505BFA213D14DE078AD90EF8059ADD5.gif)
