# 面向用户

所有用户

# 显示楼层参考

1. 切换为**2D模式**

2. 点击小眼睛按钮，**楼层参考**中选择**上层**或**下层**，选择后会显示对应的楼层户型作为参照

![](https://qhstaticssl.kujiale.com/image/png/1747971489475/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1747971654150/50B51378A88D5E0934DD2BC484112F99.gif)

# 隐藏楼层参考

1. 切换为**2D模式**

2. 点击小眼睛按钮，**楼层参考**中选择**隐藏参考**，选择后会隐藏其他的楼层户型

![](https://qhstaticssl.kujiale.com/image/png/1747971720527/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1747971822294/942D92F284C465B894CD2B8179C66B8F.gif)
