# 入口

**入口一：**进入工作台，点击**上传平面图，**选择需要的图纸导入，可选择单个、多个CAD文件导入，或选择包含多张图纸的CAD文件

* **上传平面图支持导入CAD户型文件**和**上传 pdf/ png/ jpg格式的图片，上传图片的具体操作，可见教程文档：[点击查看](https://www.coohom.com/en_US/helpcenter/how-to-upload-import-cad-floor-plan-by-image-pdf?kjl_source=hc)**
* **CAD文件的格式要求，可见教程文档：[点击查看](https://www.coohom.com/zh_CN/helpcenter/%E5%AF%BC%E5%85%A5cad%E6%88%B7%E5%9E%8B%E6%96%87%E4%BB%B6%E7%9A%84%E8%A6%81%E6%B1%82%E5%92%8C%E5%B8%B8%E8%A7%81%E9%97%AE%E9%A2%98?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1779700974232/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**入口二：**进入室内设计界面后，点击左上角**导入图纸**，选择需要的图纸导入，可选择单个、多个CAD文件导入，或选择包含多张图纸的CAD文件

* **CAD文件的格式要求，可见教程文档：[点击查看](https://www.coohom.com/zh_CN/helpcenter/%E5%AF%BC%E5%85%A5cad%E6%88%B7%E5%9E%8B%E6%96%87%E4%BB%B6%E7%9A%84%E8%A6%81%E6%B1%82%E5%92%8C%E5%B8%B8%E8%A7%81%E9%97%AE%E9%A2%98?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1758772010899/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 导入CAD户型文件

## 导入单张CAD图纸

点击导入图纸后，在面板中选择**导入方式**和**参数**，点击**确定**即可

![](https://qhstaticssl.kujiale.com/image/png/1758767883035/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 导入多张CAD图纸

1. 导入多张CAD图纸，工具会自动分割好图纸，点击**楼层管理**，新建**顶层**或**底层**

![](https://qhstaticssl.kujiale.com/image/png/1758768009156/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1758768056560/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 可以点击**预览查看**，便于设置楼层的名称和层高

![](https://qhstaticssl.kujiale.com/image/gif/1758768229255/CDD551F3C301C73386A72A73F229A3C9.gif)

3. 在**楼层管理**中新建完成后，先对每张图纸**匹配对应的楼层**，再**勾选**需要导入的图纸

![](https://qhstaticssl.kujiale.com/image/png/1758768308293/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. 在面板中选择**导入方式**和**导入单位**，点击**确定**即可

![](https://qhstaticssl.kujiale.com/image/png/1758768366677/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. 导入后户型会默认显示透明，可以根据需要手动修改**户型不透明度**

![](https://qhstaticssl.kujiale.com/image/png/1758768502448/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 常见问题

**Q:** 多层图纸导入以后上下层如何对齐？

**A:** 多层图纸导入并在工具中生成户型后，可以通过**户型对齐工具**来对齐上下层，具体操作如下：

* 打开顶部工具栏，点击**户型对齐**，选择**对齐上层/对齐下层**，在画布中点对点移动对齐

![](https://qhstaticssl.kujiale.com/image/png/1758768599540/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
