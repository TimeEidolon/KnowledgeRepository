# 基本功能

**1.预设动画与关键帧动画打通**

对同一模型，同时设置预设动画与关键帧动画

**注意：同一属性，在同一时间点，不允许同时有关键帧和预设动画**

如果出现提示，可以针对该属性的动画做修改，例如下面视频中的预设动画与关键帧动画在属性【角度Z】上存在冲突，因此同一时间点不能共存。解决方式之一是删除关键帧动画的【角度Z】属性，同时在预设动画覆盖的时间范围内，不能在【角度Z】上打关键帧

**2.预设动画转换为关键帧**：预设动画可以转化为关键帧，转化后对应属性的值直接落在时间轴上，并会与当前存在的关键帧合并

**3.新增曲线编辑器功能：**可以自由设置属性的参数值，并以函数曲线的方式显示，便于用户进行更加便捷地制作复杂动画

**曲线模式下，时间轴面板介绍：**

1. 时间轴面板切换按钮
2. 模型属性
3. 纵轴：属性的参数值
4. 横轴：时间轴
5. 属性面板：每个属性都有独立的曲线，关键帧即为曲线上的点

![](https://qhstaticssl.kujiale.com/image/png/1769567006892/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**曲线编辑视频演示：**通过曲线编辑改变模型移动的速度，并让其在移动过程中旋转和缩放

**4.关键帧类型切换：**用户根据个人需要，可以将动画关键帧类型为贝塞尔/线性/突变，切换结果可以在曲线时间轴面板下显示出来

**贝塞尔**：默认模式，其特点为模型动画的速率是可以通过手柄调节的，例如物体的缓入缓出，同时两侧的手柄可以同时调节，也可以单独调节

**线性：**其特点为模型动画的速率是恒定的，例如物体的匀速运动

**突变：**其特点为模型动画是瞬间完成的，例如物体位置的闪现
