# 数据输出设置

在数据输出设置中包含**引用名、当前值、可输出字段模板、空字段、输出精度、自定义输出字段信息、清单JSON输出**及**其他设置**项，详细内容如下：

| 功能项 | 功能描述 | 关联帮助文档 |
| --- | --- | --- |
| 引用名 | 修改模型的系统变量引用名，以适配后端软件，修改后，patameters中的变量引用名将更改为修改后值。 | / |
| 当前值 | 修改材质、样式、轮廓变量的当前值，输出JSON中这类变量的quotationValue、originalValue、value将变更为所配置的名称/ID/型号/产品编码/自定义编码。 | / |
| 可输出字段模板 | 可以自行配置方案中JSON输出的字段内容，修改配置后，在工具中下载JSON及使用OpenAPI获取JSON时，获得的文件内容将按照配置内容输出。 | / |
| 空字段 | 如不希望输出值为空的变量，可在此处修改，修改后值为空的变量将不在JSON中体现。 | / |
| 输出精度 | 修改坐标、尺寸等在JSON中的精度，注意此处修改不影响工具中的精度显示。 | / |
| 自定义输出字段信息 | 支持在方案层级新增字段并在清单JSON/方案JSON/API中输出。 | / |
| 生成类模型输出配置 | 支持对XML的生成类模型输出引用名及输出值进行二次定义，通过表达式计算实现结果值的修改。 | / |
| 其他设置-接驳余量 | 支持对生成类线条（包含顶角线及台面挡水）设置输出时是否包含接驳余量，详细逻辑可见帮助文档。 | / |
| 其他设置-线条分段输出 | 商家可按照自己的对接/报价需求，决定线条是否分段处理，不分段输出时，报价按线条整段报价，JSON中不输出profileSegments | / |
| 其他设置-组合模型归属 | 商家可设定组合模型下的子模型房间归属是否跟随组合，此设置只影响报价及生产对接JSON，不影响图纸及设计工具中的房间归属 | / |
| 其他设置- XML设置 | 支持修改XML的LineNo取值直接从产品编号中获取 | / |
| 其他设置- 布尔造型生产对接 | 商家可按需开启配置，开启后，布尔造型作用在模型上将生成可对接的轮廓数据，目前仅支持平面板件，对于异常的造型、扫掠放样线条不支持，方案中存在错误数据时可能会导致生成JSON失败 | [【企业】布尔造型生产对接配置说明](https://www.coohom.com/zh_CN/helpcenter/%E4%BC%81%E4%B8%9A-%E5%B8%83%E5%B0%94%E9%80%A0%E5%9E%8B%E7%94%9F%E4%BA%A7%E5%AF%B9%E6%8E%A5%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E?kjl_source=hc) |
| 清单JSON输出 | 支持定义在清单JSON中输出的模型变量，修改后将在清单输出的JSON中输出对应变量，用于清单报表的制作 | / |

以上内容在任意一个工具线下设置，均是对全局生效，即不区分工具线。
