公共库中的模型素材支持收藏，可分类收藏到自定义命名的文件夹中，随时取用

⭐在阅读文章前，建议您先查看视频教程，便于快速上手~

🎥 [Video URL](https://www.youtube.com/watch?v=QACh9CPSMoM)

# 面向用户

所有用户

# 收藏家具模型

## 方法一

1. 选择**模型**，在**公共库**中选择模型素材

2. 选择**公共库**中的模型素材，点击左上角☆一键**收藏**

|  |  |
| --- | --- |
|  |  |

## 方法二

点击公共库中拖出的模型素材，浮窗工具栏中点击**☆收藏**

![](https://qhstaticssl.kujiale.com/image/png/1768357998900/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 收藏夹分类

1. 浮窗工具栏中点击**收藏**，可**新建文件夹**，自定义命名进行收藏夹分类

![](https://qhstaticssl.kujiale.com/image/png/1768358095802/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1768358189140/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 选择**我的 - 收藏**，选择文件夹下的素材即可拖出来使用

![](https://qhstaticssl.kujiale.com/image/png/1768358291073/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1768358585951/7B31791A3ECC4B5A3D4149B81C41D3C4.gif)
