# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 入口

点击模型，进入**风格替换**面板，分为**柜体、掩门及把手**类目，可在相对类目下对**样式、材质等**进行选择替换

![](https://qhstaticssl.kujiale.com/image/png/1753240891018/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 风格替换面板介绍

## 切换素材库

点击右上角切换库标志，在展开栏中可切换不同的素材库

![](https://qhstaticssl.kujiale.com/image/png/1753241168056/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 查看素材库目录

1. 点击素材所在位置的展开标志，可在展开栏中查看素材库节点，可选择切换到不同的素材目录下

|  |  |
| --- | --- |
|  |  |

2. 点击排列标志，可切换素材的不同展示形式

|  |  |
| --- | --- |
|  |  |

## 展开素材库目录

1. 点击右上角展开/收起标志，一键**展开/收起素材库目录**

![](https://qhstaticssl.kujiale.com/image/png/1753241978646/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 展开后，素材面板为可移动分栏视图，纵向栏有不同库的节点，以及对应库下的具体模型分类，二级目录下的模型素材在右侧平铺展示

![](https://qhstaticssl.kujiale.com/image/png/1753242600499/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**支持修改素材栏宽度、移动视图面板**

* 可以手动拖拽中间的分隔线修改纵向栏宽度
* 拖拽素材栏右下角，修改素材栏宽度以及展示素材的列数
* 自由移动视图面板，可将素材栏面板放置在工具界面的任意位置，操作演示如下：

![](https://qhstaticssl.kujiale.com/image/gif/1753242308204/06098C0100C4354CFE399180849C9062.gif)

## 一键搜索模型素材

在搜索框中搜索所需素材相关的关键词，可以一键搜索到您想找的类似模型素材
