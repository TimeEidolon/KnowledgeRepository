# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 入口

进入全屋定制设计，选择**厨房**

**入口一：**选择顶层工具栏中的**生成-水槽**

![](https://qhstaticssl.kujiale.com/image/png/1742277179688/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**入口二：**选中柜体，浮窗工具栏中选择**生成-水槽**

![](https://qhstaticssl.kujiale.com/image/png/1742277254922/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 生成水槽

1. 进入生成水槽界面后，选择需要生成水槽的柜体，在右侧面板的基础参数中选择添加**水槽样式**、**龙头样式**，确定**安装位置**

![](https://qhstaticssl.kujiale.com/image/png/1742277369888/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1742277466375/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 选择合适的水槽样式后，在**间距设置**处修改水槽边距，再点击**生成水槽**

**注意：如果生成按钮变成灰色，需要检查是否选中需要生成水槽的柜体！**

![](https://qhstaticssl.kujiale.com/image/png/1742277612051/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 生成水槽后，点击**完成**，退出生成水槽界面

![](https://qhstaticssl.kujiale.com/image/png/1742277803073/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# **生成双水槽**

1. 点击**添加水槽**，选择水槽2的**水槽样式**、**龙头样式**，确定**安装位置**

![](https://qhstaticssl.kujiale.com/image/png/1742277846162/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1742277898215/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 在**间距设置**处修改水槽边距，再点击**生成水槽**即可

**注意：如果间距输入框变红，需要确定柜体是否能放下两个水槽，距离不能为负数！**

![](https://qhstaticssl.kujiale.com/image/png/1742278115437/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
