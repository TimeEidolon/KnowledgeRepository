# 调整台面挡水参数

1. 选中台面某一边上的挡水，在参数面板中调整挡水参数

**注意：不支持单边挡水改高度，调整单边挡水的高度后，其他边挡水高度都会变化**

![](https://qhstaticssl.kujiale.com/image/png/1740033115426/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 调整缩进参数时，可根据箭头颜色来判断（左/右）缩进方向，再设置缩进数值

**注意：**

1. **台面编辑时，未选择（前/后）挡水或未选择挡水样式，不显示缩进标识**
2. **无法调整缩进的线段（例如：曲线），不显示缩进标识**

![](https://qhstaticssl.kujiale.com/image/png/1740033758689/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1740033974855/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 切换到2D模式，可以选择线添加点，自由移动点和线的位置

![](https://qhstaticssl.kujiale.com/image/png/1740034535521/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1740034911298/A7E1DCE682A90BA6682286423F93FD86.gif)

4. 调整完毕后，点击**开始生成**

![](https://qhstaticssl.kujiale.com/image/png/1740035152752/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
