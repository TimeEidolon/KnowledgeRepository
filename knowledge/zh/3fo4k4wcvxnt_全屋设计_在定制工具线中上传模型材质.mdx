# 上传模型材质

## 方式一

1. 进入**工作台**，选择**模型**，点击**上传材质**

![](https://qhstaticssl.kujiale.com/image/png/1778207902760/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 有两种新建材质的方式，可通过**实时材质**，或按照要求的格式自行**上传材质**图片

![](https://qhstaticssl.kujiale.com/image/png/1778208057933/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 方式二

1. 打开设计工具，进入**行业库**，选择**“厨卫定制/全屋家具定制”**

![](https://qhstaticssl.kujiale.com/image/png/1753173384780/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 进入定制工具线后，有两个入口可进行**模型材质上传**，详见下表：

|  |  |
| --- | --- |
| **入口一** | **入口二** |
| 进入**厨卫定制/全屋家具定制**，选择**“我的-上传”**，点击**材质上传** | 点击定制柜体，选择参数面板中的**“风格替换-柜体材质-我的上传”**，点击**材质上传** |
|  |  |

3. 点击**上传**后，会跳转到**工作台**页面，再按照方式一的操作流程上传材质即可

![](https://qhstaticssl.kujiale.com/image/png/1778208828971/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 查看已上传的模型材质

材质上传成功后，可以在定制工具线中查看

|  |  |
| --- | --- |
| **查看方式一** | **查看方式二** |
| 进入**厨卫定制/全屋家具定制**，选择**“我的-上传”** | 点击定制柜体，选择参数面板中的**“风格替换-柜体材质-我的上传”** |
|  |  |

# 替换已上传的模型材质

**方式一：**进入**厨卫定制/全屋家具定制**，选择**“我的-上传”**，将上传的模型材质拖出便可以一键替换到定制柜体上

![](https://qhstaticssl.kujiale.com/image/gif/1753174222794/1D48CBC8681C3A878D581AAB224B4338.gif)

**方式二：**点击定制柜体，选择右侧参数面板中的**“风格替换-柜体材质-我的上传”**，点击上传的模型材质即可

![](https://qhstaticssl.kujiale.com/image/gif/1753174754478/0361E1F259AF946B2E16A6381284B7A9.gif)
