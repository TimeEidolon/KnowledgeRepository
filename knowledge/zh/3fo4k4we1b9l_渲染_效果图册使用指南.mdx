# 下载或删除效果图

1. 进入图册后，左上角可点击选择**“列表”**或**“大图”**

![](https://qhstaticssl.kujiale.com/image/png/1760667734682/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1765175950182/57D6B239C4AF39F2F9130DD3F1DB30AC.gif)

2. 点击效果图中的工具栏，可对单张图片进行下载或删除

![](https://qhstaticssl.kujiale.com/image/png/1760668277027/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 点击**列表**可多选，对多张效果图进行下载或删除

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1765175950158/2D37C2734C50A6837B78ACB4959B66FF.png)

# 分享效果图

点击效果图中的工具栏，可以以二维码或链接的形式，分享您的效果图给其他人

![](https://qhstaticssl.kujiale.com/image/png/1760668416261/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1765175950167/476F1822775EB51DE5A67AE4F26004C1.gif)

# 收藏效果图

效果出色的图片可以进行收藏，点击效果图中的收藏按钮，可以在左侧收藏夹中进行查看

![](https://qhstaticssl.kujiale.com/image/png/1760668016728/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1765175950146/3AB74445393B3E184177DF62E8947710.gif)

# 编辑效果图

选择**大图**，在下方工具栏，可以保存视图的相机视角、美化效果图和升级图片清晰度

* **注意：**升级图片清晰度后无法恢复

![](https://qhstaticssl.kujiale.com/image/png/1760669319992/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1765175950203/3AD3F3106D2924AC75ADE92878865E6C.gif)

# 恢复已删除的效果图

点击左侧栏中的**回收站**，选择需要恢复的效果图，点击**恢复**

* **注意：**回收站中的效果图只能保留15天，15天后会自动清除

![](https://qhstaticssl.kujiale.com/image/png/1760670606502/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1765175950226/3426BCE174D892DB19A0FF8712B95472.gif)
