# 生成剖面图纸

1. 选择柜体，然后点击右下角的**生成施工图纸**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1770018620338/2D37C2734C50A6837B78ACB4959B66FF.png)

* 注意：选择后所选中的柜体会在右侧面板显示，然后按住Shift可以多选成组

![](https://qhstaticssl.kujiale.com/image/gif/1770013296919/D0F6A7934A629582CB972C4F55C2A615.gif)

2. 进入图纸页面后，点击顶部的**生成视图—剖视图**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1770018620298/2D37C2734C50A6837B78ACB4959B66FF.png)

3. 在顶部选择剖切方式，然后点击**放置剖面**并在图中合适位置放置**剖切起点/终点**，放置完成点击**生成视图**

![](https://qhstaticssl.kujiale.com/image/gif/1770014406547/435B495D2CC0DB3743BE6523FD081D5D.gif)

4. 生成的剖面视图可在左侧视图列表中查看

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1770018620279/2D37C2734C50A6837B78ACB4959B66FF.png)
