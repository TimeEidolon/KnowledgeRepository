### Step 7
添加完商品后，商品将置于下方列表内，此时，对于不同的建库方式有不同的配置方法 * **示例1：**我是标准柜企业，我的柜子都是一个个建的，一种样式的柜子会有很多个，只是尺寸不同（多用于北美），配置方法如下： + 在此处选择对应想要的种类下的所需尺寸的所有柜子 + 由于当前情况下一个柜子只有一个固定尺寸，因此无需打开**缩放** * **示例2：**我是定制柜企业，我什么柜子都能生产，AI可以随机给我生产柜子的宽度，只要是在我能加工的范围就行（适用于大部分企业），配置方法如下： + 首先在企业库中选择对应想要的模型 + 打开**缩放**功能，此时如果将鼠标放置在后方的**信息图标**上，会发现你对当前柜子在参数化编辑器中设置的柜体宽度取值范围被显示在其中 + 只需要打开**缩放**，即表示，当前这个柜子的取值范围是对应参数化编辑器范围，请AI在其中随意取值生成 * **示例3：**我是定制柜企业，虽然我的柜子是定制柜，但我也有大致想要的柜子宽度偏好（适用于大部分企业），配置方法如下： + 在打开**缩放**后，用户可以在后方输入框内输入希望AI所取的宽度值（注意：此处宽度值支持手动输入或下拉选择，具体形式由用户模型配置决定；若模型配置为数值范围，可手动输入该范围内的任意数值；若模型配置为固定选项，则通过下拉菜单选择对应选项即可） + 宽度值必须在范围内，不可设置超限的参数

### Step 9
可以添加多个配置来满足不同模型的需求，可以按照国家、风格、系列等不同维度定义，按需控制（配置完成后，在定制商品模板中可以按需选择） * 针对不同的种类，分别有不同的配置技巧，详细内容如下： | | | | Base cabinet | 每个分类尽量都配置 | | Wall cabinet | 每个分类尽量都配置 | | Tall cabinet | 每个分类尽量都配置 | | Island | 配置地柜时，可以多配几种，AI会随机在里面取 | | Corner Cabinet | 目前仅支持Blind，必须都配置上，不然转角会缺失 | | Refrigerator | 分为单独冰箱、冰箱高柜和冰箱吊柜，一个配置中只能选择其中一个商品 单独冰箱 - 选择一个冰箱的3d模型 冰箱高柜 - 建模时已经将冰箱模型建在参数化柜子里时选择 冰箱吊柜 - 冰箱和吊柜分开选择，冰箱选3d模型，吊柜选对应柜体 | | Sink | 分为水槽柜+水槽模型以及水槽柜和水槽模型，根据用户建模情况选择 | | Hood | 可以自行选择烟机模型或烟机柜，如果选择烟机的3d模型，请特别注意要把对应的烟机3d模型的离地高度设置好，否则会生成到地上 | | Stove | 分为集成灶和灶台柜+灶台，请根据实际情况选择 | | Oven | 暂时只支持烤箱高柜，请选择一个高柜设置 | | Decoration | 设置顶线、脚线、台面的区域 | | Others | / |

# 定制商品配置

1. 进入**AI厨房自动布局**点击**定制商品配置**，在顶部选择需要配置的种类

![](https://qhstaticssl.kujiale.com/image/png/1778811834647/A250A84063F5E82086F9C11434B6E35C.png)

2. 在所需配置的种类下，点击**添加配置**，然后点击**更多操作**，您可以根据需要进行**重命名**

![](https://qhstaticssl.kujiale.com/image/png/1778662126060/61359EF6E42C4E9E73511D5A8C698F98.png)

![](https://qhstaticssl.kujiale.com/image/png/1778664317964/9DD6C177710717224052F2AACAA3BE74.png)

* 如果不需要该配置，可以直接点击**删除**

![](https://qhstaticssl.kujiale.com/image/png/1778664503425/531BBEA16830B5066B4C1482BC10054C.png)

3. 点击**类型**，选择需要的柜体类型，然后点击**添加商品**，在弹出的面板选择需要的素材并点击**确认**

![](https://qhstaticssl.kujiale.com/image/png/1778664469259/99065DEF00ED015975E84810A3880417.png)

![](https://qhstaticssl.kujiale.com/image/png/1778664560936/E4F5CAA959B3E1D3691EBFD1C2085816.png)

![](https://qhstaticssl.kujiale.com/image/png/1778664618014/EB93D31637FAAB702817A6EF75ECAD87.png)

4. 点击右上角的**添加类型**，然后用同样的方法添加商品

* 尽量多的添加类型，这样生成效果更佳
* 选择类型时，每个柜体分类只能选择一次
* 选择分类后，对应该分类请选择企业库中的一个种类模型柜（一个种类指的是一种样式，比如全门基础柜，请不要选择多个样式的全门柜体，AI将会在配置所选的柜子里按照类型进行选择，混乱的样式会大大影响生成效果）

![](https://qhstaticssl.kujiale.com/image/png/1778664829318/AB71479D14A0C171818FA76BF8EF9B98.png)

5. 添加完商品后，商品将置于下方列表内，此时，对于不同的建库方式有不同的配置方法

![](https://qhstaticssl.kujiale.com/image/png/1778664920152/A2ABE300D9653234CFB283224065CDB0.png)

* **示例1：**我是标准柜企业，我的柜子都是一个个建的，一种样式的柜子会有很多个，只是尺寸不同（多用于北美），配置方法如下：
  + 在此处选择对应想要的种类下的所需尺寸的所有柜子
  + 由于当前情况下一个柜子只有一个固定尺寸，因此无需打开**缩放**

![](https://qhstaticssl.kujiale.com/image/png/1778665099467/CC6866C406F125BBECA8ADD6E785C750.png)

* **示例2：**我是定制柜企业，我什么柜子都能生产，AI可以随机给我生产柜子的宽度，只要是在我能加工的范围就行（适用于大部分企业），配置方法如下：
  + 首先在企业库中选择对应想要的模型
  + 打开**缩放**功能，此时如果将鼠标放置在后方的**信息图标**上，会发现你对当前柜子在参数化编辑器中设置的柜体宽度取值范围被显示在其中

  + 宽度值必须在范围内，不可设置超限的参数
