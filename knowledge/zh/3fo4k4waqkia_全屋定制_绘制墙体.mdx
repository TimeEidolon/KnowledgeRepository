# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 入口

1. 进入全屋定制设计，选择**房间**

2. 选择**墙体**，有**直墙、弧墙、矩形墙**三种墙体可选择

![](https://qhstaticssl.kujiale.com/image/png/1740972932874/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 设置定位线

**方式一：**在参数面板**绘制参数-定位线**中设置，有**内部、中心线、外部**三种定位线可选择

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1741156198568/2D37C2734C50A6837B78ACB4959B66FF.png)![](https://qhstaticssl.kujiale.com/image/gif/1740973774508/4619C3BAED050D0F244E9EE32BEF4F6B.gif)

**方式二：按一下空格键**切换墙体的**内部、中心线、外部**定位线，快速调整墙体绘制位置

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1741156198630/B11CBAFC8E5361B3E2204FDA79F74CF3.gif)

# 设置墙体角度

绘制墙体过程中，**按一下Shift键**可快速勾选**正交**，此时墙体只能沿着**X/Y轴**方向绘制

* **Win系统快捷键：Shift**
* **mac系统快捷键：⇧**

![](https://qhstaticssl.kujiale.com/image/gif/1740981673565/B828E74601FE371CD634292F4A453E7A.gif)

# 锁定墙体角度

1. 右下角点击**锁定**，选择**户型**，勾选**角度锁定**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1741156198478/8A17B3792FB240442441786D81F62902.png)

2. 勾选**角度锁定**后，移动墙体端点时，墙体只能沿原方向进行运动

* 快捷方式：**按住Shift / ⇧键**移动墙体端点，也可以锁定墙体角度

![](https://qhstaticssl.kujiale.com/image/gif/1741156798171/542F8B725441419D89B288BFF4C99B97.gif)

# 绘制弧形墙

## 按半径绘制

按照**起点→终点→半径**的顺序绘制，弧形大小由该圆的半径尺寸决定

![](https://qhstaticssl.kujiale.com/image/gif/1741156285353/93E6669C8253382EAAB76395BAE97DB9.gif)

## **按弦高绘制**

按照**起点→终点→弦高**的顺序绘制，弧形大小由该圆的弦高尺寸决定

![](https://qhstaticssl.kujiale.com/image/gif/1741156663351/E289F041EC507A56679AF77426240698.gif)
