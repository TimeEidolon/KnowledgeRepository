# 问题

玻璃渲染出来的效果不正常

# 原因

# 解决方法

1. 可以使用【材质替换】替换一个库内的【实时材质】渲染试一下
2. 若仍然不对，请检查模型是否存在问题（重面、法线、uv等等）
