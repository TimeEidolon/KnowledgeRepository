欢迎开启全屋定制设计之旅！无论您是装修新手还是设计师，掌握户型搭建基础都是打造个性化空间的第一步。本文将带您快速上手：从新建方案、绘制房间到添加门窗结构，助您轻松完成户型框架搭建。您将学会两种快速生成房间的方式，掌握空间命名技巧，并了解如何通过添加柱子、门窗等元素让平面图更贴近真实户型

# 面向用户

**所有用户**。部分企业账号中没有全屋定制设计入口的，需要联系客户成功顾问申请

# 入口

进入**工作台**，选择**厨房和衣橱设计**

![](https://qhstaticssl.kujiale.com/image/png/1779701521850/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**入口一：**在**开始设计**界面中，可以选择**户型模板**，点击**应用**

![](https://qhstaticssl.kujiale.com/image/png/1741242175020/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**入口二：**进入全屋定制设计后，选择**房间**

![](https://qhstaticssl.kujiale.com/image/png/1742460859360/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 绘制房间

* **与绘制房间相关的具体操作，可见教程文档：[绘制房间](https://www.coohom.com/zh_CN/helpcenter/article/%E7%BB%98%E5%88%B6%E6%88%BF%E9%97%B4?hl=zh-CN&kjl_source=hc)**

**方式一：用户型模板生成**

应用户型模板后，工具界面会生成对应的户型

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1742443317545/2D37C2734C50A6837B78ACB4959B66FF.png)

**方式二：用墙体工具绘制**

进入全屋定制设计，选择**房间-墙**，可用**矩形墙工具**快速绘制闭合的房间，**墙体绘制必须是闭合的才能生成房间**

* **如何绘制墙体，可见相关教程文档：****[绘制墙体](https://www.coohom.com/zh_CN/helpcenter/article/draw-walls?kjl_source=hc)****、[调整墙体尺寸](https://www.coohom.com/zh_CN/helpcenter/article/adjust-the-wall-size?kjl_source=hc)、[拆分墙体](https://www.coohom.com/zh_CN/helpcenter/article/split-walls?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1741241621848/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 命名房间

* **与命名房间相关的具体操作，可见教程文档：[绘制房间](https://www.coohom.com/zh_CN/helpcenter/article/%E7%BB%98%E5%88%B6%E6%88%BF%E9%97%B4?hl=zh-CN&kjl_source=hc)**

1. 切换到**2D模式**

2. 选中**房间**

3. 在右侧面板**基础参数**中，设置**房间类型、房间名**

![](https://qhstaticssl.kujiale.com/image/png/1741244733604/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 添加柱子、梁和烟道

选择**结构**，有**柱子、梁、烟道**三种结构可选择

* **如何在户型中添加柱子、梁和烟道，可见相关教程文档：[在户型中添加柱子和烟道](https://www.coohom.com/zh_CN/helpcenter/article/add-columns-and-flues-to-the-floor-plan?kjl_source=hc)、[在户型中添加梁](https://www.coohom.com/zh_CN/helpcenter/article/add-beams-to-the-floor-plan?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1741682903518/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1742455812568/8A6DEC013F9E9D101FADEE0E433B3D03.gif)

# 添加门窗

选择**门/窗**，有**单开门、双开门、移门**三种门，以及**一字型窗、一字型飘窗、落地窗**三种窗可选择

* **如何在户型中添加门窗，可见相关教程文档：[在户型中添加门窗](https://www.coohom.com/zh_CN/helpcenter/article/add-doors-windows-to-the-floor-plan?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1741833714764/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1742456945068/1E09766EF276B3EB88D2C01585BA3EC5.gif)

# **注意事项**

在全屋定制设计中，**不支持使用材质/墙壁/天花板/地板编辑器**，如需使用该功能，可选择**创建空白方案**工具进行创建方案

![](https://qhstaticssl.kujiale.com/image/png/1779703968954/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
