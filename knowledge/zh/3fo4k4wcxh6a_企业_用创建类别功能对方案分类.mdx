# 面向用户

所有企业用户

# 入口

进入**工作台**，选择**我的方案**，点击**新类别**

![](https://qhstaticssl.kujiale.com/image/png/1752733320324/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# 新建方案类别

点击**新类别**后，输入**类别名称**，点击**创建**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749788404/2D37C2734C50A6837B78ACB4959B66FF.png)

# 移动方案到类别文件夹中

## 移动单个方案

1. 选择需要移动的方案，点击右下角的...标志，选择**移动到**

![](https://qhstaticssl.kujiale.com/image/png/1752734434338/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. 可以**选择已有的类别**，也可以**创建新类别和子类别**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749788448/2D37C2734C50A6837B78ACB4959B66FF.png)

3. 点击**创建子类别**后，输入**子类别名称**，点击**创建**

![](https://qhstaticssl.kujiale.com/image/png/1752734516913/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749788414/2D37C2734C50A6837B78ACB4959B66FF.png)

4. 创建后，方案便会移动到新创建的子类别文件夹中

![](https://qhstaticssl.kujiale.com/image/png/1752734694630/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 批量移动多个方案

1. 选中需要移动到同一类别的多个方案后，点击右上方的移动标志

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749788486/2D37C2734C50A6837B78ACB4959B66FF.png)

2. 可以**选择已有的类别**，也可以**创建新类别和子类别**，具体操作可见上文

![](https://qhstaticssl.kujiale.com/image/png/1752735364887/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. 创建后，选中的多个方案便会移动到对应的文件夹中

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749788464/2D37C2734C50A6837B78ACB4959B66FF.png)
