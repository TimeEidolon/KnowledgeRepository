# 绘制弧形吊顶

1. 点击顶部工具栏的**偏移**工具，然后在顶面任意位置点击，输入偏移数值，并按下**Enter**键确认

![](https://qhstaticssl.kujiale.com/image/gif/1775123148764/CD8FC63C91BC49078853BAFD719BE561.gif)

2. 选中区域，点击浮窗中的**倒角**工具，选择需要的倒角模式，输入**半径**，然后点击**Ente**r键确认，接下来点击造型面在右下角设置**凸出**值

![](https://qhstaticssl.kujiale.com/image/gif/1775123345140/BA3CF5EA969D0747CC1A36BC94ED33E3.gif)

3. 点击**偏移**工具偏移出另外两条叠级的造型线，并在右下角设置每一个造型面**凸出**值

![](https://qhstaticssl.kujiale.com/image/gif/1775123827844/DCE71965D19C1438840BBBDE410C6B53.gif)

![](https://qhstaticssl.kujiale.com/image/png/1775123948148/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. 设置完成后你可以拖拽放大右上角的视窗查看效果，观察是否需要调整

![](https://qhstaticssl.kujiale.com/image/png/1775123986665/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. 若无需调整，点击右上角的**保存并退出**，渲染后便可以得到一张满意的效果图

![](https://qhstaticssl.kujiale.com/image/png/1775124086127/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* 最终效果如下：

![](https://qhstaticssl.kujiale.com/image/png/1775206510584/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
