# 问题

渲染后有色差应该怎么解决？

# 解决方法

1. 确认且和相关方对齐颜色验收的标准，即素材上传时候，保证贴图和实物颜色对齐一致。
2. 调整显示器。
3. 明确设计师的灯光模板，未使用旧灯光模板，如有”专属灯光“模板需使用同一版本。
4. 如使用自定义打光，灯光需保持“中性白色光”。
5. 方案渲染完成，使用“美化”功能。
