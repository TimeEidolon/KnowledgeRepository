# 面向用户

**所有用户**

**影响范围：**已创建的手动灯光不受版本更新影响。在灯光版本管理中更新版本后创建的自动灯光和之后创建的手动灯光，会使用新的灯光模板进行渲染

# **版本3.1的主要更新**

**1）在保留3.0 的效果优势的基础上，3.1版本引入了实时渲染，极大优化了打光体验**

![](https://qhstaticssl.kujiale.com/image/gif/1724826283344/2A968EBC26C6C6EFA8316648A0DD8900.gif)

**2）入口更新**

手动模板移动到“实时渲染”标签下，渲染弹窗入口位置如下:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1743075261999/C0FD9DC4589BA416505C72CF6C30E544.png)

**3）****渲染****功能升级**

极速模板首次支持LUT滤镜、景深、材质替换等

**a. LUT滤镜**

手动开启滤镜 → 在公共库可自由切换不同滤镜

* **注意：也可自行上传SRGB模式滤镜，目前支持cube格式**

![](https://qhstaticssl.kujiale.com/image/gif/1724826439455/27971027C2D0C3CCBDCD30B0C883A8A2.gif)

**b. 景深**

手动开启景深，通过编辑中心点以及模糊度可以实现多种景深效果

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1743075262196/598C9506743BE0EB23B605CD80176340.gif)

**c. 材质替换**

点击材质 ，在左侧栏进行材质替换，也可以在右侧栏编辑原材质属性

![](https://qhstaticssl.kujiale.com/image/gif/1724829918960/0505BFA213D14DE078AD90EF8059ADD5.gif)

**4）全局GI升级：自3.0起，反弹次数增加，暗部细节提升**

由于GI升级，一样的灯光亮度可以照亮更大面积范围，如下图所示。自3.0起，自动灯光亮度已做相应降低处理，手动灯光也建议控制在一个合理的范围内，避免曝光问题出现
