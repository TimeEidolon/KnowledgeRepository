空間に奥行きと高級感を演出する「下がり天井」。 しかも、光る演出も簡単に！ この動画では、Coohomを使って照明付きの下がり天井を素早く設計する方法を、実際の操作画面とともにわかりやすく解説します。

Coohomなら、こんなことが可能に！

 ✅ 補助線で正確にラインをとる

 ✅ クリックで“発光する天井”を作成

✅ モデリング不要、時間短縮で提案力UP 設計提案に差をつけたいインテリアデザイナー・住宅関係者の方、必見です！

## Assistant step guide

Mintlify assistant tools may strip markdown images. Always use the plain
MEDIA_* lines below (they contain full https:// URLs).

For each MEDIA_STEP_N_TEXT, output the step, then embed every matching
MEDIA_STEP_N_IMAGE URL using:
`<img src="URL" alt="Step" style="max-width:100%;height:auto;" />`

For video: if MEDIA_VIDEO_THUMBNAIL exists, output a clickable preview
(assistant chat often strips iframes):
`<a href="MEDIA_VIDEO"><img src="MEDIA_VIDEO_THUMBNAIL" alt="Video" /></a>`
For MEDIA_VIDEO_FILE (.mp4), use `<video controls src="URL"></video>`.
