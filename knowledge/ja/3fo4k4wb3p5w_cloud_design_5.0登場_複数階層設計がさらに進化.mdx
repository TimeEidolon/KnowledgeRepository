**こんなお悩みはありませんか？**  
複数階層の設計作業が思うように進まない  
720度の全方位ビューで、複数階層をまとめて確認できない  
複数階層のCAD図面をうまく認識・インポートできない  
各階ごとのフロア属性を自由にカスタマイズできない

これらの課題は、Cloud Design 5.0の複数階層設計機能で一気に解決できます。  
スムーズな設計体験と、よりリアルで詳細なビジュアル表現をぜひご体感ください！

![](https://qhstaticssl.kujiale.com/image/png/1742958607597/76A71E3F404E90E19507F40800772CFA.png)

メゾネットタイプの住宅、共用スペース、戸建住宅（ヴィラ）など、複雑な空間構造もこれからは簡単に設計できます。もう、勘や手探りでの設計作業に悩まされることはありません。

* + かつての不便は、今や使いやすいソリューションに置き換わりました！
  + 長らくお待たせしていたアップデート機能が、ついにリリースされました。

**さあ、今すぐ新機能を活用してみましょう！**


Cloud Design 5.0 における複数階層設計の新機能：  
        1.        複数階層のフロアプレビュー  
        2.        参照階の表示／非表示切り替え機能  
        3.        階層ごとのフロア属性の作成・編集  
        4.        複数階層のCAD図面のインポート  
        5.        手動による階層図面の位置合わせ

> **注意事項：**
>
> Cloud Design 5.0 は、個人ユーザーにはすでに全機能が開放されています。法人アカウントでご利用の方は、ご契約アカウントの管理者様にお問い合わせください。

1. 複数階層のフロアプレビュー

各階ごとの表示／非表示が可能。現在編集中のフロア以外は「閲覧のみ（編集不可）」の状態でプレビューされます。フロアの透過度設定も、プレビューパネルに移動されました。

![](https://qhstaticssl.kujiale.com/image/png/1742967886612/A6CDC998604AF634258A631FB3185934.png)

2.参照階の表示／非表示切り替え機能

2D画面で下層階の間取り図を参照表示できます。表示の切り替えは「表示／非表示」オプションで設定可能です。

![](https://qhstaticssl.kujiale.com/image/png/1742968787661/20486B89EC24958847122D41990F8401.png)

3.フロア属性の作成・編集

フロア名称のカスタム設定、階層の順序変更、最上階／最下階の作成が可能になりました。

![](https://qhstaticssl.kujiale.com/image/png/1742969034381/A4E8BB19B19F4F8FB178FB53EF2CCCD3.png)

4.複数階層のCAD図面のインポート

複数のCAD図面をそれぞれの階層にインポートすることで、階層構造を自動で認識・反映します。

![](https://qhstaticssl.kujiale.com/image/gif/1742969338648/99BF555F7C7A6F52FCC3947A71834220.gif)

5.手動による階層図面の位置合わせ

複数階層をインポート、または手動作図後、各階の間取り図を手動で位置調整できます。

![](https://qhstaticssl.kujiale.com/image/png/1742969676641/D8007C1A2DFF807E3FCCFD6B0E3FE1EE.png)

![](https://qhstaticssl.kujiale.com/image/gif/1742970001290/99BF555F7C7A6F52FCC3947A71834220.gif)
