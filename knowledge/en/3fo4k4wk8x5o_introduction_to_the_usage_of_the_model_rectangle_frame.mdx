Now when you create a new model in the editor, the system will automatically add a rectangular frame, which is the model frame.  

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
The appearance and position of the model frame in the editor ​ After creating a new model, the blue cube wireframe in the visual interface of the editor is the model frame. The position of the model frame is the rectangular frame in the model frame in the structure navigation on the right.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850200934/CA9C9A5B80843AB43788E00AC73B1847.png)

### Step 2
The meaning of the frame of the model ​ Meaning: a virtual frame that sets the size of the model ​

### Step 3
The role of the model frame ​ Function: Used to realize adsorption or adaptation by identifying the size of the outer frame in the front-end tool ​

### Step 4
Function difference ​ Difference: In the cabinet mode, the outer frame of the model is used for adsorption between the cabinets. It is adsorbed on the surface of the outer frame of the model (the tool only recognizes the size of the model frame but not the actual size of the model). The frame is mainly used for adaptation, the size of the model frame is changed during adaptation ​

### Step 5
Attributes of the model frame ​ In the structure navigation, left-click the rectangular frame to display the properties of the model frame on the far right ​ The attributes of the model frame include physical attributes and other attributes. The physical attributes are the position attributes of the model frame in the coordinate axis (position XYZ), and the other attributes are the size attributes of the model frame in the coordinate axis (size XYZ) and Calling method, where the position and size attributes of the frame of the model are not allowed to be modified and kept as default. ​ There are two options for the calling method: left-back-bottom and center point. The default is left-back-bottom. The calling method is unified with the modeling method, that is, the modeling method is left-back-bottom, and the modeling method is called the center-point modeling.
