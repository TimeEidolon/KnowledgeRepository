## Steps with screenshots

### Step 1
Self-check Currently, max offers many plugins and small tools for checking overlapping faces, which can be chosen as needed. If there are no small tools, you can still check within max. Use vraylightmtl and connect vraydirt to the light color node. Assign this material to the parts that need testing and render. Gray areas indicate overlapping faces; they should normally be the color of vraylight.

# Self-check

Currently, max offers many plugins and small tools for checking overlapping faces, which can be chosen as needed.

If there are no small tools, you can still check within max.

Use vraylightmtl and connect vraydirt to the light color node. Assign this material to the parts that need testing and render. Gray areas indicate overlapping faces; they should normally be the color of vraylight.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762504660502/F6BBBB4748F3DB2EF422E4EC4DD83573.png)

# Solution

Simply select and delete the overlapping parts.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762504660484/FE8B981F4212383A82118DB8CBB37187.png)
