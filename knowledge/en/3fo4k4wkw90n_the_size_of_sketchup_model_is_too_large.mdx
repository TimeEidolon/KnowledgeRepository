**Upload error: the model size is too large, the length，width，or height cannot exceed 30,000mm.**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733899095071/44E4A71E908995CE00D232879F534BE4.png)

**Reason:**

* The dimension of length/width/height is exceed 30,000mm.
* There are other objects in the model file except the main model.
* There are some empty data in the model which cannot be deleted.

The dimension of length/width/height is exceed 30,000mm.

There are other objects in the model file except the main model.

There are some empty data in the model which cannot be deleted.

**How to solve the problem:**

1. The dimension of length/width/height is exceed 30,000mm.  
​

Use the tape Measure tool to check if the model size is larger than 30,000mm. Click the rule tool in the SketchUp, you can see the measurement.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733899098071/ADE1E29895A2964A84ED16005CAAAAB9.png)

If the dimension of the model is more than 30,000 mm, we need to choose the model and use scale tool. Click any edge of the model as a reference, then you can scale the model or enter the size you want to zoom out the model and make the longest part less than 30,000 mm.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733899100814/D21808EA61A8EBEEB7499526C7E0792A.png)

2. There are other objects in the model file except the main model.  
​

If the model size is smaller than 30,000 mm, there might be some hidden models in the file.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733899102059/0CB3A0A07E024A0EC59C70A47CB1C8CD.png)

Open the model file, Click 【Unhide】- 【all】, then delete other models and datas and left the model you need.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733899102972/386B319D36CFF02A1D89DB641E8DC7ED.png)

3. There are some empty data in the model which cannot be deleted.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733899105569/30A4F65B026471A235ECEC626CD86B89.png)

There might be other datas in the file, usually it happens when the model is converted from other softwares.In this case you need to copy the model to a new SketchUp scene, select the model and ctrl + C.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733899108253/8522B71014A0A02F488ACB2C32F0C331.png)

**Open a new scene, Ctrl+V to paste the model.**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733899110305/D701F0259E1604A5F91C0551F81E8553.png)

If you need to use the model which is larger than 30,000mm, you can zoom out in the SKP file and use scale tool in Coohom to enlarge the model.
