#### Overview

There may be some models that are not displayed correctly in the drawings. You can try the following methods to resolve this.

#### Target Users

All users

#### Solutions

I.If the legend and the model do not match, for example, if the model is a dining table but is displayed as a double bed in the drawing.

The reason could be a misclassification of the model, such as a dining table mistakenly categorized as a double bed. To resolve this:

1. If it's your own uploaded model, re-upload it and choose the correct category.
2. If it's from the public library, consider replacing the model.
3. Select the model and switch to using "Intelligent Legend" through the "Replace Legend" option.

![](https://qhstatic-us-s3.coohom.com/image/png/1747811109262/5A77B39D8182DCEF85D9D77DDA7FA2D1.png)

II.If the shape of the model changes in the drawing, for example, if a square-shaped lamp designed is displayed as a circular shape.

Resolve it by replacing the legend and switching to using "Intelligent Legend" or selecting a legend with a consistent shape.

![](https://qhstatic-us-s3.coohom.com/image/png/1747811135681/60E49DE7CCB25B7136420BB0495450F2.png)

III.If the orientation of the model changes in the drawing and doesn't match the design environment.

The reason might be incorrect main orientation when uploading the model.To resolve:

1. Adjust the model to the correct orientation and re-upload it if it's your own uploaded model.
2. Contact the model manager of the enterprise library if it's from an enterprise library.
3. Use the "Replace Legend" option to switch to using "Intelligent Legend".

![](https://qhstatic-us-s3.coohom.com/image/png/1747811158423/39C8F77281D02EB67C74A5602C295BCB.png)

1. If the size of the model changes in the drawing, for example, if a 1.8-meter bed is displayed as 1.5 meters.

Resolve it by replacing the legend and switching to using "Intelligent Legend" or selecting a legend with a consistent size.

![](https://qhstatic-us-s3.coohom.com/image/png/1747811187980/4DE43989D375FD3954D8EEA814B2E596.png)

1. If two beds appear stacked on top of each other in the same position.v

This might be because both the bed frame and mattress are placed in the same position, generating legends for both. Solution: Hide one of the legends.

![](https://qhstatic-us-s3.coohom.com/image/png/1747811212304/1A6DEC681EB121608B10630CB54B700F.png)
