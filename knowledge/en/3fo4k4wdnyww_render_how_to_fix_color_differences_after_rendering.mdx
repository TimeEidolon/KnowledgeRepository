# Problem

How to solve color distortion after rendering?

# Solution

1. Confirm and align with relevant parties on the color acceptance standards, ensuring consistency between texture upload and actual color.
2. Adjust the monitor.
3. Ensure the designer's lighting template is used and not an old version. If there is a "dedicated lighting" template, use the same version.
4. If using custom lighting, ensure the light is "neutral white light."
5. After rendering the plan, use the "beautify" feature.
