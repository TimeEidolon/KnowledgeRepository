Each step below includes the screenshot used in this article.

Enter the **workspace**, select **"Enterprise Catalog - Custom Products - Product Catalog"**, and click **Create Asset** in the upper right corner.

### Step 3
Select **"Create One Material"**. Size not exceed 5M, resolution not exceed 5000pixels Do not need to adjust except special cases. The smaller the number is, the weaker the bump effect is. When the number is negative, the bump reverse.

# Entrance

1. Enter the **workspace**, select **"Enterprise Catalog - Custom Products - Product Catalog"**, and click **Create Asset** in the upper right corner.

![](https://qhstaticssl.kujiale.com/image/png/1756886166278/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Select the **Material** under the corresponding tool.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1756887632724/C709A4E4F6A47B36A86D49088F73D677.png)

3. Select **"Create One Material"**.

![](https://qhstaticssl.kujiale.com/image/png/1756886601087/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Advanced Settings

## Adjust Bump scale

# Note

**1. Diffuse**

![](https://qhstaticssl.kujiale.com/image/png/1749536400216/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**2. Color Correction**

![](https://qhstaticssl.kujiale.com/image/png/1749536538003/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**3. Bump Scale**

The larger the number is, the stronger bump effect is.

The smaller the number is, the weaker the bump effect is.

When the number is negative, the bump reverse.

![](https://qhstaticssl.kujiale.com/image/png/1749536607308/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
