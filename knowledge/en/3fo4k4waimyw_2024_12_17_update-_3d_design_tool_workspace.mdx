#### 【3D Design Tool】Supports displaying the project name as the title of the browser tab.

**Target User:**

All enterprise users

**Description:**

When users edit a project in the 3d design tool, the project name will be displayed on the browser tab, so users can know exactly which project they are editing.

**![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1742387199484/E882B49A5A8B8C25DD6866FC08E716B2.png)**

#### 【Workspace】Supports customized tool loading content

**Target User:**

 Only admin account can access this feature and change the contents displayed on the loading screen of the 3d design tool.

**Description:**

This features allows the admin account to customize the image and text displayed on the loading screen of the 3d design tool.  Once the admin account changes the contents, all its sub-accounts will see the customized loading image and text.
