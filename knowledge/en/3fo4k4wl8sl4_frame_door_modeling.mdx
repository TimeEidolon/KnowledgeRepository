**II.Setting Parameters**

2.1 Basic Parameters:![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931488371/E07E63F2E30E1CFFC68A61CB5A95E895.png)

2.2.1 Width (W) - Represents the size of the model panel in the X-axis direction. Set the value type as a range (e.g., range 10-1000, current value: 600).![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931489313/14E88E0520C6C9FEEDCB76833ABC9F9F.png)

2.2.2 Depth (D) - Represents the size of the model panel in the Y-axis direction. Set the value type as a options (e.g., optional 18, current value: 18).![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931490386/E1D3C259D2C524AFBB1091CE4B06196C.png)

2.2.3 Height (H) - Represents the size of the model panel in the Z-axis direction. Set the value type as a range (e.g., range10-2400, current value: 1200).![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931491510/BBB6E35029E91762587545F807D95FDB.png)

2.2.4 Material - No modification needed, indicating that the Door uses the default material.

2.3 Custom Parameters

Custom Parameters are used to add some more parameters except that commonly basic parameters or formulas in the editor.

2.3.1 Frame width(FW)—Represents the outer edge frame of door,set it a range from 50 to 100 and value 73.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931493270/54E11270C6C530A51B48572C1772F49F.png)

**III.Importing Submodels**

3.1 Import a front panel from Internal parts library as a sidepanel of frame,and set its dimensions.Width: #FW; Depth:18;Height:#H.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931495213/FB63B3061F0101EF69CF108B2C27A1AD.png)

3.2 Right click to copy another same panel and move its physical attributes: set position X to #W-#FW.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931496245/96C088334EB6792A05592EBFE2AF88E2.png)

3.3 Import a front horizontal panel and set its dimensions.Width: #W-#FW\*2; Depth:18;Height:#FW.

move its physical attributes: set position X to #FW.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931498334/AD5CF86B286CFBB9BFD9DCD67410E10A.png)

3.4 Copy another front horizontal panel and move its physical attributes: set position Z to #H-#FW.The door frame is completed.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931499525/BE89A84F350103FD160E172AD38F0BEB.png)

3.5 Import a flat panel from elements as the door core,and edit its contour![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931500737/44C1EC470C0DBF3C07EBA77445DAEE7A.png)

Edit its four points: point 1(0,0),point 2(0,#W-#FW\*2),point 3(#H-#FW\*2,#W-#FW\*2),point 4(#H-#FW\*2,0)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931501718/380D2F548B8687E24F0DCABE8AD5CA42.png)

Set dimensions and physical attributes:

Height:5; Position X:#FW; Position Z:#FW; RotateY: -90; RotateZ: 90.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931503192/A67FFEB79B3D8FEFDDE5EC7DDA4FE018.png)

**Ⅳ. Test the Model**

Input values within the range of parameters W, D, and H, observe if the model changes correctly. If it does, it means the model is working fine.

**Ⅴ. Save to Library**

Click on the file in the upper right corner and choose save, select the subdirectory for saving, name it, and click save.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931506885/5DA82EAD3525D5AA113A4E4C6F6A4B60.png)

In the product management page, find the corresponding category for the model, select the model, click on "Save to Database", and wait for the model to finish rendering. ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931507814/D7AE92AC3BFC834C0AF64B793C707034.png)
