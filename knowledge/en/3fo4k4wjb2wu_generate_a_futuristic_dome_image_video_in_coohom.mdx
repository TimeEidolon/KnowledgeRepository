### **What you can dream, you can make it in Coohom.**

Different kind of models can be uploaded in Coohom and you can get fast, high quality Images, like this Dome:

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830073730/922AD80196E47ED826D81939A910901C.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830076930/30A7423B3897BD9CB2813707C4DFC7D9.png)

Please follow the steps below:

1. Download a dome's model from the 3D warehouse.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830079810/F8D89A2B8A6BF5A3ECCA7EA1E33CC10D.png)

2. Use Sketchup to check and adjust the model, removing unnecessary elements if necessary.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830084034/215FE89536BF1923B7D1B3B6674E9B87.png)

3. Upload the 3D model in Coohom under Personal Library section.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830086727/CF6D0A5B3E052B6BD1311C0902FB1A42.png)

4. Draw a **Terrace** Floorplan, set the walls to **Dwarf**, adjust the height to the minimum, set the height above the Dome's model.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830089541/83ECE2DA71EEC2DE61EAFE2CC14F9DF5.png)

5. Drag the 3D model into the floorplan.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830094671/00ABCF75DA614DF235257D4F2980A1D5.png)

6. Use the ‘**Walk**’ function to walk in the room, shortcut **QEWSAD.**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830097673/3ABA197B74147C28282A4173F9CFE239.png)

7. Go to Generate Images/Videos page, look for **Customize**, choose **Outdoor Reality Daytime.**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830100820/E44910D44BB2DD0A57DE0C4D630C239B.png)

8. Adjust lights under **Custom Lights**, make sure every room has lighting.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830104516/5C7F6FE57439A863989A6599ACD4DE47.png)

9. Save the Lights, then click on **GENERATE** button.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830107764/FE43FE7E600488AEE3BD3C86D9FD9562.png)

10. Next, check out the image effect.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830110440/CA6735EFE70D2C2E4102A83E5CD7E17E.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830113431/C2BC35E4558E234C73DDE8CFE8B0D61F.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733830118726/9BBFD1A1FED7A46A2BB7CC9C88ACED35.png)

11. If you have **Pro account** and you may share this project by duplicate the link.  
​[https://www.coohom.com/pub/saas/apps/project/share/MQEDAW5MCMIFSAABAY888888](https://www.coohom.com/pub/saas/apps/project/share/MQEDAW5MCMIFSAABAY888888)
