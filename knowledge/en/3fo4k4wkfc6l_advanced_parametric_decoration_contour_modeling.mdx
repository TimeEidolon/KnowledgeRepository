1. Definition

### Definition

Create decorative lines that can have their materials and styles replaced independently, commonly used for gilded door panels.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887447326/0764C72319605AB09820A767BE3658EE.png)

1. Create Location

### Create Location

Enterprise Backend—Kitchen&Bath/ Custom Furniture —Create Asset—Parametric Model Library—New—**Internal Parts**—**Molding**—OK

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887449851/8AA7E9EBA9D7731717AD5B0C64952721.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887450506/33B0482FD13D885AE3148856E191F573.png)

1. Parameter Setting

Parameter Setting

Custom Variables (Using Gilded Decorative Line for Door Panels as an Example):

1. Width (W) - Represents the dimension of the model in the X-axis. The value type can be set as optional or within a range (e.g., range of 200-1200, current value is 450).

2. Depth (D) - Represents the dimension of the model in the Y-axis. The value type is set as optional (e.g., optional value of 5, current value is 5. Actual factory process dimensions should be used).

3. Height (H) - Represents the dimension of the model in the Z-axis. The value type can be set as optional or within a range (e.g., range of 200-2400, current value is 700).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887452104/06444687418442E468DCFF87322F430C.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887453212/8ED95817D152373CEEB40E6C3C4601B1.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887454053/3468815BA6551049067B88782B8B6E12.png)

4. Material selection: Choose one default material that represents the color of the model.

1. Building Model

### Building Model

1. Insert the pattern from the component library.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887454625/8B1DB2C4F185D43E336723993B8C155A.png)

2. Select the contour line for the decorative line shape.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887455245/B89F3567A5A2CD30BC4916B4DDD1D115.png)

3. Edit the pattern path by using point-to-point sketching and save the changes.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887457027/9F2FEE0C33101132A7536B25A0C00E1E.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887458034/3DFC957EDF50D0AE378742764B753B95.png)

4. Set the physical properties:

Position X: 0; Position Y: 0; Position Z: #H.

Rotation X: 90; Rotation Y: 0; Rotation Z: 0.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887459199/A36BF9B42B66041E34A7F0332C666AF7.png)

5. Set the output properties of the panel according to the manufacturing requirements (e.g., regular panel output dimensions, grain direction dimension \* perpendicular to grain direction dimension \* thickness). If you want to display the dimensions of the decorative line in the quotation list, select "Yes"; otherwise, select "No".

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887460116/543608F73A9A64E53F5D987BB7A14CA2.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887460714/D76E6D3424A67037F87DB58FF25ADD66.png)

6. Click on the material and select the uploaded gold material texture.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887461546/F0C1E86557559255A0A2B7C5A0A59483.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887462673/2924D4D72889753ADE8052C82286F12C.png)

1. Building Model

### Building Model

**4. Check And Save**

1. Check the extreme dimensions of the gilded decorative line to see if the model changes correctly.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887464150/2708171C397ED2FFC25133D53DC9762B.png)

2. After confirming that everything is correct, click on the file in the top-right corner and select "Save". Choose the desired subdirectory for saving, provide a name for the file, and click on "Save".

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887465251/5F1C94686D25480940143B317118AFF5.png)

3. Go to the backend page and locate the model. Select the model and choose "Store in Inventory" or "Complete".

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733887467317/9107087C115EC06056C243A7F4987763.png)
