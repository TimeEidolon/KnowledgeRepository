In the floor plan tool, use the drop panel function to quickly achieve a ground sinking effect. This function allows not only unified height sinking for the entire selected ground area but also precise sinking design for custom-shaped local areas.

⭐Before reading the article, it's recommended to watch the video tutorial first to get started quickly~

🎥 [Video URL](https://www.youtube.com/watch?v=fsYZSmMYbzg)

# For Users

All users

# Entrance

In the 2D view, select the floor, and then you can find **Drop Panel** in the right-side room properties panel.

![](https://qhstaticssl.kujiale.com/image/png/1773294599709/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Set Drop Panel

## Overall **Drop Panel**

1. In the **2D** view, select the ground, and then directly input the sinking value needed in the **Drop Panel** section of the right-side parameter panel**,** press **Enter** to confirm after input.

![](https://qhstaticssl.kujiale.com/image/png/1773294738721/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Switch to the **3D** view to check. At this point, the entire ground area will display the sinking effect.

![](https://qhstaticssl.kujiale.com/image/png/1773628900462/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Local Area Drop Panel

1. Select the ground, place the mouse on top of **Split Area**, and then click **Floor** under the ground section.

![](https://qhstaticssl.kujiale.com/image/png/1773294771446/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the **Split the Floor** page, use the tools in the top toolbar to draw the divided area. Once the drawing is completed, click **Finish** at the upper right corner.

* **Note: The drawn area cannot exceed the selected ground/top surface range.**

![](https://qhstaticssl.kujiale.com/image/png/1773294872703/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. After completing the ground division, click the ground area that needs sinking, and directly input the requiredsinking value in the **Drop Panel** section of the right-side parameter panel. Press **Enter** to confirm.

![](https://qhstaticssl.kujiale.com/image/png/1773294289509/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. Switch to the **3D** view to check. Now, only the local area will display the sinking effect.

![](https://qhstaticssl.kujiale.com/image/png/1773628942755/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
