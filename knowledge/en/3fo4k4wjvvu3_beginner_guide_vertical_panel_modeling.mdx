1.2.2 Vertical Panel Modeling

Key Points:

1. Panel catalog – Panel

2. In the editing of the panel contour, stroke points should be traced clockwise one by one, and the modeling method is left back to bottom

3. Variable output is affected by different texture directions

4. To reduce the size of the movable shelf, the position of the plate should be changed, and the output settings should be set to the actual size
