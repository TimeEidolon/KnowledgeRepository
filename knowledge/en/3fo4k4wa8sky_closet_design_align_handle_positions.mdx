When custom cabinet design, align handles can be used to keep the handle height positions on different door panels consistent.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=c\_9GZeribGw)

# For Users

All users

# Entrance

1. Select **Kitchen＆Bath** / **Closet /** **Doors＆Windows** to enter the customization interface.

2. Select the **Toolkit-Align handles** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739268186318/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# **Align Handles**

1. Select alignment.

2. First select the panel where the target handle is located, then select the panel to which the target handle is to be aligned.

* There are three alignments: **align top, center alignment (default), align bottom**.

3. **Right-click** to reselect the handle alignment target, press **ESC** to exit.

![](https://qhstaticssl.kujiale.com/image/png/1739269750106/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Align the top edge of the handle.
