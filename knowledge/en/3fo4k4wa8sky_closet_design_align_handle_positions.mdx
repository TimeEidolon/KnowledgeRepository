When custom cabinet design, align handles can be used to keep the handle height positions on different door panels consistent.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=c\_9GZeribGw)

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Select **Kitchen＆Bath** / **Closet /** **Doors＆Windows** to enter the customization interface.

### Step 2
Select the **Toolkit-Align handles** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739268186318/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Step 1
Select alignment.

### Step 2
First select the panel where the target handle is located, then select the panel to which the target handle is to be aligned. * There are three alignments: **align top, center alignment (default), align bottom**.

### Step 3
**Right-click** to reselect the handle alignment target, press **ESC** to exit. Align the top edge of the handle. Center the handle. Align the bottom edge of the handle.

# Entrance

1. Select **Kitchen＆Bath** / **Closet /** **Doors＆Windows** to enter the customization interface.

2. Select the **Toolkit-Align handles** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739268186318/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# **Align Handles**

1. Select alignment.

2. First select the panel where the target handle is located, then select the panel to which the target handle is to be aligned.

* There are three alignments: **align top, center alignment (default), align bottom**.

3. **Right-click** to reselect the handle alignment target, press **ESC** to exit.

![](https://qhstaticssl.kujiale.com/image/png/1739269750106/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## **Align Top**

Align the top edge of the handle.

![](https://qhstaticssl.kujiale.com/image/gif/1739270234826/4E8706B269BF02EE4C2F2EACD54BFBB2.gif)

## **Center Alignment**

Center the handle.

![](https://qhstaticssl.kujiale.com/image/gif/1739270332000/F288E263BCE20DB3CAB6EEEFCD1B6ACC.gif)

## **Align Bottom**

Align the bottom edge of the handle.

![](https://qhstaticssl.kujiale.com/image/gif/1739270442239/B262C8A8C2EDB2DCA3DB408082CA50FD.gif)
