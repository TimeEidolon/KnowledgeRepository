Through the cabinet bevel cut function, designers can ensure that customized cabinets are beveled in the tool to meet the staircase cabinet design scenario. This article will provide a detailed explanation of how to use the cabinet bevel cut function.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=-bgOzRIz7L4)

# For Users

All users

# Entrance

Select **"Advanced Tool - Kitchen＆Bath / Closet"**to enter the customized tool page.

* **Individual Users:** The cabinet bevel cut entrance is open by default.
* **Enterprise Users: It is necessary to configure whether the cabinet bevel cut entrance is open. For detailed configuration operations, refer to the tutorial: [Click to view](https://www.coohom.com/en_US/helpcenter/enterprise-cabinet-fill-angle-rules-configuration-description?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1756178376508/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance One:** Select **"Cabinet Corner Cutting"** in the top toolbar.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1756262782514/2D37C2734C50A6837B78ACB4959B66FF.png)

**Entrance Two:** Click on the cabinet, and select **"More - Cabinet Corner Cutting"** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1756178884902/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Bevel Cut Function Instructions

## Bevel Cut Direction and Position

The bevel cut can be set in different cutting directions, mainly in the form of **upper left corner, lower left corner, upper right corner, lower right corner.**

![](https://qhstaticssl.kujiale.com/image/png/1756191950032/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Set Chamfer Size

You can customize the **cutting angle width** and **high cut angle** in the chamfer size bar.

![](https://qhstaticssl.kujiale.com/image/png/1756192055116/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Supplementary Angle Settings

You can change the patch board thickness and different patch board processes.

**1. Whether to Generate a Patch Board.**

a. If unchecked, only the cabinet will be cut, and no patch board will be applied.

![](https://qhstaticssl.kujiale.com/image/png/1756192348741/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

b. If checked, you can further configure the patch board process.

* **The thickness of the patch board** can be set freely.

![](https://qhstaticssl.kujiale.com/image/png/1756192454387/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* In advanced settings, you can set the **supplementary angle height, supplementary angle method**, and **extension value**. The supplement methods include **embedded** and **cover**.

![](https://qhstaticssl.kujiale.com/image/png/1756192494002/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
