### 2.Parameter configuration

1). Width

W represents the size of the coordinate axis in the X direction, the value type can be set to optional (for example: optional value 100, 200, 300, ..., 2400, current value is 1000) or interval range (for example, interval 10-1000 , The current value is 600).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850598996/EA5ED7BA76537A5D76893BD1BAB95D16.png)

2).Depth

D represents the size of the coordinate axis in the Y direction, the value type can be set optional (for example: optional value 100, 200, 300, ..., 1200, the current value is 500) or interval range (for example, interval 10-1200, , The current value is 500).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850601259/78E3DD9F91C8952C65C6AB57DAFA5AE7.png)

3).Height

H represents the size of the coordinate axis in the Z direction, the value type can be set to the interval range (for example: interval 10-100, the current value is 18 or optional (example is the optional value 18, the current value is 18 ).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850602132/9DDB14716EA04457330134BEAFBB9FBC.png)

4).Material

Set a default material.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850603851/7CBB1BDCE30DB94C1DB9EC1CD96B9828.png)
