# Problem

The same material renders inconsistently in a scene. In the image below, the two panels use the same material, but the rendering results are noticeably different.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1596539867996/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png?x-oss-process=image/resize,m_mfit,w_624/format,webp)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762133780786/98F02D1F31F14EFE13F86C69D636646B.png)

# Reason

The base material itself has a decay parameter. Using this parameter causes the front and side views to show two different mapping effects. It's commonly used for materials with soft reflections such as velvet or flocked surfaces. This results in the inconsistency we see in the camera's direct and side rendering results.

# Solution

After understanding the reason, we know this is a property of the material, not a rendering issue. If you don't want this effect, simply change the material.
