In the images tools, you can adjust the image effect by setting advanced parameters, setting color correction, affect specular, use new materials on finish light strips, light's ambient blocking, specular reflection and render complex materials. Each parameter setting corresponds to a different rendering scene. This article provides a detailed introduction to how to set advanced images parameters.

# For Users

# Set Advanced Parameters

## Color Correction

The effect of large color patches on the overall picture can be corrected.

* The color of the floor in the picture has a great impact on the ceiling, and the color can be corrected after opening it.

![](https://qhstaticssl.kujiale.com/image/png/1743147663634/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Affect Specular

Turn on or off the reflective spots of the light source (surface light source /omni light /spotlight) in the screen on the ground or on the material with higher reflections.

* **On:** Keep the spots displayed by the light source on the reflective material.
* **Off:** The spots displayed by the light source on the reflective material are not displayed

![](https://qhstaticssl.kujiale.com/image/png/1743148181749/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Use New Materials on Finish Light Strips

The new material of the light strip solves the rendering problems of old materials such as breakage, exposure, and inaccurate brightness. New materials can accurately render brightness ranges from 0% to 6000%.

![](https://qhstaticssl.kujiale.com/image/png/1743149617526/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Light's Ambient Blocking

Light's ambient blocking is enabled by default, which will make the model in the rendering effect more three-dimensional and clear.

**Tips: Increase the size and reduce the radius, which can make the corner details richer and enhance the three-dimensional effect.**

![](https://qhstaticssl.kujiale.com/image/png/1743149678187/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Specular Reflection

If objects in the mirror are lost, you can turn on this function to ensure that reflected objects are rendered correctly in the mirror, but it will increase the rendering time.

* In order to improve rendering efficiency, this function is turned off by default. Scenes without mirrors do not need to be turned on. Scenes with mirrors can be turned on.

![](https://qhstaticssl.kujiale.com/image/png/1743151237649/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Render Complex Materials

The complex materials currently supported include replacement materials and 3S materials. The above-mentioned specific materials are required in the solution to render the corresponding effect. After checking the switch, the above-mentioned complex materials can be rendered. The rendering of complex materials will increase the rendering time.

![](https://qhstaticssl.kujiale.com/image/png/1743151381757/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
