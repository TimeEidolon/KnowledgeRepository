Use the sink generation tool to quickly install sinks and taps on the cabinet.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=SHXO4Jy9-bA)

# For Users

All users

# Entrance

Select **Kitchen＆Bath** to enter the customization interface.

**Entrance One:** Select the **Generate-Sink** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739512178153/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance Two:** Select the cabinet, and select **Generate-Sink** molding in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739512289564/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Generate Sink

1. Select the cabinet that needs to be generated. In the basic parameters on the right panel, select add **Sink Style and Tap Style**. Determine the **Installation Type**.

![](https://qhstaticssl.kujiale.com/image/png/1739512899233/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1739513010843/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After selecting the appropriate sink style, modify the sink margin at the **Spacing Mode**, and then click **Generate**.

**Note: If the generate button turns gray, you need to check whether the cabinet that needs to be generated is selected!**

![](https://qhstaticssl.kujiale.com/image/png/1739513242717/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. After generating the sink, click **Finish** to exit the generate sink interface.

![](https://qhstaticssl.kujiale.com/image/png/1739513347363/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Generate Double Sinks

1. Click **Add Sink**, select the **Sink Style and Tap Style** of Sink 2 to determine the **Installation Type**.

![](https://qhstaticssl.kujiale.com/image/png/1739513435504/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1739513523949/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Modify the sink margin at the **Spacing Mode**, and then click **Generate**.

**Note: If the spacing input box turns red, you need to determine whether the cabinet can lower two sinks, and the distance cannot be negative!**

![](https://qhstaticssl.kujiale.com/image/png/1739513689514/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
