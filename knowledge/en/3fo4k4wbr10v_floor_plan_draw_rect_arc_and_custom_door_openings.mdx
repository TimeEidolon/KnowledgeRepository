# Method Two

1. Enter the 3D design interface and select **"Floor Plan - Custom Opening"**.

![](https://qhstaticssl.kujiale.com/image/png/1747211693734/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Select a wall and click **OK**.

![](https://qhstaticssl.kujiale.com/image/png/1747212677096/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Enter the shaped door opening drawing interface, you can use the tools in the toolbar above to draw the door opening shape.

![](https://qhstaticssl.kujiale.com/image/png/1747212727063/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. Use the **Rectangle** and **Fillet** tools to draw an arc opening, and use the rectangle tool to draw a rect opening first.

![](https://qhstaticssl.kujiale.com/image/png/1747212789903/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. Select the **Fillet** tool, select **two adjacent line segments** that need to be filleted, and then set the **fillet** **radius**.

![](https://qhstaticssl.kujiale.com/image/png/1747214057334/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1747214147198/FABC37AEFE2D207DF8087FC33AE51220.gif)

6. You can also use the **Cirle** and **Polygon** tools to draw custom openings, as follows:

![](https://qhstaticssl.kujiale.com/image/png/1747214259825/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1747206667348/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
