# Entrance

1. Enter the **workspace** and select the **Kitchen ＆ Closet Design**.

![](https://qhstaticssl.kujiale.com/image/png/1779701668973/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the kitchen＆closet design and choose the **Drawing**.

![](https://qhstaticssl.kujiale.com/image/png/1745227656866/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Edit Views

Edit views is a view of the design plan to be converted into drawings through BIM technology, and supports modification of these view contents, such as adding labels and other functions.

* **Specific operations related to editing views can be seen in the tutorial documentation:[[Kitchen & Closet] Edit Views Introduction](https://www.kujiale.com/en_US/helpcenter/article/kitchen-closet-edit-views-introduction)**

# Drawings Layout

Drawings layout is tool that organize the edited views in the drawing frame to output a complete set of construction drawings.

* **Specific operations related to layout drawings can be seen in the tutorial documentation: [[Kitchen & Closet] Drawings Layout Introduction](https://www.kujiale.com/en_US/helpcenter/article/kitchen-closet-drawings-layout-introduction)**
