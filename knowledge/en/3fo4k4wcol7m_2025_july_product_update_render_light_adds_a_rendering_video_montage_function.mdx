# 1. Functions for All Users

## Key Update Details

**[Kitchen & Bath] New quick material panel, you can quickly access models, [Click here for details and learn how to use](https://www.coohom.com/en_US/helpcenter/use-quick-material-panel-to-quickly-access-models?hl=en&kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1753869347062/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**[Render & Light] New rendering video montage function, [Click here for details and learn how to use](https://www.coohom.com/en_US/helpcenter/rendering-video-montage-function-guide?hl=en&kjl_source=hc)**

The rendering video montage function enhances promotional impact for finalized rendering videos. For the rendering video, you can:

* Connect multiple renderings for display
* Add company logo and watermark for promotion
* Set a video cover image
* Add subtitles and graphic elements
