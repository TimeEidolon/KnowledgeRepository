**Before**

**【Kitchen and Closet Design Tool】Supports Right panel optimization**

**Target Users**

All KC Customers

**Before**

Without the location function, it isn't very clear to replace the directory opened during generation

**After**

When the user replaces a model, the root directory of the replaced model is automatically opened so that the user can find the required model.

When it is generated, the corresponding generation categories in the Appliances&Accessories brand library will be opened for it to find the required model quickly;

Modified the display mode of the directory, users can click on each level, and view all models under the level, convenient for users to compare the whole;

**How to use**

[![
](//qhstaticssl.kujiale.com/newt/1144/image/png/1549879747859/B0556275535F7BF4FEF2DCC4F752A054.png)](https://qhstaticva-cos.kujiale.com/media/yun/help/video/UID_60fc1f1d_f37e_4232_1749549954819.mp4)

Click to select a custom model that has been added to the canvas, find the style on the right, click the small picture in front of style to expand the replacement panel, and the panel will be automatically positioned;

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733932361176/C79B7F3AD8FDC467DF5F498356DF7490.png)

The mouse moves to the navigation bar to display the new navigation information, and the user clicks on any level in the navigation to view all models under the level;

**【Kitchen and Closet Design Tool】Supports Real-time search**

**Target Users**

All KC Customers

**Before**

The search structure is inaccurate and the search loads slowly

**After**

Search content is more accurate;

After the user enters the search content, the relevant search terms will be displayed in real-time below, which is faster and more intuitive;

**How to use**

[![
](//qhstaticssl.kujiale.com/newt/1144/image/png/1549879747859/B0556275535F7BF4FEF2DCC4F752A054.png)](https://qhstaticva-cos.kujiale.com/media/yun/help/video/UID_c39d0eb0_84b0_4ebb_1749550035314.mp4)

Move the mouse to the search box on the left panel, and enter any character, the relevant content will be displayed below in real-time, and the input field will remain highlighted;

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733932367158/E186716792A4AA796C06DA602160B78F.png)

The user can choose to click on the recommendation word for precise search or directly use enter to search the input field;

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733932368097/BA317E5A7E6E09DF93F502BBA42071E7.png)
