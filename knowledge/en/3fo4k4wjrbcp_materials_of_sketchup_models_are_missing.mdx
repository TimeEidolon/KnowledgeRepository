If the model looks perfect in the SketchUp but the textures are missing after uploading and only shows the color of the material, we need to modify the model.  
​

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844950724/59BAC0E55581A5F88A969885FFDAD028.png)

Reason: Failed to read model data  
Solution：reattach the material in SketchUp

1. Double click to enter the model group, select the faces need to be edited(make sure to select in the bottom group);

Double click to enter the model group, select the faces need to be edited(make sure to select in the bottom group);

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844953557/40066F3546046803B5B743B75EFA1BE1.png)

2. Select the material and attach to the model, save and upload again.  
​

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844956664/44ED45C42773ED92158EA79E68279F57.png)

Then we can get a perfect model in Coohom.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844959530/14ACAC9668869CE7C90F358B465984CC.png)
