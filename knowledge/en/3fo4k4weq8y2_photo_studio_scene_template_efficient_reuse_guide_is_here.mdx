# Function Background

**1. Have you ever encountered these situations:**

* Spent half an hour adjusting the axis and cannot reuse it in other scenes?
* Lighting/model animation took a whole day, then operations requested for a scene change?
* Multiple videos for various stores, changing the scene means re-setting keyframes?

**2. Scene Template as a solution:**

* **Scene templates can save lighting, camera, animation, and model parameters as templates, supporting reuse across scenes and sub-account binding, reducing repetitive work and significantly improving design efficiency.**

# For Users

Designed for paid enterprise customers in Photo Studios, main accounts, and sub-accounts bound to e-commerce studio administrator roles

# Feature Details

## 1. How to Use

* Feature Access: In the hierarchy panel, select the target object and right-click to choose "Save as Scene Template", enter a template name and select a storage folder. Confirm to complete the save.

![](https://qhstatic-us-s3.coohom.com/image/png/1773627590022/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* **Template Activation:** In the left "Enterprise Library" panel - "Scene Template" - select any template and place it in a suitable position within the scene

![](https://qhstatic-us-s3.coohom.com/image/png/1773989909755/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* The multi-selected objects must include at least one model/empty object (single model/empty object is also supported for saving in the enterprise scene template library)
* The number of multi-selected objects must be controlled within 50

## 2. Operating Permissions for Different Roles

|  |  |  |
| --- | --- | --- |
| **Role** | **Operating Permissions** | **Restrictions** |
| Main accounts / E-commerce studio administrators | Create, rename, delete, migrate templates;  Manage enterprise scene template folders; | None |
| Enterprise sub-accounts | Activate templates, replace models, adjust unlocked animation parameters | Cannot modify, delete templates or manage folders |
