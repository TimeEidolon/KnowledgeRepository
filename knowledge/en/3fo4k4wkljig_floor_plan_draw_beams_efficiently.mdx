# Efficient Beam Drawing

## Draw Beams

After clicking **Beam**, select the starting point and endpoint in the floor plan to draw the beam.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773314898819/435B495D2CC0DB3743BE6523FD081D5D.gif)

## Enable Ortho

While drawing the beam, to quickly create straight and neat beam, you can hold the shortcut key **Shift/⇧ to Enable Ortho,** which automatically locks the drawing direction to horizontal or vertical to improve drawing efficiency.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773314898819/435B495D2CC0DB3743BE6523FD081D5D.gif)

* You can also manually **Enable Ortho** in the parameter panel on the right.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773315044683/CD8FC63C91BC49078853BAFD719BE561.gif)

## Switch Location Line

1. **Location Lines** include three modes: **Internal, Center, and External**.

![](https://qhstatic-us-s3.coohom.com/image/png/1773381088954/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. To quickly adjust the alignment of the beam with the wall while drawing, you can use the shortcut key **Ctrl/⌘** to switch thelocation line.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773314987813/8B92EAD58843012015ACE668614C773A.gif)

* You can also manually switch the **Location Line** in the parameter panel on the right.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773315107144/BA3CF5EA969D0747CC1A36BC94ED33E3.gif)

# Set Beam Dimensions

After completing the drawing, select the **Beam** and set the **Width** and **Height** in the parameter panel on the right.

![](https://qhstatic-us-s3.coohom.com/image/png/1773315140063/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Adjust Beam Position

The beam positions support two modes: **Ceiling** and **Floor**. There are two methods to adjust beam position:

![](https://qhstatic-us-s3.coohom.com/image/png/1773315202332/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Method 1**

Select the beam tool, first choose the desired mode in the **Position** section of the right parameter panel, then perform the drawing. At this time, the beam will automatically be generated in the specified position. You can switch to a **3D** view to check.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773315468187/BDD03DBD7B143CFD44CBE3156FF860F8.gif)

**Method 2**

In the 3D view, press and hold **Ctrl/⌘** key to select a beam, then modify it in the **Position** section in the right parameter panel. You will see the beam's position change.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773316268673/E621DA89292319CD0682934CAE911ACF.gif)

# Modify the Beam Distance from Ground/Top

## Distance from the Top Surface

Select the beam, then click **Distance from the Top Surface** in the right parameter panel, input the value and press the **Enter** key to confirm.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773315952567/5BE693C2A037151FD75670ED2B8C2A1C.gif)

## Distance from the Ground

Select the beam, then switch to **Distance from the Ground** in the right parameter panel, input the value and press the **Enter** key to confirm.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773316268673/E621DA89292319CD0682934CAE911ACF.gif)

* If it is a sunken space, you can input a negative value for downward offset.

![](https://qhstatic-us-s3.coohom.com/image/gif/1773316382175/224070BBD0FEA231C555DBA16061712F.gif)
