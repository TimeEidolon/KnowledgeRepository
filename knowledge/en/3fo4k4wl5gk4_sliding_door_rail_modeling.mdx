**Dual-track rail modeling**

Top rail and bottom rail need to model separately .

**ⅠUpload rail DXF file**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931757901/8210B34342B3A2321FD5FD248E9F65B8.png)

Upload bottom rail section CAD file and select the model category,click submit.

\*Model Category : Modeling outline

**Ⅱ.****Creat****e****rail model**

**P****art 1. Top rail modeling**

**Custom Furniture** —Create Asset—Parametric Model Library—New—Internal Parts—Dual-track top rail

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931758706/0FB9A897F23804D22D1F6BC8C4402E0F.png)

2.1 Basic Parameters:

2.1.1 Width (W) - Represents the size of the model panel in the X-axis direction. Set the value type as a range (e.g., range 200-2800, current value: 1200).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931761188/3B10AB5BDDA44CD5B437899C70A7EBA0.png)

2.1.2 Depth (D) - Represents the size of the model panel in the Y-axis direction. Set the value type as a options (e.g., optional 82, current value: 82).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931763538/5AE0796620EC0D54D2F6D456FCDC2DFD.png)

2.1.3 Height (H) - Represents the size of the model panel in the Z-axis direction. Set the value type as a options (e.g., optional 41, current value: 41).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931764812/50D85392E6279A9981B0E8E8E0195544.png)

2.1.4 Material - No modification needed, indicating that the Door uses the default material.

2.1.5 Select top rail contour from library

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931766934/88DF378B9FB7F2C3C399EC1C27216CF4.png)

2.2 Modeling

2.2.1 Select develop from elements,then edit basic properties part, reference #LK and link top rail contour.

Edit path: Point 1(0,0) ; point 2( 0,#W).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931768582/D0EFF698BABC30E7C1A0B59ACA2162EA.png)

2.2.2 Edit physical attributes：change position and rotate to make model match with contour boundary.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931770032/DABABDAF6D5A883083778A9B5D52D63F.png)

2.2.3 Save and store top rail model.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931773533/DC766790BDA69BEF76D188323F7DF901.png)
