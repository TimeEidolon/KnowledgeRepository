When configuring and calculating quotes, users can choose imperial units and complete calculations.


## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
On the **General Quotatian** page, select **Price List**, then click **Edit** after quoting.

### Step 3
During quote rule setup, input formats for imperial units can be entered for quantity comparison. * Invalid input formats include: 1.1", 1/32", etc. Note: When in use, consider the conversion relationship and use "approximately equal". Approximately equal will judge if the error between two data points is within 0.01mm range. If yes, the equation holds.

![](https://qhstaticssl.kujiale.com/image/png/1765510672610/8540669BA69831E29E68450A87C92B1B.png)

### Step 4
After setup, click **Release** in the top-right corner. In quantity or supplementary rules, expressions can only be used for input. If you want to compare a variable with an imperial result, enter the following to determine: * Method 1: #W > 42.5in ? 100 : 90—Ensure no space between the number and "in". * Method 2: #W > 42 1/2in ? 100:90. * Method 3: #W> 3ft6.5in ? 100:90—Ensure no space. * Method 4: #W>3ft6 1/2in ? 100:90.

# Usage in Quote Rule Expressions

In quantity or supplementary rules, expressions can only be used for input. If you want to compare a variable with an imperial result, enter the following to determine:

* Method 1: #W > 42.5in ? 100 : 90—Ensure no space between the number and "in".
* Method 2: #W > 42 1/2in ? 100:90.
* Method 3: #W> 3ft6.5in ? 100:90—Ensure no space.
* Method 4: #W>3ft6 1/2in ? 100:90.
