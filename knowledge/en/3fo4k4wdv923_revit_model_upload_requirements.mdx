## Steps with screenshots

Each step includes plain MEDIA_* URLs (for AI tools) plus rendered images.

# For Users

All users

# Supported Formats

1. Model versions from 2021 and earlier

2. Models in ***rvt*** and ***rfa*** formats are supported

3. Models need to be uploaded as compressed packages ***(rar or zip)***

# Model Requirements

If the materials of the Revit model have textures and the textures are not in the Revit official material library path.

Please place the textures and models in one folder and upload them as a package.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762311261665/767EAF9E6E8B7BEA07FAFC81CA1425AE.png)
