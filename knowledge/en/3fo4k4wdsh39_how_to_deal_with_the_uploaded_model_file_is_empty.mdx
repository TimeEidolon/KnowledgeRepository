# Problem

The uploaded model file is empty, and it is not allowed to upload an empty scene file.

# Reason

1. A higher version of 3D Max was used to create the model; currently, only 2021 is supported.

2. The model has only one material, and the following parameter was selected, which resulted in the exported vrscene turning into a light, without any other model data, thus considered an empty model.

# Solution

1. Deselect

2. Change the model's material to a regular vraymtl

3. Add another non-emitting component
