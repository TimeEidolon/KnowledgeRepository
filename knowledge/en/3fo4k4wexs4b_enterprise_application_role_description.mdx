To clarify system operation permissions and standardize enterprise management processes, this document specifies the role functions and responsibilities across modules including quotation, product, order, drawing, modeling, and inspiration spaces, enabling enterprises to properly assign account permissions.

# For Users

Enterprise Users

# Character Introduction

The following is a detailed description of the functionality corresponding to each role in the modules Quote, Product, Order, Drawing, Model and Inspiration Space:

| Type | Role | Function Explanation |
| --- | --- | --- |
| **Quotation Management** | Kitchen and Bath Product and Quotation Admin | Kitchen & Bathroom Products Management, Parametric Modeling, Quotation Rules Setting |
|  | Custom Furniture Product and Quotation Admin | Custom Furniture Products Management, Parametric Modeling, Quotation Rules Setting |
|  | Door and Window Product and Quotation Admin | Door & Window Products Management, Parametric Modeling, Quotation Rules Setting |
| **Custom Product Management** | Kitchen & Bath Product Admin | Kitchen & Bathroom Products Management, Parametric Modeling |
|  | Custom Furniture Product Admin | Custom Furniture Products Management, Parametric Modeling |
|  | Door and Window Product Admin | Door & Window Products Management, Parametric Modeling |
| **Industry 4.0 Interface** | JSON/XML File Download Operator | Ability to download JSON/XML files through design tools |
| **Order Management** | Inspection Allocator | Ability to manually set allocation rules to assign Inspection orders to designated reviewers |
|  | Production Allocator | Ability to manually set allocation rules to assign production orders to designated reviewers |
|  | Inspector | Obtain review orders through active claiming/passive assignment; conduct review, modification, approval of the Inspection order, and create production orders |
|  | Production inspector | Obtain production orders through active claiming/passive assignment; conduct review, modification,approval/order placement of the production order, and fill in the order number |
|  | Order Administrator | Ability to view all order information (customer orders, Inspection orders, production orders) and transfer these orders (except for those in review status) |
|  | Order Viewer | Ability to view all customer orders and Inspection orders but cannot operate on other review orders except for those submitted by oneself |
|  | Order Submitter | Allow creating customer orders/submitting Inspection orders |
| **Construction Drawing** | Construction Drawing Designer | Using/exporting construction drawings feature in design tools |
| **Inspiration Spaces** | Inspiration Spaces Designer | Inspiration Space Plan Creation |
|  | Inspiration Spaces Manager | Inspiration Space Management |
| **Model Management** | Modeling System Operator | Parametric Modeling |
|  | Parametric Paving Editor | This role allows accounts with administrator privileges to create parametric tiling products in the enterprise directory |
