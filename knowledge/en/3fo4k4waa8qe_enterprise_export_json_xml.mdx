To meet the needs of users to export JSON/XML with different tools. The export range can be selected as a single room, a single floor or all floors. One-click generation and download of JSON/XML helps users cope with different design scenarios.

# For Users

All enterprise users

# Entrance

1. Select **Kitchen＆Bath** / **Closet /** **Doors＆Windows** to enter the customization interface.

2. Select the **File-Export JSON/XML** from the top toolbar.

3. Select the tool that needs to export JSON/XML

![](https://qhstaticssl.kujiale.com/image/png/1738821705652/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Export JSON/XML

1. Set the **Quote Logic**, the export range can be selected as **a single room, a single floor or all floors**,andclick **Generate** after setting.

![](https://qhstaticssl.kujiale.com/image/png/1738821931662/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After clicking **Download**, the download will be compressed into a single package.

![](https://qhstaticssl.kujiale.com/image/png/1738824547800/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
