Using the drawing view tool, you can design layouts, customize actions, and simultaneously observe the styles and annotation effects automatically generated on the drawings in real time. It supports switching between different view nodes for comprehensive inspection and effect evaluation, making it convenient to adjust designs for drawing at any time. This article explains how to use the drawing view tool.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=skwFTwA5T1U)

# For Users

All users

# Entrance

After entering the design tool, click on the third tab in the view window at the top right corner: **Drawing**.

![](https://qhstaticssl.kujiale.com/image/png/1766110862552/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Adjust Drawing View Size

If you find the current view window too small, you can adjust the layout between the small window and the main canvas by clicking **View Arrangement**.

![](https://qhstaticssl.kujiale.com/image/png/1766110961798/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* When selecting **Vertical Sliding Window**, as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1766111184631/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* When selecting **Left and Right Window**, as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1766111266144/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* When selecting **Window Size Default**, you can restore the default layout, as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1766111461354/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1766111543784/D0F6A7934A629582CB972C4F55C2A615.gif)

# Switch Views

Click the view node entry to switch between different view nodes, making it easier to check and compare different view effects.

![](https://qhstaticssl.kujiale.com/image/png/1766111670587/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Automatic Annotation

Click the **Auto - Annotation** button at the top right corner to instantly trigger and generate the default annotation effect in the **Drawing**.

![](https://qhstaticssl.kujiale.com/image/png/1766111923147/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Quick Access to Drawing Editing

1. Click **Editing Drawing Views** to switch tools immediately and enter the drawing tool.

![](https://qhstaticssl.kujiale.com/image/png/1766112426528/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After entering the drawing tool, you can click **Back to Design** to go back to the design tool.

![](https://qhstaticssl.kujiale.com/image/png/1766112553748/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
