In Coohom, you can customize watermark settings, and the watermark in the renderings, panoramas and 720º VR tours. This article will explain in detail the four modes of setting watermarks to meet your different needs for watermark settings.

Each step includes plain MEDIA_* URLs (for AI tools) plus rendered images.

# For Users

All enterprise users

# Operation Permissions

* For Admin and Manager roles, you can switch the watermark mode for the current enterprise and upload or delete the enterprise's public watermarks.
* For any account role, you can modify your current watermark with the administrator's permission.

# Entrance

Enter **Workspace**, select **"Settings - Customization - General Settings - Watermark"**, and you can set the **watermark**.

![](https://qhstaticssl.kujiale.com/image/png/1752119268038/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Introduction to the Four Watermark Modes

**Mode 1: The entire enterprise uses the same watermark.**

If you want the entire enterprise to use the same watermark, you only need to turn on **“Global Watermark”** and upload the corresponding watermark image. If you upload multiple logos, you need to select one of them to use it.

![](https://qhstaticssl.kujiale.com/image/png/1752119477446/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Mode 2: Personalized watermark for sub-accounts. (All accounts can upload their own watermark)**

If you want all sub-accounts to be able to upload their own watermark, you only need to turn on **"Personalization for Each Account"** and select **"For Each Account"**, as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1752119696663/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Mode 3: Personalized watermark for sub-accounts. (Only specific accounts can upload their own watermark)**

If you want all sub-accounts to only use the enterprise-specified watermark and only specific accounts can use their own watermark, you only need to turn on **"Personalization for Each Account"** and select **"For Specified Accounts"**, as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1752120036217/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

For individual accounts that wish to upload their own watermarks, you can go to the **Account Management** page to grant them the role of **"Upload Own Watermark"**:

![](https://qhstaticssl.kujiale.com/image/png/1752120209376/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1752120279830/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Mode 4: Entire enterprise prohibits any watermark.**

If you don't want any watermarks to appear in the enterprise, you can turn off **“Global Watermark”** and **"Personalization for Each Account"**, as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1752120559980/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
