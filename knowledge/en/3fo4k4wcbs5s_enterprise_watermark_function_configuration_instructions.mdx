In Coohom, you can customize watermark settings, and the watermark in the renderings, panoramas and 720º VR tours. This article will explain in detail the four modes of setting watermarks to meet your different needs for watermark settings.


## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Entrance Enter **Workspace**, select **"Settings - Customization - General Settings - Watermark"**, and you can set the **watermark**.

### Step 2
Introduction to the Four Watermark Modes **Mode 1: The entire enterprise uses the same watermark.** If you want the entire enterprise to use the same watermark, you only need to turn on **“Global Watermark”** and upload the corresponding watermark image. If you upload multiple logos, you need to select one of them to use it.

### Step 3

# For Users

All enterprise users

# Operation Permissions

* For Admin and Manager roles, you can switch the watermark mode for the current enterprise and upload or delete the enterprise's public watermarks.
* For any account role, you can modify your current watermark with the administrator's permission.
