Create a louver door core that can be used as a replacement, with the option to set whether it is quoted and whether it appears in the list.

* **Note: The modeling process for louver door cores in both Kitchen & Bath and Custom Furniture is identical, and they can be shared by copying through category settings.**

**1. Creation Location**

Create Asset - Custom Furniture - Parametric Model Library

![](https://qhstaticssl.kujiale.com/image/png/1749090492274/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**2. Parameter Configuration**

Custom Parameter Settings

* **Note: As the door core is a sub-component, its size range should closely resemble the maximum and minimum values in real situations, otherwise it will affect future applications.**

**Width (W)** - represents the size of the door core model in the X direction, usually a range value (e.g., range 200-1200, current value 450).

**Depth (D)** - represents the size of the door core model in the Y direction, i.e., the thickness of the door core, with a single or multiple optional value types (e.g., optional 9, current value 9).

**Height (H)** - represents the size of the door core model in the Z direction, usually a range value (e.g., range 200-2400, current value 700).

**Material (CZ)** - represents the current material of the model, keep the default.

![](https://qhstaticssl.kujiale.com/image/png/1749090587465/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1749090617108/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1749090641804/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**3. Modeling**

3-1. Select "Develop" in the "Element library".

![](https://qhstaticssl.kujiale.com/image/png/1749090676316/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3-2. Select the Develop model, modify the right-side component attribute contour, choose the louver section contour line (previously uploaded), and click "Confirm".

![](https://qhstaticssl.kujiale.com/image/png/1749090733402/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3-3. After selecting the contour, click "Edit Contour" and enter the point information. Click "Save".

![](https://qhstaticssl.kujiale.com/image/png/1749090817866/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1749090961548/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3-4. Select the Develop model and adjust the physical properties on the right side:

* Position X: #W; Position Y: 0; Position Z: 0;

* Rotation X: 0; Rotation Y: 0; Rotation Z: -90.
