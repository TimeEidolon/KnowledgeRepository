Sometimes users will import a CAD floor plan and use it as reference image which is called basemap. To remove the floor plan image, you can click the basemap button in 2D view to either delete it or hide it.

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Sometimes users will import a CAD floor plan and use it as reference image which is called basemap. To remove the floor plan image, you can click the basemap button in 2D view to either delete it or hide it.
