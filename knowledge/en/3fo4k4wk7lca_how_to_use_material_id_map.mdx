As you know, Coohom is a powerful design tool, and Coohom supports downloading **Material ID Map.**

1. First, download the Material ID Map and Original Image.

   ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745243877405/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
2. Open the Original Image in Photoshop, then make a copy.

   ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745243919457/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
3. Put the Material ID Map in Photoshop and resize it.

   ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745243941988/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
4. Select window area

   ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745243966024/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
5. Hide the other two images and delete the selected area

   ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745243972796/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
6. Drag in the ocean image and adjust it below the Background copy image

   ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745244017268/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
7. This completes the perfect exterior replacement, which is just one of the techniques of using Material ID Map. You can also use Material ID Map to adjust the color saturation of local materials. Looking forward to your exploration.

   ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745244069648/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
