In this article, we will explain in detail how to set up the material parameters of a customized model.

# Entrance

1. Enter the **workspace**, select **"Enterprise Catalog - Custom Products - Product Catalog"**, and click **Create Asset** in the upper right corner.

![](https://qhstaticssl.kujiale.com/image/png/1756886166278/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Select the **Material** under the corresponding tool.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1756887738525/C709A4E4F6A47B36A86D49088F73D677.png)

3. Select **"Create One Material"**.

![](https://qhstaticssl.kujiale.com/image/png/1756886601087/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Advanced Settings

## Upload Diffuse Map

Size not exceed 5M, resolution not exceed 5000pixels.

![](https://qhstaticssl.kujiale.com/image/png/1749608317501/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Color Correction

Do not need to adjust except special cases. (see detail specification in the end of this tutorial for special cases)

![](https://qhstaticssl.kujiale.com/image/png/1749608355822/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Adjust Bump Scale

# Note

**1. Diffuse**

![](https://qhstaticssl.kujiale.com/image/png/1749608556950/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**2. Color Correction**

![](https://qhstaticssl.kujiale.com/image/png/1749608594337/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**3. Bump Scale**

The larger the number is, the stronger bump effect is.

The smaller the number is, the weaker the bump effect is.

When the number is negative, the bump reverse.

![](https://qhstaticssl.kujiale.com/image/png/1749541817773/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
