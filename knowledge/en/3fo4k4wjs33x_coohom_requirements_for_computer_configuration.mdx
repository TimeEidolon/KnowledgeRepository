# Version Recommendation

**64 bit browser download links are as follows:**

* Google Chrome download link (Mac applicable): [https://www.google.com/chrome/](https://www.google.com/chrome/?kjl_source=hc)
* Edge browser download link (Mac applicable): [https://www.microsoft.com/edge/](https://www.microsoft.com/edge/)

**Kind Reminder:**

* Network and system software environment are also important factors affecting the smooth operation of Coohom design tools.
* System software environment: It is recommended to regularly clean local computer caches and system junk to ensure smooth operation.
* Network troubleshooting: 1. Unstable network, check network speed. 2. Too many browsing traces, need to clean cache.
* If only for floor plans, requirements can be lowered to 32 bit CPU + 4GB memory, no special requirements for graphics card and operating system.

# Coohom Configuration Requirements

| **Category** | **Standard Configuration** | **Recommended Configuration** |
| --- | --- | --- |
| Network | **4M** for **individual user, 100M**for **enterprise** | **8M** or above for **individual user, 100M** or abovefor **enterprise** |
| Browser | Meet both of the following points:  1. **Browser:** 64 bit   2. **Chrome:** 133 and higher (The most recommended)  3. **Microsoft Edge:** 133 and higer | |
| Operating System | win 10, win 11, Mac OS 10.5 | |
| CPU | 64 bit CPU | |
| RAM | 8GB | 16GB |
| Graphics Card | Minimum Integrated Graphics Card Model: Intel(R) UHD Graphics 770, [Intel Iris Xe Graphics G7](https://www.notebookcheck.net/Intel-Tiger-Lake-U-Xe-Graphics-G7-96EUs-GPU-Benchmarks-and-Specs.462145.0.html) |  |
| Discrete Graphics Card:  1. NVIDIA GeForce GTX 1650  2. NVIDIA GeForce GTX 1660  3. NVIDIA GeForce GTX 1060 | Discrete Graphics Card:  1. NVIDIA GeForce RTX 3060/3060Laptop  2. NVIDIA GeForce RTX 3060 Ti   3. NVIDIA GeForce RTX 4060 |

If you are operating Coohom on the Web, **Google Chrome** will be strongly recommended as the browser.
