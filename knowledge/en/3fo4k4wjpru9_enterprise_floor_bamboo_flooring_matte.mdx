In this article, we will explain in detail how to set up the material parameters of a customized model.

# Entrance

1. Enter the **workspace**, select **"Enterprise Catalog - Custom Products - Product Catalog"**, and click **Create Asset** in the upper right corner.

![](https://qhstaticssl.kujiale.com/image/png/1756886166278/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Select the **Material** under the corresponding tool.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1756886858094/2D37C2734C50A6837B78ACB4959B66FF.png)

3. Select **"Create One Material"**.

![](https://qhstaticssl.kujiale.com/image/png/1756886601087/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Parameter Settings

1. Adjust diffuse map and add VR color.  
When using default map, adjust VR color will change the floor color.  
(Size not exceed 5M, resolution not exceed 5000pixels)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848449038/BE2ECCF44DA920E355BDBF5923D2D1C5.png)

2. Color correction: Do not need to adjust except special cases  
(see detail specification in the end of this tutorial for special cases)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848450103/BBF100FB8D631C72F4DF73DCEFDE74E1.png)

3.Adjust Reflection scale

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848452944/BD05513DE191F116837B092FDDFBC0C9.png)

4.Adjust Reflection glossiness

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848455196/B38773621583F54F5AD6803E41DE7576.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848456436/D135C82556C5AD9EDAC9508D8FE5964F.png)

5. Adjust Scratches

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848457507/50831F40DB41447F0FA4DB7F51C2CF32.png)
