#### **1.** **Function introduction**

How to output a professional illumination calculation report?

How to calculate the uniform glare value UGR?

#### **2.** **Operating steps**

**Step1:**Enter the working plane editor page and draw a working plane.  

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745245286266/5A255B762B84634D25FADDACABFE4B1F)

a.The number of measurement points can be adjusted via the parameter panel in the lower right corner.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745245291711/66D94FBF0C390E8A3D322D723A555D14)

b.If you need to calculate the uniform glare value UGR, check the UGR switch.

Click Close in the upper-right corner to exit the Working Plane Editor after setting.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745245293750/A45600A9B5E390B0745CAD281032D52A)

**Step2:** In the Lighting Design Tools interface, click Calculation Export.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745245297549/40D90B35E830DC724AEB8C3153FF863F)

a.Select the calculated surface and the scene mode to be calculated, and click Start Calculating.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745245299213/2A7E60ED0B9A5A0C4C9FF40716CD5E1C)

b.Wait for a while, after the calculation is finished, you can export the PDF report.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1745245299993/592DC9A936383A82D14D0DF1A0A80A59)
