The rendering video montage function enhances promotional impact for finalized rendering videos. For the rendering video, you can:

* Connect multiple renderings for display
* Add company logo and watermark for promotion
* Set a video cover image
* Include an end logo
* Add subtitles and graphic elements

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started\~

<br />

🎥 [Video URL](https://www.youtube.com/watch?v=ZkzM0LgK1hM)

<br />

## For Users

All users

## Entrance

Enter the design tool, and click on **Gallery** in the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1753846882259/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance One:** Select a video that has been rendered in **Gallery** , and click **Montage**.

![](https://qhstaticssl.kujiale.com/image/png/1753842857577/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance Two:** View videos in the gallery, and click the **Montage** tool below the video.

![](https://qhstaticssl.kujiale.com/image/png/1753842913021/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Function Guide

### Select Import Clips in a Way

After clicking **Montage** , choose **the** **way of importing clips** , selecting either **full import** or **segment import.**

![](https://qhstaticssl.kujiale.com/image/png/1753842984318/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Video Settings

1. Set the **video scene ratio**, options include 16:9, 9:16, 4:3, 3:4, 1:1.

2. Set **enterprise watermark** on/off.

* **Note:** The watermark function is only available to **enterprise users** , how to configure enterprise watermark can be seen in the tutorial document: [click to view](https://www.coohom.com/en_US/helpcenter/enterprise-watermark-function-configuration-instructions?hl=en&kjl_source=hc)

3. Set **background**, choosing black or obscure.

![](https://qhstaticssl.kujiale.com/image/png/1753843672253/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Material Function

You can add **graphic annotations, shapes**, and other materials, drag the materials onto the video track and place them at the required time points.

![](https://qhstaticssl.kujiale.com/image/png/1753843795399/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**1. Edit Annotation Content**

Click on the annotation material to edit text content.

![](https://qhstaticssl.kujiale.com/image/png/1753844101245/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**2. Adjust Material Size**

Set the **zoom** value, or drag the four adjustable nodes around the material.

![](https://qhstaticssl.kujiale.com/image/png/1753844188288/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Add Subtitles

Click **Add Subtitles.**

![](https://qhstaticssl.kujiale.com/image/png/1753844233069/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

After entering subtitle **text**, it will be displayed in the video. You can move the subtitle position in the video, and set the time points for subtitles to appear in the video track below.

![](https://qhstaticssl.kujiale.com/image/png/1753844343730/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

In the **Subtitle Settings** , set the **subtitle zoom size** and **color.**

![](https://qhstaticssl.kujiale.com/image/png/1753844435294/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Add Images/Videos

Click on the lower-left **"Add Image/Video"** to import multiple images or videos.

![](https://qhstaticssl.kujiale.com/image/png/1753844501006/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Select the **images/videos** to be added and click **Confirm the Addition.**

![](https://qhstaticssl.kujiale.com/image/png/1753844712197/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

After importing multiple videos, you can add **transition effects** between different videos.

![](https://qhstaticssl.kujiale.com/image/png/1753845896354/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Set a Video Cover

Click **Cover** to add a cover image, and set **cover title** and **title heading style**

![](https://qhstaticssl.kujiale.com/image/png/1753846009201/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Set a Video Ending (Enterprise Function)

Click **the end of the film** to set **End-of-film Style, Company Logo** , and **Text Slogan.**

![](https://qhstaticssl.kujiale.com/image/png/1753846303726/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
