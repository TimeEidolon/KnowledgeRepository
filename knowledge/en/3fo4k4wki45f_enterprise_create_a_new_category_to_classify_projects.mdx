The function of creating project categories helps users create different category folders to organize and classify projects for easier management of their own projects.

# For Users

All enterprise users

# Entrance

Enter the **Workspace**, select **My Projects**, and click **New Category.**

![](https://qhstaticssl.kujiale.com/image/png/1752747705708/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Create a New Project Category

After clicking **New Category**, enter the **Category Name**, then click **Create.**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749577618/2D37C2734C50A6837B78ACB4959B66FF.png)

# Move Projects to Category Folders

## Move a Single Project

1. Select the project to be moved, click the ... icon at the bottom right corner, then select **Move to.**

![](https://qhstaticssl.kujiale.com/image/png/1752748010702/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. You can **select an existing category** or **create a new category and subcategory.**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749577590/2D37C2734C50A6837B78ACB4959B66FF.png)

3. After clicking **Create New Category**, enter the **Subcategory Name**, then click **Create.**

![](https://qhstaticssl.kujiale.com/image/png/1752748668255/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749577520/2D37C2734C50A6837B78ACB4959B66FF.png)

4. After creation, the project will be moved to the newly created subcategory folder.

![](https://qhstaticssl.kujiale.com/image/png/1752748787461/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Batch Move Multiple Projects

1. Select multiple projects to be moved to the same category, then click the move icon at the top right.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749577605/2D37C2734C50A6837B78ACB4959B66FF.png)

2. You can **select an existing category** or **create a new category and subcategory**, for detailed operations see above.

![](https://qhstaticssl.kujiale.com/image/png/1752749083908/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. After creation, the selected multiple projects will be moved to the corresponding folder.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1752749577576/2D37C2734C50A6837B78ACB4959B66FF.png)
