# Problem

The uploaded model shows fisheye or spherical glass (usually due to single-sided glass)

# Self-check

Check if the model parts using glass material (whether the refract attribute is enabled in the material) are cubic (i.e., have length, width, and height properties)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762505534467/47999879BC99A5AB5EACAF3DACD62FCD.png)

Provide an hdri, render directly, observe the effect

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762505534496/D5D49F3EA7029D97C088FA3F79FB19CC.png)

# Solution

Apply a shell command to the model parts

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762505534450/D138A59A7A7C068AC00E6A534DEF5F07.png)

Change the refractive index in the material properties to 1.0

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762505534462/F24580C8AFAC55D41E99354B69657401.png)
