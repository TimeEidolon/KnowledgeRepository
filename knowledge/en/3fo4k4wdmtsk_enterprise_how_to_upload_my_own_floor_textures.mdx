# **For Users**

Enterprise users can upload paving data and edit related parameters.

# **Action Steps**

To upload your own Textures/Materials/Paints, you may:

1. Click Enterprise creating > Products > **Paving**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1760940819119/56DA0702CBEB1C7036E8169C1A36D4E7.png)

2. Click **New Pavings** to upload textures by selecting them from your computer.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1760940819082/20E10FECC8A4BBBDDB2F65294A0D3A53.png)

3. Please note the file upload requirements:

-Upload up to 200 paving at a time.  
-Each file should not exceed 5MB. JPG and PNG formats are allowed  
-The length and width of image should not exceed 5000 pixels.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1760940819086/93BE711A9A47718496C42C9EAAB2D4CD.png)

4. Fill in all required information. Ensure dimensions match installation specifications. Choose categories carefully, as different classifications may affect rendering results.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1760940819170/E99279CA857ECD5A98C0F9C7A024A424.png)

You can edit the paving in Edit All - Default Pattern, where you can choose from 6 different pavings:

Grid/ Brickwork/ Staggered Brickwork/ Herringbone/ Whirlwind/ Chevron

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1760940819173/10ADDD549827B42563C7BD31317C05AE.png)

|  |  |  |  |
| --- | --- | --- | --- |
| NUMBER | PAVING | PRODUCT | RENDERING |
| 1 | Staggered Brickwork |  |  |
| 2 | Grid |  |  |
| 3 | Herringbone |  |  |
| 4 | Brickwork |  |  |
| 5 | Whirlwind |  |  |
| 6 | Chevron |  |  |

5.Click **Done** to confirm.

6.You can also use the pattern tool in **Floor Editor** to modify your pattern effect. (Except for Chevron patterns)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1760940819183/DA1892EA78C22FABA5F053E2D1D0BB31.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1760940819289/741824DF98A886F00CF99196822E93A4.png)

**Note:**

For **individual users**, please refer to the following guides for uploading and importing: [How To Upload My Own Textures (Material)?](https://www.coohom.com/helpcenter/how-to-upload-my-own-textures-material?kjl_source=hc)
