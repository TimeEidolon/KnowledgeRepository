The model distance dimension line can help you accurately view, measure, and adjust the distance between models. This article will detail practical functions such as 2D/3D dimension display, numerical editing, and slanted wall measurement, making your design more convenient and precise.

⭐Before reading the article, it's recommended to watch the video tutorial first to get started quickly~

🎥 [Video URL](https://www.youtube.com/watch?v=sH2RJYXiQpM)

# For Users

All Users

# Entrance

1. In the **2D** view, click the bottom toolbar **Show/Hide** icon and select **2D Distance.**

* In the 2D view, the **2D Distance** defaults to **Margin Distance** mode.

![](https://qhstaticssl.kujiale.com/image/png/1773719593273/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the **3D** view, click the bottom toolbar **Show/Hide** icon and select **3D Distance Dimension Line.**

* In the 3D view, the **3D Distance Dimension Line** defaults to **Hide the Dimension Lines** mode.

![](https://qhstaticssl.kujiale.com/image/png/1773725944822/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Model Dimension Line Distance Settings and Hiding

## Margin Distance

When you select **Margin Distance** and a model, the dimension displayed is the distance from the selected object's border to the nearest object.

![](https://qhstaticssl.kujiale.com/image/png/1773803026460/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1773803145784/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Center Point Distance

When you select **Center Point Distance** and a model, the dimension displayed is the distance from the selected object's center point to the nearest object.

![](https://qhstaticssl.kujiale.com/image/png/1773803314507/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1773803206681/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Hide the Distance Dimension Lines

1. Select **2D Distance**, click **Invisible**, and the tool will no longer display distance dimension lines.

![](https://qhstaticssl.kujiale.com/image/png/1773803458829/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Select **3D Distance Dimension Line**, click **Hide the Dimension Lines**, and the tool will no longer display distance dimension lines.

![](https://qhstaticssl.kujiale.com/image/png/1773803560095/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Dimension Editing

1. **Basic Input**: After selecting an object, dimension lines and input boxes are displayed. You can directly input precise values in the box for accurate object movement.

![](https://qhstaticssl.kujiale.com/image/gif/1773803664465/D0F6A7934A629582CB972C4F55C2A615.gif)

2. **Basic Arithmetic**: Based on Basic Input, addition (+), subtraction (-), multiplication (\*), and division(/) arithmetic operations are supported for quick calculations and object movement.

![](https://qhstaticssl.kujiale.com/image/gif/1773803740449/435B495D2CC0DB3743BE6523FD081D5D.gif)

# Slanted Wall Measurement

There are two methods here, the details are as follows:

**Method One**

In the 2D view, you can click the bottom **Show/Hide** icon, then select **2D Distance**, click **Center**, and select the model. The vertical distance from the model to the wall will be displayed.

![](https://qhstaticssl.kujiale.com/image/gif/1773804358104/8B92EAD58843012015ACE668614C773A.gif)

**Method Two**

In the 2D view, you can directly use the shortcut key **Shift/⇧+Space** to switch from **Margin Distance** mode to **Center**mode for measuring the vertical distance from the model to the wall.

![](https://qhstaticssl.kujiale.com/image/gif/1773804473843/CD8FC63C91BC49078853BAFD719BE561.gif)

# 2D Distance Dimension Line Isolation Display

1. Move the mouse to the avatar and select **Preferences.**

![](https://qhstaticssl.kujiale.com/image/png/1773804529404/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the pop-up panel, check **Allows Dimension Lines to be Recognized Among Wall/Ceiling/Floor Models**. In the 2D view (floor view/ceiling view), wall models, surface models, and floor models will recognize distance dimension lines between each other, allowing, for example, ceiling models to identify distance dimension lines with wall, surface, or floor models.

![](https://qhstaticssl.kujiale.com/image/png/1773730243941/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1773805038247/BA3CF5EA969D0747CC1A36BC94ED33E3.gif)

3. If **Allows Dimension Lines to be Recognized Among Wall/Ceiling/Floor Models** is unchecked, wall models, surface models, and floor models in the 2D view (floor view/ceiling view) will only recognize dimension lines within the same category—for example, ceiling models will only recognize dimension lines with other ceiling models, floor models will only recognize dimension lines with other floor models, and wall models will only recognize dimension lines with other wall models.

![](https://qhstaticssl.kujiale.com/image/png/1773730221591/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1773805023053/DCE71965D19C1438840BBBDE410C6B53.gif)
