# Problem

Convert higher version Skp to lower version.

# Solution

Open the model in SketchUp, click on the upper left "File", select "save as"

![0015a05bffc44a0fc93f6699a843e97](https://qhstaticssl.kujiale.com/yuntai/image/png/1551343764334/AD3F4BA7DA8D6A520A4F1150569D3911.png?x-oss-process=image/format,webp)

In the "save as" window, you can save it as a lower version; it is recommended to save as version 2020 or below.
