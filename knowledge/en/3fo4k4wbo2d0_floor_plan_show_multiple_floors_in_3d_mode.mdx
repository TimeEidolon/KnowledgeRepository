# Show All Floors

1. Switch to **3D mode**.

2. Click the **floor navigation**, select **show all floors**.

* **Note: Show and hide a single or all floors, and other floors except the current floor only support preview, not edit.**

![](https://qhstaticssl.kujiale.com/image/png/1747893823632/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1747894357368/10CB6F18489D13C410D507C1CA06CD49.gif)

# Show a Single Floor

Set the **other floors opacity** to view effect, you can preview all floors as a whole, or you can click the small eye button to preview two floors individually.

![](https://qhstaticssl.kujiale.com/image/png/1747894515994/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1747894721571/3514A4A5553E6385693D4BEF1825C107.gif)
