After building the design scene, the render tools can help us quickly generate good images. This article will take you to learn the use of render tools step by step: start by selecting the rendering mode, add lights and environments, adjust the camera position, set image resolutions and advanced parameters, and finally complete rendering of the images. The article comes with rendering tips to help you get started with render tools efficiently.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=DFZIZQUMr9I)

# For Users

All users

# Entrance

1. Enter the **workspace** and select the **Create New Pjoject** in the upper right corner.

![](https://qhstaticssl.kujiale.com/image/png/1779343695360/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the 3D design and choose the **Render**.

![](https://qhstaticssl.kujiale.com/image/png/1750330801038/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Set Aspect Ratios

1. Select the rendering mode, with **perspective view, 360° walkthrough, and top view** options.

2. Set **aspect ratios** on the left panel.

* The 360° walkthrough renders to a 360° effect, no need to set aspect ratio.

![](https://qhstaticssl.kujiale.com/image/png/1750660042744/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Add Ambient Lights

## Select a Light Template

Select the **light template** to set **brightness and effect** parameters.

![](https://qhstaticssl.kujiale.com/image/png/1750660165617/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Add Custom Lights

1. Click **add to customize**.

2. **Name** the lighting project, select a **basic template** and customly adjust it based on the template.

* **How to add custom lights, you can see the relevant tutorial documents: [3 Steps to Light Customization](https://www.coohom.com/en_US/helpcenter/3-steps-to-light-customization?hl=en&kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1750660457229/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Add Environments

Select the appropriate environment to set the **environment brightness, tone, position and ambient lightbrightness**.

![](https://qhstaticssl.kujiale.com/image/png/1750660495324/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Adjust the Camera Position

Select the camera and the right panel can set the **camera's position, views and clipping effect**.

* **How to adjust the camera properties, you can see the relevant tutorial documents: [Set Camera Properties and Views](https://www.coohom.com/en_US/helpcenter/set-camera-properties-and-views?hl=en&kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1750660697798/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1750660803950/A92E692F248F7BA0CFE45F5FDC7C1502.gif)

# Set Image Resolutions

There are four resolutions to choose from, the higher the resolution, the longer the rendering time will be.

* **Tips: You can first select 1K SD rendering small images, initially determine the rendering effect, and then select one with high resolution for rendering.**

![](https://qhstaticssl.kujiale.com/image/png/1750660872590/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Set Advanced Parameters

Select advanced settings to adjust the image effect advanced parameters.

* **How to set advanced parameters, you can see the relevant tutorial documents: [Set Advanced Render Parameters](https://www.coohom.com/en_US/helpcenter/set-advanced-render-parameters?hl=en&kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1750403751564/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# View Images

After adjusting all rendering parameters, click **Generate**. The image can be viewed in the **gallery** in the upper right corner.

![](https://qhstaticssl.kujiale.com/image/png/1750661011352/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1750661058163/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
