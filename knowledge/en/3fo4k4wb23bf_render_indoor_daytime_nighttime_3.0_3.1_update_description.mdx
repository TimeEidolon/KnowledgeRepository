Realistic class added 4 new lighting templates [Indoor Daytime/Nighttime 3.0/ 3.1]. In order to allow users to better compare the effects, this article will explain in detail the difference comparison between the Indoor Daytime/Nighttime 1.0/2.0/2.1/3.0/3.1.

# **For Users**

**All users**

**Sphere of influence：**Manual lights that have been created are not affected by version updates. Automatic lights created after the update in Light version management and manual lights created after the update are rendered using the new light template.

# **Action Steps**

You can experience the new version of light by upgrading the light template to the new version under the light version button in the rendering interface.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1743128673273/62A13C4A4CA7DAD41DC090AA6154D394.png)

# **Rendering Parameter Upgrade**

**1）Reduce noise**

|  |  |
| --- | --- |
| **Indoor Daytime/Nighttime  1.0/2.0/2.1** | **Indoor Daytime/Nighttime 3.1/3.0** |
| Need to open manually | On by default |
|  |  |

**2）Light leakage repair**

| **Uncheck** | **Check** |
| --- | --- |
|  |  |

| **Indoor Daytime/Nighttime  1.0** | **Indoor Daytime/Nighttime  2.0/2.1/3.0** | **Indoor Daytime/Nighttime 3.0** | **Indoor Daytime/Nighttime  3.1** |
| --- | --- | --- | --- |
| No this function. | To add Light leakage repair, you need to manually select it. | Remove the [Light leakage repair] button from the effect settings and change it to default to enable the light leakage repair repair. | Compared to the old version, 99% of the default has solved this problem, enabling this feature can provide more accurate light cache calculation.  Therefore, it should be noted that the rendering time is increased after checking and the rendering effect is not different at low resolution. |

**3）From version 3.0 onwards, the number of global lighting bounces has been increased, and dark details have been improved.**

Due to the GI upgrade, the same light brightness can illuminate a larger area, as shown in the image below. Since 3.0, the automatic light brightness has been reduced accordingly, and the manual light is also recommended to be moderately reduced within a reasonable range to avoid exposure problems.

# **Major updates in version 3.1**

**1）Introduction of real-time rendering: adjust parameters to tone and render, and preview the effect with only 1s to reduce waiting. Click to learn more:****[Real-time Rendering Is Launched In Coohom](https://www.coohom.com/helpcenter/article/real-time-rendering-is-launched-in-coohom?kjl_source=hc)**

As you move the canvas, adjust materials and lighting parameters, the lens Angle is rendered in real time, allowing you to see the design effect in real time.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1743131101057/7DDD32C1361537E76BE2D159C3BEBAC3.gif)

**2）Entry Update**

Move the manual template to the "real-time Rendering" TAB, Render popup entry location is as follows:

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1743131136962/B8E7418691C496B30D2F4EC72D47E0BA.png)

**3）The noise reduction parameters were upgraded in version 3.1, and the final rendering effect not only effectively reduced the noise problem, but also better preserved the rendering details. At the same time, the problem of some material blur in version 3.0 has been solved.**

| **Indoor Daytime  3.0** | **Indoor Daytime   3.1** |
| --- | --- |
|  |  |

**4）Added post-processing [Exposure suppression] : The exposure suppression parameter is opened in the real-time rendering - Effects - Post-processing parameter.**

This parameter is used to control the brightness and detail of the exposure area. The lower the parameter,  the softer the highlights will be, thus retaining more details.Users can adjust as needed.

| **Low Parameters** | **High Parameters** |
| --- | --- |
|  |  |

**5）Other feature upgrades: Realistic templates for the first time support LUT filters, depth of field, material replacement and more.**

**a. LUT Filters**

Manually open filters → Free to switch between different filters in the public library.

* **Note: You can also upload SRGB mode filters by yourself, currently cube format is supported.**

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1743131544682/4E7EC4EA67380C60C9EFC77CAF2D8247.gif)

**b. Depth of field**

Manually open the depth of field, by editing the center point and blur can achieve a variety of depth of field effects.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1743131578623/2ED2BA8979DD11808D59DA3F391DA68A.gif)

**c. Material replacement**

Click material to replace material in the left column, you can also edit the original material properties in the right column.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1743131619223/796EED5EBB2326D01209A4AF746A4766.gif)
