### **I.Function Introduction**

**Usage Scenario:** Customers want to download a PDF file when downloading a custom list.

**Before optimization:** PDF downloading is not supported, only word and excel can be downloaded

**After optimization:** The new viewer supports downloading PDFs and enriches some interactive interface displays, such as selecting single-page/continuous/multipage display, report centering, etc
