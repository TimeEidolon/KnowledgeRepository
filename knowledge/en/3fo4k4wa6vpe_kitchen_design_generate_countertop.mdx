The countertop generation function allows efficient and flexible countertop design, which can be automatically identified or freely drawn.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=7MVX5bZhZUs)

# For Users

All users

# Entrance

Select **Kitchen＆Bath** to enter the customization interface.

**Entrance One:** Select the **Generate-Countertop** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739351042638/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance Two**: Select the cabinet, and select **Generate-Countertop** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739415754414/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Generate Countertop

1. Select the cabinet, the tool will automatically identify the cabinets that need to generate countertops. You can also **press and hold shift to select multiple cabinets** for generation. If you need to identify all cabinets, you can click **Auto-select**.

![](https://qhstaticssl.kujiale.com/image/png/1739351074369/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the basic parameters on the right panel, select add **front edge,** **back splash and countertop material**. After adjusting the specific parameters, click **Generate**.

![](https://qhstaticssl.kujiale.com/image/png/1739351114763/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. If you need to freely adjust the range of the countertop generation, click **manual edit** in the top bar to enter the manual mode.

![](https://qhstaticssl.kujiale.com/image/png/1739351524874/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. In the manual interface, use the drawing tools on the top bar in 2D to edit edge and add rectangle.

![](https://qhstaticssl.kujiale.com/image/png/1739352279901/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1739352378412/6E0CBEFB6EC6360F8238AD7475591BB0.gif)

5. You can manually edit points, lines, and surfaces, or enter values ​​for precise adjustments. After the adjustments are completed, click **Generate**.

![](https://qhstaticssl.kujiale.com/image/gif/1739352518988/CD537F53AEEBD0EC66F8137FD7649EC1.gif)

6. After generating the countertop, click **Finish** to exit the generate countertop interface.

![](https://qhstaticssl.kujiale.com/image/png/1739352650150/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* The generate countertop effect is as follows：

![](https://qhstaticssl.kujiale.com/image/png/1739352745509/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
