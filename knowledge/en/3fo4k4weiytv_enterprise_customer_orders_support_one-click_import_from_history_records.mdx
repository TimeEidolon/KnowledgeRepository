For customized order scenarios for long-term cooperative customers, considering that the basic information of such customers is relatively fixed, only order placement time and transaction amount differ, to reduce the repetitive work of filling out customer forms when creating new orders and improve order entry efficiency, in the process of creating new customer orders, relevant information can be imported from historical records. Simply select past orders to synchronize the information into the new order form.

### Step 3
On the import page, you can browse all historical customer orders. You can select one order data to import, and the new order form will automatically populate all the field values from the old order. * For example, if I have a long-term customer, each time I create a new order, I must write his information again. Now, I only need to select his previous order to directly import it, avoiding repetitive form filling.

# Import From History

1. Click **Create Customer Order**, then click **Import From History** on the pop-up page.

![](https://qhstaticssl.kujiale.com/image/png/1769501623502/1D3442FD1627A8BBBD0FC568C9B5461D.png)

2. On the import page, you can browse all historical customer orders. You can select one order data to import, and the new order form  
will automatically populate all the field values from the old order.

* For example, if I have a long-term customer, each time I create a new order, I must write his information again. Now, I only need to select his previous order to directly import it, avoiding repetitive form filling.

![](https://qhstaticssl.kujiale.com/image/png/1769507259007/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. After importing, you can see that the form automatically populates all the values from the old order, and all fields support importing (Note: The new order and the selected imported old order have no actual connection; the import function is only to help the user quickly fill out information).

* After importing, you can modify field values again and proceed to save and create a new order.

![](https://qhstaticssl.kujiale.com/image/png/1769501538519/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
