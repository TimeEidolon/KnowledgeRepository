Custom Furniture —Create Asset—Parametric Model Library—New—Internal Parts—Cupboard panel![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931551598/1B3AB0D6636939B23A1A5D1F0ADCE943.png)

**II.Setting Parameters**

2.1 Basic Parameters:

2.2.1 Width (W) - Represents the size of the model panel in the X-axis direction. Set the value type as a range (e.g., range 5-1000, current value: 600).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931553973/EEB8440E389C68210AC63643F33B4C0E.png)

2.2.2 Depth (D) - Represents the size of the model panel in the Y-axis direction. Set the value type as a options (e.g., optional 18, current value: 18).![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931555703/F860EFEE45E0112BA330FBC23AF96C8B.png)

2.2.3 Height (H) - Represents the size of the model panel in the Z-axis direction. Set the value type as a range (e.g., range10-1000, current value: 800).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931556978/CDC314D34EC1EFB81C7839A51A50DBEF.png)

2.2.4 Material - No modification needed, indicating that the Door uses the default material.

2.3 Custom Parameters

Custom Parameters are used to add some more parameters except that commonly basic parameters or formulas in the editor.

2.3.1

Groove With(GW)—Represents the width of outer edge groove of panel,set it a range from 0 to 10 and value 4.

Groove Depth(GD)—Represents the depth of outer edge groove of panel,set it a range from 0 to 10 and value 4.

**III.Importing Submodels**

3.1 Import a flat panel from elements library and edit it contour![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931558542/DD8D4DA1B0C9807C0BF6AB44256A8F54.png)

Set four points: point 1(0,0),point 2(0,#D),point 3(#W,#D),point 4(#W,0)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931560362/68D259B132AD8A1EF8DC9F0D7CC5B778.png)

Height set to #H.

Material angle #CZFX, Side material rotate #CBCZFX.
