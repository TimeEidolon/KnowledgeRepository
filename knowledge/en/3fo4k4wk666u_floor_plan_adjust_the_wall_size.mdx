After drawing walls, you can adjust the length, thickness and height of the walls to fit the actual layout.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=JGti9OcCrYM)

# For Users

All users

# Modify Wall Length

1. Enter the design tool, select the wall you want to modify, click on thedimension line for lengthannotation, andactivate the dimension line input field.

![](https://qhstaticssl.kujiale.com/image/gif/1772767688455/D0F6A7934A629582CB972C4F55C2A615.gif)

2. Enter thedimensions in the input field, choose the wall'sdirection of extension, then press the **Enter** key to confirm**.** At this point, you'll see the wall length has changed.

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1772769908424/1CA83DEA6D0928946F2BD389D15284CF.gif)

# Modify Wall Thickness

There are three ways to modify wall thickness, as detailed below:

**Method 1**

1. Select the wall you want to modify, click on thedimension line for thickness annotation, and activate the dimension line input field.

![](https://qhstaticssl.kujiale.com/image/gif/1772768108964/8B92EAD58843012015ACE668614C773A.gif)

2. Enter thedimensions in the input field, choose the wall'sdirection of extension, then press the **Enter** key to confirm**.** At this point, you'll see the wall thickness has changed.

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1772769908391/4F947A0C40AB61A3D176FB209E4463BB.gif)

**Method 2**

After selecting the wall, find the **Thickness** option in the parameters panel on the right. You can either choose the required size directly from the value bar and click apply, or manually input the values in the input box to adjust the wall thickness.

![](https://qhstaticssl.kujiale.com/image/gif/1772768446104/BA3CF5EA969D0747CC1A36BC94ED33E3.gif)

**Method 3**

When drawing walls, you can hold down the **Alt** key to switch wall thickness quickly and conveniently.

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1772769908483/01E538B0FC104B103B1396EF5612F38D.gif)

# Modify Wall Height

To modify wall height, firstselect the wall, then input the desired value in the **Height** section on the right parameter panel, andpress the **Enter** key to confirm (default height is 2800).

![](https://qhstaticssl.kujiale.com/image/png/1772769296584/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Notes

The dimension line input box supports the addition (+) subtraction (-) multiplication (\*) division (/) four operations, using the current size or input size for quick calculation, and quickly modifying wall parameters (the dimension line input box of beams and columns is also supported)

![](https://qhstaticssl.kujiale.com/image/gif/1774233225077/CD8FC63C91BC49078853BAFD719BE561.gif)
