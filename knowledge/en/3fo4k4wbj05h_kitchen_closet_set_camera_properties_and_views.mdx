# Set Camera Properties

## Adjust the Camera Height

In the parameter panel on the right, you can enter the height value, or move the camera height up and down.

* **Tips: The camera height is best between 1100-1300. If you render a local object, it depends on the actual height of the object. The camera angle is 0 degrees, don't look down or up.**
* **Height parameter reference：**

  **Home decoration — Floor height 2700-2800mm, camera height 800-1200mm**

  **Villa — Floor height 3000-3500mm, camera height 1200-1400mm**

  **Engineering decoration and outdoors — Floor height 5000mm, camera height between 1400-1600mm**

![](https://qhstaticssl.kujiale.com/image/png/1744371866582/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1744371938475/10D3E9869DAE64770510F9B37E02B51C.gif)

## Adjust the Camera FOV

Adjust the rendered view range by dragging FOV bar.

* **Tips: 60-80 degrees of FOV rendering is best in global space. A FOV of 0-30 degrees is suitable for local rendering. The FOV can be used in combination with camera clipping, making it easier to find good-looking angles.**

![](https://qhstaticssl.kujiale.com/image/png/1744372229661/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1744372758921/2942E604F30A1CAA0C5DA150AD17E192.gif)

## Set the Camera Clipping

Adjust the rendered view range by dragging the clipping value bar.

![](https://qhstaticssl.kujiale.com/image/png/1744372882259/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1744373004045/059590B2744DA6D1655EBD2EEAD17974.gif)
