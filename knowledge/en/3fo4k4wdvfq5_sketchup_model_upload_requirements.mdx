In order to facilitate smoother uploading of SketchUp models in Coohom, this article will explain in detail the requirements for uploading SketchUp models.

# For Users

All users

# Supported Formats

1. Models from version 2024 and earlier

2. ***SKP*** file uploads, support for packaged uploads

# Model Requirements

3D "File"--"Export"–"3D Model", choose obj format, check if the exported obj file exceeds 3 million faces, as shown:

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762308524484/9A165AB0735BAB5E728EDFDD012C2BEA.png)

1. Units should be in millimeters or inches. (Units can be set to mm in SketchUp through "Window"-"Model Info"-"Units")

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762308524466/940DF5A2F079E67020F45AE98DECC519.png)

2. Model file and object size should both be less than 200MB. (How to check actual object size: export obj format file from the model file)

3. Each side length of the model must not exceed 30 meters.

4. Texture formats supported are .jpg and .png. The image mode must be RGB, and texture file pixels must not exceed 5,000.

5. Material names should not have special symbols, and only default SketchUp materials are supported, not V-Ray or other plugin materials.

6. Model view should be set to front view.

7. The file name of the su model must not contain spaces.
