Rename it as "Back Panel" and set the component's position and dimensions. Set toe kick height as a fixed value of 80mm. Width: #W-36 Depth: #D-18 Height: 18 Position X: 18 Position Y: 0 Position Z: #H-18 Chamfer width: #W1-18 Chamfer depth: #D1 6.Add pentagon-shaped component, rename it as "Bottom Panel," and set the component's position and dimensions.
