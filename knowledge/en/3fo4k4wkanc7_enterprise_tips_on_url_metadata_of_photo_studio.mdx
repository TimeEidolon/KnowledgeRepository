When the user views the image, the metadata in the URL can display the name of the scene/model, and this article will explain the URL metadata in detail.

# **For Users**

All  enterprise users

**Q: Can the metadata in the URL display the name of the scene/model when the user views the image?**

**A:** **The URL can show the IDs of the model and the studio scene.**

**⭐See details as follows:**

1. A gallery of photos that are shot in the photo studio are related to a specific product model. While you browse these photos, from the metadata in the URL on top, you can tell which product model the studio photo is shot for. Then you can search for the target model based on the BrandGoodId.

* In the picture below, the ID of the model for the photo, i.e., the BrandGoodId, is highlighted in a red box. It is shown as "3FO48E5RLI8B".
* Meanwhile, the album ID, i.e. the studio ID of the gallery of photos, is shown as "3FO4K58VANHC".

![](https://qhstaticssl.kujiale.com/image/png/1748486854726/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1748486938925/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* By typing in the BrandGoodId, you can find the product model related to the photos in the model library from Enterprise Catalog. When the search is done, the name of the product model is also indicated below its preview.

![](https://qhstaticssl.kujiale.com/image/png/1748487012127/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* You can also search for the target model based on the product name.
* The picture below shows the name of the related product model:

![](https://qhstaticssl.kujiale.com/image/png/1748487057096/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* By typing in the model name, you can get the same search result.

![](https://qhstaticssl.kujiale.com/image/png/1748487109840/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Though the studio ID, unique to each studio, is shown in the URL on top, you cannot search for the studio used for the photos based on that ID in the photo studio album. In other words, the studio ID cannot help you find or locate the studio. If you want to get the search results, type in the studio name that you create.

* A failed search based on the studio ID:

![](https://qhstaticssl.kujiale.com/image/png/1748487179471/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* A successful search based on the studio name:

![](https://qhstaticssl.kujiale.com/image/png/1748487228571/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
