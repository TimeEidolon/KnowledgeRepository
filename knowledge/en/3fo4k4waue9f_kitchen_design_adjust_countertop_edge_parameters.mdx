Freely edit the front edge and back splash of the countertop to meet the diverse countertop design needs.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=6AJRQKFcOu4)

# For Users

All users

# Entrance

After generating countertop, select the countertop and select **manual edit** in the floating toolbar.

* The operation of generating countertop can be seen in the tutorial: **[Generate Countertop](https://www.coohom.com/en_US/helpcenter/article/generate-countertop?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1740036279153/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Adjust Countertop Edge Parameters

1. Select the edge on one side of the countertop and adjust the edge parameters in the parameter panel.

**Note: The change of the height of the unilateral edge is not supported. After adjusting the height of the unilateral edge, the height of the other edge will change.**

![](https://qhstaticssl.kujiale.com/image/png/1740038141428/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. When adjusting the oversize parameters, you can judge the oversize direction (left /right) based on the arrow color, and then set the oversize value.

**Note:**

1. **When editing the countertop, the edge is not selected (front / back) or the edge style is not selected, and the oversize mark is not displayed.**
2. **Unable to adjust oversize edges (for example: curves) , and the oversize mark is not displayed.**

![](https://qhstaticssl.kujiale.com/image/png/1740038601059/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1740038655330/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Switch to 2D mode, you can select the line to add points, freely move the positions of the points and lines.

![](https://qhstaticssl.kujiale.com/image/png/1740039089718/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1740039149606/A7EA2FA9B90470AA4F8D80E8269427A3.gif)

4. After the adjustment is completed, click to **Generate**.

![](https://qhstaticssl.kujiale.com/image/png/1740039293380/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
