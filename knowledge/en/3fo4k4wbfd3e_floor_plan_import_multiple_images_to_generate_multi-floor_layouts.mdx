# Importing Multiple Floor Plan Drawings

1. In the design tool, click **Import 2D Drawing**, select the drawings you need to upload from the popup panel, and click **Open**.

![](https://qhstaticssl.kujiale.com/image/png/1779186783109/4BCD1377ADBDCABF1C48D785ECEE311F.png)

2. After uploading successfully, click **Upload Pics** in the lower left corner, and continue uploading drawings using the same method.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1779260129781/A9B031268239B48604851CDCF99C7DD5.png)

* If you want to upload multiple drawings quickly, click **Import 2D Drawing**, hold down the **Ctrl** key to select multiple drawings, then click **Open**.

![](https://qhstaticssl.kujiale.com/image/png/1779187661805/BDE13ECFB002CD384A4212182F648930.png)

3. After successfully importing multiple drawings, check the floor level of each drawing to ensure each floor plan is on the correct level.

![](https://qhstaticssl.kujiale.com/yuntai/image/gif/1779260129744/3A999E1633012EFB1124D481D8A046E9.gif)

4. Align the ruler with the actual dimensions on each image and verify the values in the input box to ensure the scale is correct.

![](https://qhstaticssl.kujiale.com/image/gif/1779188079279/435B495D2CC0DB3743BE6523FD081D5D.gif)

* **Note:** Dimension annotations support modifying measurement units, as shown below:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1779260129662/7C1E5C4A9C3F1FB2189799B22764B9A7.png)

# Generating Multi-Level Floor Plans

1. By default, check **Auto Generate Floorplan** and click **Import** to automatically generate 3D floor plans. If you only need to import the images into the project, uncheck this option before clicking Import.

![](https://qhstaticssl.kujiale.com/image/png/1779188224437/28D0817BA3F1332C66E1DA437F1C1905.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1779260129900/4FB7AB3C3A2741CE36F1F4287F2C4AB4.png)

* If you want to adjust the imported floor plan drawings, detailed steps can be viewed in the online document: [Click to View](https://www.coohom.com/helpcenter/floor-plan-efficiently-adjust-imported-floor-plan-drawings?kjl_source=hc)

2. After generation, switch to **3D** mode.

![](https://qhstaticssl.kujiale.com/image/png/1779188664621/4E972C11A6FBA168318D6A9C81C68B61.png)

3. Click **Floor Navigation** and select **Show All Floors** to view the multi-floor generation effect.

* Detailed steps can be viewed in the tutorial document: [Click to View](https://www.coohom.com/zh_CN/helpcenter/%E6%88%B7%E5%9E%8B%E8%AE%BE%E8%AE%A1-3d%E6%A8%A1%E5%BC%8F%E4%B8%8B%E6%98%BE%E7%A4%BA%E5%A4%9A%E4%B8%AA%E6%A5%BC%E5%B1%82?kjl_source=hc)
