**ⅠCreate a new folder under material folder**

1.1 click advanced---range settings![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931592382/EE35FCA53A13D1AE19CAEF40ADA06B99.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931597330/DB467EA07DC91C1EB1A7782A1F002E52.png)

2.2 Choose outline package configuration---New outline package![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931597926/93CF696C8CC458111181566068C61010.png)

2.3 Enter information of outline package in information bar, name, type of product,

When object type is category:click placed product and select product catalog, then click yes;![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931598824/A01A23EC380FC0FBE2EDA0A27C50E5DE.png)

Also,select enumeration to be object type,need to select certain single material in library![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931601263/92E518208684CFA521FA47FBB7E09878.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931602259/07E99E43CDF9F96DAF7AD8532E8D89A6.png)

2.4 enter the model editer, click line contour parameters and select options as data type,then select outline package and select a certain line as value.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931603618/7B50A4087B4B99E1977B037B6542FAD0.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931606540/3B69759743FD29AD9EC2A24AFB315F93.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931608969/E9A5862EFECB577E4F96FD2DADC8AED3.png)

Now only line contour which in outline package can be select.
