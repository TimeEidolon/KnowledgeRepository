**Recognizing differences between generated design and traced floor plan.**

There are two ways to import CAD files in the Cloud Design 5.0 tool: Generating a 3D model from the CAD file (Generated Design) or tracing over the original CAD floor plan (Traced Floor Plan). The "Import CAD" button on the top menu imports as Traced Floor Plan.

Generated Design: When imported, this will clear the corresponding floor's design. For example, if there are two floors (1F and 2F) in the current design, importing a new CAD file for 2F as Generated Design will clear the existing design for 2F but not affect 1F.

Traced Floor Plan: When imported, it does not clear the existing design content. It only adds a new CAD floor plan file that can be toggled on or off.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733854360237/F57FF452206769D1033EB693B841EC77.png)

**Import process and limitations**

A maximum of 5 files can be selected for import per floor. If there are already imported files, the number of files you can add will be limited to a total of 5 per floor.

After import, you can switch and show/hide files by clicking "Floor Plan Settings" on the bottom bar.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733854363214/86B72F0FF69226AA80E4CEF7988EC1EC.gif)

**CAD functions and changes**

There have been some adjustments to the CAD floor plan setup, and some functions have moved. Changes to bottom bar buttons: They now expand the floor plan settings panel instead of opening a functional menu.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733854365673/DCD197051AB01A5189AF720F924A452F.png)

File management: In the floor plan settings panel, you can switch between different CAD files.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733854366811/9E09520897B098CA8728B0C4D64F6226.gif)

Layer management: In the floor plan settings panel, you can show or hide layers in each CAD file.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733854367846/68F3BBE131459C88CF19144384ABAD92.gif)

The position of the manual CAD alignment has changed and is now managed from the top toolbar.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733854369327/290E17E9F624A6A424BBF146F78E43B1.gif)
