In the Material & Component tools, for bathroom and kitchen thresholds, you can add a custom threshold stone, and set the size, material, and thickness of the threshold stone.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=LTVWmD0iv6k)

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Enter the material & component interface, select the threshold to which you want to add a threshold stone.

### Step 2
In the **area settings** panel on the right, turn on **"Set as Threshold"**.

![](https://qhstaticssl.kujiale.com/image/png/1747031369838/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
<img src="https://qhstaticssl.kujiale.com/image/png/1747031369838/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png" alt="Step 2" style="max-width:100%;height:auto;" />

### Step 1
After setting the threshold stone, you can change the **material** of the threshold, and set the **thickness** and **grinding edge size** of the threshold stone.

![](https://qhstaticssl.kujiale.com/image/png/1747034183542/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
<img src="https://qhstaticssl.kujiale.com/image/png/1747034183542/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png" alt="Step 1" style="max-width:100%;height:auto;" />

### Step 2
After adding the threshold stone, you can also modify the parameters of the threshold stone in **3D mode**, and click **"Applied to the whole house threshold stone"** to apply the set threshold parameters to the whole house.

![](https://qhstaticssl.kujiale.com/image/png/1747034751779/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# For Users

All users

# Entrance

Enter the 3D design interface and select **"Advanced Tool - Material & Component"**.

![](https://qhstaticssl.kujiale.com/image/png/1746771661772/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Set as Threshold Stone

1. Enter the material & component interface, select the threshold to which you want to add a threshold stone.

2. In the **area settings** panel on the right, turn on **"Set as Threshold"**.

![](https://qhstaticssl.kujiale.com/image/png/1747031369838/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Modify Threshold Stone Parameters

1. After setting the threshold stone, you can change the **material** of the threshold, and set the **thickness** and **grinding edge size** of the threshold stone.

![](https://qhstaticssl.kujiale.com/image/png/1747034183542/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After adding the threshold stone, you can also modify the parameters of the threshold stone in **3D mode**, and click **"Applied to the whole house threshold stone"** to apply the set threshold parameters to the whole house.

![](https://qhstaticssl.kujiale.com/image/png/1747034751779/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
