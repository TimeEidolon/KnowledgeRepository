# For Users

All users

# Entrance

1. Select **Kitchen＆Bath** / **Closet**to enter the customization interface.

2. Select the panel in the **Parts**, drag the panel into the cabinet, and select the **Panel Array** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1740110730834/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1740110787980/F898A55816BF89E2BC3526BE832C3B51.gif)

# **Panel Array**

1. Select the **internal zone** where the panel needs to be arrayed.

* **How to adjust the size of the internal zone, the tutorial can be found: [Use Design Internal Zone to layout the Inner Space of the Cabinet](https://www.coohom.com/en_US/helpcenter/article/use-design-internal-zone-to-layout-the-inner-space-of-the-cabinet?kjl_source=hc)**

2. Set the **number and direction** of the array panels.

3. Click **OK**.

![](https://qhstaticssl.kujiale.com/image/png/1740120626433/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1740120721916/236AC02FE1D7E90B4598218B44843848.gif)

* The effect of the panel array is as follows:
