By using the Drawing View Tool, you can perform actions like designing floor plans and customizing while observing the automatically generated styles and annotation effects in real time. Additionally, you can switch between different view nodes for thorough inspection and effects evaluation, allowing for design adjustments tailored to drawing outputs at any time. This article explains how to use the Drawing View Tool.
