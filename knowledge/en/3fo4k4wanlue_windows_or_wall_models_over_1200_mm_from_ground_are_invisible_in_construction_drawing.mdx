Construction drawings are a critical component of the design and building process, but one common issue that arises is the disappearance of windows or wall models positioned over **1200** mm from the ground. This article explains the cause of this issue, how to address it, and what improvements are on the horizon.

### Default Section Cutting Height at 1200 MM

The primary reason for the missing models is the default section cutting height setting, which is automatically set to 1200 mm in the software. This means that any windows or wall models located above this height will not be visible in the construction drawings. While this default setting is designed to standardize the display, it can lead to confusion if not properly understood or adjusted.

### Solution One: Adjust The Height Manually

For users, the solution lies in manually adjusting the section cutting height. To ensure that windows or wall models over 1200 mm are displayed, users must set the cutting height to below 1200 mm. While this is a simple fix, it requires careful attention to avoid overlooking critical design elements in the drawings.

### Solution Two:  Use Model Detection Function To Force Models To Show

Some users may not want to change the height based on the real situation. In this case, they can use the model detection function to detect the invisible models and then force them to show.
