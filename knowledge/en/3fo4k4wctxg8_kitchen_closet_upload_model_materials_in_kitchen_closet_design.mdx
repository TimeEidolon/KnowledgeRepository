# Upload Model Materials

# 

## Method One

1. Enter the workspace, select **Models**, and click **Upload Textures.**

![](https://qhstaticssl.kujiale.com/image/png/1778469753653/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. There are two ways to create new materials: generate **Real-time Material** or **Upload Textures** by yourself in the required format.

![](https://qhstaticssl.kujiale.com/image/png/1778469891150/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Method Two

1. Enter the Kitchen & Closet design, select **"Kitchen/Closet".**

![](https://qhstaticssl.kujiale.com/image/png/1753328035579/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Click the top right corner to switch the library icon, and switch to **Personal.**

![](https://qhstaticssl.kujiale.com/image/png/1753328093421/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After entering **Kitchen/Closet**, there are two entrances to **upload model materials**, as detailed in the table below:

|  |  |
| --- | --- |
| **Entrance One** | **Entrance Two** |
| Open **Personal Library of Kitchen/Closet**, select **"Upload"**, and click **Material Upload.** | Click on the custom cabinet, select the **"Style - Body Material - Uploaded"** in the parameter panel, and click **Material Upload.** |
|  |  |

3. Click **Upload**, and it will jump to the workspace page. Then follow the operation process of method one to upload the materials.

![](https://qhstaticssl.kujiale.com/image/png/1778469753653/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# View Uploaded Model Materials

After the material is successfully uploaded, you can view it in the **Personal Library.**

|  |  |
| --- | --- |
| **Viewing Method One** | **Viewing Method Two** |
| Open **Personal Library of Kitchen/Closet**, select **"Upload".** | Click on the custom cabinet, select the **"Style - Body Material - Uploaded"** in the parameter panel. |
|  |  |

# Replace Uploaded Model Materials

**Method One:** Open **Personal Library of Kitchen/Closet**, select **"Upload",** and drag the uploaded model material out to replace it with a custom cabinet with one click.

![](https://qhstaticssl.kujiale.com/image/gif/1753329182950/D850BF6641DCE70596502177B1F2F20C.gif)

**Method Two:** Click on the custom cabinet, select the **"Style - Body Material - Uploaded"** in the right parameter panel, and click on the uploaded model material.

![](https://qhstaticssl.kujiale.com/image/gif/1753329384521/BA711E4BDFE4161339DF4E428C748575.gif)
