# Problem

The glass rendering effect is not normal.

# Reason

The material UV is incorrect.

![image2022-3-29_14-28-50.png](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762134319960/D8761F4F86F7EF4504305B1C00C6DA77.png)

# Solution

1. You can use the 【Material editor】 to substitute a 【Real-time Material】 from the library and render it to test.
2. If it is still incorrect, please check if there are any issues with the model (such as overlapping faces, normals, UVs, etc.)
