# Solution

## Indoor Scene

1. First, check all the lights in the scene (including sunlight) to see if there are too many or if they are too bright. You can try deleting them all and then rendering again.

2. If deleting them results in a normal exposure, it means the lights are causing the overexposure, and you can adjust the Brightness settings.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762136066436/1DEC84D75788641443FAEE07A481D4C3.png)

3. If it's still overexposed (abnormal) after deleting all the lights, adjust the camera position to render from multiple angles and check for missing walls, ceilings, or floors (hard installations).

4. If no Finishes are missing, you can try enabling Specular reflection rendering to see if it normalizes.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762136066388/0D535192DA1267857807512F8AD559EF.png)

## Outdoor or Near Outdoor Scenes (Balcony, Window Edge)

1. Adjust the 【Environment】under 【Ambient LightBrightness】. If overexposed, reduce the value; if too dark, increase the value.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762136066438/6D268EB7E9438124EA15ABBE3FCA42DF.png)
