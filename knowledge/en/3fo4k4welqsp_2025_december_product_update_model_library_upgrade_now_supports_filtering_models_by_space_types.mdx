# 1. Functions for All Users

## Key Update Details

**[Model Materials] The model library has been upgraded to support filtering models by space types, [Click here for details and learn how to use](https://www.coohom.com/en_US/helpcenter/model-materials-guide-for-using-the-model-library?kjl_source=hc)**

The new model library has clear section divisions, including personal resource management, premium model recommendations, and precise category navigation, making every model search and use easy and efficient.

![](https://qhstaticssl.kujiale.com/image/gif/1768199875077/57332ED30DA651CA1B698BBF9BD047E4.gif)

**[Drawing Design] New drawing view tool added to construction drawings, [Click here for details and learn how to use](https://www.coohom.com/en_US/helpcenter/construction-drawings-guide-for-operating-the-drawing-view-tool?kjl_source=hc)**

With the drawing view tool, you can design the layout while observing the automatically generated styles and annotations in real-time, support switching between different view nodes for comprehensive inspection and effect evaluation, and conveniently adjust the design for output at any time.

![](https://qhstaticssl.kujiale.com/image/gif/1768200417580/517F866F25FEDDEA41AED698B1C9D351.gif)

## Function Update Summary

**[Floor Plan Design]**

* **New:** A new staircase tool has been added to floor plan design, [Click here for details](https://www.coohom.com/en_US/helpcenter/floor-plan-guide-for-using-staircase-models)

**[Kitchen & Closet Design]**

* **New:** A new drawing view tool has been added to Kitchen & Closet,[Click here for details](https://www.coohom.com/en_US/helpcenter/kitchen-closet-drawing-view-tool-operation-guide)
* **New:** The roof tool now supports placing windows and window openings,[Click here for details](https://www.coohom.com/en_US/helpcenter/kitchen-closet-draw-a-roof)

**[Model Materials]**

* **New:** The model library has been upgraded to support filtering models by space types, [Click here for details](https://www.coohom.com/en_US/helpcenter/model-materials-guide-for-using-the-model-library?kjl_source=hc)

**[Drawing Design]**

* **New:** New drawing view tool added to construction drawings, [Click here for details](https://www.coohom.com/en_US/helpcenter/construction-drawings-guide-for-operating-the-drawing-view-tool)
* **New:** Construction drawings now support switching interior design drawing templates, [Click here for details](https://www.coohom.com/en_US/helpcenter/construction-drawings-switch-interior-design-drawing-templates)

**[Project Management]**

* **New:** A new share to community entrance has been added to the upper right corner of the projects section, [Click here for details](https://www.coohom.com/en_US/helpcenter/2-steps-to-share-your-projects-to-social-community)
