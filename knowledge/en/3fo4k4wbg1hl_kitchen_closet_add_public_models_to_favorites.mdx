# Add Public Models to Favorites

## Method One

1. Select the **decors** and select the model in the **public**.

2. Select the model in the **public**, click the upper left corner ☆, One-click **add to favorites**.  

|  |  |
| --- | --- |
|  |  |

## Method Two

Click on the dragged model in the public, click **Add to Favorites** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1744786894297/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Favorites Category

1. Click **Add to Favorites** in the floating toolbar to **create folder** and customize the name to classify favorites.

![](https://qhstaticssl.kujiale.com/image/png/1745226200229/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Select the **personal - favorites**, select models under the folder and drag them out to use.

![](https://qhstaticssl.kujiale.com/image/png/1744787016728/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1744787329944/E50996A4694246B8614B8F72C8E73437.gif)
