**Function Introduction**

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
**Function Introduction** The tool provides a 2D scene with the option to display lower floor plan references. The user can toggle the reference display in the show/hide settings. **Tool Version** Cloud Design 5.0 **Entrance Introduction** Bottom bar-Show Hidden-Floor reference

### Step 2
**Detailed Functionality** By default, when there is a floor plan on the lower floor, the lower floor reference will be displayed.
