The measure tool is equivalent to a ruler. In 3D and 2D modes, the measure tool can measure the distance between any two points in the space.

# Use the Measure Tool to Measure Distance

After entering the Material & Component interface, select **"Measure"** in the upper toolbar, select the start and end points, and the distance between the two points will be displayed after measurement.

![](https://qhstaticssl.kujiale.com/image/png/1747040824027/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
