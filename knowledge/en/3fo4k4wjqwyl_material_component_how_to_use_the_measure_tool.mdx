The measure tool is equivalent to a ruler. In 3D and 2D modes, the measure tool can measure the distance between any two points in the space.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=nnlDLYgkhE0)

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Entrance Enter the 3D design interface and select **"Advanced Tool - Material & Component"**.

### Step 2
Use the Measure Tool to Measure Distance After entering the Material & Component interface, select **"Measure"** in the upper toolbar, select the start and end points, and the distance between the two points will be displayed after measurement.

# Use the Measure Tool to Measure Distance

After entering the Material & Component interface, select **"Measure"** in the upper toolbar, select the start and end points, and the distance between the two points will be displayed after measurement.

![](https://qhstaticssl.kujiale.com/image/png/1747040824027/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1747039459582/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
