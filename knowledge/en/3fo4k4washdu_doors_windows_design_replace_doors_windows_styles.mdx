# For Users

All users

# Method One: Model Replacement

1. Select **Doors＆Windows** to enter the customization interface.

2. Switch to **3D mode**, select new Doors＆Windows to drag the doors and windows to the position that needs to be replaced.

![](https://qhstaticssl.kujiale.com/image/gif/1740713985736/8A02B83C5E2CB7E0800E4232630ED912.gif)

3. Display **Successful Replacement**, Doors＆Windows types are replaced.

![](https://qhstaticssl.kujiale.com/image/png/1740713348635/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* **Note: When replacing, you need to switch to 3D mode. If you replace the model in 2D mode, a prompt will appear: Some models are overlapped. Operation failed.**

![](https://qhstaticssl.kujiale.com/image/png/1740713460097/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Method Two: **Style**

1. Click on the doors and windows whose DoorsStyle needs to be changed.

2. Select **Style** in the Parameters panel to replace the material style of doors, moldings, panels and handles.

![](https://qhstaticssl.kujiale.com/image/png/1740713707223/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1740713813548/293FCD3BF93F364153E59C6A4B8AB39E.gif)
