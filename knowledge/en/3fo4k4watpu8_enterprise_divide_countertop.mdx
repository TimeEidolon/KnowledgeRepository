After generating the countertop, the countertop can be divided and the price can be calculated separately for each divided countertop, reducing the countertop quotation error.


## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
After entering the table split interface, you can freely add the table split line. Click the split line to delete it.

![](https://qhstaticssl.kujiale.com/image/gif/1739871501499/69D5E643B9DC77EC3EF69DB79910B137.gif)

### Step 2
You can drag points or lines to move, adjust the angle of split. * Differences in adsorption inner and outer contours: | | | | --- | --- | | **Snaps the inner contour** | **Adsorbs the outer contour** | | Excluding the depth of the front edge, only the contour area of the table panel. | Including the depth of the front edge, the contour area of the table panel and the depth of the front edge. | | | |

![](https://qhstaticssl.kujiale.com/image/gif/1739872363021/093770E1E54591E3FAB4DC177E6ED7C3.gif)

### Step 3
After the split line is added, click **Divided**. In the table split interface, select the **back splash splitting** in the top toolbar to add points to the back splash to complete the splitting of the back splash.

**Entrance Two:**Enter the **manual edit** interface, select **Divide** in the top toolbar.

# Divide Countertop

1. After entering the table split interface, you can freely add the table split line. Click the split line to delete it.

![](https://qhstaticssl.kujiale.com/image/gif/1739871501499/69D5E643B9DC77EC3EF69DB79910B137.gif)

2. You can drag points or lines to move, adjust the angle of split.

![](https://qhstaticssl.kujiale.com/image/gif/1739872363021/093770E1E54591E3FAB4DC177E6ED7C3.gif)

* Differences in adsorption inner and outer contours:

|  |  |
| --- | --- |
| **Snaps the inner contour** | **Adsorbs the outer contour** |
| Excluding the depth of the front edge, only the contour area of the table panel. | Including the depth of the front edge, the contour area of the table panel and the depth of the front edge. |
|  |  |

3. After the split line is added, click **Divided**.

![](https://qhstaticssl.kujiale.com/image/png/1739873169136/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# **Back Splash Splitting**

In the table split interface, select the **back splash splitting** in the top toolbar to add points to the back splash to complete the splitting of the back splash.

![](https://qhstaticssl.kujiale.com/image/png/1739873622879/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
