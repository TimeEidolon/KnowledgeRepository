### Step 1
**Specific Operation** Clients can set a default currency in 'Price Manager' of 'Product Info'. When the 'Display currencies based on the language version of your design tool' mode is closed, a default currency could be set. You can change it from the default condition 'Not set yet' to any other currencies.
