### **Introduction**

Provide the ability to set the default currency of the product.

### **Aim User**

All Coohom Enterprise Users

### **Specific Operation**

Clients can set a default currency in 'Price Manager' of 'Product Info'. When the 'Display currencies based on the language version of your design tool' mode is closed, a default currency could be set. You can change it from the default condition 'Not set yet' to any other currencies.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848771446/BBF3688E6BB3DF5B5FF3F7E87EAE65C6.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848772375/1738AE64C445E18EFD51C1C3761F392C.png)

When the currency is changed, the default currency of products and custom products are both affected.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848773226/53952A13C9B145146AE63CE580560B3A.png)

For products, the price can only be set in default currency.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848774422/46D475EFAB176ED3896F7CDCC659F366.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848775626/A1DA711E24F1F07349DA522748AA5A6A.png)

For custom products, prices in different currencies still can be set. The default currency displayed will be the same as you set.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733848776857/28D9C5A6E3B2D267CCCF67620F694A2F.png)
