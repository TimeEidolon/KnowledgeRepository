# Problem

When uploading 3Dmax models, the following error message appears: "Model size too large," and the model needs to be reduced.

# Solution

To solve this problem, you need to measure the scene by pressing ctrl+A and then perform the following operations to check if the unit setting is in millimeters.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762313260028/6EAEF2F423704269121670F369BB2F58.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762313260004/D525AE80222107952A080B768F7ECA77.png)

There are a few reasons for this model upload prompt

1. The max model itself exceeds 30 meters, and the model size needs to be adjusted to below 30 meters.

2. In max, there are other models besides the main model

Right-click the scene to unhide all, thaw all, and then select the unnecessary items and click confirm to delete them.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762313260043/86444CE6E661C698E50A6BA713E3901D.png)

3. The model does not exceed 30 meters, and there are no other models. It is possible that other data exists in max. You can select the model, save the selected model separately, and then reopen it.

The model is actually intended to exceed 30 meters because in coohom it can be magnified twofold, allowing it to be expanded beyond 30 meters when used after import.
