**Global style** supports one-click replacement of materials, styles, hardware, etc. for all custom models in the scene, and also supports continuous replacement operations.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=Nx4DMIP2v\_w)

# For Users

All users

# Entrance

1、Select **Kitchen＆Bath** / **Closet /** **Doors＆Windows** to enter the customization interface.

2、Select the **Global edit-****Global style** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1737632349196/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# **Global Style**

Select the cabinet, and select the material and hardware to be replaced in the parameter panel on the right to complete the global style.

![](https://qhstaticssl.kujiale.com/image/png/1737632466124/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
