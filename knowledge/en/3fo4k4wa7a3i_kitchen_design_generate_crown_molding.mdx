Use the crown molding generation tool to complete the layout of the cabinet crown molding, and you can select multiple cabinets to design the cabinet crown molding efficiently.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=HakkXf9arZw)

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Select the cabinet, the tool will automatically identify the cabinets that need to generate countertops. You can also **press and hold shift to select multiple cabinets** for generation. If you need to identify all cabinets, you can click **Auto-select**.

![](https://qhstaticssl.kujiale.com/image/png/1739859201530/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Step 2
In the basic parameters on the right panel, select add **crown molding style and material**. After adjusting the specific parameters, click **Generate**. * **Generate Wrap:** Fit for cabinets with crown moldings. Adds wraps for the cabinets with crown moldings.

# Entrance

Select **Kitchen＆Bath** to enter the customization interface.

**Entrance One:** Select the **Generate-Crown molding** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739859338212/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance Two:** Select the cabinet, and select **Generate-Crown molding** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739859370024/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Generate Crown Molding

1. Select the cabinet, the tool will automatically identify the cabinets that need to generate countertops. You can also **press and hold shift to select multiple cabinets** for generation. If you need to identify all cabinets, you can click **Auto-select**.

![](https://qhstaticssl.kujiale.com/image/png/1739859201530/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the basic parameters on the right panel, select add **crown molding style and material**. After adjusting the specific parameters, click **Generate**.

![](https://qhstaticssl.kujiale.com/image/png/1739859426142/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* **Generate Wrap:** Fit for cabinets with crown moldings. Adds wraps for the cabinets with crown moldings.

![](https://qhstaticssl.kujiale.com/image/png/1739859501611/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. If you need to freely adjust the range of the crown molding generation, click **manual edit** in the top bar to enter the manual mode.

![](https://qhstaticssl.kujiale.com/image/png/1739859685864/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. In the manual interface, use the drawing tools on the top bar in 2D to edit edge and add rectangle.

![](https://qhstaticssl.kujiale.com/image/png/1739859738048/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1739859945973/4100BE281D06A44162E59C95C89C289B.gif)

5. You can manually edit points and lines, or enter values ​​for precise adjustments. After the adjustments are completed, click **Generate**.

![](https://qhstaticssl.kujiale.com/image/gif/1739860058142/180CA1D20EFF84770D7933BA7CAB5173.gif)

6. After generating the crown molding, click **Finish** to exit the generate crown molding interface.

![](https://qhstaticssl.kujiale.com/image/png/1739860188682/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* The generate crown molding effect is as follows：

![](https://qhstaticssl.kujiale.com/image/png/1739860285875/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
