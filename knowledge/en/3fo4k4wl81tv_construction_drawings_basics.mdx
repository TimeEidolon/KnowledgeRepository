# Entrance

1. Enter the **workspace** and select the **Create New Project** in the upper right corner.

![](https://qhstaticssl.kujiale.com/image/png/1779343695360/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the 3D design, select **Drawings - Construction Drawings**.

![](https://qhstaticssl.kujiale.com/image/png/1750748257782/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Edit Views

Edit views is a view of the design plan to be converted into drawings through BIM technology, and supports modification of these view contents, such as adding labels and other functions.

* **Specific operations related to editing views can be seen in the tutorial documentation: [Edit Views Introduction](https://www.coohom.com/en_US/helpcenter/construction-drawings-edit-views-introduction?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1750839421542/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1750839089917/316CE9B2ACE21216F2F4B77E12707295.gif)

# Drawings Layout

Drawings layout is tool that organize the edited views in the drawing frame to output a complete set of construction drawings.

* **Specific operations related to layout drawings can be seen in the tutorial documentation: [Drawings Layout Introduction](https://www.coohom.com/en_US/helpcenter/construction-drawings-drawings-layout-introduction?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1766139886318/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
