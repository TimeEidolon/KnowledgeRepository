* What is Protruding Value?

### What is Protruding Value?

The Protruding Value in the View Function at the bottom of the toolbar helps you to see the value in 2D Ceiling View Mode.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733845127027/3C0E038FDFAC08E73EA59C83F0DA00C0.png)

When you create a ceiling extrusion in Advanced Tool- Construction, and go back to 2D Ceiling Mode, you will see the Protruding value you set before.
