# Solution

Materials need to be reattached

1. Double-click to enter the group and select the face to which you want to attach material

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762408933772/DCC2E97C656435823AF9F8ACF461AABD.png)

2. Select the material in the toolbar

In the material panel, select the material to be used and assign it to the selected model, then save and upload.

Note: Make sure to enter the group to select the face and attach the material
