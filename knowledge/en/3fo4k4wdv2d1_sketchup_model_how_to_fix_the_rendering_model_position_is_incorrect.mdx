# Solution

1. First, check the model units. The upload model requires the unit to be millimeters. Size errors will result in incorrect upload size.

“Window” - “Model Info” - “Unit Settings”, set the unit to mm.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762313779775/940DF5A2F079E67020F45AE98DECC519.png)

2. If the units are correct, use the tape measure tool to measure the model dimensions and check the size of the model. Select the tape measure tool from the toolbar.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762313779770/F0892333DBAA1FD1FEA99D47BEC5A34D.png)

3. Select the model, choose the "Scale Tool", and click on any endpoint of the model to scale proportionally. The lower left corner shows the scaling ratio, where you can enter the desired scaling data directly.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762313779785/CC510B4711A6605C67D8D39943D3990D.png)
