# Wave Line Floor Design

1. Select the floor and click to enter **Floor Editor.**

![](https://qhstatic-us-s3.coohom.com/image/png/1775811361755/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the floor editor page, click **Tiles** on the left sidebar, then choose appropriate materials and place them on the floor.

![](https://qhstatic-us-s3.coohom.com/image/png/1775811475250/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Select the floor, and adjust the tile size via the left sidebar.

![](https://qhstatic-us-s3.coohom.com/image/png/1775811660793/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. Once adjusted, click to enter the **Edit Paving** page.

![](https://qhstatic-us-s3.coohom.com/image/png/1775811784588/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. Select the floor and click on **The Whole Wave is Wired** in the floating window. In the popup **Select Direction** and then click **Confirm**.

![](https://qhstatic-us-s3.coohom.com/image/png/1775811931393/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstatic-us-s3.coohom.com/image/png/1775811969551/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* You can also click on the **Common Accessories** in the left sidebar, select the **Wavy Line** at the top, and then select the desired style to place on the floor.

![](https://qhstatic-us-s3.coohom.com/image/png/1776076590701/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

6. Select the wavy line and change the type to **No Corner Tile** via the left sidebar.

![](https://qhstatic-us-s3.coohom.com/image/png/1775815118546/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

7. In the **Parameters** section, you can adjust wavy line width via **Side Brick Width**, or directly set it through the floating window's**Width** input box.

![](https://qhstatic-us-s3.coohom.com/image/png/1775815220099/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

8. Adjust the settings, select the wavy line, add **The Whole Wave is Wired** again.

![](https://qhstatic-us-s3.coohom.com/image/png/1775815329881/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

9. Select the newly added wavy line, adjust the type to **No Corner Tile** in the left sidebar, and modify **Side Brick Width** and **Edge Brick Offset** in the parameters section as needed.

![](https://qhstatic-us-s3.coohom.com/image/gif/1775816088403/BDD03DBD7B143CFD44CBE3156FF860F8.gif)

# Waterjet Pattern Floor Design

1. Select the **Model Catalog** from the left sidebar, click **Waterjet Pattern**, then choose the desired style and place it in the target location.

![](https://qhstatic-us-s3.coohom.com/image/png/1775816175542/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstatic-us-s3.coohom.com/image/png/1775816638294/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Select the waterjet pattern, adjust the **Dimensions** via the left panel, then center the waterjet pattern. Once adjusted, click **Exit with Save** in the upper-right corner.

![](https://qhstatic-us-s3.coohom.com/image/png/1775816764582/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstatic-us-s3.coohom.com/image/png/1775816872043/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. On the edit paving page，click **Exit with Save** once again. After rendering, you will get a satisfactory effect image.

![](https://qhstatic-us-s3.coohom.com/image/png/1775817084274/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* The final effect is as follows:

![](https://qhstatic-us-s3.coohom.com/image/jpeg/1775821071103/52880A28C153DDD7D5BDAA9779561643.jpg)
