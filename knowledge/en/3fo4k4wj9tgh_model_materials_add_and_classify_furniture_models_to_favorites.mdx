Public models and materials support add to favorites, and can be classified and add to custom named folders and can be used at any time.

⭐Before reading the article, it's recommended to watch the video tutorial first to get started quickly~

🎥 [Video URL](https://www.youtube.com/watch?v=QACh9CPSMoM)

# For Users

All users

# Add Furniture Models to Favorites

## Method One

1. Click **Models**, select a model, and move the cursor over it to hover.

2. Click the upper left corner ☆, One-click **add to favorites**.  

|  |  |
| --- | --- |
|  |  |

## Method Two

Click on the dragged model in the public, click **Add to Favorites** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1768357998900/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Favorites Category

1. Click **Select folder** in the floating toolbar to **create folder** and customize the name to classify favorites.

![](https://qhstaticssl.kujiale.com/image/png/1768358095802/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1768358189140/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Click the **Models - Favorites**, select models under the folder and drag them out to use.

![](https://qhstaticssl.kujiale.com/image/png/1768358291073/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1768358585951/7B31791A3ECC4B5A3D4149B81C41D3C4.gif)
