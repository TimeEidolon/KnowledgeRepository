Some new users who take interest in Coohom may want to try Coohom tool in a fast way. Hence, they will prefer to adopt the ready-made template. This article will illustrate several easy ways of using template function to have a quick start from Coohom. 

## Target User: All individual users


Step 1: Add **New 3D Project** to start

![](https://qhstaticssl.kujiale.com/image/png/1744196012692/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Step 2: Select from **Floor plan templates** or **Idea templates** to continue

![](https://qhstaticssl.kujiale.com/image/png/1744196102965/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

By **Floor plan template**, it will take users more effort to work out the whole layout and then do the interior but it allows more space for creative design ideas.

![](https://qhstaticssl.kujiale.com/image/png/1744196607084/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

By **Idea template**, the complete floor plan will show so that users can use to render directly to see effect or just do the minimum adjustment before rendering.

![](https://qhstaticssl.kujiale.com/image/png/1744196536643/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Hence, each user can take full advantage of the template function to meet their needs.
