Want to make the kitchen both practical and good-looking? This guide will easily unlock the core features of kitchen＆closet design! Whether you want to renovate an old kitchen or plan a new space, you only need three steps to "create a new project → select a design template → position the kitchen" to start the design. Users can combine base cabinets, wall cabinets, countertop cabinets and add them to the design plan. They can adjust the size, material and style of the cabinet model, and generate countertop and crown molding with one click~

# For Users

**All users**.If some enterprise accounts do not have the kitchen＆closet design entrance, please contact the CSC to apply.

# Entrance

1. Enter the **workspace** and select the **Kitchen＆Closet Design**.

![](https://qhstaticssl.kujiale.com/image/png/1779701668973/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the **Start Design** interface, you can select the **Design Templates** and click **Apply**.

![](https://qhstaticssl.kujiale.com/image/png/1745224754585/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Enter the kitchen＆closet design and choose the **kitchen**.

![](https://qhstaticssl.kujiale.com/image/png/1742548516879/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Filter Materials

In the **Public**, you can choose Kitchen ＆ Bath cabinets, wall cabinets, tall cabinets and other furniture materials for modular design and combination.

![](https://qhstaticssl.kujiale.com/image/png/1742548850190/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Set up Cabinets and Sizing

After selecting the cabinet model in the **Public** and dragging it out, **the parameter panel** can adjust the model parameters.

![](https://qhstaticssl.kujiale.com/image/png/1742549714804/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1742550127545/6BA9B9561C99035BC52EA3D13FBF4338.gif)

# Click Generate

Select **Generate** in the toolbar to generate cabinet countertop, crown molding, toekick molding, sink, stove, handle, furniture leg, etc. with one click.

![](https://qhstaticssl.kujiale.com/image/png/1742550362926/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Appearance Modification

## Style

1. Click on the cabinet whose BodyStyle needs to be changed.

2. Select **Style** in the Parameters panel to replace the material style of the cabinet, door and handles.

![](https://qhstaticssl.kujiale.com/image/png/1742550476578/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Global Style

1. In the top toolbar, switch to **Select cabinet**.

2. Select **Global style** in **Global edit**. 

3. Click on the cabinet, and select the material, hardware, etc. that need to be replaced in the parameter panel.

![](https://qhstaticssl.kujiale.com/image/png/1742550659223/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

After completing the above basic operations, the next step is to imaging, drawing and quotating~

# **Notice**

 In the kitchen & closet design, **the material /wall /ceiling /floor editor is not supported**. If you need to use this function, you can choose **Create Blank Project** tools to create a project.

![](https://qhstaticssl.kujiale.com/image/png/1779703833180/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
