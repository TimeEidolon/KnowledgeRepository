### Definition

Lofting Contour is the basic molding that forms any structural shape.

I. Enter Custom Furniture / Kitchen&Bath - Create Asset

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844117839/C564546B3C39A542D19F5CF582A8CF50.png)

II. Select the molding contours upload entrance

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844122678/721EBA526E14758A7970090C3449AC33.png)

III. Click on "Add File" and select the contour you want to add, or drag and drop the contour file directly into it;

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844125350/41D43A5DEF72E844F53855660C250196.png)

IV. Select the modeling contour as the molding type

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844127800/EA8910F227329EA80072BAF737BD345E.png)

V. Adjust the contour height to an integer, select the category to save, and click [Complete Upload]

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844129746/24C4178DBB0E83D1DCFCD98C46399567.png)

VI. View the uploaded contours under the corresponding category

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844132407/9CE88D73A4C75940ABA39A9468F66DF6.png)

VII. Application of unfolding contour moldings

In the editor's component library, select "unfolding", choose the desired shape, modify the path accordingly, and create the required parametric model.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844135177/04AFD3ABC09A22654BAE330F00427E69.png)
