This is the feature where **enterprise users** can add watermark; product tag with e-commerce link; QR code.

**a. How To Add Brand Watermark:**   

​  

1. Click on **Enter Tool** .  

​
![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850267532/59280FFDCC1C7EF79ADD360904B6E502.png)

<br />


2. User need to render design work before go to **Gallery**.
![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850268960/C1B1A072E4D41C1B48340020878077FD.png)

<br />


3. Picture need to be selected. Next, click on **Share** button and then **Picture settings**

button.  

​
![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850270867/31E6F788945428DED915E74BBCC16E80.png)

<br />


4. Turn on **Brand watermark** button. Click re-upload if watermark logo is not uploaded yet.
![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850272723/6DF94DEA8BF3F839F6C1D76FF5F55209.png)

<br />


​**b. How to add product tag together with details or URL links:**

If click on **Text**, fill up the product name and details as shown below:
![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850275398/97496748672899AE50B7AC10AF47A755.png)

If click on **Link**, fill up the hotspot name and link address as shown below:
![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850277205/09B0513D574977A4BCCA15C40FD1713C.png)

**c. How to set expiration for URL and set own QR code:**

Tick or untick "Link settings" when necessary. Set the expiration time.
![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850281173/7847EA4411616169B82FB6DF511B3A44.png)

If brand watermark is uploaded before, just tick the box for **Inline brand logo.**
![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850283236/0BB5D845E4CC9C6C22E69A22F0E476A1.png)
