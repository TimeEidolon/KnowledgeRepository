The **Resource Manager** displays as the form of a list, showing the soft decoration in the project. It can help users find models quickly and accurately, know how many models are in the current project, and bulk edit them.

After opening the project, move the mouse to the lower left corner of the page, there is the **Resource Manager**.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733849972586/D7DFDDA07DFF121D14A53ED78F04C3B9.png)

It supports to choose whether to aggregate by room names.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733849974967/5AE8C85D938A7B5A04988FFA01766252.png)

It also supports filtering hidden models and locked models.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733849976038/6F80188EBDE2310810D795A3303BEE14.png)

Move the mouse to the name of the model, you can see where the model locates in the project, and can also lock/hide the model.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733849978730/ADCD5962ADC5DFFAF531781A54D79E9B.png)

When clicking the model name in resource manager, it is allowed to edit the model directly.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733849979856/6BA5266632B7FE39769365A668E0FEA9.png)

Models with the same name will be lined up in order. This helps users to see the quantity of the models used in the project/room. And Hide/Lock is allowed to edit at the same time by holding on *Ctrl/Cmd* to multi-select the names of the models. (*Shift* is used to multi-select models in the project directly.)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733849980998/A67E2008F93880C3EA3D7C950AC466CB.png)
