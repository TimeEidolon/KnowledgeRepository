In the room tool, the roof function can help you quickly draw common flat roofs, shed roofs, and gable roofs. This article will explain in detail how to draw a roof.

# For Users

**All users, function is being gradually opened.** If some enterprise accounts do not have the kitchen＆closet design entrance, please contact the CSC to apply.

# Prerequisites

Historical projects do not have the roof function, only newly created kitchen＆closet projects have the roof function.

# Entrance

1. Enter the **workspace** and select the **Kitchen ＆ Closet Design**.

![](https://qhstaticssl.kujiale.com/image/png/1779701668973/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the kitchen＆closet design, click **Room**, and select the **Roof** tool in **Construction**.

![](https://qhstaticssl.kujiale.com/image/png/1761880680442/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Draw a Roof

## Select the Roof Area

Enter the roof tool and click the area where you need to generate the roof. There are two ways to select the roof area: identify and manual.

**Method One:** Use the identify area tool to select the roof area.

![](https://qhstaticssl.kujiale.com/image/png/1761880741777/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1761880846919/347BC0B0BA909867C61D0986BD5FEB01.gif)

**Method Two:** Use the drawing tool to manually draw the roof area.

![](https://qhstaticssl.kujiale.com/image/png/1761880766986/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1761880958567/81B6D704650E0F0C73E7F426B570FA82.gif)

## Select a Roof Type

There are three types of roofs to choose from: **Flat Roof, Shed Roof, and Gable Roof**. Select the roof type you need.

![](https://qhstaticssl.kujiale.com/image/png/1761880993577/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1761881148078/AC8640E6AE8611DCF71C957E1C6DE5FE.gif)

## Set Roof Parameters

Set the parameters of the roof such as **Overhang, Height, Thickness, and Slope** in the left parameter panel.

![](https://qhstaticssl.kujiale.com/image/png/1761881061008/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1761881100231/22F8A5A6B90B16A38FCB33DDEFA8A178.gif)

## Complet the Roof Design

After setting is complete, click **Finish** in the upper right corner, and it will save the roof shape you just designed.

![](https://qhstaticssl.kujiale.com/image/png/1761881369693/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Place windows on the roof

After the roof is drawn, **Windows** and **Window Openings** can be placed on the roof from a 3D , and the selected window can be resized in the parameter panel on the right.

![](https://qhstaticssl.kujiale.com/image/png/1766743667442/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1766743889413/2D37C2734C50A6837B78ACB4959B66FF.png)

# Elevation Drawings Display

For plans with roofs drawn, you need to manually adjust the view range in the drawing design to display the complete wall extending to the roof.

![KC图纸.gif](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761881447629/9597EF57BE8B9C91A2BD20938B3BCD91.gif)

# FAQ

**Q1:** How do I partition and draw in a complete room when the roof shapes in different areas are not the same?

**A1:** Select **"Room - Construction - Roof"**. After entering the roof editing page, click the drawing tool in the top bar to draw the roof outline as if you were drawing a wall. Then click on the roof outline separately, and set the roof type you need, as demonstrated below:

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761881567995/BBFBEC90EF8BF4F348CB0847BD26CBC6.gif)

**Q2:** What if the walls do not automatically extend to the roof?

**A2:** Click on the wall, select the **Connecting the Roof** tool in the floating toolbar, and then click on the roof that needs to be connected.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761881567770/2391BBDE05B81B6B027340717EAB1C02.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761881567976/B75862A8EF831F045677155E0CC1DD41.gif)

**Q3:** How can the roof be selected?

**A3:** In 2D mode, you need to switch to the ceiling view to select the roof.

![](https://qhstaticssl.kujiale.com/image/png/1761891390600/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* If there is overlap with the room outline, you need to use the overlap selector, as shown in the following figure:

![](https://qhstaticssl.kujiale.com/image/png/1761891448276/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**A3:** In 3D mode, hold down **Win - Ctrl / Mac - Command** on the keyboard, and select the roof on the canvas with the mouse. Once the roof is selected, you can adjust the parameter settings in the attribute bar.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761881567886/29C122B32E40BEFC11103DEF9D1777B6.png)
