# 1. Functions for All Users

## Key Update Details

**[Drawing Design] Continuous dimensions support batch editing in construction drawings, [Click here for details and learn how to use](https://www.coohom.com/en_US/helpcenter/construction-drawings-batch-edit-continuous-dimensions-operation-guide?kjl_source=hc)**

When marking dimensions, you can enable the batch edit function, allowing these continuous dimensions to be edited as a whole, supporting overall movement, style modification, deletion, and other functions.

🎥 [Video URL](https://www.youtube.com/watch?v=ZXSN6HleAEs)

**[Doors & Windows] Doors & Windows add curved window (customized), [Click here for details and learn how to use](https://www.coohom.com/en_US/helpcenter/doors-windows-design-design-custom-curved-windows?kjl_source=hc)**

In Doors & Windows, a new curved window material has been added, allowing you to place and adjust a suitable curved window quickly and accurately when encountering arc walls in your design.

# 2. Functions for Enterprise Users

For details on enabling and using enterprise functions, please consult your corresponding CSC.

## Key Update Details

**[Kitchen & Bath] Doors support configuration of handle style packages, [Click here for details and learn how to use](https://www.coohom.com/en_US/helpcenter/enterprise-handle-style-package-configuration-description?kjl_source=hc)**

In interior design, if the combination of doors and handles is too free, it often leads to inconsistent styles or installation conflicts, affecting design efficiency. Configuring handle style packages for doors can restrict the installation of doors and handles, so a certain door can only match specific handles, making design more efficient and precise.

![](https://qhstaticssl.kujiale.com/image/png/1758615151870/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**[Custom Quotation]** **New general quotation configuration (Available to some users), [Click here for details and learn how to use](https://www.coohom.com/en_US/helpcenter/enterprise-general-quotation-configuration-description?kjl_source=hc)**

General Quotation is a service for product pricing capabilities, offering more comprehensive and open pricing setup capabilities compared to the original custom quotation capabilities.

* Supports pricing for public catalog models and circulation models.
* Supports maintaining prices in a tabular format.
* Designed in a way that aligns better with how users price offline, reducing maintenance difficulty.

![](https://qhstaticssl.kujiale.com/image/png/1758707938439/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
