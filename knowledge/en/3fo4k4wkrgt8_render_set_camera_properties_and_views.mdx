In the render tools, setting camera properties and views is very important. Rendering different design spaces also requires adjusting different camera properties. This article will explain in detail the setting camera properties and views, with some tips.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=QSgPoO\_0P6c)

# For Users

All users

# Preconditions

The camera height is mainly determined by the room height. To adjust the maximum height of the camera, you need to adjust the room height first.

![](https://qhstaticssl.kujiale.com/image/png/1750330230011/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Entrance

Enter the 3D design and choose the **Render**.

![](https://qhstaticssl.kujiale.com/image/png/1750330801038/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Set Camera Properties

## Adjust the Camera Height

In the parameter panel on the right, you can enter the height value, or move the camera height up and down.

* **Tips: The camera height is best between 1100-1300. If you render a local object, it depends on the actual height of the object. The camera angle is 0 degrees, don't look down or up.**
* **Height parameter reference：**

  **Home decoration — Floor height 2700-2800mm, camera height 800-1200mm**

  **Villa — Floor height 3000-3500mm, camera height 1200-1400mm**

  **Engineering decoration and outdoors — Floor height 5000mm, camera height between 1400-1600mm**

![](https://qhstaticssl.kujiale.com/image/png/1744371866582/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1750330573107/2942E604F30A1CAA0C5DA150AD17E192.gif)

## Adjust the Camera FOV

Adjust the rendered view range by dragging FOV bar.

* **Tips: 60-80 degrees of FOV rendering is best in global space. A FOV of 0-30 degrees is suitable for local rendering. The FOV can be used in combination with camera clipping, making it easier to find good-looking angles.**

![](https://qhstaticssl.kujiale.com/image/png/1744372229661/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1750330521053/10D3E9869DAE64770510F9B37E02B51C.gif)

## Set the Camera Clipping

Adjust the rendered view range by dragging the clipping value bar.

![](https://qhstaticssl.kujiale.com/image/png/1744372882259/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1750330476657/204660CDD77AD8020FCD1AC1073AB04B.gif)

# Save /Select Camera Views

1. Adjust to the appropriate camera views, click **Save angle**, you can save the camera view and switch to different views for rendering.

![](https://qhstaticssl.kujiale.com/image/png/1750330606898/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. You can also select **smart views**, which can automatically switch to different views.

![](https://qhstaticssl.kujiale.com/image/png/1750330630385/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Select to enter the **top view** to set the **View and Az.**

![](https://qhstaticssl.kujiale.com/image/gif/1750330745252/059590B2744DA6D1655EBD2EEAD17974.gif)
