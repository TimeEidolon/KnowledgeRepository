Floor plan unit setting is the key to ensure accurate drawings and smooth operation. In Preferences, you can modify the floor plan units, and the units can be selected in mm(millimeter), m(meter), and ft(feet). In this article, we will explain in detail how to change the unit of measurement in preferences.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=CBqclv5Geks)

# For Users

All users

# Set Floor Plan Units

1. Enter the 3D design interface, move the mouse to the avatar, and select **preferences**.

![](https://qhstaticssl.kujiale.com/image/png/1745920856881/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the preferences interface, select **General - Unit**, units can be selected from **millimeters, meters, fractional inches, decimal inches, feet w/ fractional inches, feet w/ decimal inches**.

![](https://qhstaticssl.kujiale.com/image/png/1745921275301/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1745921438070/801C56CA0B546DDD5138F0AB73509376.gif)

3. After successful set up, for example, if it is set to imperial units, the units in the base parameter of the floor will change to imperial units.

![](https://qhstaticssl.kujiale.com/image/png/1756367579019/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. Click on the import 2D drawing, the scale ruler will show the set units, imperial input rules: support by "feet + inches" input.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1756368936610/2D37C2734C50A6837B78ACB4959B66FF.png)

![](https://qhstaticssl.kujiale.com/image/png/1756367874020/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
