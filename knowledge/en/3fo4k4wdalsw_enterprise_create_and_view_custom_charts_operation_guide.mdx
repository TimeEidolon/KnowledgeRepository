If you want to tailor data analysis to better suit your business needs, the custom chart feature will be an efficient assistant. You can visually compare different data metrics through custom charts, from selecting a data source, setting metrics and dimensions, to choosing the right chart type, optimizing display details, and then saving and integrating the chart into the dashboard, the entire process can create a unique data visualization tool for you. This article will explain in detail how to create and view custom charts.

# Create Custom Charts

## Create a New Chart

After entering the custom chart, click **Create Chart** to add a custom chart.

![](https://qhstaticssl.kujiale.com/image/png/1755845184819/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Select a Data Source

The first step in creating a chart is to select a **Data Source**, and only one specific data source can be selected at a time.

![](https://qhstaticssl.kujiale.com/image/png/1755845809634/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Set Data Metrics and Dimensions

First, without selecting a chart type, we will enter **"Data Settings"** to obtain the data we want to observe, requiring separate settings for **"Metrics"** and **"Dimension"**.

**1. Set Data Metrics**

Using Design Data as an example, first set the metrics. Suppose we first select "User Created Design Projects" to observe the metric of the number of design projects created by users under the current account.

![](https://qhstaticssl.kujiale.com/image/png/1755846500911/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**2. Set Data Dimensions**

When setting dimensions, in the example we choose "account id" and "account name" to separately view the number of projects created under each account ID and name.

![](https://qhstaticssl.kujiale.com/image/png/1755846589496/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Generate Custom Chart

After setting, click **"Query"** to obtain a table showing the number of projects created by all accounts in the current company, displayed separately by account ID and account name.

![](https://qhstaticssl.kujiale.com/image/png/1755847775161/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**1. Choose a Chart Type**

Choose the chart type to present the data in the most reasonable way.

Using "User Created Design Projects" as an example, we choose a horizontal bar chart as the display style, and by clicking **Query** again, a horizontal bar chart is obtained.

![](https://qhstaticssl.kujiale.com/image/png/1755847956508/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Furthermore, you can make the chart more complete by **adding criterias, sorting rules, setting maximum data points, and decimal places**.

![](https://qhstaticssl.kujiale.com/image/png/1755848178573/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**2. Choose Chart Display Form**

If needed, you can choose the chart display form, such as "whether to show numbers", to make the chart more readable.

![](https://qhstaticssl.kujiale.com/image/png/1755848311771/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**3. Name and Save the Chart**

Finally, you can click **Save** in the upper right corner to **name and save** the chart.

* **Note: After setting the chart form, it is necessary to click query again before it can be saved successfully.**

![](https://qhstaticssl.kujiale.com/image/png/1755848382018/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# View Custom Charts

There are two ways to view custom charts.

**Method One:** In **"Data Analytics - Custom Chart"**, you can view and edit historically saved charts.

![](https://qhstaticssl.kujiale.com/image/png/1755849147229/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Method Two:** In **"Data Analytics - Custom Dashboard"**, you can arrange the created charts on the dashboard by editing the layout for quick viewing.

![](https://qhstaticssl.kujiale.com/image/png/1755849302961/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

You can obtain the dashboard you just created from the custom charts. Of course, Coohom also presets some conventional dashboards for quick access.

![](https://qhstaticssl.kujiale.com/image/png/1755849387759/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
