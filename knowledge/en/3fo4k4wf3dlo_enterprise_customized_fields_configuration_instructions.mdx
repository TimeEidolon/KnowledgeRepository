Enterprise users can add personalized information fields for products based on their business needs through the customized fields configuration feature. The field fill type supports two forms: text and link. When link is selected, after configuration, the link click effect can be presented on the front-end tool page.

# For Users

Enterprise users

# Notes

The customized fields configuration feature is only supported for **Furniture**, **Textures**, **Paving**, and **Molding/Wall  Panels** **Products** in the **Enterprise Catalog**. It is not available for **Furniture Sets**, **Custom Modeling**, etc. **Customized Products** do not support this feature at all.

# Configure Customized Fields Types

1. In the Enterprise Workspace, click **Product Info** under the **Enterprise Catalog** on the left sidebar, select **Info Management**, and then click **Create New Field**.

![](https://qhstaticssl.kujiale.com/image/png/1776754845806/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the pop-up panel, set the **Field Name, Application Scenarios** (after checking, the configured field information can be viewed in the corresponding scenario), and **Field Number**.

![](https://qhstaticssl.kujiale.com/image/png/1776758863708/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Select **Link** as the fill type, and then fill in the **Link Text** (Text displayed when the link is shown on the front end). After completing the input, click **Create**.

![](https://qhstaticssl.kujiale.com/image/png/1776758954964/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* Note: The fill type also supports **Text**.

![](https://qhstaticssl.kujiale.com/image/png/1776758900601/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. After the setup is complete, select **Products** under the **Enterprise Catalog**, click **Furniture**, and then click on the desired material to enter the product detail page.

![](https://qhstaticssl.kujiale.com/image/png/1776762337262/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. On the product detail page, click **Edit**, then find the new field type you configured and set it. After setting it, click **Save** at the top.

![](https://qhstaticssl.kujiale.com/image/png/1776758229707/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1776758287993/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1776758311127/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

6. Once configured, enter the design tool. In the corresponding scenario, you can see the field displaying the link click effect.

![](https://qhstaticssl.kujiale.com/image/png/1776757439310/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* If the field type is set to text, the field will display text effects in the corresponding scenario.

![](https://qhstaticssl.kujiale.com/image/png/1776758518518/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
