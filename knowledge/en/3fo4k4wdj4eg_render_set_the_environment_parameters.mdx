In the render tool, you can adjust the rotate, brightness, ambient light brightness, and tone of the environment. This article will explain in detail how to set the environment parameters.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=8U80M7WklmI)

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Enter interior design, select **Render** in the toolbar on the left.

### Step 2
Hover the mouse over the selected environment to display the parameter bar icon, where you can set parameters such as environment rotate, brightness, ambient light brightness, and tone.

Move the mouse to the question mark icon next to the parameter bar to see the effects of adjusting that parameter.

# Set Environment Parameters

1. Select the desired environment scene, move the mouse to frequently used exterior scenes to reveal the **Pin** icon, allowing you to set it to the top for easy access.

![](https://qhstaticssl.kujiale.com/image/png/1761294909291/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1761294855639/37B50E299C61D71BC45AF5F04FEFB41F.gif)

2. Hover the mouse over the selected environment to display the parameter bar icon, where you can set parameters such as environment rotate, brightness, ambient light brightness, and tone.

![](https://qhstaticssl.kujiale.com/image/png/1761294966501/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Move the mouse to the question mark icon next to the parameter bar to see the effects of adjusting that parameter.

![](https://qhstaticssl.kujiale.com/image/gif/1761295262404/12884D6C945851D265E3C738C35B4FEA.gif)
