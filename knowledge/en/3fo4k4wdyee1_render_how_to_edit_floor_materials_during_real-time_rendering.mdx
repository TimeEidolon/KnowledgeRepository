# Reason

When the Finishes is selected as a whole (such as the floor), it actually selects multiple different materials in bulk. You can only replace materials but cannot edit them, so it is impossible to define which one of these materials you want to edit.

# Solution

1. Directly select non-multi-surface materials during the selection phase.

2. If a similar texture effect is necessary, manually copy the single-surface material.
