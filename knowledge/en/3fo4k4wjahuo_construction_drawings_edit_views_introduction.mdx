Edit views is a view of the design plan to be converted into drawings through BIM technology, and supports modification of these view contents, such as adding labels and other functions. This article introduces the functions and how to use the edit view tool.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=oXOnM3jorhs)
