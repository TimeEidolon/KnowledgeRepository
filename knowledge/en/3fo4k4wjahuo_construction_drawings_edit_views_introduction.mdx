Edit views is a view of the design plan to be converted into drawings through BIM technology, and supports modification of these view contents, such as adding labels and other functions. This article introduces the functions and how to use the edit view tool.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=oXOnM3jorhs)

# For Users

All users

# Entrance

Enter the 3D design, select **Drawings - Construction Drawings**, and click **Edit Views**.

![](https://qhstaticssl.kujiale.com/image/png/1750748257782/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1750753982318/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# View List

1. The views in the list will be generated based on what is covered in the project. The default includes: renovation sheets.

![](https://qhstaticssl.kujiale.com/image/png/1750754072049/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Click the **expand all /collapse all** button in the upper right corner, and the user can expand or collapse the view list with one click.

![](https://qhstaticssl.kujiale.com/image/gif/1750754138525/BA134C69B7E8A2E0A15B33A27EB46CB7.gif)

# Real-time Linkage of Views and Projects

![](https://qhstaticssl.kujiale.com/image/gif/1750754238800/16FC3881538658421E5B059F717CAF0F.gif)

# Add View Annotations

## Floor Plan Views

In floor plan view, it supports **auto - annotate, dimension annotate, text annotate and clear annotate.**

**1.** **Auto - Annotate**

**3. Text Annotate**

Text annotate can **customize editing text information**, click to generate **product information, room information** and **custom furniture information**.

## Elevation Views

The elevation view requires the elevation marks to be placed in the floor plan to regenerate the elevation view.

**1. Place Elevation Marks**

In the top toolbar, select **generate views - elevation**.

![](https://qhstaticssl.kujiale.com/image/png/1750755746356/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
