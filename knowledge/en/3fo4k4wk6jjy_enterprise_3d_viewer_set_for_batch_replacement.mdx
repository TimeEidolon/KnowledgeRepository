If you are an enterprise user, you can now replace multiple models of the same type in a 3D Viewer Set with just one click. In other words, you can change the same models as a whole group to another batch instantly without the need to rearrange the positions of the same models individually.

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
In the left-side navigation pane, click 3D Viewer Set. In the page that appears, create a new set.

### Step 5
