1. Enter **the Merchant Platform → Kitchen&Bath / Custom Furniture → Create → Molding Contours**

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Enter **the Merchant Platform → Kitchen&Bath / Custom Furniture → Create → Molding Contours**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907823478/44981513237401A6F2CC8A234A285374.png)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907823478/44981513237401A6F2CC8A234A285374.png" alt="Step 1" style="max-width:100%;height:auto;" />

### Step 2
Drag file in the box, or click to upload.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907824220/A95A04321BE1CF2992583422F7715838.png)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907824220/A95A04321BE1CF2992583422F7715838.png" alt="Step 2" style="max-width:100%;height:auto;" />

### Step 3
Modify the name, and select **Arrary Crown Molding** as Category, and **Crown Molding** as Profile Type

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907825758/1C3E36FDF5DD36667FB43A93AE5E5F92.png)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907825758/1C3E36FDF5DD36667FB43A93AE5E5F92.png" alt="Step 3" style="max-width:100%;height:auto;" />

### Step 4
Fill an integer in Contour Height. Extension Parameters are optional. Click "Complete" when you finish.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907828009/521CF6B30FD2410D5C3AE178C810C6FD.png)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907828009/521CF6B30FD2410D5C3AE178C810C6FD.png" alt="Step 4" style="max-width:100%;height:auto;" />

### Step 5
Click Edit Properties on the Edit Information page

### Step 6
Click to modify the preview image, select the prepared preview image and click save;

### Step 7
Click save, the model upload is complete

### Step 8
View the uploaded profile under the corresponding category


![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907823478/44981513237401A6F2CC8A234A285374.png)

2. Drag file in the box, or click to upload.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907824220/A95A04321BE1CF2992583422F7715838.png)

3. Modify the name, and select **Arrary Crown Molding** as Category, and **Crown Molding** as Profile Type
