# Problem 1

Through glass rendering, the model on the other side is deformed or misplaced

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762134541251/8241CF39580F155402F1F1D3CC414414.png)

# Solution 1

Replace the glass material, you can upload a glass material with a lower refractive index to use (e.g., refractive index of 0 or 1)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762134541276/2C34C469CC3233F71F1B45ACDE429D8A.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762134541188/14FC70E8E6FB470271E2EBEC8474F82C.png)

# Problem 2

Glass position renders as spherical, mirror

![](https://qhstaticssl.kujiale.com/image/png/1670326218720/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png?x-oss-process=image/resize,m_mfit,w_800/format,webp)

# Solution 2

Replace with public library's plastic PVC material or upload your own glass material with a refractive index of 0. If it is a self-modeled uploaded model, remodel it with double-sided glass and re-upload

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762134541196/A00F17A987570BAA814C7E8A9F33B048.png)

![](https://qhstaticssl.kujiale.com/image/png/1670326318304/1223B8D9DF8F6B67847407BD17015ED1.png?x-oss-process=image/resize,m_mfit,w_800/format,webp)

# Problem 3

After glass rendering, the transparency is low

![](https://qhstaticssl.kujiale.com/image/png/1670326251855/6CBF0E66193C3B8ADCCE4D0CD43CC2BE.png?x-oss-process=image/resize,m_mfit,w_800/format,webp)

# Solution 3

Replace the material, you can use ordinary glass or plastic PVC

![](https://qhstaticssl.kujiale.com/image/png/1670326356791/6D35163E40B25A9FAF3BD24BF63C6D9C.png?x-oss-process=image/resize,m_mfit,w_800/format,webp)

# Problem 4

Model UV chaos

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762134541295/148040F0984B2CE96ABD4BF0ADC4A94D.png)

# Solution 4

If it is a public library model, it is recommended to replace it with another model; if it is a self-uploaded model, it is recommended to set up UV in max and then re-upload it for use
