# Problem

The handle colors in the image appear different on light and dark cabinet panels.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1599831443047/D323A3D37306C9D5828CD183A04C77AC.jpg?x-oss-process=image/resize,m_mfit,w_880/format,webp)

# Reason

By using PS to sample the colors of the handles in two positions, it's found that the colors are actually similar. This is a classic case of visual illusion.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762133909404/5DB1D9BE4A032F799835C93F3D254C8E.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762133909365/CAD8E57CA7C4AFC5E28997465476491D.png)

This is indeed a classic visual illusion case. In the image below, are the colors of A and B the same? The answer is yes.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1599831668250/B7433092DD83FA454B6799266D9448E8.png?x-oss-process=image/resize,m_mfit,w_343/format,webp)

# Solution

It seems that if this is normal, there is no need for much solution; just keep the effect as it is.
