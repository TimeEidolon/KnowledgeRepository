In the room tools, there are three structures: columns, beams, and flues. After adding beams, the beam structure can be split and merge, and the size and position can be adjusted with one click, which is convenient for designers to restore the floor plan structure.

# For Users

**All users**.If some enterprise accounts do not have the kitchen＆closet design entrance, please contact the CSC to apply.

# Entrance

1. Enter the kitchen＆closet design and choose the **room**.

2. Select a **structure**, there are three structures: **columns, beams, and flues**.

![](https://qhstaticssl.kujiale.com/image/png/1741779870348/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Add Beams

After clicking on the **beam**, select the starting point and end point in the floor plan to draw the beam.

![](https://qhstaticssl.kujiale.com/image/gif/1741782408925/732FE26B6F3C547722778657E8B6BE8E.gif)

# Adjust the Beam Size

You can directly drag the adjustment points on the beam to modify the size, or you can modify the **location line, width and height** of the beam in the **basic** parameters in the right panel.

![](https://qhstaticssl.kujiale.com/image/png/1741782592887/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Split the Beam

Select the **beam** and select the **split** tool in the floating toolbar to split the beam into multiple segments. The **shortcuts** for the split tool are as follows:

* **Win System Shortcut：Ctrl+D**
* **mac System Shortcut：⌘+D**

![](https://qhstaticssl.kujiale.com/image/png/1741782829257/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1741782917826/16762334357EF3FF840990DA7AA33EC8.gif)

# Merge the Beam

Select the split beam and select the **merge** tool in the floating toolbar to connect the split beam into a section. The **shortcuts** for the merge tool are as follows:

* **Win System Shortcut：Ctrl+J**
* **mac System Shortcut：⌘+J**

![](https://qhstaticssl.kujiale.com/image/png/1741783699857/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1741783805072/F4C3FA0A1F0864DA193BAA4433D3F099.gif)

# Adjust the Beam Position

1. Switch to **3D mode**, press and hold **Ctrl/⌘** to select the beam, and the position of the beam in the right panel can be selected as **the ceiling and floor**.

![](https://qhstaticssl.kujiale.com/image/gif/1741784220783/FCB83ABF94FE50B9DEB0D4C9933A144A.gif)

2. You can set the value of the distance between the beam and the top surface.

![](https://qhstaticssl.kujiale.com/image/gif/1741784276783/F3F92E49131E80A65D5764BB32327DF2.gif)
