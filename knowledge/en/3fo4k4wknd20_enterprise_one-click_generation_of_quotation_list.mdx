Each enterprise has its independent pricing system, which can be complex due to factors such as craftsmanship. Calculating the price every time is a tedious and time-consuming process, and it is easy to make mistakes when applying complex formulas. When there are multiple orders, this process must be repeated, indirectly increasing the error rate. The custom quotation function of Coohom helps businesses solve these issues. All that is required is to set the system of pricing needed once and associate it with the corresponding product model, and the price is automatically generated.

After confirming the order, customers can generate a detailed list of items with a clear pricing and specifications breakdown. This saves the time and effort required to create an invoice or report, providing a seamless experience for both customers and enterprises.

# For Users

**Enterprise users** (the solution includes production capabilities). Only the main account can set a quotation list template. Sub-accounts can use the function of exporting quotation list. If you need custom configuration, you can contact CSC for consultation.

# Export the Quotation List

# 

1. After completing the project design in the design tool, click **Drawings & Schedules** in the upper right corner and select **Custom Furniture Quotes** to export the custom furniture quote of each tool.

![](https://qhstaticssl.kujiale.com/image/png/1749627991988/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After selecting the quote logic and package setting, click **Generate** quotation list.

![](https://qhstaticssl.kujiale.com/image/png/1749628533337/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

 3. After generating a quotation list, click **View**.

![](https://qhstaticssl.kujiale.com/image/png/1749628987269/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. Select the quotation list format you need and click **Save**.

![](https://qhstaticssl.kujiale.com/image/png/1749629094856/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
