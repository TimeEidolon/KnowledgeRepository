Use the toekick molding generation tool to quickly install toekick molding on cabinets.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=3KMfBUF-eQM)

# For Users

All users

# Entrance

Select **Kitchen＆Bath** to enter the customization interface.

**Entrance One:** Select the **Generate-Toekick molding** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739416218893/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance Two:** Select the cabinet, and select **Generate-Toekick molding** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739416132037/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Generate Toekick Molding

1. Select the cabinet, the tool will automatically identify the cabinets that need to generate countertops. You can also **press and hold shift to select multiple cabinets** for generation. If you need to identify all cabinets, you can click **Auto-select**.

![](https://qhstaticssl.kujiale.com/image/png/1754467656715/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the basic parameters on the right panel, select add **toekick molding style and material**. After adjusting the specific parameters, click **Generate**.

![](https://qhstaticssl.kujiale.com/image/png/1754468388869/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Different types of toekick moldings can be generated by **checking different parameter options at the same time** as required.

* **Generate Wrap:**Fit for cabinets with toekick moldings. Adds wraps for the cabinets with toekick moldings.
* **Generate only the front side of the cabinet:** After checking, it will only be generated on the front of the cabinet.
* **When you check the toekick molding is 0 above the ground, you can check the height of the cabinet from the ground to follow the height of the toekick molding at the same time.**

![](https://qhstaticssl.kujiale.com/image/png/1754468411859/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. If you need to freely adjust the range of the toekick molding generation, click **manual edit** in the top bar to enter the manual mode.

![](https://qhstaticssl.kujiale.com/image/png/1754468760202/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. In the manual interface, use the drawing tools on the top bar in 2D to edit edge and add rectangle.

![](https://qhstaticssl.kujiale.com/image/png/1739432236377/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1739432582348/B675990FB7F828179FF55D918725B8EB.gif)

5. You can manually edit points and lines, or enter values ​​for precise adjustments. After the adjustments are completed, click **Generate**.

![](https://qhstaticssl.kujiale.com/image/gif/1739433123606/AA970E1BA8719903C78E5A46B34DA5D7.gif)

6. After generating the toekick molding, click **Finish** to exit the generate toekick molding interface.

![](https://qhstaticssl.kujiale.com/image/png/1754468816662/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* The generate toekick molding effect is as follows：

![](https://qhstaticssl.kujiale.com/image/png/1739433536528/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
