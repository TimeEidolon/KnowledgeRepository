# For Users

All users

# Entrance

1. Select **Kitchen＆Bath** / **Closet /** **Doors＆Windows** to enter the customization interface.

2. Select the **Toolkit-Measure** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1738919761808/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# General Measurement

## **Select Model**

1. First select the measurement object, then select the distance object, and modify the distance to move object.

2. **Press Z** to reselect the measurement object, **Right-click** or **Esc key** to exit.

3. Part distance can be measured in **select part** mode.

![](https://qhstaticssl.kujiale.com/image/gif/1738920687574/BA3B525623D8C7EE076515306FF691FE.gif)

## **Select Point**

You can select any two points in the canvas to measure the distance, not limited to the model, and modify the distance to move object.

![](https://qhstaticssl.kujiale.com/image/gif/1738921128491/A47BFA9082DC8CC5D976C6BB2250A545.gif)

## **Select the Outline Point**

The custom tool has a proprietary function that can capture and recognize the panel outline of the customized model, measure the distance between the model and the model more accurately, and modify the distance to move model.

![](https://qhstaticssl.kujiale.com/image/gif/1738921746070/8BF1A36E1AAF3376546A69D27A604824.gif)

# Orthogonal Measurement

1. Enter the measure tool interface, **select the outline point**, and select **orthogonal distance**.

![](https://qhstaticssl.kujiale.com/image/png/1738922820095/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Click the two cabinets that need to measure the orthogonal distance to display the vertical distance between the cabinets.

![](https://qhstaticssl.kujiale.com/image/gif/1738922969095/D07B15077854C6A2518C3A58A2DFE28C.gif)

# Isolated Measurement

# 

1. Select **isolated measurements**, **press and hold Shift to multiple-select** the model or part you want to measure, and click below to **start measuring**.

![](https://qhstaticssl.kujiale.com/image/png/1738923496469/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the isolated measurement, and other unselected models will be locked and unselectable measurements. Only the isolated models or parts can be measured.

![](https://qhstaticssl.kujiale.com/image/gif/1738923850774/B4D1382C9B35CF97452B64A2C4734FA0.gif)
