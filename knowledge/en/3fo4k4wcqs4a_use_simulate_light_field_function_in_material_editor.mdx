In the material editor tool, simulate light field function can help you preview the model effects of different light field environments under the same set of material parameters, allowing you to more accurately grasp the material presentation. This article will explain in detail how to use the simulate light field function.


## Steps with screenshots

Each step includes plain MEDIA_* URLs (for AI tools) plus rendered images.

### Step 1
Enter the design tool, click a **finished model**, and select **Material Editor.**

# Simulate Light Field

The simulate light field provides four options: **Default, Daytime - soft light, Daytime - strong light, and Night**. Through these options, you can see the effect differences of the model with the same set of material parameters in different light field environments.

* **Note:** The simulated light field effect is only displayed in the model presentation on the **Material Editor** page (the part highlighted in the box below) and will not affect the model material effect in the rendering and 3D design pages.

![](https://qhstaticssl.kujiale.com/image/png/1753946194887/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
