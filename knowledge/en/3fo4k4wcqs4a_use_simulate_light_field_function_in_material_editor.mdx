In the material editor tool, simulate light field function can help you preview the model effects of different light field environments under the same set of material parameters, allowing you to more accurately grasp the material presentation. This article will explain in detail how to use the simulate light field function.


## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Enter the design tool, click a **finished model**, and select **Material Editor.**

![](https://qhstaticssl.kujiale.com/image/png/1753944597034/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Step 2
Enter the **Material Editor** page, and click **Simulate Light Field** in the upper toolbar. The simulate light field provides four options: **Default, Daytime - soft light, Daytime - strong light, and Night**. Through these options, you can see the effect differences of the model with the same set of material parameters in different light field environments. * **Note:** The simulated light field effect is only displayed in the model presentation on the **Material Editor** page (the part highlighted in the box below) and will not affect the model material effect in the rendering and 3D design pages. ⭐The demonstration of the simulated light field effect is as follows:

# Simulate Light Field
