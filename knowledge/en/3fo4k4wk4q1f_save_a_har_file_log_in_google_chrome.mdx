# Function Introduction

**Why Should I Save a HAR Log?**

In extreme cases where our Support team did not manage to reproduce an issue you reported, it is extremely helpful to provide us with your browser's network request and response log to help us get to the root cause of your problem.(video tutorial*at the bottom*)

**What is a HAR File/Log?**

HAR (HTTP Archive format) is a JSON-formatted archive file format that logs a web browser's interaction with a website. The log itself will contain all the requests and responses between the browser and the site allowing our support engineers to debug an issue by seeing exactly what you were experiencing.

* **Important! HAR files log all data requested and passed by your browser and this may include any sensitive information you have entered (such as: personal details, passwords, credit card numbers, etc.).**

# Save a HAR Log in Google Chrome

1. Prepare a clean and newly opened Google Chrome private browsing environment:

![](https://qhstaticssl.kujiale.com/image/png/1751882082603/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Open the DevTools panel by pressing **F12****(Windows)** or **Option + Command + I****(Mac)** in the current browser tab.

![](https://qhstaticssl.kujiale.com/image/png/1751882136936/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Enter the page URL (e.g., [www.coohom.com](http://www.coohom.com/)) in the address bar. Once the page has fully loaded, confirm that no new network activity is occurring in the **Network** tab of DevTools.

![](https://qhstaticssl.kujiale.com/image/png/1751882217575/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. Locate the **"Export HAR"**button in the DevTools Network tab (usually represented as "Export HAR with content"), click it, and select a folder to save the HAR file locally.

![](https://qhstaticssl.kujiale.com/image/png/1752634561389/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1751882416350/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. Locate the HAR file you just saved, and share it with Coohom support. We will pass it on to our testing and development team for further investigation.

![](https://qhstaticssl.kujiale.com/image/png/1751882508665/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
