For product models that want to be displayed on the Public Library , the name should accurately describe the product , and the category and preview image must also match with the actual product.

# For Users

All enterprise users

# Product Information

## 1.  Name

**(1)  You can choose from the following keywords:**

Brand, Style, Color, Material, Space, Type(MUST BE INCLUDED).

Don't change the order of this keywords.

**(2)  Not allowed keywords:**

Pure numbers, special symbols(\ / \_ > < \* % ^ ! ~), personal names, place names, company names, and contact information.

# Model Quality

Model displayed on the public library must be modeled correctly according to the Coohom modeling standards with clear material textures and correct structures.

## 1.  Model Material

**(1)  White model is not allowed:**

Models without any material are not allowed.

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674302/2163451E229A55FDBAAF5AB6D57184D7)

**(2)  Black surface is not allowed:**

Material turn black due to material loss, normal reversal or UVW error.

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674326/57BC1892EC4629222D3B39E84D2776CB)

**(3)  Texture error is not allowed :**

Models with broken texture or textures that do not match the actual effect.

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674657/F7E2D907227390132BA1865545ED3089)

**(4)  Using reasonable materials:**

Materials like leather or fabric should have clear textures , especially for black color material.

Effect reference:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674289/344C169690549DD5F235D7886A0B20C1)

**(5)  Unreasonable color matching is not allowed:**

Color matching should be good looking, not too bright or appear as material loss colors, like bright red, green or purple.

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674550/9BF5838CEEC9682B6C4B94D70C16E176)

**(6)  Texture size:**

It is recommended that textures file size used on the model within 1-10 M.

**(7)  Image format :**

The texture files used on the model should be .jpg, .png, or .jpeg format, with the RGB color mode.

## 2.  The model structure must be displayed clearly and accurately

**(1)  Interpenetration, overlapped faces, and broken surfaces are not allowed:**

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002675548/C66FD334B0F161AD6DC2A8184B01EAEE)

**(2)  Low precision models are not allowed:**

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674625/2A5B2955DF15A4ACD1E9902BA6CCDAF7)

**(3)  Non centered models are not allowed:**

There are unless objects around, causing the model not to be centered.

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674295/D866F9DECC8A4AB7037FED0154410C8C)

**(4)  Single-sided model is not allowed:**

Single-side faces can cause abnormal rendering effects, like fish eye distortion.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674587/099F856EAFCF65B1687B17341231E9F4)

Incorrect example: (the thickness is 0 )

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674719/762FEA27DCC50DCED7F5930AF9545246)

## 3. The model's appearance should be reasonable and following current trends and aesthetics

**(1)  Trends:**

Materials such as stone, metal, acrylic, etc., are more in line with current trends. Generally, Only models created within the last 3 years will be published to the public library.

**(2)  Proportion Coordination:**

Decorations should match the main furniture, and individual furniture within furniture sets should also maintain proportional coordination.

**(3)  High Quality Model:**

The model has a reasonable structure with seamless textures.

**(4)  Accurate Dimensions:**

The model should adhere to standard real-world sizes. The model’s dimensions should not exceed 30 meters (length, width, and height all should not exceed 30 meters). Special-sized models will be considered case-by-case.

**(1)  Opening the 3ds max source file and setting the Gamma value to 2.2.**

# Model Combination

## 1.  A single part of a model is not allowed

**(1)  For example, the left/right arm of a sofa or the corner piece.**

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674537/E6646CB987656C4C2AD0A46E0705C2C9)

**(2)  The accessories of cabinet/wardrobe models, like drawer frames, built-in baskets, back panels, are custom accessories for customized cabinets. Don’t publish them to the Design Tool public library. They can be uploaded to the Advanced Tool Custom model library.**

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674297/2245F8C7E94D80CA0C0383BD52D4CAA9)

## **2.  Model combinations can be published as appropriate**

**(1)  Inseparable combined furniture (like modular wardrobes) and furniture with accessories (like wardrobe with clothes) can be uploaded as a whole.**

Reference:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674274/D6ADD80CA3202C23A9EC090271A975AF)![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674516/642818FFDCAE32583192FE6F43D069CA)

**(2)  Furniture with a background wall needs to be separated. While TV cabinet set with necessary background can be uploaded as a whole.**

Reference:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674549/8BA64D3C0E281D7C4AA7E34A89ED4DFA)![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674249/915646BDF3C39CB58FEC06EAD8CB51C7)

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674596/CADFB8B3BF78C8693BAE9B8F40488AC2)

**(3)  Multiple individual models need to be separated and uploaded individually.**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674653/35FC79C5A73BC906D68D31A46FC84E8E)

**(4)  Multiple ceiling models for different spaces need to be separated and uploaded individually.**

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674564/E1EA3644AF96EF6DFF8F981C8519F30D)

**(5)  The furniture sets:**

5.1. The furniture sets model combination should be set at front view, and set as “Show in tool”.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674342/0EA740A2C526C39002E03BB0635A649E)

5.2. Only the following categories of furniture sets can be published: sofa combinations, TV cabinet combinations, dining table combinations, bedroom combinations, vanity table combinations, and desk combinations.

**(6)  Requirements for publishing the furniture collections:**

Models of the same series with different colors, directions, and materials can be added to a collection and be published together.

Reference:
