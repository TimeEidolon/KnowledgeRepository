# Model Quality

Model displayed on the public library must be modeled correctly according to the Coohom modeling standards with clear material textures and correct structures.

## 1.  Model Material

**(1)  White model is not allowed:**

Models without any material are not allowed.

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674302/2163451E229A55FDBAAF5AB6D57184D7)

**(2)  Black surface is not allowed:**

Material turn black due to material loss, normal reversal or UVW error.

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674326/57BC1892EC4629222D3B39E84D2776CB)

**(3)  Texture error is not allowed :**

Models with broken texture or textures that do not match the actual effect.

Incorrect example:

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674657/F7E2D907227390132BA1865545ED3089)

**(4)  Using reasonable materials:**

Materials like leather or fabric should have clear textures , especially for black color material.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1749002674289/344C169690549DD5F235D7886A0B20C1)

Color matching should be good looking, not too bright or appear as material loss colors, like bright red, green or purple.

It is recommended that textures file size used on the model within 1-10 M.

**(7)  Image format :**

The texture files used on the model should be .jpg, .png, or .jpeg format, with the RGB color mode.

## 3. The model's appearance should be reasonable and following current trends and aesthetics

**(1)  Trends:**

Materials such as stone, metal, acrylic, etc., are more in line with current trends. Generally, Only models created within the last 3 years will be published to the public library.

**(2)  Proportion Coordination:**

Decorations should match the main furniture, and individual furniture within furniture sets should also maintain proportional coordination.

**(3)  High Quality Model:**

The model has a reasonable structure with seamless textures.

**(4)  Accurate Dimensions:**

The model should adhere to standard real-world sizes. The model’s dimensions should not exceed 30 meters (length, width, and height all should not exceed 30 meters). Special-sized models will be considered case-by-case.
