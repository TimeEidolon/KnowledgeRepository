# Problem

Can a material with the same design be universally applied across the library by uploading it once?

# Solution

The texture can be uploaded to the basic material library, and after preprocessing, it can be published. At this stage, it can support use across multiple industry lines (model/material, kitchen& bath, Closet, Doors&Windows, Custom modeling). When edited or modified a second time, all libraries will be synchronized for adjustments.
