# Problem

After uploading the step/stp model, various random colors appear on the model.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762409782767/C813D3781D1E798A33AB48F92B0B2861.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762409782768/C950B9C9D0291E5B357B452DB3B3A03E.png)

# Reason

The step/stp can only be uploaded on the 2023 version of max, but 2023 and hair substitute are mutually exclusive.

# Solution

If this problem occurs, please provide the user's userid or email so that the developer can disable the user's hair substitute permission.
