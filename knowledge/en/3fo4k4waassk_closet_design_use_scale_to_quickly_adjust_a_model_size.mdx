The model can be fuzzy stretched in six directions, which is suitable for designers to quickly adjust the gap between cabinets, fuzzy stretch/reduce the size of the cabinet.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=zOQs56MC7K4)

# For Users

All users

# Scale

1、Switch to **Select cabinet**.

2、Select the cabinet, select **Scale** in theswitch control.

![](https://qhstaticssl.kujiale.com/image/png/1737029441149/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3、By dragging the direction arrow, the model can be extended, reduced or enlarged in the direction of the selected arrow.

4、When the mouse is stretched to the boundary of other models or walls, there will be an adsorption alignment effect.

![](https://qhstaticssl.kujiale.com/image/gif/1737029808586/BE392B1686CB6768370F96A339DFE625.gif)
