**Background**

When creating different models, it is necessary to repeatedly create custom Parameters. Now, you can store Parameters as standard Templates and easily call them whenever needed.

**I.Creation Location**

Custom Furniture —Create Asset—Parametric Model Library— New—

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850052812/D38EF2B2A1CA03C22706CA2908E29374.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850055145/23584930226753E3CF0F562ADAF830C9.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850057124/74D465C25964C0B1B89619AE5845D1DA.png)

**II.Creating method**

Right-click on the Parameter after creation.

A dialog box will appear, click on "Set as Template".

Saving the Template.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850060088/89B60739C99FC04BCC45B57BE881828E.png)

In the dialog box, select the appropriate folder to save the Template.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850061098/F9D4063049797C2789B95AB7DFF8CA13.png)

Successful Template Addition

The system will prompt that the Parameter Template has been successfully added.

**III.Importing Custom Parameter Templates**

Click on the "Import Parameter Template" button.

Select the folder where the Parameters are located.

Check the Parameters to be imported.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850062146/64FCC34FDF0D79B3FF47439E89ACF832.png)

Optionally modify the reference name.

Click "Import".

Successful Import.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850063136/9FE920AB889829A7842011D33B59745D.png)

The system will prompt that the import was successful.

**IV.Editing Custom Parameter Templates**

Location for editing:

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850064222/2503CB541BBACD5330A55976FBE9932E.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850065366/1028F5BD008F6855DB81F8CD063C1717.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850066178/D9891D4FC6C5C33E71673216F84B2311.png)

The folder has three functions:

Create a new group.

Rename the group.

Delete the group.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850067097/B2EA3FE40D7F1D704CDF1B31AFCDA65A.png)

Usage of the "Move" button: Move Parameters to another folder.

Select the desired folder.

The system will prompt that the move was successful.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850068745/A50CB5679D02C78CCF2A854567A3FFEE.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850071248/14894A258DECB53B9600F9A992DE81AF.png)

Usage of the "Delete" button: Delete the added custom Parameters.

Click "Delete".

Choose "Confirm" or "Cancel".

The system will prompt that the deletion was successful.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850073569/3430CD93D6403F250958EA5329FCADC7.png)

**V.Summary of Editing Custom Parameter Templates:**

1.Create a custom Parameter Template.

2.Import custom Parameters from the Template.

3.Edit the custom Parameter Template (move/delete).
