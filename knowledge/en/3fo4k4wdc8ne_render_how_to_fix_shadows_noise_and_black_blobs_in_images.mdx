# Reason 1

The overlapping of two surfaces of the model will result in the situations shown below.

1. In the first image, you can see that there is no object in the sky area, but a similar situation occurs. This is because there is a piece of transparent glass in the same position, and when rendering, the surfaces of the glass and the ceiling overlap, resulting in the situation shown below.
2. This situation often occurs in custom panels or cabinets because the model had overlapping surfaces during modeling, leading to shadows when rendering.
3. When the light source and the model overlap, especially when the ceiling height and the surface light source height overlap, rendering will result in the situation shown below, with the ceiling disappearing.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762136226675/C61A23282006CB221735C95917B5AD6E.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1762250106400/95698113CFD6B560B9927D05D92771D5.png)

# Solution 1

1. If it is the situation in Image One, simply delete the overlapping surfaces in the free construction. If this does not resolve the issue, try clearing the ceiling design.
2. Delete the overlapping surfaces in the model. If it is a custom cabinet model, try to find and delete the overlapping components in the resource list.
3. Adjust the height of the lower light source to prevent it from overlapping with the ceiling height.
