#### Overview

During the process of downloading construction drawings, various issues may arise such as inability to locate downloaded files, long waiting times, download failures, inability to open files, or missing content in downloaded drawings. This document provides solutions to some common download problems.

#### Target Users

All users

#### Solutions

**Q:** **What to do if the downloaded file cannot be found?**

A: If the construction drawing has been downloaded but cannot be found on the computer, it is likely that the file has been blocked by the browser or the file's save location cannot be found. Please refer to the document: [Unable to Find Downloaded Files](https://www.coohom.com/en_US/helpcenter/article/construction-drawing-common-issue-unable-to-find-downloaded-files?kjl_source=hc)

**Q: What to do if the download waiting time is too long?**

A: If the progress bar moves slowly or gets stuck during the download process, causing a long wait, it may be due to the complexity of the project (large area, many models, complex shapes, or numerous textures).

You can try the following methods:

1. For complex projects, it is recommended to click all the drawings to be downloaded before downloading and wait for them to load completely, which can help reduce the download time.
2. If the project is complex and there are too many drawings to download (more than 30), it is recommended to download them in batches.
3. For large area hardcover color drawings exceeding 200 square meters, it is recommended to download only one drawing at a time.
4. (For enterprise only) When downloading CAD files, click on the ①"Advanced Settings" in the download dialog box, and then select ②"No Cropping" under "Clipping Method," which can help reduce the download time.

![](https://qhstatic-us-s3.coohom.com/image/png/1747809658752/BF1269ADEE975E50479D5C2F0F7808AF.png)

If all the above methods have been tried and the download is still slow, it is recommended to wait patiently for about 10 minutes. If the download still hasn't been completed, you can try refreshing the page and downloading again.

**Q: What to do if the downloaded file cannot be opened?**

A: If you found the file but cannot open it, try the following methods:

1. If the file has a .zip extension, it is a ZIP compressed file that requires decompression software installed on your computer to extract all files.
2. If the file has a .dwg extension, it is a CAD file that requires a CAD viewer installed on your computer to open. If you do not have a CAD viewer, it is recommended to download in PDF or JPG format instead.
3. If the file has a .pdf extension, it is a PDF file that can be opened using a PDF reader or a browser.
4. If the file has a .jpg extension, it is an image file that can be opened using image preview software.
5. If double-clicking the downloaded .dwg file does not open it, right-click the file, click "Open With," and select another CAD viewer software.
6. If none of the above methods work, try refreshing the page and downloading again.

**Q: What to do if content is missing in the downloaded drawings?**

A: If you have opened the downloaded file and found missing content, you can try the following methods:

1. Ensure that the project design and drawings are complete.

* If you need elevation drawings, click ① "Edit View," then ② "Elevation Index," and check if elevation marks are already present in the view. If not, click ③ "Auto-place" or "Manu-place" to add elevation marks, and then click ④ "Generate" to generate elevation drawings.

  ![](https://qhstatic-us-s3.coohom.com/image/png/1747809684027/A1CF1DAA1164EAF780CB1334EF98FD17.png)
* For multi-level projects, switch to the target level in the ① “Current Floor” menu before entering the ② “Construction Drawings”. Only one level can be downloaded at a time.

  ![](https://qhstatic-us-s3.coohom.com/image/png/1747809733261/1C2511F56E3211DEEA81C3289F1BF8E9.png)

1. If images are missing in the CAD file (such as rendered images, details, axonometric views, or texture images), and a string of text appears instead, it may be because the images in the compressed file were not extracted. Extract the compressed file, and then open it to display the images.

![](https://qhstaticssl.kujiale.com/image/png/1747809733261/1C2511F56E3211DEEA81C3289F1BF8E9.png)

1. If the color images are still not displayed after extraction, check if the "Keep Images" switch is turned on in the Download dialog. If not, turn it on and download it again.

![](https://qhstatic-us-s3.coohom.com/image/png/1747809760127/32316B66FAF4D6C574A1277CB3177BEE.png)

1. Modifying design or switching legends during the download process may cause content loss, ensuring that no modifications are made during the download process.
2. Some missing in downloaded files may be due to unknown bugs. Click the corresponding drawing in Construction Drawings to check if it is normal.

* If the drawing is blank,  click ① "Layout Drawing" and select the desired drawing, then manually drag the required view from the ② "Insert" panel to the drawing.

![](https://qhstatic-us-s3.coohom.com/image/png/1747809784568/0261AE782301C69F77C1F65181C91D7B.png)

* If there is a model far away from the plan, it may cause the drawing to be blank. You can exit Construction Drawings, open ① "Resource Manager", check ② "Outside the room" to see if there is a model far away, delete the corresponding model, then enter Construction Drawings and drag the view into the canvas.

![](https://qhstatic-us-s3.coohom.com/image/png/1747809808135/3A032A897B1D3956FDDF116E35B8E475.png)

**Q: What to do if the download fails?**

A: If the software prompts a download failure, try the following methods:

1. If the download takes a long time and fails, it may be due to a timeout. Refer to section 2 of this document ("Long Download Waiting Time") for solutions.
2. If you are using a client, you may encounter problems with downloading. It is recommended to use the Google Chrome browser for downloading.
3. An unstable network connection may also cause failure. Try refreshing the page and downloading it again.
4. If the project was created a long time ago, try clicking "Settings," then "Synchronize System Settings" (for some enterprise users, it is "Update/Switch Drawings Templates") before downloading again.

![](https://qhstatic-us-s3.coohom.com/image/png/1747809843099/6B711DD98FFE4D987B99FB360ABB921C.png)

1. If multiple retries still fail, try copying a new project and downloading again.
2. If all the above methods fail, please contact online customer service or enterprise-specific operations.
