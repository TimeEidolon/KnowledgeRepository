# Problem

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Go to Tools--Rendering–Choose the latest lighting version (above 3.0). You can directly render by selecting a version above 3.0 through lighting version management, and the Reduce noise is enabled by default in the parameters, addressing the noise issue from the source.

### Step 2
If using an older version of indoor daytime/nighttime (below 3.0), click "Effect Settings" and check the "Reduce noise" feature. Initiate rendering, and the problem will be solved once rendering is complete.

Sometimes rendering images may have light spots, patches, grainy objects, or a frosted effect.

# Solution

Enable the Reduce noise 

1. Go to Tools--Rendering–Choose the latest lighting version (above 3.0). You can directly render by selecting a version above 3.0 through lighting version management, and the Reduce noise  is enabled by default in the parameters, addressing the noise issue from the source.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762135423588/0AAB006B80093E7E5B964C9140F10B37.png)

2. If using an older version of indoor daytime/nighttime (below 3.0), click "Effect Settings" and check the "Reduce noise" feature. Initiate rendering, and the problem will be solved once rendering is complete.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762135423591/C1A539A6475B2F4202895C9EE2DF7FE3.png)
