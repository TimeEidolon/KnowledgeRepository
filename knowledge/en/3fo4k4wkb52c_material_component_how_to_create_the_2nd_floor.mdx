### How to create the 2nd floor by Material&Component

Hey, many users have questioned how to draw the 2nd Floor. which we currently do not support, is estimated to launch in the future. As for now, we recommend you create the 2nd floor by using our Material&Component.

🎥 [Video URL](https://www.youtube.com/watch?v=nZuYADanxdU )

To complete the 2nd floor in this way, you may:

1. Draw the floor plan.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896418254/C33345B1E4DABA5F4C04D876FBF5E6AC.png)

2. Set the first level height to the height of the whole house.（ For example 6000mm）

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896419545/B9309E01C40C36F9574039F2DC312BBB.png)

3. Click Material&Component in the left panel.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896421132/92D9DBBB2BB899C13C373CE130314A09.png)

4. Draw the modeling face of the first level.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896422775/CB28A91D5E227E5F2EE8C5324C2535A0.png)

5. Duplicate the face to the position of the target second level.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896424305/C6D822F61D7F0332828EF5A0E77AF6EE.png)

6. Use the Draw/Pull tool to change the thickness of the floor

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896425783/E8FF0DB0B916619311A4E892F1F92AFD.png)

7. Now you get two floors of the house.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896426843/8F28B3FF8334184004879A3E43FBC878.png)

8. Go back to DIY, add furniture on the first floor, and adjust the height above the floor to make it located on the second floor.
