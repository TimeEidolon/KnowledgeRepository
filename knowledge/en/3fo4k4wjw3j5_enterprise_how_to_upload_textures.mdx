### Ⅰ.Upload Maps to Custom Products Catalog

**Upload Steps**

1. Go to Enterprise Catalog→Custom Products→ Product Catalog → Custom Furniture → Create Assset → Materials

![](https://qhstaticssl.kujiale.com/image/png/1747897096301/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1747898403485/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1747898432968/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Edit Material Settings → Upload Map → Enter Dimensions→Creat→Folder Settings

![](https://qhstaticssl.kujiale.com/image/png/1747897564302/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1747897643018/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Tips:

1. The file size should not exceed 5M.

2. Higher resolution is better but should not exceed 5,000 pixels.

3. Can upload up to 100 material maps at a time.

4. Supported file formats: PNG, JPG, and JPEG.

### Ⅱ.Upload Maps to Finished Texture Library

**Upload Steps**

1. Select Enterprise Catalog→Products→Textures→New Material

![](https://qhstaticssl.kujiale.com/image/png/1747899121841/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Drop files or upload models from your computer.

3. Click **Submit** to confirm.

![](https://qhstaticssl.kujiale.com/image/png/1747899391882/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Tips:

1. Textures will be uploaded toNew Product.

2. Upload up to 20 textures at a time.

3. Each file should not exceed 5MB. JPG and PNG formats are allowed.

4. The length and width of image should not exceed 8,000 pixels.
