Customized shortcuts can switch templates or edit settings according to user's operating habits. When designing a plan, you can use shortcuts to quickly call out the design tools to improve designer work efficiency.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=f7Wv24i7GFU)

# For Users

All users

# Entrance

Enter the custom tool, drag the mouse to the expansion bar on the avatar to select the **shortcuts and mouse**.

![](https://qhstaticssl.kujiale.com/image/png/1739003335842/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Shortcuts Panel Settings

1. Enter the shortcuts panel and select the corresponding tool in the upper left.

![](https://qhstaticssl.kujiale.com/image/png/1739004543101/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In order to continue users' operating habits of using CAD and SU shortcuts, there are three templates to choose from: **Default template, CAD template,** and **SU template**.

![](https://qhstaticssl.kujiale.com/image/png/1739004683408/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. You can select shortcuts with different functions to view on the left. There is keywords search function above to find the shortcuts to be found more quickly and conveniently.

![](https://qhstaticssl.kujiale.com/image/png/1739004892313/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. You can edit and modify the shortcuts by yourself. If there is a duplicate with the existing shortcuts, the corresponding prompt will appear to avoid setting duplicate shortcuts. Click to **Save** after the settings are completed.

* After modifying and editing, you need to restore the default shortcuts. Click the lower left corner to **restore the default shortcut**.

![](https://qhstaticssl.kujiale.com/image/png/1739005558005/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
