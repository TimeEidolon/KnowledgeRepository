# Method 1: Category Search

## Search by Rooms

Search for the desired model by navigating through first-, second-, and third-level categoriesbased on the space where the model is used.

For example: To find a single bed model, go to [Bedroom] → [Beds] → [Single beds].

![](https://qhstaticssl.kujiale.com/image/png/1768288395695/E1CB00AA4AD348452FB911ECA31B7A87.png)

## Search by Model Categories

If you want to search for models based on their classification, click the button on the right side of the Catalog to switch the classification mode, and then navigate through the three-level category hierarchy to find the model.

For example: To find a single bed model, go to [Furniture] → [Bed] → [Single Beds].

![](https://qhstaticssl.kujiale.com/image/png/1768292031956/C236B54237C459AA5A25A6E391899D77.png)

## Search for Commercial Models

Models intended for commercial spaces are categorized under Commercial and should be searched for there.

For example: To find a library bookshelf model, go to [Commercial Design] → [Library] → [Storage furniture].

![](https://qhstaticssl.kujiale.com/image/png/1768289449954/5BEEF1CF9EC49E3CBAF8FAE70C8C9ECA.png)

# Method 2: Keyword Search

If you don't want to browse through categories step by step, you can directly enter keywords in the search bar at the top of the model library to find the type of model you need.

For example: Simply enter the keyword **[sofa]** in the search bar.

![](https://qhstaticssl.kujiale.com/image/png/1768288806444/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Method 3: Tag Search

 If you already have a specific model type in mind, you can filter the results using tags such as texture, form, or brand.

![](https://qhstaticssl.kujiale.com/image/png/1768356515975/48DE0C3350341B41E14AE6571BBD64CE.png)

# Method 4: Browse & Explore

In addition to category search, the model interface offers several alternative ways to find models.
