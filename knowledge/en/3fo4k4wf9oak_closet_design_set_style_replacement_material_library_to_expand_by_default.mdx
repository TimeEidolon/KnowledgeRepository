In the preferences, you can set the material library to be in an expanded state by default when replacing materials and styles, eliminating the need to expand it manually, making it easier to quickly find desired materials or styles.

# For Users

All users

# Set the Style Replacement Material Library Directory to Expand by Default

1. In the customization tool, hover the mouse over the profile icon in the upper right corner, and click **Preferences.**

![](https://qhstaticssl.kujiale.com/image/png/1774923818214/82663AB520D14CF08A34B954E27E2CAF.png)

2. In the pop-up panel, click **Customize**, set the **Style Replacement Material Library Directory** to the **Expand** mode, and then click the **Close** button in the upper right corner.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1774938987335/C7E82B630764733B1C3B29B498E0E7E1.png)

3. After completing the setup, select a model, and when replacing materials or styles in the **Style** section of the parameter panel on the right, the material library directory will be in the **Expand** state.

![](https://qhstaticssl.kujiale.com/image/png/1774923849238/82EABF784593AB943AE0EBB7E8ADA674.png)

* Note: After setting it up in preferences, the material library directory will remain expanded by default when creating new projects. However, if you manual click **Collapse Categories Under Model Catalogs**, it will remain collapsed the next time you open a project.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1774938987330/6694343553A6E96E125777BE8BBF777B.png)

![](https://qhstaticssl.kujiale.com/image/png/1774923871404/6364E175E12410CD3CC5A40D0A9BF660.png)
