1. **Function Introduction**

As shown in the figure below, when designing, there is often a need to draw curved walls. So how to draw such a wall? Below is a small tip that can be used as a reference.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733883121488/D7ADB56538CDA5E2E4DCDE523B3D6A4E.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733883122078/AFFEFADBE7DE64A5461C5D6424277E96.png)

2. **Target users**

All users

3. **Operation Steps**

Step 1: Use the straight wall or rectangle wall tool to draw the wall first.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733883123432/B66DDA8FBBA6F746FD3043636BB0EB56.png)

Step 2: Select one of the wall segments and set the automatic connection to "No".

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733883125922/6F87451DC21C4E188E64A6BEA79C567A.png)

Step 3: Move the wall to separate the originally intersecting two walls and modify it to a staggered connection, as shown in the figure.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733883126524/6241E10DDE619541BCCFC679FCBA7A8B.png)

Step 4: Click the arc wall tool to draw the wall again, and modify the thickness of the wall to 239. Here, you need to subtract 1 from the original wall thickness, and the logic is that the radius cannot be equal to the wall thickness. For example: If the original wall thickness is 260, the wall thickness needs to be set to 259 when drawing an arc wall, and so on.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733883127566/316F0E64FE9DCE0A1EEA6F54939BA40C.png)

Step 5: Click the arc wall tool and use it to connect the missing corner position. Finally, you will get the effect shown in the figure.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733883130134/2798C30C2DADF03CE266DA2B3546755E.png)
