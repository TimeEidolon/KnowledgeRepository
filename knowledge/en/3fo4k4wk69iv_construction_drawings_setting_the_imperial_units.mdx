# Setting the Imperial Units

1. Enter the 3D design, move the mouse to the avatar in the upper right corner, and select **Preferences**.

![](https://qhstaticssl.kujiale.com/image/png/1750822901763/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In Preferences, select **General-Unit** and modify the interface unit to "ft".

![](https://qhstaticssl.kujiale.com/image/png/1750822985798/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Select **Drawings - Construction Drawings,** enter the construction drawings interface.

![](https://qhstaticssl.kujiale.com/image/png/1750748257782/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. After the setup is completed, the units of room area, size and perimeter in the construction drawings will be changed to the imperial unit.

![](https://qhstaticssl.kujiale.com/image/png/1750823411016/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1750823500860/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. The view scale in the drawings layout is also displayed as an imperial unit.

![](https://qhstaticssl.kujiale.com/image/png/1750823589828/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
