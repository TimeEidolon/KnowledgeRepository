In the manufacturer catalog's search box, search for the relevant keywords of the required product, and you can search all products with one click. This article will explain in detail how to search for products in the manufacturer catalog.


## Steps with screenshots

Each step includes plain MEDIA_* URLs (for AI tools) plus rendered images.

Entrance Enter kitchen & closet design and select **Kitchen.**

# Switch Catalogs

Click the switch catalogs icon at the top right corner to switch between different catalogs in the drop-down menu.

# Search Products

In the search box, search for **the model number or brand** of the desired product, and you can search for the product you want with one click.

![](https://qhstaticssl.kujiale.com/image/png/1753431721037/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

|  |  |
| --- | --- |
| **Search for the model number of the product** | **Search for the brand of the product** |
|  |  |

In the search results, you can not only select the retrieved products but also click on the product's **manufacturer catalog** to switch to the **manufacturer catalog** to view products.
