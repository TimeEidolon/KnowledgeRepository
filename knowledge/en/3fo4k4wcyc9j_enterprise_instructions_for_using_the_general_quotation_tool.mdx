# For Users

Enterprise users with general quotation configuration enabled

# Prerequisites

* The enterprise has enabled general quotation and set up a quotation system (necessary)
* The enterprise has enabled cross-enterprise quotation application configuration, selecting public treasury or supply chain library.
* Enterprise enables compatible custom quotation 1.0.
* **Related configuration operations can be found in the tutorial document: [Click to view](https://www.coohom.com/en_US/helpcenter/enterprise-general-quotation-configuration-description?kjl_source=hc)**

# Generate Quotation List

1. Enter the customization tool, select **"Drawings & Schedules - Custom Furniture Quotes".**

![](https://qhstaticssl.kujiale.com/image/png/1759052563790/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1759052498561/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the quotation list export interface, you can select the export scope and quotation system

**a. Export Scope**

* **All Models:** Quoting all models in the current tool.
* **Current Product Library:** Quoting all models in the current library.
* **the Selected Model:** Quoting selected models separately.

**b. Quotation System**

* Enterprises with **general quotation configuration** enabled will use the general quotation configuration of the measurement list for quotation calculation, including finished and custom products.
* Enterprises with **cross-enterprise quotation application configuration** enabled can choose the quotation system of the public treasury or supply chain library.

![](https://qhstaticssl.kujiale.com/image/png/1759053042792/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* Enterprises that have enabled the **Compatible Custom Quotation 1.0** switch can use the custom quotation method for price calculation.

![](https://qhstaticssl.kujiale.com/image/png/1764746112794/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Note: For enterprises that have enabled the **Compatible Custom Quotation 1.0** switch, the quotation system in the quotation board can also be set to use the custom quotation method.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1764746583769/2D37C2734C50A6837B78ACB4959B66FF.png)

3. After clicking **Generate** quotation, you can view the list and download it to modify and send to customers.

![](https://qhstaticssl.kujiale.com/image/png/1759053141422/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
