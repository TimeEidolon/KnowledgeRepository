After the cabinet design is completed, you can use the custom smart decor to place decors in one click, reducing the time to select and place decors, and improving the design efficiency.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=aEgdJxfW-HI)

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Select **Kitchen＆Bath** / **Closet** to enter the customization interface.

### Step 2
Select the **AI-Custom Smart Decor** from the top toolbar.

### Step 1
Select the cabinet where decors are needed.

### Step 2
Select the corresponding custom tool from the **Smart Decors** panel.

### Step 3
Select the appropriate decors template and click **Apply**.

# For Users

All users

# Entrance

1. Select **Kitchen＆Bath** / **Closet** to enter the customization interface.

2. Select the **AI-Custom Smart Decor** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739259775741/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Custom Smart Decor

1. Select the cabinet where decors are needed.

2. Select the corresponding custom tool from the **Smart Decors** panel.

3. Select the appropriate decors template and click **Apply**.

![](https://qhstaticssl.kujiale.com/image/png/1739260614949/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1739260986132/95E8947B0BAF02091EC6FFEE311EBAAF.gif)
