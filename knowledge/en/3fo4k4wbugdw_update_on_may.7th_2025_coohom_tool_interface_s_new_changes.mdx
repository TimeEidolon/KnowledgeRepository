Dear Coohomers,

To improve user experiences, Coohom has recently changed the tool interfaces but the functions remain the same. This article will introduce the new changes of several frequently used tool buttons. 

**Target Users: All individual users**


**Part 1: Public library** has been renamed as **Models**. Users can check the **Public** models or their own uploaded models here. 

![](https://qhstaticssl.kujiale.com/image/png/1746702047971/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Part 2**: **Material&Component** button has been moved into **Advanced tool.**

![](https://qhstaticssl.kujiale.com/image/png/1746756166068/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Part 3: Render** button has been moved to the left navigation bar.

![](https://qhstaticssl.kujiale.com/image/png/1746701823660/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Part 4**: **Construction drawings/Floor plan drawings/Bill of materials** have been moved to the left navigation bar under **Drawings** button. The bill of materials is renamed as **Quotation**.

![](https://qhstaticssl.kujiale.com/image/png/1746702190455/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
