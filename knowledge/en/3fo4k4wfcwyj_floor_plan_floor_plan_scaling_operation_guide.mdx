# Floor Plan Scaling

**Method 1:** In the floor plan **Interior Area** input box on the right-side floor properties panel, directly input the desired floor plan area to automatically scale the floor plan size with one click.

![](https://qhstaticssl.kujiale.com/image/gif/1774258529686/8B92EAD58843012015ACE668614C773A.gif)

**Method 2:** When there is a base drawing in the design, click **Basemap Settings** in the lower-right corner of the bottom bar, then **Set Scale** as needed and check **Floorplan Follows Scaling** (this option is checked by default). Once selected, the floor plan will automatically scale accordingly.

![](https://qhstaticssl.kujiale.com/image/png/1774259025587/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1774258847713/CD8FC63C91BC49078853BAFD719BE561.gif)

# Notes

1. The result of floor plan scaling is not precisely scaled to the input value but provides an approximate scale value (Example: 60.4 → 124.5).

|  |  |
| --- | --- |
| Before Scaling | After Scaling |
|  |  |

2. Components such as **Closet**, **Doors/Windows**, and **Custom Modeling** in the **Advanccd Tool** are not supported for adjustment and will be cleared.
