# For Users

All users

# Entrance

On the floor plan tool page, hover the mouse over the **Staircase** on the left sidebar, then select the desired model.

![](https://qhstaticssl.kujiale.com/image/png/1768529900970/0A023EAFB607833B33BE84603170BB0C.png)

# Placing Staircases

Click on the desired model and drag it to the target location.

![](https://qhstatic-us-s3.coohom.com/image/gif/1766976974394/D0F6A7934A629582CB972C4F55C2A615.gif)

# Adjusting Model Parameters

1. Select the model and adjust its **Length, Width, Height** in the right sidebar property panel.

![](https://qhstaticssl.kujiale.com/image/png/1768529939730/62A177C04AB11A38EF1BB12CFCC69612.png)

* **Note:** When the input box is grayed out, parameters cannot be adjusted as shown in the image below:

![](https://qhstaticssl.kujiale.com/image/png/1768529963819/2CF76332F43CFC97F7AD4470B9E513DF.png)

2. Select a model, click **Customize** in the floating window to enter the customization page. Then select the model again, and use the right-side properties panel to adjust parameters such as **Number of Staircase Steps, Railing Height and Tread Depth**.

![](https://qhstaticssl.kujiale.com/image/png/1768529988072/8A82F8ECBABF3240272B52A74D05F37D.png)
