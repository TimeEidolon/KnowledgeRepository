When selecting a model for scaling in the design tool, you can enable scale by checking the scale option or using the shortcut "Shift/⇧." Once scale is disabled, you can scale freely as needed. This article provides a detailed explanation of how to use the scale tool to adjust model size.

⭐Before reading the article, it's recommended to watch the video tutorial first to get started quickly~

🎥 [Video URL](https://www.youtube.com/watch?v=ZeOy8LuGT0c)

# For Users

All users

# Prerequisite

Only certain models (prefabricated models and customized 3D accessory models) support proportional scaling

# Entrance

Enter the design tool, select a model, click the floating **Switch Control** button, then click **Scale**

![](https://qhstaticssl.kujiale.com/image/png/1764212228225/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Proportional Scaling

## Enable  Scale

1. In the 3D view, select a model and click the floating **Switch Control** button, choose **Scale**, and check the **Scale** option on the right panel. Then drag any axis of the model to scale, achieving proportional scaling along the X/Y/Z axes.

![](https://qhstaticssl.kujiale.com/image/png/1764212519237/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1764212864069/68D177736ADEF82983E0B18DC15823C5.gif)

2. In the 2D view, select a model and check the **Scale** option on the right panel. Then directly drag any point of the selected model to achieve proportional scaling along the X and Y axes.

![](https://qhstaticssl.kujiale.com/image/png/1764212561609/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1764212851483/D0F6A7934A629582CB972C4F55C2A615.gif)

## Using the Shortcut "Shift/⇧" for Proportional Scaling

1. In the 3D view, select the model you want to scale, ensuring that the **Scale** option on the right panel is unchecked. Then press and hold **"Shift/⇧"** and dragany axis to achieve proportional scaling along the X/Y/Z axes.

![](https://qhstaticssl.kujiale.com/image/gif/1764212881057/435B495D2CC0DB3743BE6523FD081D5D.gif)

2. In the2D view, select a model and make sure the **Scale** option on the right panel is unchecked. Then press and hold **"Shift/⇧"** and directly drag any point of the selected model to achieve proportional scaling along the X and Y axes.

![](https://qhstaticssl.kujiale.com/image/gif/1764212900884/8B92EAD58843012015ACE668614C773A.gif)

# Disabling Proportional Scaling

1. In the 3D view, uncheck **Scale** to freely scale the model along any axis as needed.

![](https://qhstaticssl.kujiale.com/image/gif/1764212919627/CD8FC63C91BC49078853BAFD719BE561.gif)

2. In the 2D view, after unchecking **Scale**, dragging the four corner points still allows proportional scaling, while dragging the top, bottom, left, or right edges enables free scaling.

![](https://qhstaticssl.kujiale.com/image/gif/1764212928477/BA3CF5EA969D0747CC1A36BC94ED33E3.gif)

3. In the 2D/3D views, while using the **"Shift/⇧"** for **Scale**, releasing the **"Shift/⇧"** will revert to free scaling.

![](https://qhstaticssl.kujiale.com/image/gif/1764212951639/DCE71965D19C1438840BBBDE410C6B53.gif)
