# Prerequisites

Curved windows can only be placed on arc walls. For how to draw arc walls, see the tutorial document: [Click to view](https://www.coohom.com/en_US/helpcenter/floor-plan-draw-an-arc-wall-with-outer-curve?kjl_source=hc)

# Place a Custom Curved Window

**Method One:** Enter the **Floor Plan** tool, place the mouse on the **Window** model, select **Curved Window (Customized)**, and drag it to an **arc wall.**

![](https://qhstaticssl.kujiale.com/image/png/1758607575854/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1758607762331/E892FB9FED1C8A3D8A1508D68D11F863.gif)

**Method Two:**

1. Select **Advanced Tool**, click to enter **Doors & Windows Tool.**

![](https://qhstaticssl.kujiale.com/image/png/1758607800445/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After entering the customization interface, click on the left **Public Catalog**, select **Whole Wood Doors and Windows -** **Arc-shaped Window**, and drag it to an **arc wall.**

![](https://qhstaticssl.kujiale.com/image/png/1758608107069/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1758608023788/D3082194BAAA09DAC2F359C1F51E4DEE.gif)

# Modify Curved Window Parameters

1. Select the **Curved Window**, click **Start Customize** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1758608898577/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the customization interface, **Select Model**, and set parameters such as **overall window size, glass installation method, frame size** in the right **Parameters** panel.

![](https://qhstaticssl.kujiale.com/image/png/1758609245864/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Switch to **Style** to replace **window style, glass material, frame material** and more.

![](https://qhstaticssl.kujiale.com/image/png/1758609207429/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
