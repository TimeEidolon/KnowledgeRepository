# Step-by-Step Instructions

## **Locate Compatible Cabinet Models**

1. Navigate to the **Kitchen & Bath**.

2. Search for cabinet models that align with your appliance's specifications.

![](https://qhstaticssl.kujiale.com/image/png/1741860543636/0022C66B9F1CD5E59BB7E7A90CB50AC9.png)

![](https://qhstaticssl.kujiale.com/image/png/1741860560740/D7A7BEFF092189F304491D0733559057.png)

![](https://qhstaticssl.kujiale.com/image/png/1741860575348/738345F29131E4653EB5D1FB0B745C62.png)

## **Remove Preloaded Appliances**

Delete any default appliances included in the selected cabinet model.

![](https://qhstaticssl.kujiale.com/image/png/1741860608498/20247E574033D3F4C413768AB6D4334D.png)

## **Adjust Cabinet Dimensions**

Modify the cabinet height to match the appliance's exact measurements. Ensure precise alignment for seamless integration.

![](https://qhstaticssl.kujiale.com/image/png/1741860620776/83C4900C933ED32210F93512DB66BE60.png)

## **Use Custom Furniture Tools**

1. Select **Customize** from the toolbar.

2. Click to modify the internal dimensions.

![](https://qhstaticssl.kujiale.com/image/png/1741860627235/F117FB38B3EEE9DA848F280B11748A1E.png)

![](https://qhstaticssl.kujiale.com/image/png/1741860637324/D12012A65A29213893AEB5F8115351AF.png)

## **Fine-Tune Measurements**

1. Adjust the cabinet's internal dimensions to fully accommodate the appliance.

2. Utilize **2D/3D view switching** for spatial verification if necessary.

![](https://qhstaticssl.kujiale.com/image/png/1741860648932/8F9DFEEDB4E0BFAB34D2F191E20D04BE.png)

# Important Notes

1. Edited K&B Public Library models and Personal Library models cannot be grouped or moved together.

2. Access advanced tools like **Snap Adjustment** and **Group Movement** are unique to **Enterprise Accounts**.To explore additional K&B functionality, [contact sales to request a demo](https://www.coohom.com/b2b/schedule-a-demo?kjl_source=hc).
