# Entrance

1、Click **Advanced Tool**.

2、Select **Closet** to enter the customization interface.

![](https://qhstaticssl.kujiale.com/image/png/1735898074092/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Select a Model

1、Click **Public Catalog**.

2、Select the appropriate cabinet model from the **Product Catalogue**.

![](https://qhstaticssl.kujiale.com/image/png/1736826561407/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Drag out a Model

**Click and drag** the model, **long press to pull it into** the design scene, and the cabinet is dragged in to complete

![](https://qhstaticssl.kujiale.com/image/gif/1736827680575/CB2B3982EA1DD293844C9F6871EBC492.gif)
