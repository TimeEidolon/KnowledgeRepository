Coohom supports uploading CAD floor plan drawings, automatically identifying and generating walls, doors and windows. A single drawing and multiple drawings can be imported into the floor plan. This article will explain in detail the operation of importing CAD floor plan files. It is recommended to read it before importing the drawings: **[Requirements and FAQ for Uploading CAD Floor Plan Files](https://www.coohom.com/en_US/helpcenter/requirements-and-faq-for-uploading-cad-floor-plan-files?kjl_source=hc),** which facilitates the import of drawings more efficiently.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=ZKjg6CDonCM)

# For Users

All users

# Entrance

**Entrance One:**Enter the workspace, click **upload floor plan** and select the required drawings to import, you can choose to import a single or multiple CAD files, or select a CAD file containing multiple drawings.

* Uploading floor plan supports **importing CAD floor plan files and images in pdf/ png/ jpg format. The specific operation of uploading images can be seen in the tutorial document: [Click to view](https://www.coohom.com/en_US/helpcenter/how-to-upload-import-cad-floor-plan-by-image-pdf?kjl_source=hc)**

* **The format requirements for CAD files can be found in the tutorial document: [Click to view](https://www.coohom.com/en_US/helpcenter/requirements-and-faq-for-uploading-cad-floor-plan-files?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1779700054148/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance Two:** After entering the 3D design interface, click on the **import 2D drawing** in the upper left corner, select the required drawing to import, you can choose to import a single or multiple CAD files, or select a CAD file containing multiple drawings.

* **The format requirements for CAD files can be found in the tutorial document: [Click to view](https://www.coohom.com/en_US/helpcenter/requirements-and-faq-for-uploading-cad-floor-plan-files?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1758771878873/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Import CAD Floor Plan Files

## Import a Single CAD Drawing

After clicking import 2D drawing, select the **import options** and **parameters** in the panel, and click **Confirm**.

![](https://qhstaticssl.kujiale.com/image/png/1758771317078/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Import Multiple CAD Drawings

1. Import multiple CAD drawings, the tool will automatically divide the drawings, click on **Floor Management**, and create a new **upper** or **lower floor**.

![](https://qhstaticssl.kujiale.com/image/png/1758771389398/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1758771414912/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. You can click on the **preview to view**, which is convenient to set the name and height of the floor.

![](https://qhstaticssl.kujiale.com/image/gif/1758771458730/699213F43475E1F50FFEBCA90D0FF5B6.gif)

3. After the new floor is created in **Floor Management**, **match the corresponding floor** to each drawing, and then **select** the drawings to be imported.

![](https://qhstaticssl.kujiale.com/image/png/1758771534399/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. Select the **import options** and **import unit** in the panel, and click **Confirm**.

![](https://qhstaticssl.kujiale.com/image/png/1758771590398/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. After importing, the floor plan will display transparent by default. You can manually modify **the opacity of the floor plan** according to your needs.

![](https://qhstaticssl.kujiale.com/image/png/1758771693128/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# FAQ

**Q:**How to align the upper and lower layers after importing multi-layer drawings?

**A:** After importing multi-layer drawings and generating a floor plan in the tool, you can align the upper and lower layers through the **align multi-floor units tool**. The specific operations are as follows:

* Open the top toolbar, click **align multi-floor units**, select **align with upper / align with lower**, and move the point-to-point in the canvas.

![](https://qhstaticssl.kujiale.com/image/png/1758771733098/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
