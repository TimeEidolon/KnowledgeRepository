# Production overview

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1742392594820/78E8CE2030532ECA0C85921729714DC7.png)

## a. Production introduction

Coohom Architect is an **all-in-one design tool** that integrates **architectural/exterior** and interior design. It provides an exclusive solution for a complete house design, from concept to construction drawing.

The new tool makes it easier to design both exterior and interior in one project and all interior design features remain consistent with Coohom.

## b. Operation entry

* **Individual user：**[Jump to Coohom with one click](https://www.coohom.com/)

🅰️Step1: Enter the **workbench**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1742392596771/6A62D58D63A49D42388C985DE03D647A.png)

🅱️Step2：Hover on **'New project'** , then select **'New House design Project'**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1742392598834/DA6B0D810BC4ED0DF8EA9479214E1B4C.png)

# This section describes the operations and functions

## a. Architectural design

* Import 2D drawings to create floorplan fast by auto-generate or trace image.

[![
](//qhstaticssl.kujiale.com/newt/1144/image/png/1549879747859/B0556275535F7BF4FEF2DCC4F752A054.png)](https://43661589.fs1.hubspotusercontent-na1.net/hubfs/43661589/1%20%E6%88%B7%E5%9E%8B%E5%AF%BC%E5%85%A5-1.mp4)

* Space planner : Arrange the rooms like placing bubbles to quickly generate the architectural floor plan.

[![
](//qhstaticssl.kujiale.com/newt/1144/image/png/1549879747859/B0556275535F7BF4FEF2DCC4F752A054.png)](https://43661589.fs1.hubspotusercontent-na1.net/hubfs/43661589/2%20%E6%B3%A1%E6%B3%A1%E5%9B%BE-%E5%B8%83%E5%B1%80%E6%88%B7%E5%9E%8B.mp4)

## b. Appearance design

* Decorate easily: In the "Decorate" part, you can quickly add a variety of facade decoration to enrich the exterior details of the building
* Materialize quickly: Provide a rich material library to achieve personalized exterior atmosphere for the house, applying the material with one click

[![
](//qhstaticssl.kujiale.com/newt/1144/image/png/1549879747859/B0556275535F7BF4FEF2DCC4F752A054.png)](https://43661589.fs1.hubspotusercontent-na1.net/hubfs/43661589/3%20%E5%A4%96%E8%A7%82%E8%AE%BE%E8%AE%A1-1.mp4)
