# **Function Overview**

* **Function Background:** Some Photo Studio users mentioned that they hope their team members using the Photo Studio can share the good studios and well-done plans they find, so that everyone in the company using the studio can see and copy them.
* **Enterprise Studio Library:** All Photo Studio accounts within a system can submit their preferred plans to the enterprise library for everyone to view and copy.

# **For Users**

All Photo Studio Users

# **Detailed Description**

1. Entrance to submit plans to the enterprise library: Photo Studio Workbench - My Project - Select Target Plan - “...” - Submit to the Enterprise Library - Select the corresponding group - click “Confirm”

![](https://qhstatic-us-s3.coohom.com/image/png/1769583566647/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After submission, all studio accounts in the company can see and use the plan/studio in the Photo Studio Workbench - Enterprise Studio Library.

![](https://qhstatic-us-s3.coohom.com/image/png/1769583608472/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
