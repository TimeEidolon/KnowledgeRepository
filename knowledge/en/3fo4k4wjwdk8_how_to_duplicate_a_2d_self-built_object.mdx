This topic describes how to duplicate a 2D self-built object.

**Prerequisites**

You have already built a 2D object by using **Construction** in **Advanced Tool**.

**Procedures**

1. Click an object that you have buit and click **Duplicate** in the toolbar that appears.

Click an object that you have buit and click **Duplicate** in the toolbar that appears.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839281738/FB571A0710A89B8E9F2F56AF935B3D71.png)

2. Click somewhere to set a reference point and move the pointer to display the duplicated object.  
​

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839284787/E98D188FDE85B65390CDD15DC15EF84C.png)

3. Click again to place the duplicated object.  
​

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839287691/2FAEAA08AFA54EEBB1323DC8D47434F4.png)
