# Feature Introduction

1. The model and camera can now simultaneously set both preset animations and keyframe animations, and support preset animations to be converted into keyframes, enhancing the model animation production capabilities.

2. Added curve editor function, allowing for more complex model animations, such as adjusting object movement speed variations, changing keyframe types, etc.

# Basic Functions

**1. Connect preset animations with keyframe animations**

Set both preset and keyframe animations on the same model

**Note: For the same attribute, at the same time point, keyframes and preset animations cannot coexist**

If a prompt appears, you can modify the animation for that attribute. For example, in the video below, preset animations and keyframe animations conflict on the attribute 【angle Z】, so they cannot coexist at the same time point. One solution is to delete the keyframe animation for the 【angle Z】 attribute and not create keyframes on 【angle Z】 within the time range covered by the preset animation.

**2. Convert preset animation to keyframe**: Preset animations can be converted into keyframes, and the converted attribute values will be directly placed on the timeline and will merge with existing keyframes.

**3. New curve editor function:** Freely set parameter values for attributes and display them in a function curve manner, facilitating more convenient and complex animation production for users.

**Introduction of the timeline panel in curve mode:**

（1）Timeline panel switch button

（2）Model attributes

（3）Vertical axis: parameter values of attributes

（4）Horizontal axis: timeline

（5）Attribute panel: Each attribute has an independent curve, and keyframes are points on the curve

![](https://qhstaticssl.kujiale.com/image/png/1769567006892/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Curve editing video:** Change the speed of the model movement through curve editing and make it rotate and scale during movement

**4. Switch keyframe types:** Users can switch animation keyframe types to Bezier/Linear/Mutation according to personal needs, and the results can be displayed on the curve timeline panel.

**Bezier:** Default mode, characterized by adjustable animation speed using handles, such as easing in and out. Both handles can be adjusted simultaneously or individually.

**Linear:** Characterized by a constant animation speed, such as uniform motion.

**Mutation:** Characterized by instant completion of the animation, such as the flashing of an object's position.
