Coohom supports uploading CAD floor plan images, automatically identifying and generating walls, doors and windows. This article will explain in detail the operation of importing CAD floor plan images.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=ZEsq7kywjrI)

# For Users

All users

# Entrance

**Entrance One:**Enter the workspace, click on the **Upload Floor Plan** and select the required drawings to import**.**

* Uploading floor plan supports **importing CAD floor plan files and images in pdf/ png/ jpg format. The specific operation of uploading files can be seen in the tutorial document: [Click to view](https://www.coohom.com/en_US/helpcenter/how-to-upload-import-cad-floor-plan-files?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1779699628343/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance Two:** After entering the 3D design interface, click on the **import 2D drawing** in the upper left corner, select the required drawing to import.

![](https://qhstaticssl.kujiale.com/image/png/1779699883530/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Auto Generate

## Import Drawings

**Method One: Scale Length Measurement**

After uploading the floor plan image, the floor plan area will be automatically identified. You can use the scale **length** to set the basemap and generate the scale of the floor plan.

![](https://qhstaticssl.kujiale.com/image/png/1747125053907/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Method Two: Scale Area Measurement**

When there is no clear dimension information in the image, you can enter the expected building **area** to complete the floor plan generation.

![](https://qhstaticssl.kujiale.com/image/png/1747125298360/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Generate a Floor Plan

1. After the scale setting is completed, click **Confirm** to generate the floor plan, as shown in the figure below:

![](https://qhstaticssl.kujiale.com/image/png/1747125736189/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. You can switch to **3D mode** to check the effect of the floor plan generation.

![](https://qhstaticssl.kujiale.com/image/png/1747125810230/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Manual Trace

If you choose to **manual trace**, you need to draw the floor plan manually based on the floor plan image as the basemap. Therefore, it is recommended to use the recognition and generating floor plan function.

![](https://qhstaticssl.kujiale.com/image/png/1747126011151/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1747126151283/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Import Images Multiple Times to Replace the Basemap

1. If you already have a floor plan, import the floor plan image again. The import method will select **"manual trace"** by default. After identifying the floor plan, the floor will be cleared. Floor information needs to be set manually.

![](https://qhstaticssl.kujiale.com/image/png/1747126632842/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. If you generate a floor plan again when there is a floor plan, you will be prompted with the message **"Identification import will clear the existing apartment types on the current floor"**. You can click **continue importing** to generate a new floor plan or **import later** and keep it as it is.

![](https://qhstaticssl.kujiale.com/image/png/1747127104524/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
