**Definition**

A Decorative Lintel Panel is a component that serves as a decorative element in the whole-house customization tool.

Taking the shape in the following figure as an example, let's create a model for a Decorative Lintel Panel.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896481952/420D5C5D4ACAD69D8DC7FE672B0E8400.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896484120/5013730231C4E5EEA548AD3FD80556BE.png)

**I. Creation Location**

Custom Furniture—Create Asset—Parametric Model Library— New—

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896485471/0856B27A409AC7454759C7721C27B3F3.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896486505/7432BA2470170F9FF113E588940CC5B5.png)

**II. Parameter Configuration - Custom parameters**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896489096/54E6BF37EE5C08746E92D481830D4D28.png)

Width (W) - Represents the size of the Panel in the X-axis direction, with value type set as a range (e.g., range from 300 to 2400, current value is 700).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896490202/95565F3973B1F84B564C53E942CBF1A5.png)

Depth (D) - Represents the size of the Panel in the Y-axis direction, with value type set as optional (e.g., optional values: 18, current value is 18).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896492235/F81C55A7F9F43B94CF07352B6E1795BA.png)

Height (H) - Represents the size of the Panel in the Z-axis direction, with value type set as a range or optional (e.g., optional value: 120, current value is 120).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896493403/81A3E8D15CA00603529C5F5F00B2DBE2.png)

Material does not need to be changed, indicating that the default material will be used.

(Note: The range of parameter values mentioned above can be set based on actual process requirements and is not fixed.)

Creating an advanced parameter for controlling the size of the cornice arc, with reference name-R and value type as a range or optional (e.g., range from 50 to 70, current value is 60).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896494980/5964B982B84BDDEC1DEA0123CEBA4327.png)

**III. Calling the Flat Panel Component**

Element Library - Parametric Model - Flat Panel

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896496613/A736064866F2E27D952342C8E7066D0D.png)

**IV. Setting the Physical Properties of the Irregular Cuboid**

Physical Properties - Edit contour

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896499502/CD126DFBCAB8CA2E9BE6508B081EE6E6.png)

Insert Endpoints - Edit Endpoints, starting from the bottom-left point (0, 0), write down the position of each endpoint:

Point 1 (0, 0) Point 2 (0, #H)

Point 3 (#W, #H) Point 4 (#W, 0)

Point 5 (#W-50, 0) Point 6 (#W-50-#R, #R)

Point 7 (50+#R, #R) Point 8 (50, 0)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896500843/70A6ED4DF8463E1A7D7D48EEADC8D3D6.png)

Change the segment types of lines 7-8 and 5-6 to arcs, with an arc radius of #R.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896501811/ECFDBFCE9A0C5E5546EBDA99B269948F.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896503017/6F1A44A8D6DF5B5428349CDAA9C098A8.png)

Adjust the position and rotation (to make the model appear within the frame): Position Z: #H, Rotation X: 90.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896504110/BC6DC618B82A10F5BB6583C3D360C550.png)

Setting System parameters

Currently, when the model is dragged out for use, it defaults to full coverage. Therefore, the overlay methods are as follows:

Top overlay: Full Coverage, Bottom overlay: Embedded, Left overlay: Full Coverage, Right overlay: Full Coverage.

Top overlay: 18, Bottom overlay: 0, Left overlay: 18, Right overlay: 18.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896505626/AC5113E96DD76820D2AC7CF566163F5E.png)

**V. Setting Output Attributes**

The dimensions displayed in the export list correspond to the texture direction of the Panel. Width corresponds to the size in the texture's direction (X-axis) displayed in the editor, so width is #W;

Depth corresponds to the size in the vertical texture direction (Z-axis) displayed in the editor, so depth is #H;

Height corresponds to the thickness direction of the Panel displayed in the editor (Y-axis), so height is D.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896506964/4046F5D6AB1CE21A396373CCCE411D15.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896508472/4712528EFB2E115C6A788123F262FF0F.png)

**VI. Model Testing**

Enter values within the optional or range limits of the input parameters and check if the model's changes align with the parameter changes. If they match, it indicates that the model is correct. To ensure accuracy, every parameter must be thoroughly tested. (In the given example, only the width W and arc radius R need to be tested, as depth D and height H have one optional value each and do not require testing.)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896509890/12421A0B1F3D419D745C65171FEF3846.png)

**VII. Saving to the library**

Click on the file in the upper-right corner, select Save, choose the subdirectory for saving, and click Save.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896512734/A8D3EDB396E1A63B57E6B0426F5348E9.png)

In the corresponding category on the Product page, find the model, select it, and click Store to wait for the model rendering to complete.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896513871/5B6FBAD139A89942902F1F5DD987BD05.png)
