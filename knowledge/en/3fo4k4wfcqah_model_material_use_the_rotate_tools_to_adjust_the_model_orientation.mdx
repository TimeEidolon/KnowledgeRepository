In the design tool, various methods can be used to achieve the requirement for model rotation. During the rotation process, the 2D/3D views will synchronize in real-time.

⭐Before reading the article, it's recommended to watch the video tutorial first to get started quickly~

🎥 [Video URL](https://www.youtube.com/watch?v=4mECnxu63uI)

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Select the model, and you will see the blue arc line surrounding the model material. Then, long-press the arc line and drag to rotate it. Release the mouse once it reaches the desired angle.

### Step 2
When the model rotates, the bottom right **2D Display** and the top right **3D** view will synchronize and rotate in real-time.

### Step 1
Select the model, hold down the **Ctrl/⌘+R** key. Each press rotates the model by 45°, two presses rotate it by 90°.

![](https://qhstaticssl.kujiale.com/image/gif/1774333565849/8B92EAD58843012015ACE668614C773A.gif)

### Step 2
When the model is in the hovering state (following the mouse), you can also hold down **Ctrl/⌘+R** to rotate. Rotate it to the desired angle and click to place it. **Method 1** Select the model, click on **Switch Control** in the floating window, select **Rotate**, then click on the corresponding arc line as needed to rotate. **Method 2** Select the model, access the **Angle** section in the right-side properties bar, and input the desired value for rotation on the corresponding axis. Select the model, click on flip in the floating window. The flip tool has two modes: **Mirror or Back & Forth**. Click the desired mode to successfully flip the model.

# Rotate the Model

## Manual Rotation

1. Select the model, and you will see the blue arc line surrounding the model material. Then, long-press the arc line and drag to rotate it. Release the mouse once it reaches the desired angle.

![](https://qhstaticssl.kujiale.com/image/gif/1774332635676/D0F6A7934A629582CB972C4F55C2A615.gif)

2. When the model rotates, the bottom right **2D Display** and the top right **3D** view will synchronize and rotate in real-time.

![](https://qhstaticssl.kujiale.com/image/gif/1774333436591/A5103715D99933353788DA96625BF3FF.gif)

## Shortcut Key Rotation

1. Select the model, hold down the **Ctrl/⌘+R** key. Each press rotates the model by 45°, two presses rotate it by 90°.

![](https://qhstaticssl.kujiale.com/image/gif/1774333565849/8B92EAD58843012015ACE668614C773A.gif)

2. When the model is in the hovering state (following the mouse), you can also hold down **Ctrl/⌘+R** to rotate. Rotate it to the desired angle and click to place it.

![](https://qhstaticssl.kujiale.com/image/gif/1774333784250/CD8FC63C91BC49078853BAFD719BE561.gif)

## Three-Axis Rotation

**Method 1**

Select the model, click on **Switch Control** in the floating window, select **Rotate**, then click on the corresponding arc line as needed to rotate.

![](https://qhstaticssl.kujiale.com/image/png/1774334153030/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1774334082276/BA3CF5EA969D0747CC1A36BC94ED33E3.gif)

**Method 2**

Select the model, access the **Angle** section in the right-side properties bar, and input the desired value for rotation on the corresponding axis.

![](https://qhstaticssl.kujiale.com/image/png/1774334283402/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1774334228209/DCE71965D19C1438840BBBDE410C6B53.gif)

# Flip the Model

Select the model, click on flip in the floating window. The flip tool has two modes: **Mirror or Back & Forth**. Click the desired mode to successfully flip the model.

![](https://qhstaticssl.kujiale.com/image/png/1774334329454/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1774337795328/BDD03DBD7B143CFD44CBE3156FF860F8.gif)
