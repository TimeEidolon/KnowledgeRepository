In the custom productcatalog, you can add selected products to a new folder or move them from the original folder to a new folder. It supports adding products to other tools, for example: copying products from the custom furniture to the kitchen＆bath product catalog.


## Steps with screenshots

Each step includes plain MEDIA_* URLs (for AI tools) plus rendered images.

### Step 1
Entrance Enter the **Workspace**, select **"Enterprise Catalog - Custom Products - Product Catalog"**, and choose a tool.

# For Users

All enterprise users

# Entrance

Enter the **Workspace**, select **"Enterprise Catalog - Custom Products - Product Catalog"**, and choose a tool.

![](https://qhstaticssl.kujiale.com/image/png/1752222934578/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Copy Enterprise Catalog Products

1. In the product list, select the product that needs a new folder location, click **"Set Folder - Add to the Folder".**

![](https://qhstaticssl.kujiale.com/image/png/1752223013552/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the right folder list, select the tool you want to copy to and the folder location to add to, check it and click **Save**.

![](https://qhstaticssl.kujiale.com/image/png/1752223092939/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. After adding is complete, the selected products will be saved to the folder of the newly selected tool, as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1752223154484/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Note:

* Click the **"Delete"** button in the directory of the folder where the left product is located to quickly delete the product from the corresponding folder.
* Click **"Empty the Folder"** to clear all the products from the current folder location.

![](https://qhstaticssl.kujiale.com/image/png/1752223567511/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Move Enterprise Catalog Products

1. In the product list, select the product that needs to move folder location, click **"Set Folder - Move to the Folder".**

![](https://qhstaticssl.kujiale.com/image/png/1752224127212/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the right folder list, select the folder location to add to, check it, and click **Save.**

![](https://qhstaticssl.kujiale.com/image/png/1752224308679/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
