### Step 1
Uploading Process Waiting for the rendering Rendering category selection as a handle Once the category is determined, you can adjust the default distance parameter of the handle. In this case, the system will default to X: 30 and Y: 30 as the distance from the handle model's edge to the edge of the door panel. These values can be modified later for further use. - X represents the horizontal distance between the edge of the handle model and the edge of the door panel. - Y represents the vertical distance between the edge of the handle model and the edge of the door panel.
