Dear Coohomers,

We are excited to announce that the basemap display button is added in the right panel for users to clearly see and operate while users are importing the floor plan as basemap. (Sometimes users will import a CAD floor plan and use it as reference image which is called basemap. )

**Users**

All users

**How to get to the basemap display button?**

Step 1: Open or create a project

Step 2: Go to 2D view

Step 3: Proceed to the right panel to find the button.

![](https://qhstaticssl.kujiale.com/image/png/1750159222852/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

Tips: If you cannot see this button, it means that you did not import the floor plan as basemap.
