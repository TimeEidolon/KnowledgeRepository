When downloading drawings, users can set the generated directory drawings to display only the drawings exported this time, rather than showing all drawings. This article will explain in detail how to generate a dynamic drawing directory.

# For Users

All elite users

# Entrance

1. Enter Interior Design, select **Drawings - Construction Drawings.**

![](https://qhstaticssl.kujiale.com/image/png/1762337484082/A17A899ADED35F082C782B22E0C9C18A.png)

2. Enter the **Drawings** **Layout**page, click the **Settings** button in the upper right corner, and select **Preferences.**

![](https://qhstaticssl.kujiale.com/image/png/1762337495973/76B92E78F8573E4AA93317D14485094C.png)

3. In the **Preferences** panel, enable **Update Directory When Downloading.**

![](https://qhstaticssl.kujiale.com/image/png/1762337687413/2A4F899661BF1257460A4877B7328C51.png)

# Generate Dynamic Drawing Directory

1. Click the **Download** button in the upper right corner.

![](https://qhstaticssl.kujiale.com/image/png/1762337771199/EAADC6727F836BAF5C39E04508559CED.png)

2. In the pop-up download drawings panel, **check the drawings to download**, then select **PDF format**, and click **Download.**

![](https://qhstaticssl.kujiale.com/image/png/1762338019238/D92221750551B0CA8B2F4F0DACD7D534.png)

3. After the download is complete, click the **Download Icon** in the upper right corner, and then open the corresponding file in Download Records.

![](https://qhstaticssl.kujiale.com/image/png/1762338521162/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. After opening the file, you can see that the **Directory Drawings** only show the name of the exported drawings checked this time.

![](https://qhstaticssl.kujiale.com/image/png/1762338582489/9102C39A87DDD4DD2980FCFF5B30AC6B.png)
