**III. Procedures**

2-1. Width (W): Represents the dimension along the X-axis, usually set as a range (e.g., 300 to 2400, current value: 600).

2-2. Depth (D): Represents the dimension along the Y-axis, usually set as a range (e.g., 300 to 1200, current value: 600).

2-3. Height (H): Represents the dimension along the Z-axis, with optional value types. Can be a single value or multiple values (e.g., single optional value: 18, current value: 18).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907349538/9CC451572131BFAA5E57A18975C83F11.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907350212/7BA2672B608474D7E32C90E231E08FAE.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907350819/1CC92C407298D3AE869C47CAB311BE24.png)

2-4. Add advanced parameter: Name: Left Width, Reference Name: W1, value type range (e.g., range, minimum value: 100, maximum value: #W-100, current value: 300), click [Create];

2-5. Add advanced parameter: Name: Right Depth, Reference Name: D1, value type range (e.g., range, minimum value: 100, maximum value: #D-100, current value: 300), click [Create];

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907351450/EADB4CABEF6A21F8DCB01BA91641A074.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907352048/A0C2A27308BF2387C0524E6732ED8923.png)

2-6. Keep material default settings or modify them according to actual needs.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907354396/6E4177E1CC01413B25030263F7497541.png)

1. Inserting Flat Panel Modeling - Two methods are explained below. First Method: (Recommended)

Inserting Flat Panel Modeling - Two methods are explained below. First Method: (Recommended)

3-1. Insert flat panel component from the component library.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907355220/80E9A570C16D6BA82EBFC5A77EB4CB22.png)

3-2. Click on "Edit Contour," delete the default four points, and redraw or modify them directly.

Point 1: 0,0; Point 2: 0,#D;

Point 3: #W,#D; Point 4: #W,0;

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907357260/E5F89EFBAE6DA3217BE6E64BAC040B86.png)

②After that, click on point 4, select "Chamfer" as the corner type, and set the chamfer length 'a' as #D-#D1 and chamfer length 'b' as #W-#W1. Then, save the changes.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907358828/35A74622452DC00D8325172E6B505344.png)

3-3. Set height: #H

3-4. Set other properties, material angle: #CZFX, side material angle: #CBCZFX (The texture direction of the panel can be modified when calling the panel, but the quotation output remains unchanged).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907360539/B7B4C6E0633F92CCAACF0E8D1D9DF48C.png)

Second Method:

3-1. Insert flat panel component from the component library.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907361488/69F0EBDFD0AABAD1C2A28D70EFAE033C.png)

3-2. Click "Edit Contour" to draw points:

Select import pre-drawn CAD points (saved in DXF format).

Imported coordinates should be replaced with corresponding reference values.

Click "Save."

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907363868/07FBF99A7A2D3CC74941620126EC2A4A.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907364814/DEFC099FE9B7F5721A12565A30BB7E04.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907367236/F7B9C1972B6AD0922737E4432A9D39ED.png)

3-3. Set other properties, material angle: #CZFX, side material angle: #CBCZFX (The texture direction of the panel can be modified when calling the panel, but the quotation output remains unchanged).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907368429/E0D24D7D280ECB18FC94D6882FB53382.png)

**4.Test the Model**

1.Replace values for W, D, H, Left Width, and Right Width variables to different sizes on the left side. Check if the model has any issues.

2.After testing, restore these values to default.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907369616/39957458EA44C9F3A9D0CF490905E7DA.png)

**5.Save to Library**

5-1. Click on the top right corner "File," select "Save," choose a subdirectory, name it, and click "Save."

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907371615/38F845140A3C1B13D83B539C421F75D9.png)

5-2. On the backend page, find the model, select it, and choose to save it to the library.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733907374121/A21A33B1618AFA592570A3A056ADA3E9.png)
