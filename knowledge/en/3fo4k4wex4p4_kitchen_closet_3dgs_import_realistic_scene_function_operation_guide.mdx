No manual drawing needed, unlock realistic scene restoration and creative design with one click! This feature supports uploading realistic scene images, panoramic images, 3dgs file, automatically recognizing and generating floor plans. Subsequent design modifications can be freely done within the tool, and new renderings can be completed with one click. Moreover, when generating a floor plan, users can choose whether to remove soft furnishings and custom furniture: if not removed, the layout of the real scene from the uploaded image can be accurately restored; if removed, a clean floor plan base will be generated, flexibly accommodating various design transformation needs, helping users efficiently start design work.
