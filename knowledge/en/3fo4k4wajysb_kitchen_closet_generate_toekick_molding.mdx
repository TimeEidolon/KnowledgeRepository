Use the toekick molding generation tool to quickly install toekick molding on cabinets.


## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
After entering the generate toekick molding interface, select the cabinet, the tool will automatically identify the cabinets that need to generate countertops. You can also **press and hold shift to select multiple cabinets** for generation. If you need to identify all cabinets, you can click **Auto-select**.

### Step 2
In the basic parameters on the right panel, select add **toekick molding style and material**. After adjusting the specific parameters, click **Generate**. * **Generate Wrap:** Fit for cabinets with toekick moldings. Adds wraps for the cabinets with toekick moldings.

![](https://qhstaticssl.kujiale.com/image/png/1742205495420/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

### Step 3
If you need to freely adjust the range of the toekick molding generation, click **manual edit** in the top bar to enter the manual mode.

In the manual interface, use the drawing tools on the top bar in 2D to edit edge and add rectangle.

# Entrance

Enter the kitchen & closet design and choose the **kitchen**.

**Entrance One:** Select the **Generate-Toekick molding** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1742204463423/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance Two:** Select the cabinet, and select **Generate-Toekick molding** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1742204808029/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Generate Toekick Molding

1. After entering the generate toekick molding interface, select the cabinet, the tool will automatically identify the cabinets that need to generate countertops. You can also **press and hold shift to select multiple cabinets** for generation. If you need to identify all cabinets, you can click **Auto-select**.

![](https://qhstaticssl.kujiale.com/image/png/1742204852535/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the basic parameters on the right panel, select add **toekick molding style and material**. After adjusting the specific parameters, click **Generate**.

![](https://qhstaticssl.kujiale.com/image/png/1742205428179/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* **Generate Wrap:** Fit for cabinets with toekick moldings. Adds wraps for the cabinets with toekick moldings.

![](https://qhstaticssl.kujiale.com/image/png/1742205495420/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. If you need to freely adjust the range of the toekick molding generation, click **manual edit** in the top bar to enter the manual mode.

![](https://qhstaticssl.kujiale.com/image/png/1742205565507/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. In the manual interface, use the drawing tools on the top bar in 2D to edit edge and add rectangle.

![](https://qhstaticssl.kujiale.com/image/png/1742205605084/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1742205677194/C6382D49D7F5A1A08E88E1DA69C519ED.gif)

5. You can manually edit points and lines, or enter values ​​for precise adjustments. After the adjustments are completed, click **Generate**.

![](https://qhstaticssl.kujiale.com/image/gif/1742205749745/5861E0F8D1D739DF91EABB8C03448357.gif)

6. After generating the toekick molding, click **Finish** to exit the generate toekick molding interface.

![](https://qhstaticssl.kujiale.com/image/png/1742205793981/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* The generate toekick molding effect is as follows：

![](https://qhstaticssl.kujiale.com/image/png/1742205867441/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
