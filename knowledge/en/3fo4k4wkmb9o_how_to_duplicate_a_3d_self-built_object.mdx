**Prerequisites**

You have already built a 3D object by using Construction.

1. Hold Shift and then hold the mouse to select all surfaces of the 3D object that you have built.  
​

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733854679668/6DA46A08B04605296486ECF276FB183F.png)

2. Release the mouse and click **Group** in the toolbar that appears to group the surfaces.  
​

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733854681601/4BB63805B09A3DDFC20393CBE0844A2F.png)

3. Click the object and click **Move** in the toolbar that appears.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733854684446/6E1B163727BC811C26E7170DEBFF0EB1.png)

4. Click the object and move the pointer to move the object.  
​

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733854685914/6B4D5B6C6AB1233A6C6BC349ECDAC10D.png)

5. Click again to place the object.  
​

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733854686954/115FF28CD60710C5997D5B37B208C2E2.png)
