In the floor plan drawings, you can modify the color scheme of the floor plan by switching the Color Combinations and modifying the object styles, making the expression of the Floor Plan Drawings more diverse.

# For Users

All users

# Entrance

Enter the 3D design, select **Drawings - Floor Plan Drawings**.

![](https://qhstaticssl.kujiale.com/image/png/1750917425461/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Switch the Color Combinations

Click the **Color Styles** card to switch the color combinations.

![](https://qhstaticssl.kujiale.com/image/png/1734525210538/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1734525263672/48F7D46E6EE55977A660D81F522FEFB6.gif)

# Modify the Object Styles

1. Click the **Annotate** icon in the top menu or **Double-click the Floor Plan Drawings** to enter the Label Mode.

![](https://qhstaticssl.kujiale.com/image/png/1734525548199/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Select an object.

3. Modify the style of the object in the parameters panel.

![](https://qhstaticssl.kujiale.com/image/png/1734525597963/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
