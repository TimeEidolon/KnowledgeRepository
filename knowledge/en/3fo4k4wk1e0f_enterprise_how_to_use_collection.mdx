### Collections help users to combine models of the same style, quickly to pick or replace when use.

### **Collections**

**Collections** help users to combine models of the same style, quick to pick or replace when used.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733853546764/76E18E8A1F9CB2EC69FD2993BEBB522D.gif)

1. Merchant Platform > Enterprise Library > 3D Models, choose a model, **add to collections**

*only 3D models can be added to collections*

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733853548228/0372EB136442418172AE415960395BE7.png)

2. Create a new collection.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733853549250/32BFFB5191C02A0F8D98E9294203DB92.png)

Also, we can select several models to bulk edit. Add to collections.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733853551156/C2BE185BAD3D4E7221B188540A4B7679.png)

When the model adds to collection, it is automatically removed from the original category.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733853553213/F1C1F50792D09647E201ABAB6CC59955.png)

3. Click this collection, view all the models in it.

Also can rename, delete, or ungroup this collection.

**Ungroup** means,

All products will be removed out if this collection and get back to their original categories.

But **delete** means, the whole collection with all models will be deleted.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733853554379/D1ED5755422C1DD503434DC260624BB4.png)

4. Edit models in collections. Set as starred model/remove/delete.  
​  
The model set as starred models can’t be removed.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733853555249/5BACE185E5844A5BEF50F9282DB7C70A.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733853556128/09A0759937DD8D2C6CE20BF463A8FC02.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733853557907/B4943902B85323752FA02D2D1D1A43B2.png)

5. This is how to create a collection.

Go back to floor planner to use.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733853561087/691C0E4BF605C0B4E734BEC4B05C1F29.gif)
