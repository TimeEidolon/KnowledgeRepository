**There are no quotation category options for 3D models, so quoting rules cannot be set for 3D models. Quoting for 3D models is done by setting finished product prices.**

**I. Setting Entrance**

Custom Products-Product Catalog - Find the hardware model to be quoted - Click to view the model - Edit the model.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733834187159/DAE80AAD7E1E97F8BE62465AD9DB1BE7.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733834190020/69A5E9CAE17846551AEC3D397ED4D62E.png)

**II. Quotation Setting**

Fill in the quotation of the hardware model in the price column, and click save.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733834193384/9F8648357ABAC2CF04B76A889B52AC7C.png)

**III. Testing Prices**

Drag the hardware model or the model that contains (calls) the hardware from the tools to test if the quote is correct.
