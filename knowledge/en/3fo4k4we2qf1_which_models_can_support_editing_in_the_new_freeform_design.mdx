# Problem

Users have discovered that certain models support custom modeling tool.

![](https://qhstaticssl.kujiale.com/image/png/1764731807796/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Reason

There are currently two types of models that support custom modeling tool.

1. Upload the SKP file to your personal library. custom modeling tool is supported. Once in the custom modeling tool, you can delete, stretch, deform, and perform other operations on various parts of the model.([SKP file upload requirements](https://www.coohom.com/helpcenter/sketchup-model-upload-requirements?kjl_source=hc))

2. Some public library models support custom modeling tool.

# Usage

Convert model to "Editable model"

Converting to editable model is an irreversible process and will lose the product data of the model, but after conversion, multiple edits can be made on the style
