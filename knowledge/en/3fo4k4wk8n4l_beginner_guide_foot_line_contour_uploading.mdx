### Definition

Foot line contour is the basic line that forms any foot line shape.

Enter Custom Furniture / Kitchen&Bath - Create Asset

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850481479/F0CFA6F4A6BC04EABD94D344C8D2CDFE.png)

Select the contour upload entrance

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850484180/82B0475E3194FD4BC18ECF2B8EEAF407.png)

Click on "Add File" and select the corresponding contour or drag and drop the contour directly into it.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850485062/CB0B3DAA6553FC1965A8257EE032BF06.png)

Modify the name, select the foot line type

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850487568/118A271F0C2EFC5238711CE1388C6875.png)

Adjust the contour height to an integer (no adjustment needed if it's already an integer), select the category to save, leave the extension parameter blank, and click on Complete Upload

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850488243/02A30E01C33A1C933C569FD6872D1BCF.png)

Modify the preview image (optional): click on Attribute Editing on the Edit Information page

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850489946/88602915F1F35D71C848DD781538F168.png)

Click on Modify Preview Image, select the created preview image, and click on [Save] to complete the upload;

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850490964/D09356FA56ECD95204078CC7F25D33B7.png)

View the uploaded contours under the corresponding category;

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850491915/901267435108D284A3FCA988E13B76E3.png)

Application of foot line contours

In the frontend Full House Furniture Customization / Cabinet Customization - Cabinet Mode, click on Generate - Skirting, select the desired generation method, cabinet, foot line style, and parameter settings, then confirm.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850494691/665DC34DE05A2904E7AC07800735E029.png)
