In the kitchen＆closet design, there are two ways to draw rooms, which can quickly use floor plan templates to generate, or use wall tools to draw rooms. Supports custom named rooms, making it easy to quickly divide different room types in the floor plan.


## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
In the **Start Design** interface, you can select the **Floor Plan Template** and click **Apply**.

# Draw Rooms

## Use Floor Plan Templates to Generate

1. In the **Start Design** interface, you can select the **Floor Plan Template** and click **Apply**.

![](https://qhstaticssl.kujiale.com/image/png/1741318182455/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After the floor plan template is applied, the tool interface will generate the corresponding floor plan.

![](https://qhstaticssl.kujiale.com/image/png/1741333416403/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Use Wall Tools to Draw

Enter the kitchen＆closet design, select the **Room-Wall**, and use the **rect wall** to quickly draw closed rooms. **The wall drawing must be closed to generate rooms**.

* **How to draw a wall, you can see the relevant tutorial documents: [Draw Walls](https://www.coohom.com/en_US/helpcenter/article/draw-walls?kjl_source=hc)****、[Adjust a Wall Size](https://www.coohom.com/en_US/helpcenter/article/adjust-a-wall-size?kjl_source=hc)、[Split a Wall](https://www.coohom.com/en_US/helpcenter/article/split-a-wall?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1741334662364/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Name the Room

1. Switch to **2D mode**.

2. Select the **room**.

3. In the **basic** parameters of the right panel, set the **room type** and **room name**.

![](https://qhstaticssl.kujiale.com/image/png/1741335337118/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Room Type

1. Select the recommended type directly, or **search** for the type name.

![](https://qhstaticssl.kujiale.com/image/png/1741335595641/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1741336989743/7C44BF1C7D2485048F7A3DFBDB618F66.gif)

2. **Custom** named rooms.

![](https://qhstaticssl.kujiale.com/image/gif/1741337060352/8813EF59C55A4811DB7CE99980BDE32C.gif)

3. Set the **room ceiling**, you can choose whether the room **with ceiling or no ceiling**.

* Each room has a default ceiling setting, where the **default no ceiling** room is **terrace**.

![](https://qhstaticssl.kujiale.com/image/gif/1741337134875/8813EF59C55A4811DB7CE99980BDE32C.gif)
