II. How to Use 3D Partition Tool

### **I. Definition of 3D Partition Tool**

The 3D model partition tool is a tool that cuts the model into smaller modules, which facilitates the conversion of the original model into a parameterized model during the parametric modeling process. It can only cut 3D uploaded models.

### **II. How to Use 3D Partition Tool**

1. Enter the workspace-Custom Products-Product Catalog-Create Asset

Enter the workspace-Custom Products-Product Catalog-Create Asset

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844236814/AF6805527343EF41247F5A15AC53A1D6.png)

2. Select 3D Partition.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844239813/C099EEEACEDD40BB89139877943C10DD.png)

3. Select the model to be cut - confirm.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844243036/655A631713024A7E9A30DBFA961A7FDE.png)

4. Enter the partition interface, and select the appropriate view orientation on the right side. Add a cutting plane on the left side, and select coloring mode and wire and frame mode in the lower right corner **(the material color of the imported model is fixed by the system, and there is no need to worry about the model material transformation).**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844245627/FF81D353067B6C8EE8BA12C8ED551709.png)

5. Add a partition plane and move it to the appropriate position. The cutting plane is generated perpendicular to the direction axis. For example, if the X axis is selected, it means that the cutting surface is generated perpendicular to the X axis. (Up to 9 cutting planes can be added.)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844247767/8873DB3FE0165637EC2AF269EFC54687.png)

6. Set the position of the partition plane. The position can be input with positive and negative values. A positive number means that the cutting plane moves in the positive direction of the quadrant, and a negative number means that the cutting plane moves in the negative direction of the quadrant.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844250818/D090FA15CCDA5522E782C02BCFB67C52.png)

7. Take the cutting door panel as an example.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844253377/B862424A7AA1EE53CA52DE22225EFAD0.png)

8. After adding the partition plane, click "Save" in the upper right corner, and select a directory to save the cutting model.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844256045/C35B89BD24567287D950F8D11CBC9041.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844258560/48BA5257FC817012BFF679C49D0C1402.png)

9. Enter the workspace - Custom Products-Product Catalog, find the saved directory file library, wait for rendering and conversion, and then it can be used.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733844262829/5FE6E63B05DA5BD63830EBE53F7E6984.png)
