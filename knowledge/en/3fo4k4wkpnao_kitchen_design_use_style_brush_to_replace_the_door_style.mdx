In the custom tools, you can use the style brush to quickly replace the door style, and support the display of the door style figure after absorption.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=s493t42\_dvY)

# For Users

All users

# Style Brush

1. Enter the custom tools and select **"Toolkit - Style Brush"** in the top toolbar. (the shortcut key is N)

![](https://qhstatic-us-s3.coohom.com/image/png/1749024242288/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Place the style brush on the door style you want to copy, and then paste it on other door panels, the specific operation is as follows:

**Note:**

1. **The true classification of the absorption model belongs to the door/sliding door core/waistline, and the style and material can be successfully absorbed.**
2. **Absorb Range: Must be the same true classification model (texture pack, style pack, size range) as the absorption object.**

![](https://qhstatic-us-s3.coohom.com/image/gif/1749026812509/CFAD239D5E6A89212C5AA0969D6958F1.gif)

* When using the style brush for the door, you can also brush the handle style on the door to the target door, as shown below:

![](https://qhstatic-us-s3.coohom.com/image/gif/1749027345551/23FEA6553AABAB269AB306106DF6E5F3.gif)

# FAQ

**Q: How to solve the problem that the current object style range is limited and the replacement failed is always prompted when using the style brush?**

**A: Turn off the size restriction under the corresponding tool.**

![](https://qhstatic-us-s3.coohom.com/image/png/1749027955324/FF2B225FFF263B23739F3399DDF2CB3F.png)
