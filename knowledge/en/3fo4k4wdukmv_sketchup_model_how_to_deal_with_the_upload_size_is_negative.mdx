# Problem

The model is lines, and the uploaded model size is displayed as a negative number. Currently, line uploads are not supported, and surfaces need to be generated for upload.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1555309219641/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png?x-oss-process=image/format,webp)

# Solution

1. Create a new model.

2. Use the SUAPP plugin and the weld line tool to weld scattered points.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762314116525/5F28DDFB739037B61B8C2DA190BAF9B4.png)

3. Use the SUAPP plugin to select the surface generation tool, generate the surface, then extrude the thickness, and save and upload.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762314116545/8B5705A0626C2907B1B631CFA9E8953F.png)
