# For Users

All users

# Supported Formats

1. Models from Rhino7 and earlier versions.

2. Supports models with the .3dm file extension.

Rhino models can be uploaded in the supported 3dm format.

If you wish to export the model from Rhino for modification in 3ds Max, save it as an obj.

The following will list the settings for 3dm and obj models:***3dm***

# Model Requirements

1. The unit for model creation is millimeters.

2. Select “File” – “Save As” to save as a .3dm model.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762310477677/7D6976680C5AC08CF51E8FDC24077CEB.png)

3. Material names should not contain special symbols.

4. File names should not contain spaces.
