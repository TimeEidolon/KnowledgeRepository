Use the stove generation tool to install the stove and chimney hood on the corresponding cabinets.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=AwhMBLBtshA)

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
Select the cabinet that needs to be generated. In the basic parameters on the right panel, select add **Stove Style and Chimney Hood Style**.

![](https://qhstaticssl.kujiale.com/image/png/1739518816568/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
<img src="https://qhstaticssl.kujiale.com/image/png/1739518816568/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png" alt="Step 1" style="max-width:100%;height:auto;" />

### Step 2
After selecting the appropriate style, determine the **Installation Type** **and** **Alignment**, modify the stove margin at the **Spacing Mode**, and then click **Generate**. **Note: If the Generate button turns gray, you need to check whether the cabinet that needs to be generated is selected!**

![](https://qhstaticssl.kujiale.com/image/png/1739519058069/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
<img src="https://qhstaticssl.kujiale.com/image/png/1739519058069/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png" alt="Step 2" style="max-width:100%;height:auto;" />

### Step 3
After generating the stove and chimney hood, click **Finish** to exit the generate stove interface. **Q：**How to modify the stove size? **A：**Exit the generation environment and click to select the **stove** to modify the **size and position** of the stove in the parameter panel.

# For Users

All Users

# Entrance

Select **Kitchen＆Bath** to enter the customization interface.

**Entrance One:** Select the **Generate-Stove** from the top toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739518113230/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Entrance Two:** Select the cabinet, and select **Generate-Stove** in the floating toolbar.

![](https://qhstaticssl.kujiale.com/image/png/1739518303960/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Generate Stove

1. Select the cabinet that needs to be generated. In the basic parameters on the right panel, select add **Stove Style and Chimney Hood Style**.

![](https://qhstaticssl.kujiale.com/image/png/1739518816568/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After selecting the appropriate style, determine the **Installation Type** **and** **Alignment**, modify the stove margin at the **Spacing Mode**, and then click **Generate**.

**Note: If the Generate button turns gray, you need to check whether the cabinet that needs to be generated is selected!**

![](https://qhstaticssl.kujiale.com/image/png/1739519058069/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. After generating the stove and chimney hood, click **Finish** to exit the generate stove interface.

![](https://qhstaticssl.kujiale.com/image/png/1739519188645/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Q&A

**Q：**How to modify the stove size?

**A：**Exit the generation environment and click to select the **stove** to modify the **size and position** of the stove in the parameter panel.

![](https://qhstaticssl.kujiale.com/image/png/1739519268969/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
