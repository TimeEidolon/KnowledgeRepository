# **Detailed Description**

## Lighting Animation Function Supports Keyframe Setting for Various Light Sources Including Sunlight/Lighting

1. Parameters of sunlight can be adjusted and keyframes set, such as azimuth, elevation angle, color temperature, and brightness.

2. Parameters of various light sources can be adjusted and keyframes set, including light color, brightness, number of faces, length and width, angle, position, switch, etc.

3. You can find the corresponding light source object in the hierarchy/design scene/right parameter panel, and then add keyframes to the object.

![](https://qhstaticssl.kujiale.com/image/png/1772614332669/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

By setting keyframes for lighting, various lighting animations can be output, such as dynamic changes in natural light at different times within a scene/room light switch effect/commercial space scene various light effects change/digital product surrounding light consistent change effect, etc.
