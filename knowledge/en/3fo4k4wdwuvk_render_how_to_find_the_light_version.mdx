# Problem

1. Can't find a specific light version?
2. Where to switch between different light versions?

# Solution

In general, you can find light version here.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762135632506/03E477D2E02134A2C2065840077CE396.png)

Light Version Management

If the version you want is not in the previous step, click Light Version Management to find other version in this series. Click Apply to set your favorite version to "Current".

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762135632467/92246DD73DF82C1991A7BED67E9F7F1F.png)

Note, there are multiple series of light templates, such as Ultra-Fast Daylight, Ultra-Fast Nightlight, Realistic Indoor Daytime, Realistic Indoor Nighttime, etc. For each series, you need to click the version you like to set it as "Current".

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762135632501/FADE53058254C89119C0A72350334B24.png)
