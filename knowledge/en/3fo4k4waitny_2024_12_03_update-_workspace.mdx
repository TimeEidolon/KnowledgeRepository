**Target Users:**

All Coohom enterprise users (Limited-time free offer)

**Before:**

For customers selling paving products, when they enter the Coohom design tool, they can only perform regular continuous paving.

**After:**

Now, customers just need to set up complex paving projects in advance, and then they can use that project with only one click in the Coohom design tool. It also supports flexible modification of size parameters, greatly enhancing the flexibility of tiling and allowing customers to present more complex and varied paving project.

**![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1742385798208/189B60A507DC48B708E0F6232BE58F6D.png)**

**![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1742385800882/6ED6B33199A848CB547FEF2D3E7DCA16.png)**

**How to use**

**Enable Features**

You need to contact your Customer Success Consultant to obtain the corresponding permissions for the "Paving Parametric Editor" feature (available for a limited time free of charge).

**Feature Access**

Go to Enterprise Catalog - Products - Paving. When creating a new tiling plan, select "Create Parametric Paving" to start creating parametric paving projects.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1742385802863/E245F3EF2496B798F34FC38EA9CC5ADA.png)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1742385806886/BDF08DA58CA9B2AF3FA73A10D2C9419A.png)

**Granting Subaccount Permissions**

You need to go to “Account Management”, select the target subaccount, and assign it the "Parametric Paving Editor (Coohom)“ role. As long as this subaccount can manage hard furnishing products, it will be able to create parametric paving."

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1742385808679/CC8DDB54CA5BF224028AE3B1AAE4193F.png)
