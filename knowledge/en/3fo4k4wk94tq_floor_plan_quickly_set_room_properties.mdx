In the floor plan tool, you can set room properties, such as room type, name, and whether the room has a ceiling.

⭐Before reading the article, it's recommended to watch the video tutorial first to get started quickly~

🎥 [Video URL](https://www.youtube.com/watch?v=3mKmlluR6jg)

# For Users

All Users

# Entrance

In the 2D view, select a room, and the related properties settings will be displayed on the right-side panel.

![](https://qhstaticssl.kujiale.com/image/png/1773298616156/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Set Room Type

1. Select a room, click **Room Type** in the right-side properties panel, and choose the desired type.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1773301557938/2D37C2734C50A6837B78ACB4959B66FF.png)

* You can also use the search tool for quick access. Enter keywords to find and select the desired room type with one click.

![](https://qhstaticssl.kujiale.com/image/png/1773299310119/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Customize Room Name

Click on a room and input the desired name in the **Room Name** field located in the right-side attribute panel, then confirm.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1773301557864/2D37C2734C50A6837B78ACB4959B66FF.png)

# Set Room Ceiling

Click the room, and in the right-side properties panel under the **Room Ceiling** section, choose either **With Ceiling** or **No Ceiling** as needed.

![](https://qhstaticssl.kujiale.com/image/png/1773299564357/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

* **Note:** Each room can be set to with ceiling or no ceiling in the right-side panel. When the room type is set to terrace or outdoors, it defaults to no ceiling.
