Some users comes to service team to complain that the desktop app doesn't work after install it and get a message of room under repair.

![](https://qhstaticssl.kujiale.com/image/png/1750699534639/E3762FE6EC3DEE50E19C087C1B095FCC.png)

## Target User: All users


To solve this issue:

**You need to delete the "config-v2.json" file in the following file path.**

**How to find this file:**

**step1: post in the C folder as per route below:C:\Users\xxx(your system name)\AppData\Roaming\Coohom**

![](https://qhstaticssl.kujiale.com/image/png/1750700318492/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

also you can click 'view',tick the 'Hidden items'and you will find the appdata folder in C:\Users\xxx(your system name)

![](https://qhstaticssl.kujiale.com/image/png/1750700555078/6CBF0E66193C3B8ADCCE4D0CD43CC2BE.png)

Step2: **delete the "config-v2.json"**

![](https://qhstaticssl.kujiale.com/image/png/1750700992672/66437958F5F85A2B9FEFC94E78EA6AB2.png)

**Step3:  reboot coohom and try again.**
