**Ⅰ.****Creation Position**

Custom Furniture —Create Asset—Parametric Model Library—Furniture—Kitchen&bath base cabinet     ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931641759/2934A926B3E041B61CE8561F2ACC390E.png)

**II.Setting Parameters**

2.1 Basic Parameters:![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931642888/8002814D19872C07373C64AEB7E66E72.png)

2.1.1 Width (W) - Represents the size of the model panel in the X-axis direction. Set the value type as a range (e.g., range 200-1200, current value: 500).![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931644017/EFD5571DF4430FFAE93111C5B5EFC655.png)

2.1.2 Depth (D) - Represents the size of the model panel in the Y-axis direction. Set the value type as a range (e.g.range 200-1200, current value: 450).![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931647125/7C402396D1377E4F6B5AE3765D685637.png)

2.1.3 Height (H) - Represents the size of the model panel in the Z-axis direction. Set the value type as a range (e.g., range500-2400, current value: 1600).![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931649434/4B9A157E7A0FE4641549121BED2AAE28.png)

2.1.4 Material - No modification needed, indicating that the Door uses the default material.

2.2 Custom Parameters

Custom Parameters are used to add some more parameters except that commonly basic parameters or formulas in the editor.

2.2.1 toekick height(TH)—Represents the height of toe kick of cabinet,set it a range from 0 to 100 and value 80.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931652640/7E4B38CC7D53D11035000809D9B4E127.png)

**III.Importing Submodels**

3.1 Import a front panel from Internal parts library as a sidepanel of cabinet,and set its dimensions.Width: #D; Depth:18;Height:#H.PositionY:-#D;RotateZ:90.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931659071/E5FB09C4F835D6E403BF6818AD6CD173.png)

3.2. Click on the panel, right-click and select "Copy," then change the Position X to #W-18.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931661288/146CB97DBA13ECEC6CDB677327A9BDB6.png)

3.3. Select a panel as back panel and set its physical attributes. Width: #W-18\*2, Depth: 18, Height: #H; Position X: 18.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931662653/6F5EA172FE63EEFD1C00BD086E42F578.png)

3.4. Select a panel as bottom panel and set its physical attributes. Width: #W-18\*2, Depth: #D-18, Height: 18; Position X: 18,PositionY:-18.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931663812/AEBB9BE386B73AE14B91EE15C9D1FA11.png)

3.5. Click on bottom panel, right-click and select "Copy" ,then change this top panel’s Position Z to #H-18.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931666451/523CB1AA0D0EEBBC886F104506235018.png)

3.6 Add toe kick panel

3.6.1 Move bottom panel Position Z attribute up to #TH,which is height of toekick.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931669403/74DD6128A5AD3520B4F1B41F29D92DCC.png)

3.6.2 Modify back panel ‘s dimension,leave some distance for toe kick.Change Height to:#H-#TH-18, PositionZ：#TH-18.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931670504/20829107CD50F0995C681A8038F80A95.png)

3.6.3 Select a front panel from parts library as front panel of toekick part.And modify its dimensions: Width:#W-18\*2,Height:#TH,PositionX:18，PositionY:-#D+18.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931673114/61F0635D0F93E5AD51F6545593568578.png)

3.6.4 Click on this panel, right-click and select "Copy" ,then change Position Y to 0 and change its name to back.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931676564/B59EDB8275787FB871828161482183A3.png)

**Ⅳ. Test the Model**

Input values within the range of parameters W, D, H and TH, observe if the model changes correctly. If it does, it means the model is working fine.

**Ⅴ. Save to Library**

Click on the file in the upper right corner and choose save, select the subdirectory for saving, name it, and click save.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931677824/886BD3FFCAD34520E664FF5A1EF50B69.png)
