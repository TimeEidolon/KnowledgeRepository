# **Feature Introduction**

1. Added the function to edit pivot points, allowing users to move model pivot points within the plan as needed
2. Supports two pivot point resetting methods: pivot centering to object, object centering to pivot
3. Based on the pivot point editing feature, users can create more diverse model display animations, such as door opening and closing animations

# Operational Entry

Expand the Hierarchy panel —— Right-click the model/group you want to edit —— 【Axis】

# For Users

All Photo Studio Users

# Basic Functions

**1. Pivot Point Editing**

Through the pivot point editing feature, users can move the model pivot points according to their needs. Subsequent rotation and scaling of the model will be based on the new pivot points

After editing, press the Esc key to exit the pivot point editing tool

**2. Pivot Point Reset**: Pivot point reset can restore the model pivot points to their original positions, and there are two reset methods:

* Pivot centering to object: Clicking will reset the pivot point without changing the current position of the model
* Object centering to pivot: Clicking will reset the model without changing the current position of the pivot point

# Derived Functions

**1. Pivot Point Editing and Video Production**: Through pivot point editing, users can create more diverse model display animations, such as door opening and closing actions

**Note: When making videos, if you need to edit model pivot points, please do so before recording keyframes or preset animations. If the object already has keyframe data, modifying the pivot point data will clear the current keyframes**

[![
](//qhstaticssl.kujiale.com/newt/1144/image/png/1549879747859/B0556275535F7BF4FEF2DCC4F752A054.png)](https://qhstaticva-cos.kujiale.com/media/yun/help/college/1721806204421_L1d0s144b28endpo7viwiuwldqwlufx7ifksq8.mp4)

**2. Model Splitting and Pivot Point Editing:**

After model splitting, if the model is not decomposed, the pivot point position remains unchanged

If the model is decomposed after splitting, the pivot points of the decomposed combined models remain unchanged, and the pivot points of the disassembled single models reset to the model's center point

**3. Pivot Point Editing Data Saving:** When saving a plan, it records the edited pivot point information

**Note: After editing model pivot points, saving or adding to favorites and then dragging to use in a plan will restore the pivot point to its default position**

# Others

**Model pivot points in historical plans remain unchanged: Modifying model pivot points will change the data of already created video plans, so historical plans' model pivot points won't be updated (still at the model's bottom), while model pivot points in new plans and new canvas additions default to the model's center**
