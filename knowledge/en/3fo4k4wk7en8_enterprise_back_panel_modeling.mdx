2. Parameters - Base Variables

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850884784/A3DDDC1C68B31D5A62774E9FAB71106C.png)

Witch

W indicates the size of the panel in the X-axis direction, and the value type is set to the range (range 100-1200, the current value is 350).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850885631/F5CAC6C1B350F634259346AE88859141.png)

Depth

D indicates the size of the panel in the Y-axis direction, the value type is set to range from 5-25 (optional values 5, 9, 12, ... can also be set) (for example, the current value of the interval is 18)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850887045/F5E58EDCA4C8EBFE60965EA7CB19AD74.png)

Height

H indicates the size of the plate in the Z-axis direction, and the value type is set to the interval (range 100-2400, the current value is 600).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733850888380/FCB65D458F3C1AB206B5350033D254F4.png)

The material does not need to be changed, which means that the model uses the default material.  
(Ps: The range of the above parameters can be set according to the actual manufacture requirements, it's not fixed.)
