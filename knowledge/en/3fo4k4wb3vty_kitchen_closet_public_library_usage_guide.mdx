Coohom public library collects millions of model resources to build diverse projects for users to protect them, and can quickly obtain models, collect them with one click or use them directly into the project. This tutorial explains how to quickly find the latest models on the public library, how to quickly find models and model ideas.

# For Users

**All users**.If some enterprise accounts do not have the kitchen＆closet design entrance, please contact the CSC to apply.

# Entrance

1. Enter the **workspace** and select the **Kitchen ＆ Closet Design**.

![](https://qhstaticssl.kujiale.com/image/png/1779701668973/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the kitchen＆closet design and choose the **Decors**.

![](https://qhstaticssl.kujiale.com/image/png/1742891346114/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Find Models

## Method One: Category Search

According to the required model type, according to the category, first-level, second-level and third-level classifications are carried out for searching according to the category.

* For example: use a twin bed model, search through "Furniture - Beds - twin Beds".

|  |  |
| --- | --- |
|  |  |

## Method Two: Keyword Search

If you don't want to search by category layer by layer, you can directly enter keywords in the search bar above the model library to search. There are also popular search keyword recommendations to choose.

|  |  |
| --- | --- |
|  |  |

## Method Three: Tag Search

If there is already a clear type of model, you can filter it through labels such as material, shape, or size.

![](https://qhstaticssl.kujiale.com/image/png/1742898727925/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1742898840020/BFABA97733FEB1D17AA85ED6F4ECA5CD.gif)

## Method Four: Brands Search

If the model in the plan wants to use brand furniture, you can directly search for the corresponding brand in the brands to select.

# Find the Latest Models

After selecting the model category, select the **latest** in the sort on the left to view the latest models listed.

# Model Ideas

The model ideas gather the latest models in the public library, and regularly update the hottest theme models of the season, festival hotspots and other related content in the form of special topics.
