We can divide the entire face to several different parts with personalized design.

## Steps with screenshots

Each step below includes the screenshot used in this article.

### Step 1
We can divide the entire face to several different parts with personalized design.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733896383764/1B153E0AFB6111B3D2E6945AC1D3AEF7.gif)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733896383764/1B153E0AFB6111B3D2E6945AC1D3AEF7.gif" alt="Step 1" style="max-width:100%;height:auto;" />

### Step 2
Click **Line** or **Rect** to draw.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896384925/2AAD85595CA905283DA3FA789606436B.png)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896384925/2AAD85595CA905283DA3FA789606436B.png" alt="Step 2" style="max-width:100%;height:auto;" />

### Step 3
Select anywhere as the start point, then move the mouse, input the length of the line. Draw continuous lines until you draw a closed face. *Notice, all the faces should be closed.*

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896387275/19D04DDFCFEE9FC1089649E6483C1BCD.png)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896387275/19D04DDFCFEE9FC1089649E6483C1BCD.png" alt="Step 3" style="max-width:100%;height:auto;" />

### Step 4
Click any lines, then you can input the distance to the edge to make the face you drew more accurately.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896389171/747C43524CBAB0F9608F92A2785B6A00.png)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896389171/747C43524CBAB0F9608F92A2785B6A00.png" alt="Step 4" style="max-width:100%;height:auto;" />

### Step 5
Rect tool is similar to this. With these tools, we can draw any shapes of the faces as we need. Then pave materials to different areas.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733896390810/2B2D44015AE054AF23CE9A6E07D33B8C.gif)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733896390810/2B2D44015AE054AF23CE9A6E07D33B8C.gif" alt="Step 5" style="max-width:100%;height:auto;" />

### Step 6
Also, each area can be edited separately. Select one area, you can see **Area Settings** on the left. There, you can control the start point of paving, adjust the angle of materials.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896393460/83A1FC5A684980AE6BCB1885B33DF783.png)
<img src="https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896393460/83A1FC5A684980AE6BCB1885B33DF783.png" alt="Step 6" style="max-width:100%;height:auto;" />
