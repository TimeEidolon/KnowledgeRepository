We can divide the entire face to several different parts with personalized design.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733896383764/1B153E0AFB6111B3D2E6945AC1D3AEF7.gif)

Click **Line** or **Rect** to draw.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896384925/2AAD85595CA905283DA3FA789606436B.png)

Select anywhere as the start point, then move the mouse, input the length of the line. Draw continuous lines until you draw a closed face.

*Notice, all the faces should be closed.*

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896387275/19D04DDFCFEE9FC1089649E6483C1BCD.png)

Click any lines, then you can input the distance to the edge to make the face you drew more accurately.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896389171/747C43524CBAB0F9608F92A2785B6A00.png)

Rect tool is similar to this.

With these tools, we can draw any shapes of the faces as we need. Then pave materials to different areas.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733896390810/2B2D44015AE054AF23CE9A6E07D33B8C.gif)

Also, each area can be edited separately. Select one area, you can see **Area Settings** on the left. There, you can control the start point of paving, adjust the angle of materials.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733896393460/83A1FC5A684980AE6BCB1885B33DF783.png)

**Rotate** can be used in the float bar, too. But this option can only rotate the material with 45 degrees or multiples of 45 degrees.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/gif/1733896395431/31413BD8CF822BE04AF641C4286719B8.gif)

This is how to draw lines and rectangles in face editor.
