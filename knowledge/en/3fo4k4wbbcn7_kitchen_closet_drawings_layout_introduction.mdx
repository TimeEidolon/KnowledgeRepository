Drawings layout is tool that organize the edited views in the drawing frame to output a complete set of construction drawings. This article introduces the functions and how to use the drawings layout tool.

Each step includes plain MEDIA_* URLs (for AI tools) plus rendered images.

# For Users

**All users**.If some enterprise accounts do not have the kitchen & closet design entrance, please contact the CSC to apply.

# Entrance

Enter the kitchen & closet design and select **Drawing - Drawings Layout.** 

![](https://qhstaticssl.kujiale.com/image/png/1745229593641/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Drawings List

1. The views in the list will be generated based on what is covered in the project, and only **custom drawings** are included in the kitchen & closet tool.

![](https://qhstaticssl.kujiale.com/image/png/1745229610078/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Click the **expand all /collapse all** button in the upper right corner, and the user can expand or collapse the drawing list with one click.

![](https://qhstaticssl.kujiale.com/image/gif/1745229698499/99E02D4C871988BACF2F2AEFEDD7728E.gif)

# View Splice

1. Open the right view manager, find the view that needs to be spliced, and drag it directly into the canvas. The view supports alignment function.

2. The scale above the sheet adjusts the view scale size.

![](https://qhstaticssl.kujiale.com/image/gif/1745229803065/80EA06ACA640D848944F4A6041DB9A8E.gif)

# Edit Views

**Method One:** Click **edit views** on the left side of the top bar to switch to the edit view environment.

![](https://qhstaticssl.kujiale.com/image/png/1745229831836/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Method Two:** Click on the toolbar above **go to edit views** and return to the edit view environment.

![](https://qhstaticssl.kujiale.com/image/png/1745229855481/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Method Three: Double-click** the view in the drawing, or select the view and click the **edit view** button to return to the edit view environment.

![](https://qhstaticssl.kujiale.com/image/png/1745229876569/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1745229969303/2C0F7B89224407F3C64E3E3FB425FBB0.gif)

# Download Drawings

Click **Download** in the upper right corner, **select the drawing you want to download** on the left side of the pop-up panel, **set the format of the exported drawing** on the right, and click **Download** after the setting is complete.

![](https://qhstaticssl.kujiale.com/image/png/1745230070361/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1745324821848/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
