# Solution

1. Ungroup.Group——Ungroup/Explode

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762409377492/1652494048F2EC6439CA5FAFE946D8E6.png)

2. Right-click to transform all selected models into editable polygons

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762409377446/59884C5E44ED466FEFC91C65EDEF31B4.png)

3. Reset Model

Select all models, utility panel, reset transformation—reset selected model

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762409377493/38A029F52B46642942AF85EA741E5B3A.png)

4. Transform into editable polygon again (it must be transformed into editable polygon at last)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762409377446/59884C5E44ED466FEFC91C65EDEF31B4.png)

5. Grouped file archive upload
