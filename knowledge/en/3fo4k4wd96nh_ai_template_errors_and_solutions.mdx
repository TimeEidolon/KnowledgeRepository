Our Coohom users may meet some unexpected errors while applying the AI template to their designs. This article will illustrate some commonly met errors.

**Users: All users**


**Type 1**: While applying the AI template, user gets the message "The layout of the room is under development, so stay tuned".

![](https://qhstaticssl.kujiale.com/image/png/1756285762719/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

This happens to Dinning room or other room types that are still under development. Hence, kindly manually add the interior designs while meeting this error.

**Type 2**: User applied one room type such as living room but got the bed room design.

![](https://qhstaticssl.kujiale.com/image/png/1756286376159/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1756286407788/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

This occurs due to the fact that the AI template logic is based on the room type and that the user only set the room name but did not set the room type correctly. Here is the reference image of correct setting.

![](https://qhstaticssl.kujiale.com/image/png/1756286581101/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Type 3**: While applying the AI template, user gets the message "There is a mistake between the templates. It is recommended to replace".

![](https://qhstaticssl.kujiale.com/image/png/1756287725053/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

This error happens to a few old templates uploaded quite a long time ago. Kindly change to other templates while meeting this error.

**Type 4**: While applying the AI template, the user gets the error "Exceeded the limit on the number of matching rooms".

![](https://qhstaticssl.kujiale.com/image/png/1756287948627/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

This is because the current maximum number is 40 for a whole project. Once exceeding this number, it will fail. Hence, please control the room number below 40 before applying.

Although there are some errors while using the AI template function, our product is trying to optimize this. We sincerely appreicate your kind understanding on this.

For any further concerns, please feel free to contact Customer Service team via: [support@coohom.com](mailto:support@coohom.com).
