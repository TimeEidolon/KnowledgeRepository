During the arraignment process, how to flexibly control the association method of drawings directly affects the work efficiency and process norms of the enterprise. This article will provide you with a detailed introduction to the various key functions in Commodity Audit Settings—from whether to allow or mandatory association of drawings, to configuring the type of automatically generated drawings, and managing the manual association permissions for specific accounts. By reasonably configuring these options, make the arraignment process more efficient~

## 3. Set whether drawings are automatically generated during arraignment

1. **When selecting to automatically generate drawings**, users can choose the type of drawings to be automatically generated: cabinet drawings or space drawings. **When selecting cabinet drawings,** the automatic grouping method can be defined. **When selecting space drawings,** no need to define automatic grouping method.

* **Smart grouping:** The effect is the same as smart grouping in tool product drawing generation, grouping is done according to cabinet space relationships and independent model background configuration, and then the drawing is generated. Smart grouping can define whether the independent model generates drawings after grouping.
* **Group by room:** Grouping according to space affiliation, models in the same room are in one group. Note that doors and windows do not support grouping by room.
* **Group by relationship:** Group according to the existing grouping relationships of the models.
* **Not group automatically:** Generate drawings according to the actual grouping condition of the models in the order.

![](https://qhstaticssl.kujiale.com/image/png/1758684248460/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. **When the application range is set to all accounts**, in the arraignment, accounts with automatic drawing will have corresponding prompts. **When the application range is for specific accounts**, only these accounts will follow the automatic drawing logic, other accounts not on the select list will not automatically generate drawings, as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1758684321646/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## 4. Set whether the selected account is disabled for manually associating drawings

1. **Click to select an account**, and in the pop-up panel select the required account as needed and click **Confirm**. At this point, you will see the option for **whether the selected accounts is disabled for manually associating drawings.**

![](https://qhstaticssl.kujiale.com/image/png/1758684842635/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1758685045939/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1758685078588/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. When the enterprise needs **only these accounts with automatic drawing to be disabled for manually associating drawings**, just select **"Yes"**. The manual drawing association selector will be disabled for these accounts, and a **red disable marker** will appear in the arraignment, as shown below:

* **Note:** Other accounts not on the automatic drawing list will not be affected and can still use the manual drawing association function.

![](https://qhstaticssl.kujiale.com/image/png/1758685218179/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
