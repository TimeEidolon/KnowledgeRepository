Dear Coohom Users,

Here illustrates several situations you might come across when you create a terrace without ceiling.

In the first situation, when you create a room and name it **Terrace** but it still has a ceiling, please check whether you have selected the **Without ceiling** button from the drop-down list.

![](https://qhstatic-us-s3.coohom.com/image/png/1747807634799/3688EE0CB3F9E16637C1AB2F6F37F1DE.png)

![](https://qhstatic-us-s3.coohom.com/image/png/1747807654453/CFCB8F8FC74BA844E0E104F876075443.png)

In the second situation, you create a room named Terrace and also select the Without ceiling button, but you still see a ceiling. When you are having this issue, please check whether you are having **Custom-height** wall connecting the neighboring two rooms, of which one is named Terrace and the other one is a normal room.

![](https://qhstatic-us-s3.coohom.com/image/png/1747807687228/AE63EC8B73273A00E155877EB0F76CAA.png)

![](https://qhstatic-us-s3.coohom.com/image/png/1747807721456/10BED4ED5BFB8FA662797F72A230D0D9.png)

![](https://qhstatic-us-s3.coohom.com/image/png/1747807744938/DA8D79863A6F19920D5515F23EE30257.png)

To resolve this issue, kindly scroll down to the **Split the top wall** button to select **Yes** from the drop-down list.

![](https://qhstatic-us-s3.coohom.com/image/png/1747807768775/978C811157E3F9C9FCB27F845E29C6ED.png)

![](https://qhstatic-us-s3.coohom.com/image/png/1747807801875/CFCB8F8FC74BA844E0E104F876075443.png)
