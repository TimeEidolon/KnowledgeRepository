1. 3D model requirements  
​  
When uploading the handle, you need to lay the handle horizontally with the installation side facing back.

As shown

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733908176201/6E0493792CFD3C5C76F1003921610A93.png)

Move the model to world coordinates 0,0,0 and then archive

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733908177009/C756D01B0188C0B88BCBEF155EECC2FC.png)

2. Model upload  
​  
Choose to add files

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733908178132/4B81B740BE84D1D058B798561CE82AA6.png)

Render classification is selected as **handle bar**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733908179725/9CD9C1CF5D0338FBD4E31BE6FE54ED6A.png)

After confirming the classification, the default margin parameters of the handle can be adjusted or not modified. The system default margin X: 30 Y: 30 can be modified after use.  
​  
X represents the horizontal distance between the edge of the handle model and the edge of the door panel  
​  
Y represents the longitudinal distance between the edge of the handle model and the edge of the door panel

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733908180569/8BE41AB205C245B009C6759639631BEC.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733908181139/21062939F4B9950525FE77C2B1AE102F.png)
