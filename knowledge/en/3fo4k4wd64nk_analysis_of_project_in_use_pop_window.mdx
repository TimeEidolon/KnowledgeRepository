Some Coohom users will meet this pop-up window of **"Project In Use"** from time to time and thus are worried whether their accounts are hacked. This article will illustrate the reasons behind this and how to avoid this.

**Users: All Users**


**Reasons:** Coohom has banned the multiple login for account and project for safety purpose. Here are several common situtations that will trigger the message.

* Open the same project again on the current broswer while working on that project
* Log in the account and open the same project on a new different broswer such as Microsoft edge browser while working on the Chrome broswer.
* Log in the account and open the same project on the web broswer while working on that project in the PC app.

**Conclusion:** Hence, kindly always work on one project on one device at a time.
