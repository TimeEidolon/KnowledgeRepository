# Problem 1

The originally existing model disappears after rendering.

# Solution 1

1. Try to check Specular reflection and see if the model appears.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761814129686/43CA0DD4D29E9746780423E087DAED30.png)

2. For occasional issues, users can switch the lighting version and try rendering again.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761814129865/5A47D33F7E8EB4D20677C90BF71F6D24.png)

3. Check if some parts of the model are hidden.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761814129771/5B9F919A2496BAEA21E7EEFADBADEB3F.png)

# Problem 2

There was originally a door, but after rendering, only a doorway is visible.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761814129738/910FF4C51982E2D833EF3BFBDF2C1358.png)

# Solution 2

During rendering, the single-room rendering option was selected, so the model from the adjoining room is not visible. Switching to full house mode rendering will display it normally.

# Problem 3

The originally placed door has disappeared.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761814129684/1CA14CF2BB877EC2DE5C3CD3FA9985A8.png)

![image-202302181712252.png](https://qhstaticssl.kujiale.com/image/png/1761814359861/D4D1942F0834CCEB3CB690BED057B098.png)

![](https://qhstaticssl.kujiale.com/image/png/1761814382913/1100306906D11CB4420D189FD738238A.png)

# Solution 3

The overlapping of wall structures at the door location is causing the issue; removing the overlapping walls can resolve it.
