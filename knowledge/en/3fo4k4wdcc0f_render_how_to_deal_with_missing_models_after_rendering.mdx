# Problem 2

There was originally a door, but after rendering, only a doorway is visible.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1761814129738/910FF4C51982E2D833EF3BFBDF2C1358.png)

# Solution 2

During rendering, the single-room rendering option was selected, so the model from the adjoining room is not visible. Switching to full house mode rendering will display it normally.
