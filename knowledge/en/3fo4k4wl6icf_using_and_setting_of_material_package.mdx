Effect of material package:

Limit the types of materials used by the model. When the model replaces materials, it can only replace one or several specified materials. By setting a material package for the model, the purpose of restriction can be achieved.

Take door panels as an example. Depending on the requirements, all materials can be applied to a certain door panel, only some materials can be used to a certain door panel, or only a certain material can be used. In this case, the setting method is as follows:

**ⅠCreate a new folder under material folder![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931612175/AC85F2A15C620BBAB9E4E26934911191.png)**

Place the materials that may be used in this corresponding folder.

**Ⅱ Create material package**

2.1 click advanced---range settings![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931614253/86D693AE02E5FED41D400E97A5C5B146.png)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931616450/A7AEB2F74C8847B4CB95F1FB048D64DF.png)

2.2 Choose material package configuration---New material package![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931618517/F511929A34F1AB6614A11D2CC34E45C9.png)

2.3 Enter information of material package in information bar, name, type of product,

When object type is category:click placed product and select product catalog, then click yes;![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931619263/630E4FB14478E23A5137056A14386226.png)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931622127/9403A5BE4C026CE34F4B8FF17129A4F7.png)

Also,select enumeration to be object type,need to select certain single material in library![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931624704/4CE6F8DA1EF3ED730B882CED740C4D1D.png)

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931626952/A818C4AE7965E42F13B92638CACD88F9.png)

Click Yes when finished.

**Ⅲ Add material package in editer**

3.1 Modify material type in model edit page,data type select options; material range select a material package,

And select a certain material, this model is given a certain material.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931629165/591FB08D621C688DD4323F4DDE16012B.png)

3.2 For sub-models in the model, if also need to set a material package, need to set the corresponding material package at the advanced parameters of the corresponding sub-model. Of course, for different models using the same sub-model, need to set different When adding a material package, it is recommended to save this sub-model and then set the material package again.

**Ⅳ** Following the above steps, the material package of the model is added. If want to add materials to the model, need to add the additional materials to the material package.
