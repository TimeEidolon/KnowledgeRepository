In exterior design, there are many types of roofs, the most common of which are hip roofs and gable roofs. Both of these roofs can be designed in Coohom. This article will explain in detail how to draw hip roofs and gable roofs.

![](https://qhstaticssl.kujiale.com/image/png/1746759707629/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=bO5P\_eEYPE0)

# For Users

All users

# Entrance

Enter the 3D design interface and select **"Advanced Tool - Material & Component"**.

![](https://qhstaticssl.kujiale.com/image/png/1746771661772/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Add Guides

After entering the material & component interface, use the **add guides** tool to draw guides that match the roof size.

![](https://qhstaticssl.kujiale.com/image/png/1746772436473/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Draw a Roof with the Straight Line Tool

## Draw a Hip Roof

According to the position of the guide, use the **straight line** tool to draw a **hip roof**, as shown in the following figure:

![](https://qhstaticssl.kujiale.com/image/png/1746773185457/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1746773142525/554FC9757C7D68C6C297FF96282238D9.gif)

## Draw a Gable Roof

According to the position of the guide, use the **straight line** tool to draw a **gable roof**, as shown in the following figure:

![](https://qhstaticssl.kujiale.com/image/png/1746774478181/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1746774443512/CB8D130D2C5C188C60CFF0593E38E45B.gif)

# Pave Material

After the roof shape is drawn, select the **material** in the material bar on the right and apply it to the roof.

![](https://qhstaticssl.kujiale.com/image/png/1746774658554/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1746774632460/737741696C05F269F295D5B670986B51.gif)
