# Problem

The issue arises because the model file is too large.

# Solution

This may be because max contains other information, which can be set within max.

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762313501969/41FF94BE730FF8C6558A52DB135DDA2B.png)

![](https://qhstaticssl.kujiale.com/yuntai/image/jpeg/1762313501776/E2EB45F9CA85CFB2B6E316AA9DA2D742.png)

Once you check the option to compress upon saving, you can archive it. After setting this, the model file will be smaller upon saving and it will not affect model quality.
