**II.Setting Parameters**

2.1 Basic Parameters:

2.1.1 Width (W) - Represents the size of the model panel in the X-axis direction. Set the value type as a range (e.g., range 10-2400, current value: 400).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931727613/AB18A0BFB69CB1C51E63C392944BC4E6.png)

2.1.2 Depth (D) - Represents the size of the model panel in the Y-axis direction. Set the value type as a options (e.g., optional 18, current value:18).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931730463/D7A7FB350F86E32FBE2AF0810BE93625.png)

2.1.3 Height (H) - Represents the size of the model panel in the Z-axis direction. Set the value type as a range (e.g., range 10-2400, current value: 800).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931733531/04121630B664BA27631DFDDA588BE414.png)

2.1.4 Material - No modification needed, indicating that the Door uses the default material.

**Ⅴ. Test the Model**

Input values within the range of parameters W, D, and H, observe if the model changes correctly. If it does, it means the model is working fine.

**Ⅵ. Save to Library**

Click on the file in the upper right corner and choose save, select the subdirectory for saving, name it, and click save.
