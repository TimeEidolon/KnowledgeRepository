1. Tap 3D model requirements

1. There are no extra models in the max file, that is, only the models that need to be uploaded in the file (excess models will cause upload failure)  
2. The model is centered in 3D MAX  
3. The max file size should be less than 100M  
4. The number of polygons in the model should not exceed 800,000 polygons (click '7' to view after all converted to editable mesh)  
5. The size of the model should be (mm) mm; (the setting path is: custom menu-unit setting)  
6. The size of the model in the scene should not exceed 10 meters.  
7. Archiving and uploading the leading model into a compressed package (recommended)  
​  
Note: If there is an angle at the outlet of the faucet model, there is no need to adjust it, just save it as normal

### 2. Upload steps

1. Merchant background-commodity management-commodity management-Custom Furniture -upload material-3D model

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733911053717/C96DADEC5E2C3A7D337CD2E34F8F7CEB.png)

2. Jump to the 3D model upload page and click "+Add File", find the 3D model file that needs to be uploaded, enter the material name (the name will automatically select the 3D file name or you can modify it yourself), and select the rendering category. The 3D model rendering takes a certain amount of time, and you need to wait for the successful processing in the blank box below to upload it.

3. Select the rendering classification **Tap**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733911054805/9E641EF41458C545CB832255548F7089.png)

4. Enter the required parameter information for the faucet model

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733911055434/C4E8B5A11E3455596876425A937132ED.png)

Installation location of the faucet example in the tutorial: 30,230.76,0  
​  
The input of the installation position of the faucet requires the form of X, Y, Z.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733911056563/E70176A8795832B9605946AFF3556E88.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733911057307/0212F262AB45D96DA68CC1321B9014CF.png)

As shown  
​  
Installation Location X: the distance between the center of the water pipe of the faucet and the lower left front and bottom measurement base point of the largest outer frame of the faucet on the X axis.  
​  
Installation Location Y: The Y-axis distance dimension between the center of the water pipe of the faucet and the bottom left front and bottom of the largest outer frame of the faucet.  
​  
Installation Location Z: the distance between the bottom of the faucet model and the installation of the faucet on the sink surface. As shown in the figure, if there is a water pipe, the size of the water pipe needs to be added; if not, it can be directly 0

5. After confirming the information, complete the upload
