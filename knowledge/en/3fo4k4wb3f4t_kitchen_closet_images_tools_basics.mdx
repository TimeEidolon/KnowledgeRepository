After building the design scene, the images tools can help us quickly generate good images. This article will take you to learn the use of images tools step by step: start by selecting the rendering mode, add lights and environments, adjust the camera position, set image resolutions and advanced parameters, and finally complete rendering of the images. The article comes with rendering tips to help you get started with images tools efficiently~

# For Users

**All users**.If some enterprise accounts do not have the kitchen＆closet design entrance, please contact the CSC to apply.

# Entrance

1. Enter the **workspace** and select the **Kitchen ＆ Closet Design**.

![](https://qhstaticssl.kujiale.com/image/png/1779701668973/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the kitchen＆closet design and choose the **Images**.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1743153164757/2D37C2734C50A6837B78ACB4959B66FF.png)

# Set Aspect Ratios

1. Select the rendering mode, with **perspective view, 360° walkthrough, and top view** options.

2. Set **aspect ratios** on the left panel.

* The 360° walkthrough renders to a 360° effect, no need to set aspect ratio.

![](https://qhstaticssl.kujiale.com/image/png/1743153741082/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Add Ambient Lights

## Select a Light Template

Select the **light template** to set **brightness and effect** parameters, and adjust the light's ambient blocking to increase the clear three-dimensional sense of the space.

![](https://qhstaticssl.kujiale.com/image/png/1743153870419/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Add Custom Lights

1. Click **add to customize**.

2. **Name** the lighting scheme, select a **basic template** and customly adjust it based on the template.

* **How to add custom lights, you can see the relevant tutorial documents: [[Kitchen & Closet] Custom Lighting Operation Tips](https://www.coohom.com/en_US/helpcenter/kitchen-closet-custom-lighting-operation-tips?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1743154196679/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Add Environments

Select the appropriate environment to set the **environment brightness, tone, position and ambient lightbrightness**.

![](https://qhstaticssl.kujiale.com/image/png/1743154977904/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1743155110180/4BEFBD6998C40E870F6376167B3F07BA.gif)

# Adjust the Camera Position

Select the camera and the right panel can set the **camera's position, views and clipping effect**.

* **How to adjust the camera properties, you can see the relevant tutorial documents: [[Kitchen & Closet] Set Camera Properties and Views](https://www.coohom.com/en_US/helpcenter/kitchen-closet-set-camera-properties-and-views?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1743155331093/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/gif/1743155533130/1BCCB318D22B4EAB6CACB77290FF8BFB.gif)

# Set Image Resolutions

There are four resolutions to choose from, the higher the resolution, the longer the rendering time will be.

* **Tips: You can first select 1K SD rendering small images, initially determine the rendering effect, and then select one with high resolution for rendering.**

![](https://qhstaticssl.kujiale.com/image/png/1743155824020/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Set Advanced Parameters

Select advanced settings to adjust the image effect advanced parameters.

* **How to set advanced parameters, you can see the relevant tutorial documents: [[Kitchen & Closet] Set Advanced Images Parameters](https://www.coohom.com/en_US/helpcenter/kitchen-closet-set-advanced-images-parameters?kjl_source=hc)**

![](https://qhstaticssl.kujiale.com/image/png/1743156042043/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# View Images

After adjusting all rendering parameters, click **Generate**. The image can be viewed in the **gallery** in the upper right corner.

![](https://qhstaticssl.kujiale.com/image/png/1743156240109/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1743156285056/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
