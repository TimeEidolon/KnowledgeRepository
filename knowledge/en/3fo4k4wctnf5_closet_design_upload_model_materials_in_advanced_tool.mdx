In the advanced tool, designers can upload model materials by themselves, and the uploaded materials can be replaced onto customized cabinets. This article will explain in detail how to upload model materials in the advanced tool.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=TtDGUasvb\_k)

# For Users

All users

# Upload Model Materials

# 

## Method One

1. Enter the **Workspace**, select **Models**, and click **Upload Textures.**

![](https://qhstaticssl.kujiale.com/image/png/1778206619280/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. There are two ways to create new materials: **Real-time Material** or **Upload Textures** by yourself in the required format.

![](https://qhstaticssl.kujiale.com/image/png/1778209337679/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Method Two

1. Open the design tool, enter **Advanced Tool**, select **"Kitchen & Bath / Closet".**

![](https://qhstaticssl.kujiale.com/image/png/1753337092031/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After entering the custom tool, there are two entrances to **upload model materials**, as detailed in the table below:

|  |  |
| --- | --- |
| **Entrance One** | **Entrance Two** |
| Enter **Kitchen & Bath / Closet**, select **"Personal Catalog - Upload"**, click **Upload Material.** | Click on the custom cabinet, select the **"Style - Body Material - Uploaded"** in the parameter panel, and click **Material Upload.** |
|  |  |

3. Click **Upload**, and it will jump to the **Workspace** page. Then follow the operation process of method one to upload the materials.

![](https://qhstaticssl.kujiale.com/image/png/1778209530507/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# View Uploaded Model Materials

After the material is successfully uploaded, you can view them in the custom tool.

|  |  |
| --- | --- |
| **Viewing Method One** | **Viewing Method Two** |
| Enter **Kitchen & Bath / Closet**, select **"Personal Catalog - Upload".** | Click on the custom cabinet, select the **"Style - Body Material - Uploaded"** in the parameter panel. |
|  |  |

# Replace Uploaded Model Materials

**Method One:** Enter **Kitchen & Bath / Closet**, select **"Personal Catalog - Upload",** drag the uploaded model material out to quickly replace the customized cabinet.

![](https://qhstaticssl.kujiale.com/image/gif/1753338659155/71CD83851CA94A98428B3F0ED58F438C.gif)

**Method Two:** Click on the custom cabinet, select the **"Style - Body Material - Uploaded"** in the right parameter panel, and click on the uploaded model material.

![](https://qhstaticssl.kujiale.com/image/gif/1753338609329/3BFAB72671EB0F76B5B255B28864AE42.gif)
