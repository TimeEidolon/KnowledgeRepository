**Entrance One:** Go to **Workspace - Projects**, select the project you want to share with others, and click **Share.**

**Entrance Two:** Go to **Project****Details** and click the share icon in the upper right corner.

# Share Your Project Links

## Set Permissions

You can set viewing, duplicating, and editing permissions for anyone with this link.

* **Basic** and **Pro** users can only set the view permissions as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1777282715316/70318DBFFFC751687E2D183851C2D714.png)

* **Elite** and **Enterprise** users can set up **View, Duplcate** and **Edit** permissions, as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1777358243823/2C099648ACC2C7D70C7D59944478E702.png)

## Set Valid Period

1. Your shared project link can have a vaild period set to **permanent, 30 days, 7 days, or 24 hours.**

![](https://qhstaticssl.kujiale.com/image/png/1777282729442/3BC5FC2D735D90AB572CAC6F6E32F786.png)

2. Select one of the option, and the expiry date and time will be shown as below:

![](https://qhstaticssl.kujiale.com/image/png/1777282740020/906BE270F0DE0F5295B206CA569DD5CB.png)

## Set Access Password

You can set a password for your shared project link to enhance the security and privacy of scenario sharing.

1. Tick on **Need Password**. Once tickas shown below:

![](https://qhstaticssl.kujiale.com/image/png/1777282748812/499652F53E116F1BB43A553863B97FE9.png)

2. Click on **generate new password**.

![](https://qhstaticssl.kujiale.com/image/png/1777282761828/284A805384A297D781D3DDF3BC3F725E.png)

## Copy Link and Password

1. After the setup is complete, click on the **Copy Link and Password.**

![](https://qhstaticssl.kujiale.com/image/png/1777282783260/DFAE8FAC7376591D655BBA37E4EF15CC.png)

* Send the project link and password to others by paste it. Below is the example of project link together with password:

Link: https://www.coohom.com/pub/saas/apps/share/NB42Y5AKQDDICAABBM888888 Password:iwt2

2. When you select the **Preview Link**, you will see the page that the shared party sees, enter the password and click **Confirm** to receive the shared plan.

![](https://qhstaticssl.kujiale.com/image/png/1777283122379/F8CA7F1CB218EB38C5EFE4D1E70FA317.png)

![](https://qhstaticssl.kujiale.com/image/png/1777282834207/C4EEB8AB5CE6D14CDE900E857871E294.png)

## Remove/Generate Link

Click **Remove Link**, and the person you shared it with will no longer be able to open this link. If you need to reopen the permissions, click **Generate Permanent can View Link** again.

![](https://qhstaticssl.kujiale.com/image/png/1777282873115/7B46DAD7DBEDBACF768C8CCF8FA979DB.png)

![](https://qhstaticssl.kujiale.com/image/png/1777282880172/986D5FABA063CB81223856F95A49749A.png)
