**Ⅲ****.Setting Parameters**

3.1 Basic Parameters:![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931525246/B31314E4F25C0BEF9DCA9D71A9808622.png)

3.1.1 Width (W) - Represents the size of the model panel in the X-axis direction. Set the value type as a range (e.g., range 200-600, current value: 500).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931527405/FABCC562E133C91A82B40B6A1A09812D.png)

3.1.2 Depth (D) - Represents the size of the model panel in the Y-axis direction. Set the value type as fixed value (e.g., fixed value 20).![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931530161/FFCDB81AD6CE59AD4228F2360CDF4BEF.png)

3.1.3 Height (H) - Represents the size of the model panel in the Z-axis direction. Set the value type as a range (e.g., range 20-2400, current value: 800).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931532755/DF5B4D01D558499F38EE2CBDE152526E.png)

3.1.4 Material - No modification needed, indicating that the Door uses the default material.

3.2 Intermediate Parameters

Intermediate Parameters are used to replace commonly used numbers or formulas in the editor. They are only displayed in the editor and not in the front-end tools.

3.2.1 Height of G Handle(A)—Represents the height of handle,set it value to 35.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931534123/7D5D984E8B3B7E36B32208B4DA64B839.png)

**Ⅳ.****Importing Submodels**

4.1 Import a flat panel from elements library as door panel.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931536691/E62B9320B7AEBCE8B3A4EE81D9B1CF3A.png)

Edit its contour.

Keep four points, point1 (0,0),point2 (0,#W), point3 (#H-#A,#W), point4 (#H-#A,0).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931538070/AF8E8764DF9C57B230F2EAC538F9CD73.png)

Physical Properties:

Height: #D; RotateY: -90;Rotate Z:90.

Material: keep #CZ default material.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931539547/F529C8CF2690A5BE6F149EC332C3FF03.png)

4.2 Import G handle from parts library and set its dimensions,width:#W,depth: 21(default),height:45(added the part inserted into the door panel).Position Z: #H-45.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931541612/E185E8AEEA63527EF50C997147F3032A.png)

**Ⅴ****. Test the Model**

Input values within the range of parameters W, D, and H, observe if the model changes correctly. If it does, it means the model is working fine.

**Ⅵ****. Save to Library**

Click on the file in the upper right corner and choose save, select the subdirectory for saving, name it, and click save.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931542668/4DE895645FCC75E8CA7DF198E50570E2.png)
