# Entrance

1. Enter the **workspace** and select the **Create New Project** in the upper right corner.

![](https://qhstaticssl.kujiale.com/image/png/1779343695360/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Enter the 3D design, select **Drawings - Floor Plan Drawings**.

![](https://qhstaticssl.kujiale.com/image/png/1750917425461/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Tool Interface Introduction

The layout of the floor plan drawings tool is as follows:

![](https://qhstaticssl.kujiale.com/image/png/1750930883618/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# One-click Generation of Floor Plan Drawings

In the floor plan drawing tool, you can adjust the color style, set the size scale, add text annotations, and share download drawings.

* **How to generate floor plan drawings, please see the tutorial document for the specific operation: [One-click Generation of Floor Plan Drawings](https://www.coohom.com/en_US/helpcenter/floor-plan-drawings-one-click-generation-of-floor-plan-drawings?hl=en&kjl_source=hc)**
