1. **Function Introduction**

Support selecting multiple drawings to import into corresponding floors and generate multi-story plans.

2. **Tool Version**

Cloud Design 5.0

3. **Entry Introduction**

Import floor plan on the left sidebar - Import CAD

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839560795/5AB175FC8AC86A4B731C34794597F29C.png)

4. **Detailed Functionality**

You can view the CAD import requirements to help improve the accuracy of import and recognition.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839563475/09C6092B6F8B14999ED9CE0FFF16EAD3.png)

Continue to import CAD, you can select CAD drawings of multiple floors.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839568343/A8D861F9423A245C9B9F819622D108A5.png)

You can select different floors for import, but only one drawing is allowed for each floor. If the floor is not in the list, you can manually add it before making a selection.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839572078/D43387B5C05B2BB2386768DFD607137D.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839574198/102A6151F112CD8C74A50A625D51A9ED.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839577722/BC13AC8D2C4E6ACCBAF549FFED432889.png)

Check the desired multiple drawings, click "Auto Recognize", and multiple-level floor plans will be automatically generated.

You can check whether the automatically generated floor plan is complete and the dimensions are correct by comparing it with the base map. The base map may interfere with the checking process. You can turn off the display of the base map or adjust its transparency first.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839581689/38A67EBB61E186F9D3C49949CFEECDD9.png)

The multi-story floor plan will be automatically aligned based on the grid in the drawing. If there is no grid, and you find that the generated multi-story floor plan is not aligned, you can use the main menu-tool-floor plan alignment command to manually align it.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733839584129/AC3042A42E4AA3D6703874189D9982FF.png)
