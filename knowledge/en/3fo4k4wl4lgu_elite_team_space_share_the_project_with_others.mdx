You can share the design project link with others for viewing; however, sharing within the team does not support external copying or editing, and can only be copied and edited within the team scope. If you wish to share this project for copying or editing, please invite others to join this team space.

# For Users

All New Elite users, gradually opening up~

# Entrance

**Entrance** **1:** Go to **Workspace - Team/My Projects**, select the project you want to share with others, and click **Share.**

![](https://qhstaticssl.kujiale.com/image/png/1777275875163/E0EBD063B069FDB5F603A0D6DEEDB235.png)

**Entrance 2:** Go to **Project Details**, and click the **Share** button in the top right corner.

![](https://qhstaticssl.kujiale.com/yuntai/image/png/1777275550106/2D37C2734C50A6837B78ACB4959B66FF.png)

![](https://qhstaticssl.kujiale.com/image/png/1773048752518/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Share the Project with Others

## Set an Valid Period

1. You can set an **Valid Period** for your shared project link, which can be set to **Permanen, 30 days, 7 days, or 24 hours.**

![](https://qhstaticssl.kujiale.com/image/png/1777278610409/00A214A91A473BBA42D511727B8E1869.png)

2. After selecting an option, the expiration date and time will be displayed as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1777278634699/47ECB39F876B5D1A3B53209DA559E95E.png)

## Set a Password

You can set a password for your shared project link to enhance security and privacy.

1. Check **Need Password**. After selecting, it will appear as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1777278657963/ED229DB2CAAA161FC551E96AD961D023.png)

2. You can click **Generate New Password.**

![](https://qhstaticssl.kujiale.com/image/png/1777278693185/070A3ACC690708E7071A45D27EEF958D.png)

## Copy Link and Password

1. After completing setup, click **Copy Link and Password,** paste and share the project link and password with others.

![](https://qhstaticssl.kujiale.com/image/png/1777278722971/ADE363E8FE4887D3D67BAF9BBBDB37AE.png)

* When you select the **Preview Link**, you will see the page that the shared party sees, enter the password and click **Confirm** to receive the shared plan.

![](https://qhstaticssl.kujiale.com/image/png/1777445737204/6F59AF3BB35CCD3EA8C0285DF0B1A99A.png)

![](https://qhstaticssl.kujiale.com/image/png/1773049175228/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

## Remove/Generate Link

Click **Remove 7Days Can View** link, and the shared users will no longer be able to open this link. If you need to re-enable access, click **Generate Permanent Valid can View Link**.

![](https://qhstaticssl.kujiale.com/image/png/1777278839095/0E1D0AAA429D94450D3AA5813F7B2A7C.png)

![](https://qhstaticssl.kujiale.com/image/png/1777278921578/3C16298615A931FF58A3F380965D7087.png)
