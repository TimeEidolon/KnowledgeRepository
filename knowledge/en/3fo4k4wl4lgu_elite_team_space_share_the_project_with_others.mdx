You can share the design project link with others for viewing; however, sharing within the team does not support external copying or editing, and can only be copied and edited within the team scope. If you wish to share this project for copying or editing, please invite others to join this team space.

![](https://qhstaticssl.kujiale.com/image/png/1777278610409/00A214A91A473BBA42D511727B8E1869.png)

## Set a Password

You can set a password for your shared project link to enhance security and privacy.

1. Check **Need Password**. After selecting, it will appear as shown below:

![](https://qhstaticssl.kujiale.com/image/png/1777278657963/ED229DB2CAAA161FC551E96AD961D023.png)

2. You can click **Generate New Password.**

![](https://qhstaticssl.kujiale.com/image/png/1777278693185/070A3ACC690708E7071A45D27EEF958D.png)

## Copy Link and Password

1. After completing setup, click **Copy Link and Password,** paste and share the project link and password with others.

![](https://qhstaticssl.kujiale.com/image/png/1777278722971/ADE363E8FE4887D3D67BAF9BBBDB37AE.png)

* When you select the **Preview Link**, you will see the page that the shared party sees, enter the password and click **Confirm** to receive the shared plan.

![](https://qhstaticssl.kujiale.com/image/png/1777445737204/6F59AF3BB35CCD3EA8C0285DF0B1A99A.png)
