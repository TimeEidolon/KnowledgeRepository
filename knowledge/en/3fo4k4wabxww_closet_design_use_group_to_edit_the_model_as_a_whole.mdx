Designers can group and ungroup cabinet models designed by combining multiple cabinets. After grouping, it is convenient to move the cabinet model as a whole. When the combination is not needed, it can be ungrouped at any time.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=Q4uOHPMHVnE)

# For Users

All users

# Group

1、After **holding shift** to select multiple cabinets, the **Group** button will appear in the floating toolbar. Click **Group** to complete the combination of multiple models.

* **Win shortcut key: Ctrl+G** (release the shift key and press Ctrl+G)
* **Mac shortcut key: Command+G**

![](https://qhstaticssl.kujiale.com/image/png/1737431832864/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1737432034066/205426BC10D5EBA143BF32355AEEBB52.gif)

2、After grouping, pay attention to standardizing the name of the combination cabinet (space name + cabinet name, such as: master bedroom wardrobe), and pay attention to check whether the size of the combination cabinet meets the installation site size.

![](https://qhstaticssl.kujiale.com/image/png/1737432210240/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Ungroup

Select the model to be ungrouped, and the **Ungroup** button will appear in the floating toolbar. Click **Ungroup** to complete the model ungrouping.

* **Win shortcut key: Ctrl+Shift+G**
* **Mac shortcut key: Command+Shift+G**
