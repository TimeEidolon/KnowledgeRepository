The import and export functions of the enterprise catalog products allow for the batch summary, statistics, and modification of enterprise products. This function allows you to add or edit details for multiple products at once. You can edit product details using existing fields in the template, as well as add customized fields and fill in details within them. This article will explain in detail how to maintain enterprise catalog product information in batches.

# For Users

All enterprise users

# Entrance

Enter the **Workspace**, select **"Enterprise Catalog - Products"**, and click **Import & Export.**

![](https://qhstaticssl.kujiale.com/image/png/1752200748050/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Export Products

Product export allows for the batch output of product information from the enterprise catalog.

1. On the **Import & Export** page, click **Export.**

![](https://qhstaticssl.kujiale.com/image/png/1752200866153/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. In the pop-up window, select **Category** and then click **Next.**

![](https://qhstaticssl.kujiale.com/image/png/1752200918304/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. Select **Export Properties**. The product ID is exported by default, and other properties are selected by the user. If you want to batch modify a product field, you need to export that field. After selection, click **Generate Excel.**

![](https://qhstaticssl.kujiale.com/image/png/1752201008134/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

4. After the table is successfully generated, click **Download.**

![](https://qhstaticssl.kujiale.com/image/png/1752201305221/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

5. Open the downloaded Excel spreadsheet and supplement the corresponding content.

**Note:**

* We use product "ID" to identify a product. This column is automatically generated in our backend and will be automatically shown in this template. Please don't delete or modify this column.
* Only fields that are currently available in product detail page will be recognized.

![](https://qhstaticssl.kujiale.com/image/png/1752201436291/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/png/1752201590941/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Add Customized Fields

If you want to add customized fields, select **"Enterprise Catalog - Product Info - Product Info Management - Customized Fields"**, and click **Create Customized Field** to add customized fields. Customized fields will be added to **Export Properties**. After selecting this property, it will be added to the exported Excel spreadsheet.

![](https://qhstaticssl.kujiale.com/image/png/1752202033505/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1752202082599/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1752202150240/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Import Products

Product import allows for the batch modification of product information within the enterprise catalog.

**Prerequisites:** Use the product export function to download the original product information EXCEL spreadsheet **(this spreadsheet needs to include the products to be modified, and confirm that the fields to be batch modified are in the export fields)**, then batch modify the fields in EXCEL. **See the above export product operation steps for details.**

1. After modifying the exported spreadsheet information, return to the **Import & Export** page, click **"Import Product Information"**, select the modified spreadsheet in the pop-up file selection box, and click **Open.**

![](https://qhstaticssl.kujiale.com/image/png/1752204123745/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1752204218899/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. After displaying **Import Completed**, the product information within the enterprise catalog is successfully modified in batches.

* **Note: If the import fails, it will show how many items are incorrect or incomplete. You can download the error information to view it.**

![](https://qhstaticssl.kujiale.com/image/png/1752204561983/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

3. After a successful import, you can return to the **Import & Export** page, where you will find the recent import history and results at the bottom of the page.

![](https://qhstaticssl.kujiale.com/image/png/1752204791094/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
