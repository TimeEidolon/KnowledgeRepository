Drawings layout is tool that organize the edited views in the drawing frame to output a complete set of construction drawings. This article introduces the functions and how to use the drawings layout tool.

⭐Before reading the article, it is recommended to watch the video tutorial first for easy getting started~

🎥 [Video URL](https://www.youtube.com/watch?v=JN9BfXD5HRU)

All users

# Entrance

Enter the 3D design, select **Drawings - Construction Drawings**, and click **Drawings Layout**.

![](https://qhstaticssl.kujiale.com/image/png/1750748257782/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1750748410027/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

# Drawings List

1. The views in the list will be generated based on the content covered in the project. The default includes: sheets and renovation sheets, where the drawings layout will have one more **sheets** folder than the edit view list, which contains the cover and design general description.

![](https://qhstaticssl.kujiale.com/image/png/1750748904861/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

2. Click the **expand all /collapse all** button in the upper right corner, and the user can expand or collapse the drawing list with one click.

![](https://qhstaticssl.kujiale.com/image/gif/1750749265530/EE1024AC42C0B6B50FA3F5D4B6A8491F.gif)

# View Splice

1. Open the right view manager, find the view that needs to be spliced, and drag it directly into the canvas. The view supports alignment function.

2. The scale above the sheet adjusts the view scale size.

![](https://qhstaticssl.kujiale.com/image/gif/1750752590934/6DA840BBDDA3CC0451E6C0BC134ED2CB.gif)

# Edit Views

**Method One:** Click **edit views** on the left side of the top bar to switch to the edit view environment.

![](https://qhstaticssl.kujiale.com/image/png/1750752711165/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Method Two:** Click on the toolbar above **go to edit views** and return to the edit view environment.

![](https://qhstaticssl.kujiale.com/image/png/1750752737890/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

**Method Three: Double-click** the view in the drawing, or select the view and click the **edit view** button to return to the edit view environment.

![](https://qhstaticssl.kujiale.com/image/png/1750752766284/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)

![](https://qhstaticssl.kujiale.com/image/gif/1750752603628/898FC69A8AB74A330318A3FBED119255.gif)

# Download Drawings

Click **Download** in the upper right corner, **select the drawing you want to download** on the left side of the pop-up panel, **set the format of the exported drawing** on the right, and click **Download** after the setting is complete.

![](https://qhstaticssl.kujiale.com/image/png/1750752845086/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)![](https://qhstaticssl.kujiale.com/image/png/1750752938366/A8CEBABD0663ECBAEEE7516DF7A7D2B5.png)
