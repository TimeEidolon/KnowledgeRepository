The production of a parameterized model for the house requires some auxiliary support from 3D models, such as carvings and sculpted shapes, hence the need for 3D model upload.

1. **Basic Requirements for 3D Models**

**Basic Requirements for 3D Models**

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733851233126/8B8A7A73BDEDE7C9863D7EDBE047818E.png)

Acceptable file formats include 3Dmax models and SketchUp models (more commonly used for 3D).

Both file formats should be smaller than 200 MB.

**2.Specifications for 3Dmax Models**

The model should not exceed 800,000 polygons (check by converting all to editable meshes and pressing '7').

Dimensions of the model should be in millimeters (mm) (set in the custom menu under unit settings).

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733851234330/D76E68FC97C8C89681F6A36739CC20E0.png)

The size of the model within the scene must not exceed 10 meters.

Supported texture formats include jpg, png, and tif; color mode should be RGB, and texture files should not exceed 5000 pixels in size. UVW mapping should be recognized (recommended).

**3.File Requirements**

After completing the 3Dmax model, move the model as a whole to the original world coordinates of the scene (0,0,0).

The front of the model should face the XZ plane (with the "Front" side towards oneself, located at the upper right corner by default).

All handles should be placed flat/horizontally, as the system defaults to the horizontal orientation.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733851234918/9633D61F3BC4A6D54FD190D0B125D82E.png) ![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733851235487/EA1ABE127CBAC37D5CD7409D3D14A923.png)

Once confirmed, create a compressed archive file by going to File → Archive, and upload it.

![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733851237542/8E28CD4F501127C753A3645DBAE2BE1C.png)
