**Ⅰ Create material package**

1.1 click advanced---range settings![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931630712/0469F85BBF317732783712360026FAB8.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931631481/54675AEF3D43AFB5A944B719E5FF3F0D.png)

1.2 Choose cabinet package configuration---New cabinet package![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931632136/7706BA4893D60311B905B2F62D206486.png)

1.3 Enter information of cabinet package in information bar, name, type of product,

When object type is enumeration: click placed product and select cabinet, then click yes;![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931633413/93471AB1586ED7D467EA8413252F778C.png)

Also,select category to be object type,need to select cabinet category from cabinet library，as follows：![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931634468/CAE64A55197822D50E22A99B3D7F207C.png)

1.4 Click edit attributes of the corresponding model,next set up furniture pack and select cabinet package added before.![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931636071/785044752DB0F12D76E2F99E217C40F0.png)![](https://qhstatic-us-s3.coohom.com/yuntai/image/png/1733931637852/1CE7293B12D4043575B537A9E28D67F8.png)

Now cabinet package is set completed. When cabinet using on design tool is meeting requirements of cabinet package, this cabinet package can be use successful.
